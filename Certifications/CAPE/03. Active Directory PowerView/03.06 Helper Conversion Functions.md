# 03.06 Helper Conversion Functions

> Convert SIDs and UAC flags with PowerView helpers; `Get-DomainPolicy` returns password + Kerberos policy in one call.

## Overview

PowerView utility functions for SID/name conversion, UAC flag decoding, ADName conversion, and domain policy retrieval. SharpView.exe is the C# port for use from non-PS contexts. ConvertFrom-UACValue decodes numeric UAC values into human-readable flags with + marking set values. Get-DomainPolicy returns password policy, Kerberos policy, and more.

## Steps

- Use ConvertTo-SID to resolve a username to its SID; use Convert-SidToName to reverse.
- Use Convert-ADName to convert between SID, DN, and samAccountName formats.
- Use ConvertFrom-UACValue -showall to display all UAC flags; set flags are marked with +.
- Use Get-DomainPolicy to retrieve password policy and Kerberos ticket policy.
- SharpView mirrors all PowerView functions but uses strings not PS objects — cannot pipe to Select-Object.

## Commands

### Convert username to SID (SharpView)

```powershell
.\SharpView.exe ConvertTo-SID -Name sally.jones
```

### Convert SID to display name (SharpView)

```powershell
.\SharpView.exe Convert-ADName -ObjectName S-1-5-21-2974783224-3764228556-2640795941-1724
```

### Decode UAC flags for a user

```powershell
Get-DomainUser harry.jones | ConvertFrom-UACValue -showall
```

### Get domain password and Kerberos policy

```powershell
Get-DomainPolicy
```

### Convert username to SID (PowerView)

```powershell
ConvertTo-SID joe.evans
```

### Convert SID to name (PowerView)

```powershell
Convert-SidToName S-1-5-21-888139820-103978830-333442103-1602
```
