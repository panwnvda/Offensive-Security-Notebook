# 03.02 Computer Enumeration PowerView

> Hunt `TRUSTED_FOR_DELEGATION` / `TRUSTED_TO_AUTH_FOR_DELEGATION` hosts and stale-OS targets with `Get-DomainComputer`.

## Overview

Get-DomainComputer enumerates all domain hosts with OS, UAC flags, and delegation settings. Key flags to hunt: TRUSTED_FOR_DELEGATION (unconstrained), TRUSTED_TO_AUTH_FOR_DELEGATION (constrained), and ACCOUNTDISABLE. Stale LastLogonTimeStamp and old OS versions identify high-value/low-security targets.

## Steps

- Enumerate all hosts with OS, UAC, and lastlogon with Get-DomainComputer -Properties.
- Export computer data to CSV with Export-Csv for offline analysis.
- Find unconstrained delegation computers with -Unconstrained.
- Find constrained delegation computers with -TrustedToAuth.
- Correlate LastLogonTimeStamp, OperatingSystem, and WhenCreated to prioritize targets.

## Commands

### Enumerate all computers with key fields (SharpView)

```powershell
.\SharpView.exe Get-DomainComputer -Properties dnshostname,operatingsystem,lastlogontimestamp,useraccountcontrol
```

### Export computer data to CSV

```powershell
Get-DomainComputer -Properties dnshostname,operatingsystem,lastlogontimestamp,useraccountcontrol | Export-Csv .\<DOMAIN>.csv -NoTypeInformation
```

### Find unconstrained delegation computers

```powershell
.\SharpView.exe Get-DomainComputer -Unconstrained -Properties dnshostname,useraccountcontrol
```

### Find constrained delegation computers

```powershell
Get-DomainComputer -TrustedToAuth | select -Property dnshostname,useraccountcontrol
```
