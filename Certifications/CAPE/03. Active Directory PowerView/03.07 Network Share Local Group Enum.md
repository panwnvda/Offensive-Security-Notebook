# 03.07 Network Share Local Group Enum

> Loop `Find-DomainUserLocation` across domain hosts to map live sessions and pinpoint where users hold local admin.

## Overview

PowerView network functions: enumerate local groups and members on remote hosts, find where domain users are logged in, test local admin access, map open shares, and check share ACLs. The foreach loop iterates all domain computers to find where a specific user has local admin. Find-DomainUserLocation shows active sessions across the domain.

## Steps

- List local groups on a remote host with Get-NetLocalGroup -ComputerName.
- Enumerate local group members with Get-NetLocalGroupMember (SharpView) or loop across all computers.
- Use Convert-NameToSid + foreach loop to find all machines where a specific user is local admin.
- Use Test-AdminAccess -ComputerName to confirm local admin rights on a target host.
- Enumerate open shares with Get-NetShare -ComputerName (or SharpView).
- Check share ACLs with Get-PathAcl on a UNC path.
- Use Find-DomainUserLocation to find domain machines where users are currently logged in.

## Commands

### List local groups on a host

```powershell
Get-NetLocalGroup -ComputerName WS01 | select GroupName
```

### Enumerate local group members (SharpView)

```powershell
.\SharpView.exe Get-NetLocalGroupMember -ComputerName WS01
```

### Find all machines where user is local admin

```powershell
$sid = Convert-NameToSid harry.jones
```

### Test local admin access on a host

```powershell
Test-AdminAccess -ComputerName SQL01
```

### Enumerate open shares (SharpView)

```powershell
.\SharpView.exe Get-NetShare -ComputerName DC01
```

### Enumerate open shares (PowerView)

```powershell
Get-NetShare -ComputerName SQL01
```

### Check share path ACLs

```powershell
Get-PathAcl "\\SQL01\DB_backups"
```

### Find where domain users are logged in

```powershell
Find-DomainUserLocation
```
