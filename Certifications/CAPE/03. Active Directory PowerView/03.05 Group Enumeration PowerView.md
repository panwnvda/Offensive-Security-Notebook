# 03.05 Group Enumeration PowerView

> Find `adminCount=1` protected groups and managed groups whose non-admin manager has GenericWrite to add members.

## Overview

Enumerate all domain groups, check membership in specific groups, identify protected groups (adminCount=1), and find managed security groups where a delegated non-admin manager has GenericWrite to add members. Cross-reference managed group managers against ACLs to find lateral movement paths.

## Steps

- List all group names with Get-DomainGroup -Properties Name.
- Use Get-DomainGroupMember -Identity to enumerate members of a specific group.
- Use -AdminCount to find protected groups (adminCount=1).
- Use Find-ManagedSecurityGroups to find groups with a managedBy manager set.
- Use Get-DomainManagedSecurityGroup to see manager name, DN, and ManagerCanWrite status.

## Commands

### List all group names

```powershell
Get-DomainGroup -Properties Name
```

### Enumerate group members (SharpView)

```powershell
.\SharpView.exe Get-DomainGroupMember -Identity 'Help Desk'
```

### Find protected groups (adminCount=1)

```powershell
.\SharpView.exe Get-DomainGroup -AdminCount
```

### Find managed security groups

```powershell
Find-ManagedSecurityGroups | select GroupName
```

### Get managed security group details

```powershell
Get-DomainManagedSecurityGroup
```
