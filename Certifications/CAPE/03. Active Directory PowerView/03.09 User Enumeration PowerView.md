# 03.09 User Enumeration PowerView

> Filter `Get-DomainUser` properties to surface AS-REP-roastable, delegation, SPN, and description-field-password accounts in one pass.

## Overview

Get-DomainUser with property filters exposes AS-REP roastable accounts, constrained/unconstrained delegation targets, Kerberoastable SPNs, description-field passwords, and cross-domain foreign group membership. Export all users to CSV for offline analysis. Password set-time analysis informs targeted password spray planning.

## Steps

- Pull key properties for a single user with Get-DomainUser -Identity.
- Export all users with key properties to CSV using Export-Csv -NoTypeInformation.
- Find AS-REP roastable users with -KerberosPreauthNotRequired.
- Find constrained delegation users with -TrustedToAuth.
- Find unconstrained delegation users via LDAP filter userAccountControl:1.2.840.113556.1.4.803:=524288.
- Hunt description-field passwords with Where {$_.description -ne $null}.
- Find Kerberoastable users with -SPN flag.
- Use Find-ForeignGroup + Convert-SidToName to identify cross-domain admin group membership.
- Cross-forest Kerberoasting: query SPNs in trusted domain with -Domain freightlogistics.local.
- Sort by pwdlastset to plan password sprays; filter stale with where { $_.pwdlastset -lt (Get-Date).addDays(-90) }.

## Commands

### Enumerate a single user (full properties)

```powershell
Get-DomainUser -Identity harry.jones -Domain <DOMAIN>.local | Select-Object -Property name,samaccountname,description,memberof,whencreated,pwdlastset,lastlogontimestamp,accountexpires,admincount,userprincipalname,serviceprincipalname,mail,useraccountcontrol
```

### Export all users to CSV

```powershell
Get-DomainUser * -Domain <DOMAIN>.local | Select-Object -Property name,samaccountname,description,memberof,whencreated,pwdlastset,lastlogontimestamp,accountexpires,admincount,userprincipalname,serviceprincipalname,mail,useraccountcontrol | Export-Csv .\<DOMAIN>.csv -NoTypeInformation
```

### AS-REP roastable users

```powershell
.\SharpView.exe Get-DomainUser -KerberosPreauthNotRequired -Properties samaccountname,useraccountcontrol,memberof
```

### Constrained delegation users

```powershell
.\SharpView.exe Get-DomainUser -TrustedToAuth -Properties samaccountname,useraccountcontrol,memberof
```

### Unconstrained delegation users (LDAP filter)

```powershell
.\SharpView.exe Get-DomainUser -LDAPFilter "(userAccountControl:1.2.840.113556.1.4.803:=524288)"
```

### Hunt password in description field

```powershell
Get-DomainUser -Properties samaccountname,description | Where {$_.description -ne $null}
```

### Kerberoastable users (SPN set)

```powershell
.\SharpView.exe Get-DomainUser -SPN -Properties samaccountname,memberof,serviceprincipalname
```

### Cross-forest Kerberoastable users

```powershell
Get-DomainUser -SPN -Domain freightlogistics.local | select samaccountname,memberof,serviceprincipalname | fl
```

### Find foreign group membership

```powershell
Find-ForeignGroup
```

### Passwords sorted by set date

```powershell
Get-DomainUser -Properties samaccountname,pwdlastset,lastlogon -Domain <DOMAIN>.local | select samaccountname, pwdlastset, lastlogon | Sort-Object -Property pwdlastset
```

### Passwords not changed in 90+ days

```powershell
Get-DomainUser -Properties samaccountname,pwdlastset,lastlogon -Domain <DOMAIN>.local | select samaccountname, pwdlastset, lastlogon | where { $_.pwdlastset -lt (Get-Date).addDays(-90) }
```
