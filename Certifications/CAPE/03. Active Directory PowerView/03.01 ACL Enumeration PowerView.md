# 03.01 ACL Enumeration PowerView

> Surface GenericAll, GenericWrite, WriteOwner, WriteDACL, ForceChangePassword, and DCSync ACEs with `Find-InterestingDomainAcl`.

## Overview

Get-DomainObjectAcl and Find-InterestingDomainAcl surface abusable ACEs. Key rights to identify: GenericAll, GenericWrite, WriteOwner, WriteDACL, ForceChangePassword, and DCSync rights. Built-in Get-ACL can also enumerate ACLs on AD objects without PowerView.

## Steps

- Use Get-DomainObjectAcl -ResolveGUIDs to decode extended rights GUIDs on a specific object.
- Filter ACEs by SecurityIdentifier using a SID from ConvertTo-SID to find rights for a specific principal.
- Use Find-InterestingDomainAcl -ResolveGUIDs to sweep the whole domain for interesting ACEs.
- Use built-in (Get-ACL "AD:...").access to enumerate ACLs without PowerView.
- Use Get-ObjectACL on the domain root to find DCSync rights (Replication-Get-Changes).
- Map ACE types to attack functions: ForceChangePassword→Set-DomainUserPassword, GenericAll→Set-DomainObject, WriteDACL→Add-DomainObjectACL, WriteOwner→Set-DomainObjectOwner.

## Commands

### Get ACL for a user (resolved GUIDs)

```powershell
Get-DomainObjectAcl -Identity harry.jones -Domain <DOMAIN>.local -ResolveGUIDs
```

### Filter ACL by principal SID

```powershell
$sid = ConvertTo-SID joe.evans
```

### Sweep domain for interesting ACEs

```powershell
Find-InterestingDomainAcl -Domain <DOMAIN>.local -ResolveGUIDs
```

### Built-in ACL query for a user

```powershell
(Get-ACL "AD:$((Get-ADUser daniel.carter).distinguishedname)").access | ? {$_.IdentityReference -eq "<DOMAIN>\cliff.moore"}
```

### Find DCSync rights on domain root

```powershell
Get-ObjectACL "DC=<DOMAIN>,DC=local" -ResolveGUIDs | ? {($_.ActiveDirectoryRights -match 'GenericAll') -or ($_.ObjectAceType -match 'Replication-Get')} | Select-Object SecurityIdentifier | Sort-Object -Property SecurityIdentifier -Unique
```

### Convert DCSync SIDs to names

```powershell
$dcsync = Get-ObjectACL "DC=<DOMAIN>,DC=local" -ResolveGUIDs | ? {($_.ActiveDirectoryRights -match 'GenericAll') -or ($_.ObjectAceType -match 'Replication-Get')} | Select-Object -ExpandProperty SecurityIdentifier | Select -ExpandProperty value
```

### Convert a single SID

```powershell
convertfrom-sid S-1-5-21-2974783224-3764228556-2640795941-1883
```
