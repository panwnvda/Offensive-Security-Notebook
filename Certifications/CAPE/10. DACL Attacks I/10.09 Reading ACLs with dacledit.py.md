# 10.09 Reading ACLs with dacledit.py

> Read DACLs with `dacledit.py -action read -principal` to spot abusable ACEs before exploiting them.

## Overview

Read the DACL on a target object. -principal filters to ACEs belonging to a specific principal. Reveals abusable rights before exploitation.

## Steps

- **-target user** — read ACEs on a user object.
- **-principal user** — filter to show only that principal's ACEs.

## Commands

### dacledit.py read

```bash
python3 examples/dacledit.py -target <USER> -dc-ip <TARGET> <DOMAIN>.local/<USER>:'HTB_@cademy_stdnt!'
python3 examples/dacledit.py -principal pedro -target Rita -dc-ip <TARGET> <DOMAIN>.local/pedro:SecuringAD01
```
