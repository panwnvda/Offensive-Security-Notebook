# 10.03 dacledit.py Setup Installation

> Install ShutdownRepo's `dacledit.py` fork in a virtualenv to read/write/backup/restore AD DACLs from Linux.

## Overview

dacledit.py from the ShutdownRepo impacket fork reads and writes Active Directory DACLs from Linux. Install it into a Python virtual environment to avoid conflicts with the system impacket package. The tool supports read, write, remove, backup, and restore actions against any AD object's DACL.

## Steps

- Clone the ShutdownRepo dacledit branch of impacket from GitHub.
- Create a Python virtual environment inside the cloned repo to isolate dependencies.
- Activate the virtual environment before every use.
- Install the forked impacket package with pip inside the venv.
- Confirm the tool works by running --help.
- Key flags: -principal (attacker account), -target (victim object), -action (read/write/restore), -rights (FullControl/ResetPassword/WriteMembers/DCSync), -target-dn (Distinguished Name for domain-level objects).

## Commands

### Clone ShutdownRepo impacket

```bash
git clone https://github.com/ShutdownRepo/impacket -b dacledit
```

### Create and activate venv

```bash
python3 -m venv .dacledit
```

### Install into venv

```bash
source .dacledit/bin/activate
```

### Verify installation

```bash
python3 examples/dacledit.py --help
```
