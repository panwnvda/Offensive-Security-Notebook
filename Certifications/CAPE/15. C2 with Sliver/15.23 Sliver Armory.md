# 15.23 Sliver Armory

> Pull pre-compiled `Seatbelt`, `Rubeus`, `Certify`, `nanodump`, `bof-roast`, and `delegationbof` from Sliver's armory.

## Overview

Sliver's armory provides pre-compiled .NET aliases and native BOF extensions. Key tools used throughout the module include Seatbelt, SharpUp, SharpView, Rubeus, SharPersist, sharp-hound-4, certify, sharpsh, nanodump, bof-roast, c2tc-kerberoast, c2tc-domaininfo, and delegationbof.

## Steps

- Run `armory` with no arguments to list all available packages and their versions.
- Use `armory install <name>` for a single tool or `armory install all` for everything.
- Armory tools are invoked using their alias name followed by `--` then the tool's own arguments.
- Extensions (native BOFs) differ from aliases (.NET wrappers around sideload); both are installed the same way.

## Commands

### List armory packages

```text
sliver > armory
```

### Install single tool

```text
sliver > armory install seatbelt
```

### Install all tools

```text
sliver > armory install all
```

### Use armory tool (double-dash syntax)

```text
sliver (HIGH_RISER) > seatbelt -- -group=all
```

### SharpUp audit via armory

```text
sliver (HIGH_RISER) > sharpup -- audit
```
