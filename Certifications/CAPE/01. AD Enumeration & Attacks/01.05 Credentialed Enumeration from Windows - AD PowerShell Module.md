# 01.05 Credentialed Enumeration from Windows - AD PowerShell Module

> Use the signed `ActiveDirectory` PowerShell module on a domain-joined host for trusted, third-party-free AD enumeration.

## Overview

The built-in ActiveDirectory PowerShell module provides trusted enumeration of domain objects without third-party tools. Covers domain info, users with SPNs, trust relationships, group membership, and Protected Users.

## Steps

- Import-Module ActiveDirectory — required first step.
- Get-ADDomain — domain functional level, PDC, RID master.
- Get-ADUser -Filter SPN — users configured for Kerberoasting.
- Get-ADTrust -Filter * — enumerate all domain trusts.
- Get-ADGroup / Get-ADGroupMember — group membership.
- Get-ADUser -Filter userAccountControl-band 128 — reversible encryption accounts.

## Commands

### Import and domain info

```powershell
Import-Module ActiveDirectory; Get-ADDomain
```

### Users with SPNs

```powershell
Get-ADUser -Filter {ServicePrincipalName -ne "$null"} -Properties ServicePrincipalName
```

### Enumerate trusts

```powershell
Get-ADTrust -Filter *
```

### Group members

```powershell
Get-ADGroup -Identity "Backup Operators"; Get-ADGroupMember -Identity "Backup Operators"
```

### Reversible encryption accounts

```powershell
Get-ADUser -Filter 'userAccountControl -band 128' -Properties userAccountControl
```
