# 01.26 Snaffler

> Hunt domain-wide SMB shares for cleartext credentials, connection strings, and keys with `Snaffler.exe`.

## Overview

Snaffler automatically hunts shares across the domain for files containing credentials, connection strings, keys, and other sensitive data. Run from a domain-joined Windows host or an attack host with domain credentials. Output to a log file for offline review.

## Steps

- s enables share discovery; -d specifies the domain.
- v data shows only data-level findings (credentials, etc.).
- o writes results to a log file.
- Review log for cleartext passwords, connection strings, private keys.

## Commands

### Snaffler run

```powershell
Snaffler.exe -s -d <DOMAIN>.local -o snaffler.log -v data
```

### Snaffler alternate form

```powershell
.\Snaffler.exe -d <DOMAIN>.LOCAL -s -v data
```
