# 01.04 BloodHound Collection

> Run `bloodhound-python` or `SharpHound.exe` to ZIP every AD relationship and load it into BloodHound for attack-path analysis.

## Overview

bloodhound-python collects all AD object relationships from Linux. SharpHound.exe is the Windows ingestor. Ingest the ZIP into BloodHound for attack-path analysis including trust relationships. Collect against multiple domains for cross-forest analysis.

## Steps

- bloodhound-python -c All — full Linux collection (sessions, ACLs, trusts, GPOs).
- SharpHound.exe -c All — Windows collection with zip output.
- For cross-forest: update /etc/resolv.conf per domain and run bloodhound-python per domain.
- zip -r to combine JSON files before ingest.

## Commands

### bloodhound-python collect

```bash
sudo bloodhound-python -u 'forend' -p 'Klmcargo2' -ns 172.16.5.5 -d <DOMAIN>.local -c all
```

### SharpHound Windows

```powershell
.\SharpHound.exe -c All --zipfilename ILFREIGHT
```

### bloodhound-python full domain

```bash
bloodhound-python -d <DOMAIN>.LOCAL -dc ACADEMY-EA-DC01 -c All -u forend -p Klmcargo2
```

### bloodhound-python cross-forest

```bash
bloodhound-python -d FREIGHTLOGISTICS.LOCAL -dc ACADEMY-EA-DC03.FREIGHTLOGISTICS.LOCAL -c All -u forend@<DOMAIN>.local -p Klmcargo2
```

### zip results

```bash
zip -r ilfreight_bh.zip *.json
```
