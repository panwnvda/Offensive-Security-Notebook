# 01.06 Credentialed Enumeration from Windows - PowerView SharpView

> Pull deep ACL, SPN, GPO, and trust intel with `PowerView` — fall back to `SharpView.exe` when PowerShell is locked down.

## Overview

PowerView provides deep AD enumeration: detailed user/group info, SPN accounts, trust mapping, local admin access testing, ACL abuse, and GPO rights. SharpView is the compiled .NET equivalent for when PowerShell is blocked.

## Steps

- Get-DomainUser -Identity — pull full user attribute set.
- Get-DomainGroupMember -Recurse — nested group membership.
- Get-DomainTrustMapping — map all trusts recursively.
- Test-AdminAccess — check local admin on remote hosts.
- Get-DomainUser -SPN — enumerate Kerberoastable accounts.
- Get-DomainGPO — list all GPO display names.
- SharpView.exe — same API as PowerView for constrained environments.

## Commands

### PowerView user detail

```powershell
Get-DomainUser -Identity mmorgan -Domain <DOMAIN>.local | Select-Object -Property name,samaccountname,description,memberof,whencreated,pwdlastset,lastlogontimestamp,accountexpires,admincount,userprincipalname,serviceprincipalname,useraccountcontrol
```

### Recursive group members

```powershell
Get-DomainGroupMember -Identity "Domain Admins" -Recurse
```

### Trust mapping

```powershell
Get-DomainTrustMapping
```

### Test local admin

```powershell
Test-AdminAccess -ComputerName ACADEMY-EA-MS01
```

### Get SPN accounts

```powershell
Get-DomainUser -SPN -Properties samaccountname,ServicePrincipalName
```

### SharpView identity

```powershell
.\SharpView.exe Get-DomainUser -Identity forend
```
