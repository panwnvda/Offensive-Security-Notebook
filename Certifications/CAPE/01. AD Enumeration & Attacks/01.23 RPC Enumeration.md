# 01.23 RPC Enumeration

> Drive RPC enumeration over SMB null or authenticated sessions with `rpcclient` to dump users, SIDs, and domain info.

## Overview

rpcclient provides direct RPC access for domain info, user detail, and SID lookups over SMB null sessions or with credentials.

## Steps

- **querydominfo** — domain name, user count, lockout threshold.
- **queryuser RID** — detailed user info by RID.
- **enumdomusers** — full user list via RPC.

## Commands

### rpcclient queries

```bash
rpcclient -U "" -N 172.16.5.5
rpcclient $> querydominfo
rpcclient $> queryuser 0x457
rpcclient $> enumdomusers
```
