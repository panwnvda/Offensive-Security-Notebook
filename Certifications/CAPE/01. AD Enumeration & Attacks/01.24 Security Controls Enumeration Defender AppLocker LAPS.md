# 01.24 Security Controls Enumeration Defender AppLocker LAPS

> Map Defender, AppLocker, PowerShell language mode, and LAPS readers with `LAPSToolkit` before deploying tooling.

## Overview

Before running tools, enumerate AV and EDR status, AppLocker restrictions, PowerShell language mode, and LAPS deployment. LAPSToolkit identifies who can read LAPS passwords and which computers have them set.

## Steps

- Get-MpComputerStatus — Windows Defender status and signatures.
- sc query windefend — service-level Defender check.
- Get-AppLockerPolicy -Effective — see what is blocked.
- $ExecutionContext.SessionState.LanguageMode — detect Constrained Language Mode.
- LAPSToolkit: Find-LAPSDelegatedGroups, Find-AdmPwdExtendedRights, Get-LAPSComputers.

## Commands

### Defender status

```powershell
Get-MpComputerStatus
```

### Defender service

```cmd
sc query windefend
```

### AppLocker policy

```powershell
Get-AppLockerPolicy -Effective | select -ExpandProperty RuleCollections
```

### PS language mode

```powershell
$ExecutionContext.SessionState.LanguageMode
```

### LAPS delegated groups

```powershell
Find-LAPSDelegatedGroups
```

### LAPS extended rights

```powershell
Find-AdmPwdExtendedRights
```

### LAPS computers

```powershell
Get-LAPSComputers
```
