# 06.16 SMB Enumeration Options

> Pull active sessions, logged-on users, local groups, WMI queries, and RID brute-force data via authenticated `nxc smb` flags.

## Overview

Authenticated SMB enumeration covers active sessions, logged-on users, computers (via LDAP), disk drives, local/domain groups (with member listing), WMI queries (including custom namespaces), local SAM dump, and RID brute-force. Most options require domain user privileges; `--sam` and `--loggedon-users` typically require local admin.

## Steps

- `--loggedon-users` shows users with active interactive sessions; add `--loggedon-users-filter <name>` to target specific users.
- `--sessions` shows users whose credentials are in use on the target (not necessarily logged in).
- `--computers` enumerates domain computers via LDAP query (available in SMB protocol).
- `--disks` lists drive letters on the target.
- `--local-groups` enumerates local groups with member counts; `--groups` enumerates domain groups.
- `--groups <GROUP NAME>` lists members of a specific domain group.
- `--sam` dumps local SAM hashes (requires local admin / Pwn3d!).
- `--wmi "SELECT ..."` issues WMI queries; `--wmi-namespace "root\WMI"` specifies a non-default namespace.

## Commands

### Logged-on users

```bash
nxc smb <TARGET> -u robert -p <PASS> --loggedon-users
```

### Filter for specific logged-on user

```bash
nxc smb <TARGET> -u robert -p <PASS> --loggedon-users --loggedon-users-filter julio
```

### Enumerate domain computers

```bash
nxc smb <TARGET> -u robert -p <PASS> --computers
```

### Enumerate disk drives

```bash
nxc smb <TARGET> -u robert -p <PASS> --disks
```

### Dump local SAM hashes

```bash
nxc smb <TARGET> -u robert -p <PASS> --local-auth --sam
```

### Enumerate local groups

```bash
nxc smb <TARGET> -u robert -p <PASS> --local-groups
```

### Enumerate domain groups

```bash
nxc smb <TARGET> -u robert -p <PASS> --groups
```

### List members of a domain group

```bash
nxc smb <TARGET> -u robert -p <PASS> --groups Administrators
```

### WMI query (default namespace)

```bash
nxc smb <TARGET> -u robert -p <PASS> --wmi "SELECT Caption,ProcessId FROM Win32_Process WHERE Caption LIKE '%sysmon%'"
```

### WMI query (custom namespace)

```bash
nxc smb <TARGET> -u robert -p <PASS> --wmi "SELECT * FROM MSPower_DeviceEnable" --wmi-namespace "root\WMI"
```

### RID brute-force (authenticated)

```bash
nxc smb <TARGET> -u grace -p <PASS> --rid-brute
```
