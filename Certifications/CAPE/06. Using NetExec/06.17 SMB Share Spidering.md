# 06.17 SMB Share Spidering

> Crawl shares with NetExec's `--spider` + `--pattern/--regex/--content` or the `spider_plus` JSON inventory in `/tmp/cme_spider_plus/`.

## Overview

CME provides multiple methods to search SMB shares: `--spider` with `--pattern`, `--regex`, and `--content` for targeted searches; `spider_plus` module creates a full JSON inventory at `/tmp/cme_spider_plus/<IP>.json`. Use `--get-file` and `--put-file` with `--share` for file transfer. `spider_plus` with `READ_ONLY=false` downloads all accessible files.

## Steps

- `--shares` first to identify accessible shares and their READ/WRITE permissions.
- `--spider <SHARE> --pattern txt` recursively searches a share for filenames matching the pattern.
- `--spider <SHARE> --regex .` lists ALL files and directories in the share.
- `--spider <SHARE> --content --regex <word>` searches file contents for a pattern.
- `--share <SHARE> --get-file <remote_path> <local_path>` downloads a specific file.
- `--share <SHARE> --put-file <local_path> <share_path>` uploads a file to a writable share.
- `-M spider_plus -o EXCLUDE_DIR=IPC$,...` generates JSON inventory without touching system shares.
- `-M spider_plus -o EXCLUDE_DIR=ADMIN$,... READ_ONLY=false` downloads all accessible files.

## Commands

### List shares

```bash
nxc smb <TARGET> -u grace -p <PASS> --shares
```

### Spider share for .txt files

```bash
nxc smb <TARGET> -u grace -p <PASS> --spider IT --pattern txt
```

### List all files with regex

```bash
nxc smb <TARGET> -u grace -p <PASS> --spider IT --regex .
```

### Search file contents

```bash
nxc smb <TARGET> -u grace -p <PASS> --spider IT --content --regex Encrypt
```

### Download file from share

```bash
nxc smb <TARGET> -u grace -p <PASS> --share IT --get-file Creds.txt Creds.txt
```

### Upload file to share

```bash
nxc smb <TARGET> -u grace -p <PASS> --share IT --put-file /etc/passwd passwd
```

### spider_plus JSON inventory

```bash
nxc smb <TARGET> -u grace -p '<PASS>' -M spider_plus -o EXCLUDE_DIR=IPC$,print$,NETLOGON,SYSVOL
```

### spider_plus download all files

```bash
nxc smb <TARGET> -u grace -p <PASS> -M spider_plus -o EXCLUDE_DIR=ADMIN$,IPC$,print$,NETLOGON,SYSVOL READ_ONLY=false
```
