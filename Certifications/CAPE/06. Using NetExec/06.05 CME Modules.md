# 06.05 CME Modules

> Extend NetExec with `-M user-desc`, `daclread`, `laps`, `gpp_password`, `spider_plus`, and dozens more enumeration modules.

## Overview

CME modules extend enumeration via `-M <module>`. List modules with `nxc <protocol> -L`; view options with `-M <module> --options`; pass options with `-o KEY=value`. Key LDAP modules: user-desc (passwords in descriptions), groupmembership, MAQ, daclread (DACL/ACE enumeration including DCSync rights), laps, get-network, adcs, ldap-checker. Key SMB modules: gpp_password, gpp_autologin, spider_plus, slinky, drop-sc, rdp, keepass_discover, and many more.

## Steps

- `nxc ldap -L` and `nxc smb -L` list all available modules per protocol.
- `nxc <proto> -M <module> --options` shows module-specific options.
- Options are passed with `-o KEY=value` (multiple options space-separated).
- gpp_password searches SYSVOL GPO XML files for cpassword values.
- gpp_autologin searches for Registry.xml autologon entries.
- user-desc queries AD descriptions; default keywords: pass, creds, creden, key, secret, default.
- groupmembership requires `-o USER=<username>` to query a specific user's groups.
- MAQ retrieves the MS-DS-Machine-Account-Quota attribute.
- daclread reads DACLs; use `TARGET=<sAMAccountName>` or `TARGET_DN=<DN>` plus `ACTION=read`.
- To find DCSync rights: use `TARGET_DN="DC=<DOMAIN>,DC=htb" ACTION=read RIGHTS=DCSync`.

## Commands

### GPP password search

```bash
nxc smb <TARGET> -u grace -p <PASS> -M gpp_password
```

### GPP autologin search

```bash
nxc smb <TARGET> -u grace -p <PASS> -M gpp_autologin
```

### User descriptions (default keywords)

```bash
nxc ldap dc01.<DOMAIN>.htb -u grace -p <PASS> -M user-desc
```

### User descriptions (custom keywords)

```bash
nxc ldap dc01.<DOMAIN>.htb -u grace -p <PASS> -M user-desc -o KEYWORDS=pwd,admin
```

### Group membership for a user

```bash
nxc ldap dc01.<DOMAIN>.htb -u grace -p <PASS> -M groupmembership -o USER=julio
```

### Machine Account Quota

```bash
nxc ldap dc01.<DOMAIN>.htb -u robert -p '<PASS>' -M maq
```

### Read DACL for a user object

```bash
nxc ldap dc01.<DOMAIN>.htb -u '' -p '' -M daclread -o TARGET=grace ACTION=read
```

### Find principals with DCSync rights

```bash
nxc ldap dc01.<DOMAIN>.htb -u grace -p <PASS> -M daclread -o TARGET_DN="DC=<DOMAIN>,DC=htb" ACTION=read RIGHTS=DCSync
```
