# 02.10 RSAT Installation

> Install RSAT to unlock `Get-ADUser`, `Get-ADGroup`, and ADUC for native AD enumeration from Windows.

## Overview

Install Remote Server Administration Tools on Windows to get the AD PowerShell module and ADUC. Required for Get-ADUser, Get-ADGroup, etc.

## Steps

- **Get-WindowsCapability** — list available RSAT features.
- **Add-WindowsCapability** — online install from Windows Update.

## Commands

### List + install RSAT

```powershell
Get-WindowsCapability -Name RSAT* -Online | Select-Object -Property Name, State
Add-WindowsCapability -Name Rsat.ActiveDirectory.DS-LDS.Tools~~~~0.0.1.0 -Online
```
