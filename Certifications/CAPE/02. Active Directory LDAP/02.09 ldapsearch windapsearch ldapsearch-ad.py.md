# 02.09 ldapsearch windapsearch ldapsearch-ad.py

> Drive Linux-side LDAP enumeration with `ldapsearch` (raw), `windapsearch` (common queries), and `ldapsearch-ad.py` (structured AD output).

## Overview

Linux-side LDAP enumeration without Kerberos tickets. ldapsearch for raw queries; windapsearch for common AD objects; ldapsearch-ad.py for structured AD-aware output.

## Steps

- **ldapsearch -x** — anonymous/simple-auth base query.
- **windapsearch --functionality** — enumerate LDAP capabilities.
- **ldapsearch-ad.py -t kerberoast** — list Kerberoastable SPNs.

## Commands

### ldapsearch anonymous

```bash
ldapsearch -H ldap://<TARGET> -x -b "dc=<DOMAIN>,dc=local"
```

### windapsearch

```bash
python3 windapsearch.py --dc-ip <TARGET> -u "" --functionality
python3 windapsearch.py --dc-ip <TARGET> -u "" -U
python3 windapsearch.py --dc-ip <TARGET> -u "" -C
```

### ldapsearch-ad.py queries

```bash
python3 ldapsearch-ad.py -l <TARGET> -t info
python3 ldapsearch-ad.py -l <TARGET> -d <DOMAIN> -u james.cross -p Summer2020 -t pass-pols
python3 ldapsearch-ad.py -l <TARGET> -d <DOMAIN> -u james.cross -p Summer2020 -t kerberoast | grep servicePrincipalName:
python3 ldapsearch-ad.py -l <TARGET> -d <DOMAIN> -u james.cross -p Summer2020 -t asreproast
```
