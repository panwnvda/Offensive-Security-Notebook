# 02.12 UAC Attributes Enumeration PowerShell PowerView

> Decode raw UAC integers (NORMAL_ACCOUNT, PASSWD_NOTREQD, DONT_REQ_PREAUTH, etc.) — PowerView's `Get-DomainUser` returns flags pre-resolved.

## Overview

User-Account-Control attributes control domain account behaviour (NORMAL_ACCOUNT, ACCOUNTDISABLE, PASSWD_NOTREQD, DONT_EXPIRE_PASSWORD, DONT_REQ_PREAUTH, etc.). Built-in AD cmdlets return raw integer values that must be converted. PowerView's Get-DomainUser returns human-readable flag names directly.

## Steps

- Get-ADUser -Filter {adminCount -gt 0} with useraccountcontrol property returns raw integer UAC values.
- Convert-UserAccountControlValues.ps1 converts the raw integer to named flag components.
- Get-DomainUser * -AdminCount (PowerView) returns flags as readable strings (PASSWD_NOTREQD, DONT_REQ_PREAUTH, etc.).
- Import PowerView from c:\tools: Import-Module .\PowerView.ps1.
- Watch for DONT_REQ_PREAUTH (AS-REP roastable) and PASSWD_NOTREQD combined with adminCount=1.

## Commands

### Enumerate admin accounts with UAC values

```powershell
Get-ADUser -Filter {adminCount -gt 0} -Properties admincount,useraccountcontrol | select Name,useraccountcontrol
```

### Convert UAC integer to flag names

```powershell
.\Convert-UserAccountControlValues.ps1
```

### PowerView — admin accounts with readable UAC flags

```powershell
Get-DomainUser * -AdminCount | select samaccountname,useraccountcontrol
```
