# 02.08 LDAPFilter UAC Bit-Field Queries

> Test specific UAC flags with LDAP bitwise OID `1.2.840.113556.1.4.803` (disabled=2, trusted-for-delegation=524288, PASSWD_NOTREQD=32).

## Overview

LDAP OID 1.2.840.113556.1.4.803 is bitwise AND. Use it to test specific userAccountControl flags: disabled=2, trusted-for-delegation=524288, passwd-not-required=32.

## Steps

- **:1.2.840.113556.1.4.803:=2** — disabled accounts.
- **:1.2.840.113556.1.4.803:=524288** — unconstrained delegation.
- **:1.2.840.113556.1.4.803:=32** — password not required.
- **:1.2.840.113556.1.4.1941:** — recursive group membership (LDAP_MATCHING_RULE_IN_CHAIN).

## Commands

### UAC bit-flag LDAPFilter queries

```powershell
Get-ADUser -LDAPFilter '(userAccountControl:1.2.840.113556.1.4.803:=2)' | select name
Get-ADGroup -LDAPFilter '(member:1.2.840.113556.1.4.1941:=CN=Harry Jones,OU=Network Ops,OU=IT,OU=Employees,DC=<DOMAIN>,DC=LOCAL)'
Get-ADUser -Properties * -LDAPFilter '(&(objectCategory=user)(description=*))' | select samaccountname,description
Get-ADUser -Properties * -LDAPFilter '(userAccountControl:1.2.840.113556.1.4.803:=524288)' | select samaccountname,useraccountcontrol
Get-ADComputer -Properties * -LDAPFilter '(userAccountControl:1.2.840.113556.1.4.803:=524288)' | select samaccountname,useraccountcontrol
Get-AdUser -LDAPFilter '(&(objectCategory=person)(objectClass=user)(userAccountControl:1.2.840.113556.1.4.803:=32))'
```
