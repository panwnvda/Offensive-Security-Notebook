# 02.03 Credentialed LDAP Enumeration windapsearch ldapsearch-ad.py

> Extract Domain Admins, delegation users, password policy, and Kerberoastable SPNs with credentialed `windapsearch` / `ldapsearch-ad.py`.

## Overview

With valid domain credentials, windapsearch and ldapsearch-ad.py extract much richer AD data: Domain Admins membership, unconstrained delegation users and computers, password policy, Kerberoastable SPNs, and AS-REP-roastable accounts. These tools are particularly useful during Linux-based AD assessments.

## Steps

- windapsearch --da — enumerate all Domain Admins with credentials.
- windapsearch --unconstrained-users — find accounts trusted for unconstrained Kerberos delegation.
- ldapsearch-ad.py -t info — pull domain/forest functional level and DC info (no creds needed).
- ldapsearch-ad.py -t pass-pols — retrieve default and fine-grained password policies.
- ldapsearch-ad.py -t kerberoast — list all accounts with SPNs set (Kerberoasting candidates).
- ldapsearch-ad.py -t asreproast — list accounts with DoesNotRequirePreAuth set.

## Commands

### windapsearch — enumerate Domain Admins

```bash
python3 windapsearch.py --dc-ip <TARGET> -u <DOMAIN>\\james.cross --da
```

### windapsearch — unconstrained delegation users

```bash
python3 windapsearch.py --dc-ip <TARGET> -d <DOMAIN>.local -u <DOMAIN>\\james.cross --unconstrained-users
```

### ldapsearch-ad.py — anonymous info

```bash
python3 ldapsearch-ad.py -l <TARGET> -t info
```

### ldapsearch-ad.py — password policy

```bash
python3 ldapsearch-ad.py -l <TARGET> -d <DOMAIN> -u james.cross -p Summer2020 -t pass-pols
```

### ldapsearch-ad.py — Kerberoastable SPNs

```bash
python3 ldapsearch-ad.py -l <TARGET> -d <DOMAIN> -u james.cross -p Summer2020 -t kerberoast | grep servicePrincipalName:
```

### ldapsearch-ad.py — AS-REP-roastable accounts

```bash
python3 ldapsearch-ad.py -l <TARGET> -d <DOMAIN> -u james.cross -p Summer2020 -t asreproast
```
