# 04.07 Custom Cypher Queries

> Extend BloodHound with shortest-path-from-owned Cypher, description-password hunts, and per-host local-admin enumeration.

## Overview

Custom Cypher queries extend BloodHound beyond its built-in set. The most powerful general-purpose query finds all shortest paths from any owned or named node to any other node. Other useful queries: find users with passwords in their description, enumerate all local admins per computer, find nodes reachable via a specific edge type. Queries are typed into the Raw Query box (visible only when Query Debug Mode is enabled in Settings).

## Steps

- **allshortestPaths from named user** — most important custom query per the course (credited to PlainText).
- **Users with non-blank description** — passwords are often stored in AD descriptions; run in Neo4j console, not BH.
- **All local admins per computer** — identifies who has admin rights and how many admins each machine has.
- **Find any edge type** — replace the edge name to hunt for any specific privilege.
- **Dangerous rights for Domain Users** — finds if all domain users have write-level rights over computers.
- **Replace shortestPath with allshortestPaths** for more results (slower).
- **When allshortestPaths returns nothing** — fall back to PowerView `Get-DomainObjectAcl`.

## Commands

### allshortestPaths from any node matching 'peter' to anything

```text
MATCH p = allshortestPaths((n)-[*1..]->(c)) WHERE n.name =~ '(?i)peter.*' AND NOT c=n RETURN p
```

### shortestPath variant (faster, single path)

```text
MATCH p = shortestPath((n)-[*1..]->(c)) WHERE n.name =~ '(?i)peter.*' AND NOT c=n RETURN p
```

### Find users with non-blank description (Neo4j console only)

```text
-- Run at http://localhost:7474/browser/
MATCH (u:User) WHERE u.description IS NOT NULL RETURN u.name,u.description
```

### All local admins per computer (sorted by admin count)

```text
MATCH (c:Computer) OPTIONAL MATCH (u1:User)-[:AdminTo]->(c) OPTIONAL MATCH (u2:User)-[:MemberOf*1..]->(:Group)-[:AdminTo]->(c) WITH COLLECT(u1) + COLLECT(u2) AS TempVar,c UNWIND TempVar AS Admins RETURN c.name AS COMPUTER, COUNT(DISTINCT(Admins)) AS ADMIN_COUNT,COLLECT(DISTINCT(Admins.name)) AS USERS ORDER BY ADMIN_COUNT DESC
```

### Dangerous rights for Domain Users over computers

```text
MATCH p=(g:Group)-[r:Owns|WriteDacl|GenericAll|WriteOwner|ExecuteDCOM|GenericWrite|AllowedToDelegate|ForceChangePassword]->(c:Computer) WHERE g.name STARTS WITH "DOMAIN USERS" RETURN p
```

### Find any node with a specific edge (replace WriteSPN with any edge name)

```text
MATCH p=((n)-[r:WriteSPN]->(m)) RETURN p
```

### Two-stage: allshortestPaths from any owned node

```text
-- Stage 1 (returns list of owned nodes):
MATCH (n {owned:true}) RETURN n.name

-- Stage 2 (uses $result from stage 1 selection):
MATCH p = allshortestPaths((n)-[*1..]->(c)) WHERE n.name = $result AND NOT c=n RETURN p
```
