# 04.06 BlueHound Blue Team Monitoring

> Schedule SharpHound + Cypher queries with `BlueHound` for ongoing AD posture monitoring and dashboards.

## Overview

BlueHound is an open-source blue team tool that automates SharpHound collection, runs BloodHound Cypher queries on a schedule, and generates reports for AD security posture monitoring. It supports custom dashboards, importable/exportable configurations, and highlights areas like dangerous Domain Users privileges, Kerberoastable paths, and sessions on non-DC machines.

## Steps

- **Data Import tab** — configure tool path to SharpHound.exe, set collection arguments and schedule.
- **Configurations tab** — define Domain Controllers, Domain Admins group, crown jewel servers.
- **Query Runner** — click 'RUN ALL' to populate all built-in report tiles.
- **Custom queries** — add tiles with a + button; define name, type, size, and Cypher.
- **Import/Export dashboards** — share configuration JSON between instances.
- Areas to monitor: admins with sessions on non-DC machines; dangerous privileges in Domain Users; paths from Kerberoastable users; AS-REP roastable users; users with elevated computer privileges.
- Related tools: **ImproHound** (AD tier model analysis), **GoodHound** (prioritize remediation by busiest paths to high-value targets).

## Commands

### Extract and run BlueHound

```powershell
PS C:\Tools> Expand-Archive .\BlueHound-win32-x64-1.1.0.zip .
# Open BlueHound.exe → Login with neo4j / Password123
```

### Import custom dashboard

```text
# File: C:\Tools\bluehound_dashboard_htb.json
# In BlueHound: Configurations → Import Config → select the JSON file
```

### BlueHound custom Cypher query example

```text
-- Count enabled users:
MATCH (u:User {enabled:true}) RETURN COUNT(u)
```
