# 04.04 BloodHound GUI Navigation

> Mark Owned / High-Value nodes, run pre-built queries, and right-click edges to read built-in Abuse Info + Opsec considerations.

## Overview

The BloodHound GUI consists of a graph canvas, search bar, pathfinding tool, Node Info panel, Analysis tab (pre-built queries), and Raw Query box. On login it auto-runs 'Find all Domain Admins'. Nodes can be marked Owned (skull icon) or High Value (crown). Every edge has built-in Abuse Info and Opsec Considerations accessible by right-click.

## Steps

- **Search prefixes**: user:, group:, computer:, OU:, GPO:, container:, domain:, AZUser:, AZGroup:, AZVM:, AZApp:, AZRole:, AZDevice:, AZKeyVault:, AZServicePrincipal:, AZSubscription:, AZTenant:, AZManagementGroup:, AZResourceGroup:.
- **Pathfinding**: click the two-node icon in the search bar — enter starting node and ending node.
- **Edge abuse help**: right-click any edge → Abuse Info (exploitation steps) + Opsec Considerations (detection risk).
- **Mark as Owned**: right-click node → Mark as Owned (skull appears); re-run 'Shortest Path from Owned Principals' after each compromise.
- **Query Debug Mode**: Settings → Query Debug Mode — dumps the Cypher for every built-in query into the Raw Query box.
- **Database Info tab**: shows object counts; use 'Warming Up Database' to load DB into RAM for faster queries.
- **Keyboard shortcuts**: CTRL = cycle node labels, Spacebar = spotlight, Backspace = go back, S = toggle info panel.

## Commands

### Search bar node type prefixes

```text
# AD types:
domain:<DOMAIN>.HTB      # domain overview
user:peter                    # find user named peter
group:domain admins           # find group
computer:dc01                 # find computer

# Azure types:
AZUser:  AZGroup:  AZVM:  AZApp:  AZRole:
AZDevice:  AZKeyVault:  AZServicePrincipal:
AZSubscription:  AZTenant:  AZManagementGroup:  AZResourceGroup:
```

### Right-click node context menu options

```text
Set as Starting Node
Set as Ending Node
Shortest Paths to Here
Shortest Paths to Here from Owned
Edit Node
Mark as Owned          # adds skull icon
Mark as High Value     # adds crown icon
Delete Node
```

### Node Info tab sections

```text
# For User nodes:
Overview > Node Properties > Extra Properties*
Group Membership > Local Admin Rights > Execution Rights
Outbound Object Control > Inbound Control Rights

# For Domain nodes:
Map OU Structure (high-level AD layout)
Foreign Members > Inbound Trusts > Outbound Trusts

# For GPO nodes:
Affected Objects (which computers/users the GPO applies to)

# *Extra Properties only populated with --collectallproperties
```
