# 16.06 AppLocker Bypass RegAsm.exe LOLBIN

> Run code via `RegAsm /U` after extending a COM-registration class and overriding `RegisterClass()` — another `%WINDIR%` LOLBIN.

## Overview

RegAsm.exe (Registry Assembly) is another %WINDIR%-whitelisted LOLBIN. Similar to InstallUtil: extend a COM-registration class, override RegisterClass(). RegAsm /U triggers Unregister().

## Steps

- Create Console App, extend ComRegisterFunctionAttribute decorated class.
- Use `[ComRegisterFunction]` decorated method for /regfile, or override Unregister for /U.
- Compile x64 Release, execute: `RegAsm.exe /U <file>`

## Commands

### RegAsm bypass execution

```powershell
# Skills Assessment I template:
C:\Windows\Microsoft.NET\Framework64\v4.0.30319\RegAsm.exe /U <YOUR_FILE>
# Structure: class decorated with [ComRegisterFunction]/[ComUnregisterFunction]
# Unregister method contains shellcode execution
# Compile as Console App x64 Release
```
