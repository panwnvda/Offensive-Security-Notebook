# 16.05 AppLocker Bypass InstallUtil

> Drop an `InstallUtil`-loadable .NET class with `Uninstall()` override — `InstallUtil /U` runs your shellcode under the whitelisted LOLBIN.

## Overview

InstallUtil.exe lives under %WINDIR% and is whitelisted by default AppLocker rules. Extend System.Configuration.Install.Installer, override Uninstall(). InstallUtil /U triggers the Uninstall method — runs your shellcode.

## Steps

- Create Console App (.NET Framework), add reference to System.Configuration.Install.dll.
- Extend Installer class, override Uninstall() with shellcode execution code.
- Compile x64 Release. Run with 64-bit InstallUtil.exe using /U flag.
