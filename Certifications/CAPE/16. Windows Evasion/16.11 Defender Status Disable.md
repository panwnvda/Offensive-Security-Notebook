# 16.11 Defender Status Disable

> Check Defender state with `Get-MpThreatDetection` — `ThreatID` pinpoints exactly which file tripped the signature.

## Overview

Check Windows Defender status and threat history. Disable real-time monitoring during dev/test. Get-MpThreatDetection with ThreatID pinpoints exactly which file triggered a detection.

## Steps

- **Get-MpComputerStatus** — signature date, RT protection state, tamper protection.
- **Get-MpThreat** — history of detected threats (ThreatName, CategoryID).
- **Get-MpThreatDetection -ThreatID** — exact file path and detection timestamp.
- **Set-MpPreference -DisableRealTimeMonitoring $true** — disable for dev (admin required).

## Commands

### Defender status + disable

```powershell
Get-MpComputerStatus
Get-MpThreat
Get-MpThreatDetection -ThreatID 2147894794
Set-MpPreference -DisableRealTimeMonitoring $true
# Clear detection history
Remove-Item -Recurse 'C:\ProgramData\Microsoft\Windows Defender\Scans\History\Service\*'
```
