# 16.17 Signature Analysis with ThreatCheck

> Binary-search a flagged file with `ThreatCheck` (wraps `MpCmdRun`) to pinpoint the exact bytes triggering Defender detection.

## Overview

ThreatCheck binary-searches a file by feeding chunks to Defender's MpCmdRun until it finds the exact bytes triggering detection. Iterative process: find trigger → rename/encrypt/change → repeat until clean.

## Steps

- **ThreatCheck.exe -f binary** — finds exact bytes triggering detection.
- Disable RT protection first so ThreatCheck can scan without deleting the file.
- Iterate: ThreatCheck → modify source → recompile → repeat.
