# 07.04 Enumeration lookupsid samrdump GetADUsers findDelegation smbclient

> Brute-force SIDs (`lookupsid.py`), dump users (`samrdump.py`), and find delegation (`findDelegation.py`) from Linux.

## Overview

Impacket includes several enumeration tools: lookupsid for SID brute-force, samrdump for user/group enumeration via SAMR, GetADUsers for AD user listing, findDelegation for delegation discovery, smbclient for SMB share interaction.

## Steps

- lookupsid: brute-forces SIDs to enumerate users/groups — works on null sessions
- samrdump: dumps users, groups, shares via SAMR protocol — older but stealthy
- GetADUsers: quick dump of all AD user attributes via LDAP
- findDelegation: finds all unconstrained/constrained/RBCD delegation entries
- smbclient: interactive SMB client; supports get/put/ls across shares
- All tools support PTH and Kerberos auth

## Commands

### lookupsid — SID brute force / user enum

```bash
impacket-lookupsid DOMAIN/user:'P@ss'@TARGET_IP
impacket-lookupsid -hashes :NThash DOMAIN/user@TARGET_IP
# Null session (if allowed):
impacket-lookupsid -no-pass WORKGROUP/anonymous@TARGET_IP
```

### samrdump — user / group / share enum

```bash
impacket-samrdump DOMAIN/user:'P@ss'@TARGET_IP
impacket-samrdump -hashes :NThash DOMAIN/user@TARGET_IP
impacket-samrdump -csv DOMAIN/user:'P@ss'@TARGET_IP > users.csv
```

### GetADUsers — AD user dump

```bash
impacket-GetADUsers DOMAIN/user:'P@ss' -dc-ip DC_IP -all
impacket-GetADUsers DOMAIN/user:'P@ss' -dc-ip DC_IP
```

### findDelegation — discover delegation

```bash
impacket-findDelegation DOMAIN/user:'P@ss' -dc-ip DC_IP
impacket-findDelegation -hashes :NThash DOMAIN/user -dc-ip DC_IP
impacket-findDelegation DOMAIN/user:'P@ss' -dc-ip DC_IP -target-domain DOMAIN.LOCAL
```

### smbclient — interactive SMB shell

```bash
impacket-smbclient DOMAIN/user:'P@ss'@TARGET_IP
impacket-smbclient -hashes :NThash DOMAIN/user@TARGET_IP
# Shell commands: shares, use C$, ls, cd Windows, get file.txt, put evil.exe, rm, mkdir
```

### GetComputerStatus — check host reachability

```bash
impacket-GetComputerStatus DOMAIN/user:'P@ss' -dc-ip DC_IP
```
