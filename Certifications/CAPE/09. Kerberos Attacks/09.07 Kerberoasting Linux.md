# 09.07 Kerberoasting Linux

> Request and crack TGS hashes from Linux with `GetUserSPNs.py -request` — mode 13100 for RC4, AES is much slower.

## Overview

Impacket's GetUserSPNs.py enumerates SPN-bearing user accounts and requests TGS hashes from Linux. Without -request it only lists; with -request it returns crackable hashes. Hash mode 13100 cracks RC4 TGS tickets. AES tickets ($krb5tgs$18$*) require the /tgtdeleg equivalent or are significantly slower to crack.

## Steps

- Run GetUserSPNs.py domain/user without -request to list Kerberoastable accounts.
- Add -request to retrieve TGS hashes for all SPN accounts.
- Note Delegation column output to identify further attack paths from the results.
- Crack with hashcat -m 13100 targeting RC4 ($krb5tgs$23$*) hashes.

## Commands

### GetUserSPNs list accounts

```bash
GetUserSPNs.py <DOMAIN>.local/pixis
```

### GetUserSPNs request hashes

```bash
GetUserSPNs.py <DOMAIN>.local/pixis -request
```

### hashcat crack TGS

```bash
hashcat -m 13100 hashes.txt rockyou.txt
```
