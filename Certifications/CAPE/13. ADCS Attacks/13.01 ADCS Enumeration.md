# 13.01 ADCS Enumeration

> Enumerate CAs, templates, and vulnerable configs with `certipy find -vulnerable` or `Certify.exe /vulnerable` — BloodHound data optional.

## Overview

Certify.exe and certipy enumerate CA servers, certificate templates, and flag vulnerable configurations. NetExec can also detect ADCS presence via LDAP. The -vulnerable flag on certipy and /vulnerable on Certify.exe narrow output to exploitable templates. BloodHound data can be generated alongside text output.

## Steps

- Use `net localgroup "Cert Publishers"` or LDAP to confirm ADCS presence in the domain
- Run `netexec ldap` with `-M adcs` to identify PKI Enrollment Servers from Linux
- Run `certipy find` with `-stdout` to enumerate all CAs and templates; add `-vulnerable` to filter
- Run `Certify.exe find` (no /vulnerable) to list all templates; use `Certify.exe find /vulnerable` to filter
- Run `Certify.exe cas` to list CA flags including EDITF_ATTRIBUTESUBJECTALTNAME2
- Certipy outputs .txt, .json, and BloodHound zip by default; use `-bloodhound` to force only BH output
- Combine `bloodhound-python` AD data with `certipy find` BH zip in the ly4k fork of BloodHound for attack-path visualization

## Commands

### netexec ADCS module

```bash
netexec ldap <TARGET> -u "blwasp" -p "Password123!" -M adcs
```

### certipy find (all templates)

```bash
certipy find -u 'blwasp@lab.local' -p 'Password123!' -dc-ip <TARGET> -stdout
```

### certipy find (vulnerable only)

```bash
certipy find -u 'blwasp@lab.local' -p 'Password123!' -dc-ip <TARGET> -vulnerable -stdout
```

### certipy find BloodHound output

```bash
certipy find -u 'blwasp@lab.local' -p 'Password123!' -dc-ip <TARGET> -bloodhound
```

### Certify.exe find all

```powershell
.\Certify.exe find
```

### Certify.exe find vulnerable

```powershell
.\Certify.exe find /vulnerable
```

### Certify.exe list CAs

```powershell
.\Certify.exe cas
```

### BloodHound-Python collection

```bash
bloodhound-python -u 'blwasp' -d 'lab.local' -p 'Password123!' --zip -ns <TARGET> --dns-tcp
```

### PowerShell ESC1 template query

```powershell
Get-ADObject -LDAPFilter '(&(objectclass=pkicertificatetemplate)(!(mspki-enrollment-flag:1.2.840.113556.1.4.804:=2))(|(mspki-ra-signature=0)(!(mspki-ra-signature=*)))(|(pkiextendedkeyusage=1.3.6.1.4.1.311.20.2.2)(pkiextendedkeyusage=1.3.6.1.5.5.7.3.2)(pkiextendedkeyusage=1.3.6.1.5.2.3.4))(mspki-certificate-name-flag:1.2.840.113556.1.4.804:=1))' -SearchBase 'CN=Configuration,DC=lab,DC=local'
```
