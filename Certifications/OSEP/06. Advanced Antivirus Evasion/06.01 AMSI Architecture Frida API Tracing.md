# 06.01 AMSI Architecture Frida API Tracing

> Section 7.2 · AMSI.DLL APIs · frida-trace · result values 1 / 32768

## Overview

AMSI.DLL is loaded into every PowerShell and PowerShell_ISE process (and JScript, VBScript, VBA Office 2019+, .NET 4.8+). The API chain is: AmsiInitialize (once at startup, creates amsiContext) → AmsiOpenSession (per command, creates amsiSession) → AmsiScanBuffer (passes content to AV, returns AMSI_RESULT) → AmsiCloseSession. A result of 1 = clean; 32768 = malicious (AMSI_RESULT_DETECTED). Use Frida to hook all Amsi* exports and inspect exactly what is passed to AmsiScanBuffer and what Windows Defender returns. The string 'AmsiUtils' is itself a Defender signature — split it: 'Am'+'siUtils' to bypass.

## Steps

- **AmsiInitialize** — takes appName and *amsiContext; called before any PS commands, cannot be influenced.
- **AmsiOpenSession** — takes amsiContext, creates *amsiSession per command.
- **AmsiScanBuffer** — takes amsiContext, buffer, length, contentName, amsiSession, *result; result 1=clean, 32768=malicious.
- **Frida trace** — `frida-trace -p <PID> -x amsi.dll -i Amsi*`; auto-generates JS handler files.
- **onEnter hook** — print args[1] buffer content with Memory.readUtf16String; store args[5] result pointer.
- **onLeave hook** — print Memory.readUShort(resultPointer) to see AMSI_RESULT.
- **'AmsiUtils' is flagged** — result 32768; split to 'Am'+'siUtils' → result 1 (clean).

### AmsiScanBuffer.js hooks

```javascript
// Hook AmsiScanBuffer onEnter — log all arguments including the buffer being scanned
onEnter: function (log, args, state) {
  log('[*] AmsiScanBuffer()');
  log('|- amsiContext: ' + args[0]);
  log('|- buffer: ' + Memory.readUtf16String(args[1]));
  log('|- length: ' + args[2]);
  log('|- contentName ' + args[3]);
  log('|- amsiSession ' + args[4]);
  log('|- result ' + args[5] + "\n");
  this.resultPointer = args[5];
},

// Hook AmsiScanBuffer onLeave — print the AMSI_RESULT value (1=clean, 32768=malicious)
onLeave: function (log, retval, state) {
  log('[*] AmsiScanBuffer() Exit');
  resultPointer = this.resultPointer;
  log('|- Result value is: ' + Memory.readUShort(resultPointer) + "\n");
}
```
