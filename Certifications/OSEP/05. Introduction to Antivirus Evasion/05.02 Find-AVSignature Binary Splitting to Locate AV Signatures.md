# 05.02 Find-AVSignature Binary Splitting to Locate AV Signatures

> Section 6.3 · bisect binary with ClamAV to isolate signature bytes

## Overview

Split a suspicious binary into progressively smaller segments and ClamAV-scan each piece to narrow down which exact bytes trigger detection. Start at 10,000-byte intervals, then 1,000, 100, 10, and finally 1. Once the offending bytes are found, patch them in memory with PowerShell and repeat for every remaining signature. Note: patching arbitrary bytes will break executable functionality — this technique teaches where signatures live, not how to produce a working bypass on its own.

## Steps

- **Import Find-AVSignature** — open a PowerShell prompt with -Exec bypass and import the script.
- **First pass (10,000-byte intervals)** — scan the whole file with -StartByte 0 -EndByte max -Interval 10000; scan output with ClamAV.
- **Narrow iteratively** — reduce -Interval to 1000, 100, 10, then 1 to isolate exact byte offset.
- **Patch the byte** — read all bytes with ReadAllBytes, zero out (or change) the offending offsets, write back with WriteAllBytes.
- **Repeat** — re-run Find-AVSignature on the modified file to find remaining signatures; iterate until clean.
- **Caveat** — patching bytes often breaks execution (Meterpreter failed to run after patching); the real value is learning which bytes to avoid when writing a custom loader.

## Commands

### Import and first scan

```powershell
PS C:\Users\Offsec> cd C:\Tools

PS C:\Tools> Import-Module .\Find-AVSignature.ps1

PS C:\Tools> Find-AVSignature -StartByte 0 -EndByte max -Interval 10000 -Path C:\Tools\met.exe -OutPath C:\Tools\avtest1 -Verbose -Force

PS C:\Windows\system32> cd 'C:\Program Files\ClamAV\'

PS C:\Program Files\ClamAV> .\clamscan.exe C:\Tools\avtest1
C:\Tools\avtest1\met_0.bin: OK
C:\Tools\avtest1\met_10000.bin: OK
C:\Tools\avtest1\met_20000.bin: Win.Trojan.MSShellcode-7 FOUND
...
```

### Narrow intervals

```powershell
PS C:\Tools> Find-AVSignature -StartByte 10000 -EndByte 20000 -Interval 1000 -Path C:\Tools\met.exe -OutPath C:\Tools\avtest2 -Verbose -Force

PS C:\Tools> Find-AVSignature -StartByte 18000 -EndByte 19000 -Interval 100 -Path C:\Tools\met.exe -OutPath C:\Tools\avtest3 -Verbose -Force

PS C:\Tools> Find-AVSignature -StartByte 18800 -EndByte 18900 -Interval 10 -Path C:\Tools\met.exe -OutPath C:\Tools\avtest4 -Verbose -Force
```

### Patch offending bytes

```powershell
# First signature patch — zero out the detected byte
$bytes  = [System.IO.File]::ReadAllBytes("C:\Tools\met.exe")
$bytes[18867] = 0
[System.IO.File]::WriteAllBytes("C:\Tools\met_mod.exe", $bytes)

# All detected signature bytes patched out
$bytes  = [System.IO.File]::ReadAllBytes("C:\Tools\met.exe")
$bytes[18867] = 0
$bytes[18987] = 0
$bytes[73801] = 0xFF
[System.IO.File]::WriteAllBytes("C:\Tools\met_mod.exe", $bytes)
```
