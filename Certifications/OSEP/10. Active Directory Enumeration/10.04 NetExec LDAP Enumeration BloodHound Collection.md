# 10.04 NetExec LDAP Enumeration BloodHound Collection

> nxc ldap · --bloodhound · --users / --groups · --pass-pol · --rid-brute

## Overview

NetExec's LDAP module queries Active Directory directly. Collect full BloodHound datasets with a single command, enumerate users/groups/computers, pull the password policy (critical before spraying), and find AdminCount=1 accounts. The SMB module's --rid-brute enumerates domain users without credentials when null sessions are permitted.

## Steps

- **Password policy first** — `--pass-pol` reveals lockout threshold and observation window before any spray.
- **BloodHound collection** — `-c All` collects Sessions, Trusts, ACLs, ObjectProps; outputs a ready-to-import zip.
- **--rid-brute** — enumerate domain users without credentials if null session is allowed on any host.
- **--admin-count** — finds accounts with AdminCount=1 (typically high-privilege).
- **Domain vs local auth** — add `-d <domain>` for domain accounts; `--local-auth` for local accounts.

## Commands

### Password policy (check before any spray)

```bash
# Always run this before spraying — know the lockout threshold
nxc smb <DC> -u <USER> -p <PASS> --pass-pol
nxc smb <DC> -u '' -p '' --pass-pol     # sometimes works unauthenticated
```

### BloodHound data collection

```bash
# Collect all data — creates <domain>_<timestamp>_BloodHound.zip
nxc ldap <DC> -u <USER> -p '<PASS>' --bloodhound -ns <DC> -c All
nxc ldap <DC> -u <USER> -H <HASH>  --bloodhound -ns <DC> -c All

# Import the zip into BloodHound GUI:
#   BloodHound CE: Administration → File Ingest
#   Classic:       drag-drop the zip onto the UI
```

### Enumerate AD objects

```bash
# Users
nxc ldap <DC> -u <USER> -p <PASS> --users

# Groups
nxc ldap <DC> -u <USER> -p <PASS> --groups

# Computers
nxc ldap <DC> -u <USER> -p <PASS> --computers

# AdminCount=1 accounts (high-value targets)
nxc ldap <DC> -u <USER> -p <PASS> --admin-count

# RID brute — enumerate users with no credentials (null session)
nxc smb <DC> -u '' -p '' --rid-brute 10000

# Extract usernames from RID brute output
nxc smb <DC> -u '' -p '' --rid-brute 10000 \
  | grep SidTypeUser | awk '{print $6}' | cut -d'\\' -f2 > users.txt
```
