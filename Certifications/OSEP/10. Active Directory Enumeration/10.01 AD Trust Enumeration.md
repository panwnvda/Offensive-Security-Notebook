# 10.01 AD Trust Enumeration

> T1482 · Domain Trust Discovery

## Overview

Cross-domain / cross-forest attacks require mapping trust direction + transitivity + SID history. Hunt with PowerView Get-DomainTrust, nltest, or BloodHound's 'TrustedBy' edge.

## Steps

- **Enumerate trusts** — PowerView, nltest /domain_trusts, BloodHound's trust map.
- **Identify forest root** — Get-Domain | Select Forest.
- **Map transitive trusts** — BloodHound's `MATCH p=(d1:Domain)-[:TrustedBy*1..]->(d2:Domain)`.
- **Look for SID history** — possible cross-forest Golden ticket abuse.

## Commands

### Quick trust dump

```powershell
Get-DomainTrust
Get-DomainTrustMapping  # recursive crawl
nltest /domain_trusts /v
```
