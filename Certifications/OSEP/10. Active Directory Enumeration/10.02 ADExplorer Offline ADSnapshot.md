# 10.02 ADExplorer Offline ADSnapshot

> T1213 · Data from Information Repositories

## Overview

Sysinternals ADExplorer creates a snapshot of the entire LDAP tree (`.dat`) that you can read offline with no live LDAP queries. SharpADWS / ADExplorerSnapshot.py parses the snapshot, lets you feed it to BloodHound.

## Steps

- **Run ADExplorer.exe** — File → Create Snapshot.
- **Exfil .dat** — could be 100 MB+; chunk transfer.
- **Convert to BloodHound** — ADExplorerSnapshot.py.
- **Analyze offline** — no live LDAP / DCSync telemetry.

## Commands

### Take + convert snapshot

```bash
# Windows
ADExplorer.exe -snapshot "" snapshot.dat
# Linux
pip install adexplorersnapshot
ADExplorerSnapshot.py snapshot.dat -o bh_output/
```
