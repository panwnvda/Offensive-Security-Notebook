# 10.01 Hash Cracking

> Identify the hash then pick the mode: NTLM `-m 1000`, NetNTLMv2 `5600`, TGS `13100`, AS-REP `18200`, bcrypt `3200`.

## Overview

Identify the hash format, then pick the hashcat mode. Common OSCP hashes: NTLM (-m 1000), NetNTLMv2 (-m 5600), Kerberos TGS (-m 13100), AS-REP (-m 18200), bcrypt (-m 3200), SHA-512crypt (-m 1800).

## Steps

- **Identify** — hashid / hashcat --example-hashes.
- **Hashcat with rockyou + best64**.
- **If slow** — try john's --format flag, or a smaller targeted list.

## Commands

### Common modes

```bash
hashcat -m 1000 ntlm.txt rockyou.txt -r best64.rule       # NTLM
hashcat -m 5600 ntlmv2.txt rockyou.txt                     # NetNTLMv2 (responder)
hashcat -m 13100 tgs.txt rockyou.txt -r best64.rule       # Kerberoast
hashcat -m 18200 asrep.txt rockyou.txt                     # AS-REP
hashcat -m 1800 shadow.txt rockyou.txt -r best64.rule    # /etc/shadow
```
