# 10.04 Wordlist Mutation

> Build company-specific candidates (`Name+Season+Year+!`) and multiply them with hashcat rules like `best64` and `dive`.

## Overview

Targeted wordlists beat rockyou: company name + season + year + special chars. Use hashcat rules (best64, dive) to multiply candidates from a small base.

## Steps

- **Build base list** — CeWL the company website + names.
- **Apply rules** — best64.rule for general, dive.rule for big GPU rigs.
- **Track results** — hashcat --show; john --show.

## Commands

### CeWL + rules

```bash
cewl https://<TARGET-DOMAIN> -d 2 -m 5 -w base.txt
hashcat -m 1000 hashes.txt base.txt -r /usr/share/hashcat/rules/best64.rule
john --wordlist=base.txt --rules=Jumbo hashes.txt
```
