# 03.02 Web App Enum Methodology

> Browse every feature with Burp in passive mode while inspecting headers, cookies, source, `robots.txt`, and `sitemap.xml`.

## Overview

Combine automated spidering with manual exploration. Always run Burp in passive mode while clicking through every visible feature. Inspect headers, cookies, source, robots.txt, sitemap.xml.

## Steps

- **Configure Burp proxy** — install CA cert in browser.
- **Browse every page authenticated + unauthenticated**.
- **Inspect cookies + headers** for hints (X-Powered-By, Set-Cookie attributes).
- **Read JS source** for hidden API routes.
- **Dir-brute** — gobuster, ffuf with content-aware extensions.

## Commands

### Web app first-pass triage

```bash
curl -sI http://<TARGET>
curl -s http://<TARGET>/robots.txt
curl -s http://<TARGET>/sitemap.xml
whatweb http://<TARGET>
wafw00f http://<TARGET>
```

### Dir-brute & vhost

```bash
ffuf -u http://<TARGET>/FUZZ -w /usr/share/seclists/Discovery/Web-Content/raft-large-words.txt -e .php,.html,.txt
ffuf -u http://<TARGET> -H 'Host: FUZZ.<TARGET-DOMAIN>' -w subdomains.txt -fs 1234
```
