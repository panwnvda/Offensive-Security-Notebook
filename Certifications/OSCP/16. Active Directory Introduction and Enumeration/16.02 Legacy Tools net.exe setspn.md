# 16.02 Legacy Tools net.exe setspn

> Fall back to `net.exe`, `setspn`, `dsquery`, and `ldapsearch` when PowerView triggers AV or PowerShell is blocked.

## Overview

When PowerShell is blocked or PowerView triggers AV, fall back to native Windows binaries — net.exe, setspn, dsquery, ldapsearch.

## Steps

- **Net commands** — net user /domain, net group 'Domain Admins' /domain.
- **setspn -T** — list SPNs across domain.
- **dsquery / ldp / ldapsearch** — raw LDAP.

## Commands

### Native enum

```cmd
net user /domain
net group "Domain Admins" /domain
net accounts /domain
setspn -T <DOMAIN> -Q */*
dsquery user -name *
dsquery computer
```
