# 19.02 Internal Recon Post-Pivot

> Read `ifconfig`/`ipconfig` and routes on the perimeter box, then enum internal subnets through the tunnel.

## Overview

Once on the perimeter box, discover internal subnets via ifconfig/ipconfig + routes, then enum hosts/services through the tunnel.

## Steps

- **Find new subnets** — ip a, route, /etc/hosts.
- **Pivot** — chisel reverse SOCKS.
- **proxychains nmap -sT** — only TCP through SOCKS.

## Commands

### Internal sweep

```bash
# On pivot
ip a; ip route; cat /etc/hosts; arp -a
# Attacker proxychains scan
proxychains -q nmap -sT -p 22,80,135,139,389,445,3389 -Pn --open <SUBNET>
```
