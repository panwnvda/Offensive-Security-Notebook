# 01.04 SMB RPC NetBIOS Enumeration

> Drop a null session against 445/139 with `enum4linux-ng` to dump shares, users, OS info, and password policy.

## Overview

SMB (445) and NetBIOS (139) leak shares, users, password policy, and OS info. Anonymous (null) sessions still work on lots of lab boxes. enum4linux-ng is the fastest single-command dump; smbclient/smbmap fill the gaps.

## Steps

- **Null session probe** — empty user/password.
- **Share list** — smbclient -L, smbmap -H, nxc smb -u '' -p ''.
- **RID brute** — enumerate users via RID cycling.
- **Read shares** — mount or get files anonymously.

## Commands

### enum4linux-ng full dump

```bash
git clone https://github.com/cddmp/enum4linux-ng
./enum4linux-ng.py -A <TARGET>
```

### smbclient + smbmap

```bash
smbclient -L //<TARGET>/ -N
smbmap -H <TARGET> -u '' -p ''
smbclient //<TARGET>/share -N
# Inside: get file.txt
```

### nxc / netexec

```bash
netexec smb <TARGET> -u '' -p ''
netexec smb <TARGET> -u guest -p '' --shares --pass-pol
netexec smb <TARGET> --rid-brute 4000
```
