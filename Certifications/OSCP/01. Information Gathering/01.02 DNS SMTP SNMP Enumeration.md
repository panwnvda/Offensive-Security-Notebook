# 01.02 DNS SMTP SNMP Enumeration

> Sweep AXFR, SMTP `VRFY`/`EXPN`, SNMP community strings, and FTP banners to catch leaks nmap defaults miss.

## Overview

DNS zone transfers, SMTP VRFY/EXPN, SNMP community strings, and FTP banner grabs leak surprisingly often on OSCP-style boxes. A small enum sweep catches issues missed by nmap script defaults.

## Steps

- **DNS** — try AXFR against each NS; iterate sub-domain wordlist with dnsrecon.
- **SMTP** — VRFY/EXPN reveal valid usernames.
- **SNMP** — `public`/`private` strings still common; walk MIB-2 + sysName.
- **FTP** — anonymous login + banner.

## Commands

### DNS

```bash
dig axfr @ns1.<TARGET-DOMAIN> <TARGET-DOMAIN>
dnsrecon -d <TARGET-DOMAIN> -t std,brt -D /usr/share/wordlists/dnsmap.txt
```

### SMTP user enum

```bash
smtp-user-enum -M VRFY -U /usr/share/seclists/Usernames/Names/names.txt -t <TARGET>
# Manual
nc -nv <TARGET> 25
VRFY root
```

### SNMP walk

```bash
onesixtyone -c /usr/share/seclists/Discovery/SNMP/snmp.txt <TARGET>
snmpwalk -v2c -c public <TARGET>
snmp-check <TARGET> -c public
```
