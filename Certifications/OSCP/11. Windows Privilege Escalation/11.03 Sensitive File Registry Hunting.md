# 11.03 Sensitive File Registry Hunting

> Grep registry (`Winlogon AutoAdminLogon`), `unattend.xml`, `sysprep.inf`, and `C:\Users\*` for plaintext credentials.

## Overview

Hunt the registry (`Winlogon AutoAdminLogon`, `unattend.xml`, `sysprep.inf`) and `C:\\Users\\*` for plaintext credentials. Many OSCP boxes have a credential in a config file.

## Steps

- **Search files** — `findstr /si password *.config *.xml *.ini *.txt`.
- **Registry** — common credential locations.
- **PowerShell history** — `%APPDATA%\\Microsoft\\Windows\\PowerShell\\PSReadLine\\ConsoleHost_history.txt`.

## Commands

### Credential hunt

```cmd
findstr /si password *.txt *.xml *.config *.ini
reg query 'HKLM\\SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion\\Winlogon' /v DefaultPassword
reg query 'HKCU\\Software\\SimonTatham\\PuTTY\\Sessions' /s
type %APPDATA%\\Microsoft\\Windows\\PowerShell\\PSReadLine\\ConsoleHost_history.txt
```
