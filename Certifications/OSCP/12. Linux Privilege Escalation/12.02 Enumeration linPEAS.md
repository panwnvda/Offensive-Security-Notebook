# 12.02 Enumeration linPEAS

> Run `linPEAS` for color-coded SUID/capabilities/cron/perm/kernel-CVE findings, then cross-verify with manual checks.

## Overview

linPEAS prints color-coded findings: SUID, capabilities, cron, weak file perms, kernel CVE candidates. Cross-verify with manual checks.

## Steps

- **Drop + run** — `curl ... | sh` if egress works.
- **Sort red/yellow** lines for high-likelihood escalation paths.
- **Always run sudo -l early** — first thing to try.

## Commands

### Quick run

```bash
curl -sL https://github.com/peass-ng/PEASS-ng/releases/latest/download/linpeas.sh | bash -s -- -a
sudo -l
id; cat /etc/os-release; uname -a
```
