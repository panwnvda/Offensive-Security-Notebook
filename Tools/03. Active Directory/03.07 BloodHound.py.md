# 03.07 BloodHound.py

> Python BloodHound collector — run from Linux without domain join.

## Overview

BloodHound.py is the Linux/cross-platform collector. Runs over LDAP (and optionally SMB for session/local-admin collection) — no Windows or domain join required. Use from a Kali attack box once you have domain credentials. Generates the same JSON format as SharpHound for BloodHound ingest.

## Steps

- **Step 1** - Install via pip.
- **Step 2** - Have domain creds + reachable DC.
- **Step 3** - Run with collection method matching your OPSEC budget.
- **Step 4** - ZIP the resulting JSON files and upload to BloodHound.

## Commands

### Install

```bash
pip3 install bloodhound
# or from source
git clone https://github.com/dirkjanm/BloodHound.py
cd BloodHound.py && pip3 install .
```

### Default collection

```bash
bloodhound-python -u <USER> -p <PASS> -d <DOMAIN> -ns <DC-IP> -c All --zip
```

### Stealth DCOnly

```bash
bloodhound-python -u <USER> -p <PASS> -d <DOMAIN> -ns <DC-IP> -c DCOnly --zip
```

### With Kerberos auth (ccache)

```bash
export KRB5CCNAME=/tmp/ccache
bloodhound-python -u <USER> -k -no-pass -d <DOMAIN> -dc <DC-FQDN> -c All --zip
```

### With hash (Pass-the-Hash)

```bash
bloodhound-python -u <USER> --hashes :<NT-HASH> -d <DOMAIN> -ns <DC-IP> -c All --zip
```
