# 03.04 ADRecon

> Comprehensive AD reconnaissance report — single PowerShell script that exports 30+ AD object types to Excel/CSV/JSON.

## Overview

ADRecon dumps every interesting class of object in the domain into a multi-tab Excel file or a folder of CSVs: users, computers, groups, GPOs, OUs, ACLs, trusts, sites, schemas, password policy, fine-grained password policies, gMSA, LAPS-enabled computers, Kerberoastable accounts, ASREProast-able accounts, unconstrained delegation, computers with old OS, etc. Run early to get the lay of the land before going surgical with PowerView or BloodHound.

## Steps

- **Step 1** - Get ADRecon.ps1 from GitHub.
- **Step 2** - Run from a domain-joined Windows host with any user creds.
- **Step 3** - Wait — ADRecon is slow on large domains (15-30 min).
- **Step 4** - Excel file gets dropped to current directory.
- **Step 5** - Review tabs in order of severity findings.

## Commands

### Get script

```powershell
iwr 'https://raw.githubusercontent.com/adrecon/ADRecon/master/ADRecon.ps1' -OutFile ADRecon.ps1
```

### Run with current creds

```powershell
.\ADRecon.ps1
# Outputs to ADRecon-Report-<DATE>.xlsx or CSV folder if Excel not installed
```

### Specific DC

```powershell
.\ADRecon.ps1 -DomainController <DC-FQDN>
```

### Alternate credentials

```powershell
$cred = Get-Credential
.\ADRecon.ps1 -DomainController <DC-FQDN> -Credential $cred
```

### Specific collection modules

```powershell
.\ADRecon.ps1 -Collect Domain,Forest,Trusts,DCs,Users,UserSPNs,Groups,GroupMembers,OUs,GPOs,GPOReport,DNSZones,DNSNodes,Computers,LAPS,BitLocker,ACLs,GPOLocalGroup -OutputType CSV
```

### JSON output for parsing

```powershell
.\ADRecon.ps1 -OutputType JSON
# Each section becomes a JSON file
```
