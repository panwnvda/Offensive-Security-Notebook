# 03.02 ADCollector

> C# AD attack-surface collector — runs from a domain user host and dumps actionable AD intel to terminal.

## Overview

ADCollector is a quicker alternative to ADRecon for getting actionable intel on first foothold. Runs as a single .NET binary; outputs interesting findings to console: domain admins, kerberoastable accounts, ASREProast accounts, machine accounts in privileged groups, unconstrained delegation, MAQ, password policy, trust info. Faster than ADRecon, less verbose, no Excel dependency.

## Steps

- **Step 1** - Get binary or compile.
- **Step 2** - Run from any domain-joined Windows host.
- **Step 3** - Review terminal output.

## Commands

### Get binary

```powershell
iwr 'https://github.com/dev-2null/ADCollector/releases/latest/download/ADCollector.exe' -OutFile ADCollector.exe
```

### Run

```powershell
ADCollector.exe
# Or via execute-assembly
# [sliver] $ execute-assembly /opt/tools/ADCollector.exe
```

### Specify domain

```powershell
ADCollector.exe -d <DOMAIN>.local
```
