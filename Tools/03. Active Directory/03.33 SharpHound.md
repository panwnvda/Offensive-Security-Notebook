# 03.33 SharpHound

> .NET collector for BloodHound — runs from Windows AD-joined host, dumps everything to JSON.

## Overview

SharpHound is the canonical BloodHound collector. Runs as a .NET binary on a domain-joined Windows host. Multiple collection methods: 'All' (full collection), 'DCOnly' (LDAP only, fast and quiet), 'Session' (live user sessions), 'Trusts', 'LoggedOn'. Output is a ZIP of JSON files to upload to BloodHound. Stealthier than full collection — use DCOnly first.

## Steps

- **Step 1** - Get SharpHound.exe from BloodHound's GitHub releases.
- **Step 2** - Run from a domain user context (current user or `runas /netonly`).
- **Step 3** - Pick collection method based on OPSEC budget.
- **Step 4** - Upload ZIP to BloodHound UI.

## Commands

### Get SharpHound

```bash
wget https://github.com/SpecterOps/SharpHound/releases/latest/download/SharpHound-v2.7.1.zip
unzip SharpHound-v2.7.1.zip
```

### Full collection (loud)

```powershell
SharpHound.exe -c All --zipfilename bh.zip
```

### Stealth DCOnly (LDAP only)

```powershell
SharpHound.exe -c DCOnly --zipfilename bh_dc.zip
```

### Skip LoggedOn collection (very loud)

```powershell
SharpHound.exe -c Default,GPOLocalGroup,Trusts --zipfilename bh_clean.zip
```

### Specific domain (cross-trust)

```powershell
SharpHound.exe -c All -d <DOMAIN>.local --zipfilename bh_<DOMAIN>.zip
```

### Throttled (less detectable)

```powershell
SharpHound.exe -c All --throttle 1000 --jitter 30 --zipfilename bh_slow.zip
```

### Run as alternate user

```cmd
runas /netonly /user:<DOMAIN>\<USER> SharpHound.exe -c DCOnly
```
