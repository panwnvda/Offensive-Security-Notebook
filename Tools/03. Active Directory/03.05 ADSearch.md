# 03.05 ADSearch

> C# raw LDAP query tool — execute arbitrary LDAP filters and return results, runnable via execute-assembly.

## Overview

ADSearch (tomcarver16) is a compact C# tool for running raw LDAP queries against AD. Useful when you need a specific custom filter that PowerView/SharpView doesn't expose conveniently, or when running from a Sliver/Cobalt beacon via execute-assembly. Returns formatted results to stdout.

## Steps

- **Step 1** - Get binary or compile.
- **Step 2** - Run with --filter and --attributes flags.
- **Step 3** - For ad-hoc one-offs, easier than PowerView.

## Commands

### Get binary

```powershell
iwr 'https://github.com/tomcarver16/ADSearch/releases/latest/download/ADSearch.exe' -OutFile ADSearch.exe
```

### Find kerberoastable

```powershell
ADSearch.exe --search '(&(samAccountType=805306368)(servicePrincipalName=*))' --attributes samaccountname,serviceprincipalname
```

### Find AS-REP roastable

```powershell
ADSearch.exe --search '(&(samAccountType=805306368)(userAccountControl:1.2.840.113556.1.4.803:=4194304))' --attributes samaccountname,distinguishedname
```

### Find unconstrained delegation

```powershell
ADSearch.exe --search '(userAccountControl:1.2.840.113556.1.4.803:=524288)' --attributes samaccountname,distinguishedname
```

### Sliver execute-assembly

```text
[sliver] $ execute-assembly /opt/tools/ADSearch.exe --search '(servicePrincipalName=*)' --attributes samaccountname,servicePrincipalName
```
