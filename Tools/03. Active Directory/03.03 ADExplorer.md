# 03.03 ADExplorer

> Sysinternals AD GUI browser — snapshot and offline diff entire AD database.

## Overview

Microsoft Sysinternals ADExplorer (signed, official) browses AD via LDAP with a tree-view GUI. Most useful feature: snapshot mode dumps the entire directory to a .dat file for offline analysis. Pair with ADExplorerSnapshot.py (Python parser) or BloodHound's adexsnapshot ingestor to import into BloodHound without running active collection. Stealthier than SharpHound for initial enumeration.

## Steps

- **Step 1** - Download ADExplorer from Sysinternals.
- **Step 2** - Connect to DC with creds.
- **Step 3** - File → Create Snapshot → save .dat file.
- **Step 4** - Process snapshot offline with ADExplorerSnapshot.py.

## Commands

### Get tool

```powershell
iwr 'https://download.sysinternals.com/files/AdExplorer.zip' -OutFile AdExplorer.zip
Expand-Archive AdExplorer.zip
```

### GUI use

```text
1. Open ADExplorer.exe
2. Connect to: <DC-FQDN>
3. Username: <DOMAIN>\<USER>, Password: <PASS>
4. File → Create Snapshot... → save <DOMAIN>.dat
5. The .dat file contains the full directory; analyze offline
```

### ADExplorerSnapshot.py — parse for BloodHound

```bash
pip3 install ADExplorerSnapshot.py
ADExplorerSnapshot.py <DOMAIN>.dat -o bh_output/
# Outputs BloodHound-compatible JSON; zip and upload to BloodHound CE
```

### Offline diff between snapshots

```text
1. Take snapshot1.dat
2. Wait some time → take snapshot2.dat
3. ADExplorer.exe → File → Compare Snapshots
4. Shows added/removed/modified objects + attributes
```
