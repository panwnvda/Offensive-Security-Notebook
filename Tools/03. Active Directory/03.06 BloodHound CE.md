# 03.06 BloodHound CE

> Graph-database AD enumeration tool — shows attack paths from any user to Domain Admin.

## Overview

BloodHound CE (Community Edition, v5+) replaces legacy BloodHound. Ingests AD data and visualizes attack paths in a graph (Neo4j backend, web UI). Click any user/computer/group → query 'shortest path to Domain Admin' → see exactly which ACL edges, sessions, or admin rights chain together. Critical for any AD assessment. Pair with a collector — SharpHound (.NET), BloodHound.py (Linux), RustHound (Rust), SOAPHound (SOAP-based stealth).

## Steps

- **Step 1** - Install via Docker Compose (cleanest path for CE).
- **Step 2** - Set up web UI + Neo4j.
- **Step 3** - Collect AD data with appropriate collector.
- **Step 4** - Upload JSON to BloodHound UI.
- **Step 5** - Run pre-built queries: Shortest Path to DA, Find Kerberoastable, Find Workstations Where Domain Users Are Local Admin.

## Commands

### Install via Docker Compose

```bash
curl -L https://ghst.ly/getbhce | docker compose -f - up -d
# Default UI at http://localhost:8080
# First run prints admin password — save it
```

### Manual install (Linux)

```bash
# Pull from https://github.com/SpecterOps/BloodHound
wget https://github.com/SpecterOps/BloodHound/releases/latest/download/bloodhound-cli-linux-amd64.tar.gz
tar -xzf bloodhound-cli-linux-amd64.tar.gz
./bloodhound-cli install
```

### Reset admin password

```bash
docker exec -it bloodhound-bloodhound-1 bhapi user-reset-password admin
# follow prompts
```

### Upload data

```text
Web UI → Administration → File Ingest → Upload the SharpHound/BHpy ZIP
# Or API:
curl -X POST 'http://localhost:8080/api/v2/file-upload/start' \
  -H 'Authorization: Bearer <TOKEN>'
# Then PUT the ZIP
curl -X PUT 'http://localhost:8080/api/v2/file-upload/<JOB-ID>' \
  -H 'Authorization: Bearer <TOKEN>' --data-binary @bloodhound.zip
```

### Pre-built queries (from UI)

```text
Find all Domain Admins
Find shortest paths to Domain Admins
Find shortest paths from owned users to Domain Admins
Find kerberoastable accounts
Find ASREProast-able accounts
Find workstations where Domain Users have admin rights
Find users with delegation permissions
Find ADCS templates vulnerable to ESC1/ESC2/ESC3/etc.
```
