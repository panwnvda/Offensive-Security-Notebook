# 01.11 Shodan CLI

> Search the Shodan internet-wide service index from your terminal.

## Overview

Shodan crawls the entire internet and indexes every service banner. Search by org name, ASN, hostname, SSL cert subject, vulnerability ID, or product version. Free tier limits filters; paid tier unlocks full search. Critical for finding <ORG>'s exposed services without touching their network.

## Steps

- **Step 1** - Install: `pip install shodan`.
- **Step 2** - Init with your API key.
- **Step 3** - Use search with filters and facets.

## Commands

### Install + init

```bash
pip3 install shodan
shodan init <SHODAN-KEY>
```

### Org / ASN sweep

```bash
shodan search --fields ip_str,port,product,version 'org:"<ORG-NAME>"' --limit 1000
shodan search 'asn:AS<ASN>' --limit 1000
shodan search 'ssl.cert.subject.cn:*.<DOMAIN>' --limit 1000
```

### CVE search

```bash
shodan search 'vuln:CVE-2024-21887' --limit 100
```

### Facet aggregation

```bash
shodan stats --facets 'port:25' 'org:"<ORG-NAME>"'
shodan stats --facets 'product:10' 'asn:AS<ASN>'
```

### Host details

```bash
shodan host <IP>
```
