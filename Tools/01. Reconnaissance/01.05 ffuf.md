# 01.05 ffuf

> Fast web fuzzer for directory/file/parameter/vhost discovery — the modern Gobuster replacement.

## Overview

ffuf is Go-based, blazingly fast, supports concurrency tuning, recursion, multi-payload positions, header injection, and JSON output. Use it for content discovery, parameter brute-force, vhost discovery, and login-form brute (with care). The `FUZZ` keyword marks where the wordlist substitutes.

## Steps

- **Step 1** - Install via Go.
- **Step 2** - Get good wordlists from SecLists.
- **Step 3** - Use `-mc` (match code) and `-fc/-fs/-fw` (filter) for noise reduction.
- **Step 4** - Recurse with `-recursion` for nested directories.
- **Step 5** - OPSEC — throttle with `-rate` to avoid IDS detection.

## Commands

### Install

```bash
go install -v github.com/ffuf/ffuf/v2@latest
# Or apt install ffuf
```

### Directory discovery

```bash
ffuf -u https://<TARGET>/FUZZ -w /usr/share/seclists/Discovery/Web-Content/raft-medium-directories.txt \
  -mc 200,301,302,403 -fc 404 -rate 50 -o dirs.json -of json
```

### File discovery with extensions

```bash
ffuf -u https://<TARGET>/FUZZ -w /usr/share/seclists/Discovery/Web-Content/raft-medium-words.txt \
  -e .php,.aspx,.jsp,.html,.bak,.old,.zip,.tar.gz,.config \
  -mc 200,301,302,401,403 -fc 404 -rate 30
```

### Virtual host fuzz

```bash
ffuf -u https://<TARGET>/ -H 'Host: FUZZ.<DOMAIN>' \
  -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-110000.txt \
  -fs <BASELINE-RESPONSE-SIZE>
```

### Parameter brute force

```bash
ffuf -u 'https://<TARGET>/api/data?FUZZ=test' \
  -w /usr/share/seclists/Discovery/Web-Content/burp-parameter-names.txt \
  -fs <BASELINE-SIZE>
```

### POST form fuzz

```bash
ffuf -u https://<TARGET>/login -X POST \
  -H 'Content-Type: application/x-www-form-urlencoded' \
  -d 'username=admin&password=FUZZ' \
  -w passwords.txt \
  -fs <FAILED-LOGIN-SIZE>
```

### Recursive scan

```bash
ffuf -u https://<TARGET>/FUZZ -w raft-medium.txt -recursion -recursion-depth 2 -mc 200,403
```
