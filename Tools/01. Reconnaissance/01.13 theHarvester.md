# 01.13 theHarvester

> OSINT email and subdomain harvester from public sources — Google, Bing, LinkedIn, GitHub, certificate logs.

## Overview

theHarvester pulls names, emails, IPs, subdomains, employee profiles from public data sources. Free tier covers Google/Bing/DuckDuckGo/Yahoo, paid sources need API keys (SecurityTrails, Shodan, BinaryEdge). Useful for the initial profile build before deeper tooling.

## Steps

- **Step 1** - Install via apt or pip.
- **Step 2** - Run with `-b all` for broad sweep across all configured sources.
- **Step 3** - For LinkedIn employee scraping, use linkedin2username instead — theHarvester's LinkedIn is unreliable.

## Commands

### Install

```bash
sudo apt install -y theharvester
# or pip3 install theharvester
```

### Broad sweep

```bash
theHarvester -d <DOMAIN> -b all -l 1000 -f harvest.html
```

### Per-source for control

```bash
for src in google bing duckduckgo crtsh certspotter; do
    theHarvester -d <DOMAIN> -b $src -l 500 -f harv_$src.html
done
```

### With Shodan

```bash
theHarvester -d <DOMAIN> -b shodan -l 500
```
