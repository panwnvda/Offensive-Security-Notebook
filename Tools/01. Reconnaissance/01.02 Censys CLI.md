# 01.02 Censys CLI

> Search Censys hosts and certs indexes — strong CT-log and TLS cert search.

## Overview

Censys is Shodan's competitor with stronger certificate and TLS data. Particularly good for finding hosts by SSL cert CN/SAN, JARM hash, ASN. Free tier limited but generous enough for one engagement.

## Steps

- **Step 1** - `pip install censys`.
- **Step 2** - Configure API ID + secret.
- **Step 3** - Search hosts or certificates indexes.

## Commands

### Install + config

```bash
pip3 install censys
censys config set --auth basic --api-id <CENSYS-ID> --api-secret <CENSYS-SECRET>
```

### Find IPs by cert CN

```bash
censys search 'services.tls.certificates.leaf_data.subject.common_name: "*.<DOMAIN>"' --pages -1 | jq -r '.ip' | sort -u
```

### Search by ASN + service

```bash
censys search 'autonomous_system.asn: <ASN> AND services.service_name: HTTP' --pages -1
```

### View host details

```bash
censys view <IP> --index-type hosts | jq '.services[] | {port, service_name, software}'
```
