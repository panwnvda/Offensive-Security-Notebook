# 01.14 TruffleHog

> Secret scanner for git, S3, filesystems, Docker images — verifies discovered secrets are still live.

## Overview

TruffleHog scans git repos, S3 buckets, GCS buckets, file systems, Docker images, and Jenkins/CircleCI logs for live secrets. Unlike Gitleaks, it can VERIFY that secrets are currently active by hitting the relevant provider's API. The `--only-verified` flag drops dead/expired credentials.

## Steps

- **Step 1** - Install via Docker or binary.
- **Step 2** - Pick a source (github, s3, filesystem, docker).
- **Step 3** - Use `--only-verified` for live-only results.

## Commands

### Install

```bash
# Docker
docker pull trufflesecurity/trufflehog:latest
# Or binary
curl -sSfL https://raw.githubusercontent.com/trufflesecurity/trufflehog/main/scripts/install.sh | sudo sh -s -- -b /usr/local/bin
```

### Scan GitHub org

```bash
docker run --rm trufflesecurity/trufflehog:latest github \
  --org=<ORG-NAME> --token=<GH-PAT> --concurrency=1 --only-verified --json > th_org.json
```

### Scan local filesystem

```bash
trufflehog filesystem /path/to/code --only-verified --json
```

### Scan S3 bucket

```bash
trufflehog s3 --bucket <BUCKET> --only-verified --json
```

### Scan Docker image

```bash
trufflehog docker --image <IMAGE>:<TAG> --only-verified --json
```
