# 01.09 masscan

> Internet-scale TCP/UDP scanner — scan all 65535 ports across a /16 in minutes.

## Overview

masscan is for breadth, not depth. It transmits raw packets at line-rate and doesn't bother with TCP state — so it discovers open ports across enormous IP ranges fast. Pair with Nmap (masscan finds open ports, Nmap fingerprints them). Always throttle the rate — at 100k pps you'll saturate links and trip IDS instantly.

## Steps

- **Step 1** - Install: `apt install masscan` or build from github.com/robertdavidgraham/masscan.
- **Step 2** - Choose target list (CIDR, file, ASN-derived range).
- **Step 3** - Pick a rate sane for your engagement: 1000 pps is gentle, 10000 pps is loud, 100000+ is saturating.
- **Step 4** - Pipe Grep output to Nmap for service detection.
- **Step 5** - OPSEC — large netblocks scanned fast = guaranteed alert. Throttle, source from burnable VPS, randomize.

## Commands

### Install

```bash
sudo apt install -y masscan
# or build from source for latest
git clone https://github.com/robertdavidgraham/masscan && cd masscan && make
```

### All-ports scan single host

```bash
sudo masscan -p1-65535 --rate 1000 <TARGET> -oG masscan.gnmap
```

### /16 sweep gentle rate

```bash
sudo masscan -p22,80,443,445,3389,8080,8443 --rate 1000 <CIDR>/16 -oG sweep.gnmap
```

### From IP list

```bash
sudo masscan -p1-65535 --rate 2000 -iL targets.txt -oG result.gnmap
```

### Convert masscan output to Nmap follow-up list

```bash
awk '/Ports:/ {ip=$2; gsub(/\/open.*/, "", $4); print ip":"$4}' masscan.gnmap > targets_with_ports.txt
```
