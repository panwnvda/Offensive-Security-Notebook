# 01.15 waybackurls and gau

> Pull every URL ever crawled for a domain from Wayback Machine, Common Crawl, URLScan.

## Overview

Two complementary tools that pull historical URL data from web archives. waybackurls hits archive.org; gau (getallurls) pulls from archive.org + Common Crawl + URLScan + AlienVault OTX. Use them to find deprecated endpoints, leaked JS files with API keys, removed admin paths, and old vhosts that may still be live.

## Steps

- **Step 1** - Install both via Go.
- **Step 2** - Run against target domain.
- **Step 3** - Merge outputs, dedupe, filter for interesting extensions.
- **Step 4** - Probe each URL with httpx to find still-live endpoints.

## Commands

### Install

```bash
go install github.com/tomnomnom/waybackurls@latest
go install github.com/lc/gau/v2/cmd/gau@latest
```

### Pull all archived URLs

```bash
echo <DOMAIN> | waybackurls > wb.txt
echo <DOMAIN> | gau --threads 5 > gau.txt
sort -u wb.txt gau.txt > all.txt
```

### Filter interesting

```bash
grep -E '\.(js|json|env|bak|config|xml|sql|log|old|orig|swp|conf)(\?|$)' all.txt > interesting.txt
```

### Probe live status

```bash
httpx -l interesting.txt -mc 200,301,302,403 -silent -o still_live.txt
```
