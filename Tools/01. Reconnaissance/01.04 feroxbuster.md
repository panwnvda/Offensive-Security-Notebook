# 01.04 feroxbuster

> Rust-based recursive content discovery — alternative to ffuf and dirsearch, with built-in recursion and filtering.

## Overview

feroxbuster is similar in spirit to ffuf but Rust-native with built-in recursion. It auto-recurses into discovered directories without explicit flags. Has a TUI-ish progress display and resumable scans. Use when you want hands-off recursive discovery vs surgical ffuf-style targeting.

## Steps

- **Step 1** - Install via cargo or apt.
- **Step 2** - Run with sensible defaults.
- **Step 3** - Tune with `-d` (depth), `-t` (threads), `-x` (extensions).

## Commands

### Install

```bash
sudo apt install -y feroxbuster
# or
cargo install feroxbuster
```

### Default recursive scan

```bash
feroxbuster -u https://<TARGET> -t 10 -d 3 -x php,html,asp,aspx,bak,old -A -o ferox.txt
```

### Custom wordlist + filter

```bash
feroxbuster -u https://<TARGET> -w /usr/share/seclists/Discovery/Web-Content/big.txt \
  -t 5 -d 2 --filter-status 404,403 -o results.txt
```

### Resume previous scan

```bash
feroxbuster --resume-from ferox.state
```
