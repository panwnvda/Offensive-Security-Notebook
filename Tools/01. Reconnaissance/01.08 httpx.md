# 01.08 httpx

> ProjectDiscovery's fast HTTP toolkit for probing live hosts, tech detection, and screenshot.

## Overview

httpx checks which of a long list of hostnames/IPs respond on HTTP/HTTPS, what status they return, what tech stack they run, what title they show, what TLS cert they present. Replaces the old `httprobe`/`screenshot-utility` combo with one fast tool.

## Steps

- **Step 1** - Install via Go.
- **Step 2** - Pipe subdomain list in via stdin.
- **Step 3** - Add detection flags as needed: `-tech-detect`, `-tls-grab`, `-title`, `-server`, `-status-code`.
- **Step 4** - For screenshot capture: `-ss` (uses chromium headless).

## Commands

### Install

```bash
go install -v github.com/projectdiscovery/httpx/cmd/httpx@latest
```

### Probe live hosts from list

```bash
httpx -l subs.txt -mc 200,301,302,403 -silent -o live.txt
```

### Full enumeration probe

```bash
httpx -l subs.txt -tech-detect -status-code -title -server -cdn -tls-grab -json -silent -o probe.jsonl
```

### Screenshot capture

```bash
httpx -l live.txt -ss -srd screenshots/ -silent
# Outputs PNG screenshots into screenshots/
```

### Match against header value

```bash
httpx -l hosts.txt -match-string 'Server: nginx' -silent
httpx -l hosts.txt -match-regex '<title>.*admin' -silent
```
