# 01.03 exiftool

> Pull every metadata field from any image, document, or media file — author, software, GPS, internal paths.

## Overview

exiftool reads metadata from PDFs, DOCX, XLSX, PPTX, JPG, PNG, MP4, almost everything. Bulk-extract authors, software versions, internal Windows usernames, machine names, file paths from a target's public documents — a single PDF can leak the author's AD samAccountName and the OS path of the original file.

## Steps

- **Step 1** - Install via apt or brew.
- **Step 2** - Download target's public docs (metagoofil helps).
- **Step 3** - Run exiftool recursively in JSON mode.
- **Step 4** - Parse unique Authors/Creator/LastSavedBy fields.

## Commands

### Install

```bash
sudo apt install -y libimage-exiftool-perl
```

### Single file extract

```bash
exiftool report.pdf
exiftool -a -G1 -s report.pdf  # all fields with groups
```

### Bulk JSON extract

```bash
exiftool -j -r ./docs/ > all_meta.json
```

### Parse unique authors

```bash
jq -r '.[] | .Author // empty, .Creator // empty, .LastSavedBy // empty' all_meta.json | sort -u > authors.txt
```

### GPS / location from images

```bash
exiftool -GPSLatitude -GPSLongitude -j ./photos/
```
