# 10.03 John the Ripper

> CPU-focused password cracker — strong at format auto-detection and hybrid attacks; jumbo edition adds 100+ formats.

## Overview

John the Ripper (JtR) is older than hashcat. CPU-only (mostly), but format auto-detection is excellent and the 'jumbo' edition supports many obscure formats hashcat doesn't. Use john for non-GPU systems, exotic hashes (KeePass kdbx, OpenSSH private keys, archive files), and the `john --show` post-crack format display.

## Steps

- **Step 1** - Install (jumbo edition is what you want).
- **Step 2** - For exotic formats, use the bundled `*2john` helpers (zip2john, ssh2john, keepass2john, etc.).
- **Step 3** - Run with `--wordlist` or `--incremental`.

## Commands

### Install jumbo

```bash
sudo apt install -y john
# or build from source for jumbo features
git clone https://github.com/openwall/john
cd john/src && ./configure && make -j$(nproc)
```

### Auto-detect format

```bash
john --list=formats | head
john hash.txt --wordlist=rockyou.txt
```

### Force format

```bash
john --format=NT hash.txt --wordlist=rockyou.txt
john --format=krb5tgs hash.txt --wordlist=rockyou.txt
```

### Crack OpenSSH private key

```bash
ssh2john id_rsa > id_rsa.hash
john id_rsa.hash --wordlist=rockyou.txt
```

### Crack KeePass DB

```bash
keepass2john database.kdbx > kp.hash
john kp.hash --wordlist=rockyou.txt
```

### Crack ZIP

```bash
zip2john archive.zip > zip.hash
john zip.hash --wordlist=rockyou.txt
```

### Crack 7z

```bash
7z2john archive.7z > sz.hash
john sz.hash --wordlist=rockyou.txt
```

### Show cracked

```bash
john --show hash.txt
```

### Rules + incremental

```bash
john hash.txt --wordlist=rockyou.txt --rules=All
john hash.txt --incremental=Digits  # brute digits only
```
