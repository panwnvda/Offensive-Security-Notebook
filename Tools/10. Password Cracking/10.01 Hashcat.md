# 10.01 Hashcat

> GPU-accelerated password cracker — handles every common hash type with masks, rules, hybrid modes.

## Overview

Hashcat is the standard offline password cracker. GPU-accelerated, supports 300+ hash modes. Five attack modes: dictionary (-a 0), combinator (-a 1), brute/mask (-a 3), hybrid (-a 6/7). Rules let you mutate dictionary entries; masks let you do targeted brute on character classes. Hash modes: 0 (MD5), 100 (SHA1), 1000 (NTLM), 1800 (sha512crypt), 5600 (NetNTLMv2), 13100 (Kerberoast), 18200 (AS-REP).

## Steps

- **Step 1** - Install via apt or download standalone.
- **Step 2** - Get a wordlist (rockyou.txt) + rules (best64.rule, OneRuleToRuleThemAll).
- **Step 3** - Identify hash type (`hashid <HASH>` helps).
- **Step 4** - Run with `-m <MODE> -a <ATTACK>`.

## Commands

### Install

```bash
sudo apt install -y hashcat
# Or download standalone with OpenCL drivers
# Verify GPU works
hashcat -I
```

### Most common modes

```text
0     MD5
100   SHA1
1000  NTLM
1100  Domain Cached Credentials (DCC)
2100  Domain Cached Credentials 2 (DCC2)
3000  LM
5500  NetNTLMv1
5600  NetNTLMv2
7500  Kerberos 5 AS-REQ Pre-Auth (KRB-AS-REQ)
10000 Django PBKDF2-HMAC-SHA256
13100 Kerberoast TGS-REP RC4-HMAC
18200 AS-REP Roast (Kerberos 5 AS-REP RC4-HMAC)
22000 WPA-PBKDF2-PMKID+EAPOL
1800  sha512crypt (Linux /etc/shadow)
3200  bcrypt
```

### Dictionary attack

```bash
hashcat -m 1000 -a 0 ntlm_hashes.txt /usr/share/wordlists/rockyou.txt -O
```

### Dictionary + rules

```bash
hashcat -m 1000 -a 0 ntlm_hashes.txt rockyou.txt -r /usr/share/hashcat/rules/best64.rule -O
hashcat -m 1000 -a 0 ntlm_hashes.txt rockyou.txt -r OneRuleToRuleThemAll.rule -O
```

### Mask attack (brute with structure)

```bash
# 8-char password starting with capital, 6 lowercase, 1 digit
hashcat -m 1000 -a 3 hashes.txt '?u?l?l?l?l?l?l?d' -O
# Wordlist + appended mask hybrid
hashcat -m 1000 -a 6 hashes.txt rockyou.txt '?d?d?d?d' -O
# Mask + wordlist (prepended)
hashcat -m 1000 -a 7 hashes.txt '?u?u' rockyou.txt -O
```

### Kerberoast TGS

```bash
hashcat -m 13100 -a 0 kerb_hashes.txt rockyou.txt -r best64.rule -O
```

### AS-REP Roast

```bash
hashcat -m 18200 -a 0 asrep_hashes.txt rockyou.txt -r best64.rule -O
```

### NetNTLMv2 (Responder catch)

```bash
hashcat -m 5600 -a 0 netntlmv2.txt rockyou.txt -r best64.rule -O
```

### Show cracked

```bash
hashcat -m 1000 hashes.txt --show > cracked.txt
```
