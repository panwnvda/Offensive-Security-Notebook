# 12.01 Ghidra

> NSA's free reverse engineering platform — decompiler, disassembler, scripting, collaborative analysis.

## Overview

Ghidra is the most-used free reverse engineering tool. Decompiler produces readable C from x86/x64/ARM/MIPS/PPC binaries. Headless mode lets you script analysis. Project files (.gpr) are shareable across team. Use for: malware analysis, vulnerability research, exploit development.

## Steps

- **Step 1** - Install (requires Java 17+).
- **Step 2** - Create a project.
- **Step 3** - Import binary.
- **Step 4** - Auto-analyze (default settings are fine).
- **Step 5** - Browse functions, decompiler view, write scripts.

## Commands

### Install

```bash
sudo apt install -y openjdk-17-jdk
wget https://github.com/NationalSecurityAgency/ghidra/releases/latest/download/ghidra_*.zip
unzip ghidra_*.zip
./ghidra_*/ghidraRun
```

### Headless analysis

```bash
./ghidra_*/support/analyzeHeadless /tmp/ghidra_project ProjectName -import binary.exe -postScript MyScript.py
```

### Common keyboard shortcuts

```text
F4         Decompile current function
L          Rename variable / function
;          Add comment
G          Go to address
Ctrl+Shift+E  Find references
Ctrl+L     Re-type variable
P          Edit struct
```

### Useful scripts (in script manager)

```text
FindCryptography.java       — flags AES/RSA/MD5 patterns
DecompileSampleScript.java  — decompile all functions to text
ExportFunctionSignatures    — dump function prototypes
```
