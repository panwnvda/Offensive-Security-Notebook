# 02.11 XSStrike

> Advanced XSS detection — context-aware payload generation, WAF detection, DOM XSS fuzzer.

## Overview

XSStrike is smarter than naive XSS scanners. It analyzes the injection context (HTML attribute, JS string, etc.), generates payloads matched to that context, detects WAFs by behavior, and supports DOM XSS scanning via headless browser. Use after manual confirmation — XSStrike works best when you know the parameter is reflected.

## Steps

- **Step 1** - Clone + install requirements.
- **Step 2** - Identify a reflected parameter.
- **Step 3** - Run XSStrike against it.

## Commands

### Install

```bash
git clone https://github.com/s0md3v/XSStrike
cd XSStrike && pip3 install -r requirements.txt
```

### Basic single-param scan

```bash
python3 xsstrike.py -u 'https://<TARGET>/search?q=test'
```

### POST with body

```bash
python3 xsstrike.py -u 'https://<TARGET>/search' --data 'q=test' --headers 'Cookie: session=<SESSION>'
```

### Crawler mode

```bash
python3 xsstrike.py -u 'https://<TARGET>' --crawl -l 3 --headers 'Cookie: session=<SESSION>'
```

### Fuzzer mode (WAF probe)

```bash
python3 xsstrike.py -u 'https://<TARGET>/search?q=test' --fuzzer
```
