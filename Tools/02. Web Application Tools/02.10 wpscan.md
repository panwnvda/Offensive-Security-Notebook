# 02.10 wpscan

> WordPress reconnaissance and vulnerability scanner — enumerates users, plugins, themes, and known CVEs.

## Overview

wpscan is the WordPress-specific scanner. Enumerates users via author archive, identifies installed plugins and themes, cross-references known CVEs from its database. Use the free WPScan API token for vulnerability lookups. For weak-password testing, supports throttled username enumeration and credential brute.

## Steps

- **Step 1** - Install via apt or gem.
- **Step 2** - Get a free API token from wpscan.com.
- **Step 3** - Scan target, pull vulnerability data.

## Commands

### Install

```bash
sudo apt install -y wpscan
# or gem install wpscan
```

### Basic scan

```bash
wpscan --url https://<TARGET> --api-token <WPSCAN-API-TOKEN> -e ap,at,u
```

### Enumerate users

```bash
wpscan --url https://<TARGET> --enumerate u
```

### Password brute (throttled)

```bash
wpscan --url https://<TARGET> --usernames admin,editor \
  --passwords /usr/share/seclists/Passwords/Common-Credentials/10-million-password-list-top-1000.txt \
  --max-threads 3 --throttle 500
```

### Plugin/theme deep enum

```bash
wpscan --url https://<TARGET> --plugins-detection aggressive --themes-detection aggressive
```
