# 02.09 wafw00f

> Identify which WAF a target uses so you can tune payloads accordingly.

## Overview

wafw00f sends probe payloads and fingerprints the response to identify which WAF product is sitting in front of the target. Supports detection of Cloudflare, Akamai, F5/imperva, AWS WAF, Sucuri, ModSecurity, etc. Knowing the WAF lets you select the right tamper scripts and evasion techniques.

## Steps

- **Step 1** - Install via pip.
- **Step 2** - Probe target.
- **Step 3** - Use WAF identity to inform payload tuning.

## Commands

### Install

```bash
pip3 install wafw00f
```

### Single target

```bash
wafw00f https://<TARGET>
```

### List of targets

```bash
wafw00f -i targets.txt -o waf_results.json -f json
```

### All detection techniques

```bash
wafw00f -a https://<TARGET>
```
