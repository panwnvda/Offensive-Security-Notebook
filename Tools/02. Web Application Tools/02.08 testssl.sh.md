# 02.08 testssl.sh

> Comprehensive TLS configuration audit — enumerate ciphers, find known vulns, weak protocols.

## Overview

testssl.sh performs every known TLS check against a target. Probes SSLv2/3, TLS1.0/1.1/1.2/1.3, every cipher suite, certificate strength, Heartbleed, ROBOT, POODLE, BEAST, CRIME, BREACH, LOGJAM, DROWN, Ticketbleed, CCS injection. The canonical first tool to run against any TLS endpoint.

## Steps

- **Step 1** - Clone from GitHub.
- **Step 2** - Run against target HTTPS host.
- **Step 3** - Review color-coded output for HIGH severity findings.

## Commands

### Install

```bash
git clone --depth 1 https://github.com/drwetter/testssl.sh.git
cd testssl.sh
```

### Standard audit

```bash
./testssl.sh https://<TARGET>:443
```

### Specific vuln check

```bash
./testssl.sh --heartbleed https://<TARGET>:443
./testssl.sh --robot https://<TARGET>:443
./testssl.sh --crime https://<TARGET>:443
```

### Severity filter

```bash
./testssl.sh --severity HIGH --quiet --color 3 https://<TARGET>:443 > testssl_high.txt
```

### JSON output

```bash
./testssl.sh --jsonfile-pretty result.json https://<TARGET>:443
```
