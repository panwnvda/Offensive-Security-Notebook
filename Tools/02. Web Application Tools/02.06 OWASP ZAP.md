# 02.06 OWASP ZAP

> Free open-source web pentest proxy — Burp Community alternative, scriptable, has good active scanner.

## Overview

OWASP ZAP is the free-tier-equivalent of Burp Pro for many use cases. Active scanner is decent (better than Burp Community), HUD overlay sits in your browser, scriptable in Python/JavaScript/Groovy. Use for: automated initial scans, CI/CD security gates, and as a Burp alternative when Pro license isn't available.

## Steps

- **Step 1** - Install via apt or Java JAR.
- **Step 2** - Configure browser proxy.
- **Step 3** - Use the Quick Start panel for automated scan, or Manual Explore for hands-on.
- **Step 4** - Run active scan after exploring.

## Commands

### Install

```bash
sudo apt install -y zaproxy
# or download from https://www.zaproxy.org/download/
```

### Headless API mode (CI/CD)

```bash
zap.sh -daemon -port 8080 -config api.disablekey=true &
# In Python:
python3 -c "
from zapv2 import ZAPv2
zap = ZAPv2(proxies={'http':'http://127.0.0.1:8080','https':'http://127.0.0.1:8080'})
target = 'https://<TARGET>'
zap.spider.scan(target)
zap.ascan.scan(target)
print(zap.core.alerts())
"
```

### Quick automated scan

```bash
zap-baseline.py -t https://<TARGET> -r baseline.html
```

### Full scan with auth

```bash
zap-full-scan.py -t https://<TARGET> -r full.html -j -z '-config replacer.full_list(0).description=auth -config replacer.full_list(0).enabled=true -config replacer.full_list(0).matchtype=REQ_HEADER -config replacer.full_list(0).matchstr=Cookie -config replacer.full_list(0).replacement=session=<SESSION>'
```
