# 02.05 Nikto

> Old-school web server scanner — finds outdated software, dangerous files, default configs.

## Overview

Nikto is loud, signature-heavy, and dated — but still catches default admin panels, old IIS files, weak SSL, etc. that more modern scanners might miss. Don't use against modern web apps (false-positive rich); use against legacy infrastructure or for quick low-hanging fruit checks.

## Steps

- **Step 1** - Install via apt.
- **Step 2** - Run against target, save output.
- **Step 3** - Sift results — many are noise; look for outdated software banners.

## Commands

### Install

```bash
sudo apt install -y nikto
```

### Standard scan

```bash
nikto -h https://<TARGET> -o nikto.txt
```

### Authenticated

```bash
nikto -h https://<TARGET> -id 'admin:password'
```

### Tuning for stealth

```bash
nikto -h https://<TARGET> -Tuning 1,2,3 -evasion 1 -useragent 'Mozilla/5.0'
```
