# 07.09 SysWhispers2 SysWhispers3

> Generate direct syscall stubs for evading EDR's userland hooks on ntdll.

## Overview

EDRs hook ntdll.dll functions in user space to monitor syscalls. Direct syscall stubs bypass these hooks by calling the kernel directly via the `syscall` instruction. SysWhispers2/3 generates C/asm stubs for every NtXxx function, dynamically resolves syscall numbers per Windows version, and integrates into C# / C++ loaders. SysWhispers3 adds 'EgeBalcı' jumping technique for added stealth.

## Steps

- **Step 1** - Clone repo.
- **Step 2** - Generate stubs for the syscalls your payload needs (NtAllocateVirtualMemory, NtWriteVirtualMemory, NtCreateThreadEx, etc.).
- **Step 3** - Compile your loader against the generated stub files.

## Commands

### Get SysWhispers3

```bash
git clone https://github.com/klezVirus/SysWhispers3
cd SysWhispers3 && pip3 install -r requirements.txt
```

### Generate stubs

```bash
python3 syswhispers.py -f NtAllocateVirtualMemory,NtWriteVirtualMemory,NtProtectVirtualMemory,NtCreateThreadEx,NtWaitForSingleObject -o stubs
```

### Output files

```text
Generated files:
  stubs.h
  stubs.c
  stubs-asm.x64.asm   (for MSVC)
# Compile into your C/C++ loader:
cl loader.c stubs.c stubs-asm.x64.asm
```

### Different stub variants

```bash
# Direct syscalls (classic)
python3 syswhispers.py -m jumper -f NtAllocateVirtualMemory -o stubs
# Indirect syscalls (call via legitimate ntdll jmp)
python3 syswhispers.py -m jumper_randomized -f NtAllocateVirtualMemory -o stubs
```
