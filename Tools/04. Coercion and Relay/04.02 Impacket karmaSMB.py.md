# 04.02 Impacket karmaSMB.py

> SMB MITM — answer 'yes' to every SMB connection, capture credentials and file requests.

## Overview

karmaSMB.py is a malicious SMB server that responds positively to every connection and authentication attempt. Captures NetNTLMv2 hashes and inbound file request data. Useful in MITM scenarios where you can redirect traffic — pair with mitm6 or arpspoof.

## Steps

- **Step 1** - Run karmaSMB on attack box.
- **Step 2** - Redirect SMB traffic to your IP.
- **Step 3** - Captured hashes saved to logfile.

## Commands

### Start karmaSMB

```bash
sudo impacket-karmaSMB -smb2support /tmp/responses
```
