# 04.05 Impacket rpcmap.py

> Map RPC interfaces to ports — find which port hosts a specific RPC service.

## Overview

rpcmap.py is the RPC endpoint mapper client. Given an interface UUID, returns the dynamic port hosting that interface. Useful when a specific RPC attack requires connecting to a non-standard port. Complements rpcdump.

## Steps

- **Step 1** - Identify target RPC interface UUID.
- **Step 2** - Run rpcmap to find its port.

## Commands

### Map interface

```bash
impacket-rpcmap ncacn_ip_tcp:<TARGET>
```

### With auth

```bash
impacket-rpcmap -username <USER> -password <PASS> -domain <DOMAIN> ncacn_ip_tcp:<TARGET>
```

### Specific UUID

```bash
impacket-rpcmap ncacn_ip_tcp:<TARGET> -auth-level 6 -brute-uuids
```
