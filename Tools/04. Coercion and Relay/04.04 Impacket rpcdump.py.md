# 04.04 Impacket rpcdump.py

> Enumerate available RPC endpoints on a target — find Print Spooler, EFSRPC, MS-DFSNM for coercion.

## Overview

rpcdump.py queries the RPC endpoint mapper (port 135) to list all available RPC interfaces. Critical for identifying which coercion attacks apply: PrinterBug needs spoolss, PetitPotam needs efsrpc, DFSCoerce needs ms-dfsnm. Pair with coerce-r script before launching coercion.

## Steps

- **Step 1** - Run against target.
- **Step 2** - Look for spoolss, efsr, dfsnm, etc. in output.
- **Step 3** - Use that intel to pick coercion method.

## Commands

### List RPC endpoints

```bash
impacket-rpcdump <TARGET>
impacket-rpcdump <DOMAIN>/<USER>:<PASS>@<TARGET>
```

### Grep for coercion-relevant

```bash
impacket-rpcdump <TARGET> | grep -iE 'efsr|spoolss|dfsnm|fsrvp|even|wkssvc'
```
